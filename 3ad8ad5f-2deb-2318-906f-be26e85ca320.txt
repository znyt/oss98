　　不过这也不是太难理解的事情，毕竟之前从博士那里，他就已经见识过这种技术了。
<br/>　　原理刘哔并不是很懂，只是知道是和病毒中的一种物质有关，地球上那些丧尸之所以失去了思维能力和智慧，大概就是因为缺少这种物质。
<br/>　　这样看来的话，如果进化速度一致，那这个世界就真的有些危险了。
<br/>　　结合丧尸和人类的所有优点，又没有寿命的限制，这种生物能造成的影响，不知道会恐怖到什么地步。
<br/>　　就连地球那种没有思维的丧尸，都将人类实力逼迫到了几乎山穷水尽的地步，如果有了智慧，这样的后果，又有谁能想象呢。
<br/>　　过不……和他也没有什么鸟关系。
<br/>　　就算这个位面真的被毁灭，那也不是一时半会能做到的事情，到时候，他早就重新回归自己的位面了。
<br/>　　“嘿嘿，我要吃了那个戴绿帽子的。”
<br/>　　壁虎怪右侧的头颅开口，目光绿油油的，盯着全副武装的保安小哥，一双眼里，全是贪婪。
<br/>　　“啊……快来帮我！”
<br/>　　被壁虎怪的尾巴缠住，白小飞拼命挣扎，可因为缺少趁手的武器，无论他如何反抗，都无法从这束缚中走出来。
<br/>　　刘哔眉头皱了一下，随手将白小飞之前掉落在步行楼道内的长刀抛去，与此同时，手中的黑色棍子涌上一股赤色的火焰，猛然变长，一棒敲向庇护怪的尾巴衔接处。
<br/>　　空中，烤肉的香味弥漫，即使刘哔已经放慢了速度，但也不是这壁虎怪物可以跟上的。
<br/>　　只是稍稍一接触，很快壁虎怪物的尾巴就被从中阶段，白小飞也被拯救了出来。
<br/>　　“谢谢了，这怪物力气好大，根本挣不开。”接过武器，白小飞缓了一缓，继而转身和刘哔一起面对壁虎怪物。
<br/>　　此时，一阵噼里啪啦的枪声响起，保安小哥瞄准壁虎怪的头颅，疯狂扣动着扳机。
<br/>　　连续扫射而带来的强大后坐力，将他整个人震的摇摇欲坠，可即使这样，保安小哥依旧很难得的保持着击中率。
<br/>　　血花不断从壁虎背上的两个头颅上冒出，顿时一阵凄厉的惨叫声响起，面对这猛烈的火力，壁虎怪唯一能做的，就只有将前肢挡在头颅前方，苦苦支撑着。
<br/>　　这两个头颅，果然是它的弱点。
<br/>　　“弟弟，我好疼啊。”
<br/>　　“哥哥，看来咱们要先把他们的武器给缴了。”
<br/>　　壁虎怪发出痛苦的嘶吼声，随后，一道长长的肉舌吐出，急速卷上了保安小哥的自动步枪。
<br/>　　突入而来的变化让保安小哥来不及反应，随后壁虎怪舌头一卷，竟将自动步枪直接吞入了腹中。
<br/>　　与此同时，白小飞和刘哔也冲到了壁虎怪面前，但一条崭新的尾巴，挡下了两人全部的攻击。
<br/>　　“这，竟然能再生？！”白小飞惊呼一声，看着面前摇曳的尾巴，有些不敢置信：“可恶，这些就麻烦了。”
<br/>　　刘哔紧抿着嘴唇，快速挥动手中的火焰长剑，连续数下，将壁虎新长出来的尾巴整个截断。
<br/>　　“能再生？又能如何，长一条给切掉一条，我就不信他能一直长出来！”
<br/>　　声音有些发狠，刘哔挥动着手中的武器，如同打桩机一般向壁虎怪的断尾处搅动。
<br/>　　霎时，比一开始还要凄惨的叫声响起，他的这次攻击，给壁虎怪造成了很严重的伤势。
<br/>　　“唰！”
<br/>　　两天鲜红的舌头瞬间弹射出，卷上了刘哔的身体，这是从速度上，壁虎怪唯一能和刘哔相比的地方。
<br/>　　但是，毫无作用。
<br/>　　一团赤红的火焰从刘哔的武器中蔓延，直至覆盖上全身，壁虎怪的舌头受到灼烧，还未来得及撤回，就被整个烤焦。
<br/>　　有SSS武器的加成，刘哔根本不惧怕任何火焰，此时他整个人沐浴在烈火中，如同火焰战神一般。
<br/>　　壁虎怪却是说不出话来了，看着这样可怕的对手，几乎没再有任何犹豫，转身就逃，甚至连被砍掉的尾巴都没去修复。
<br/>　　跑完了，可是要命的啊！
<br/>　　“咦，还会跑？有趣。”刘哔嘴角轻轻一扯，不紧不慢的伸出了一只手掌：“跑得掉么？”
<br/>　　一张火焰巨掌在空中凝聚，狠狠拍向了壁虎怪物，连惨叫声都没有，壁虎怪物就被拍成了一滩碎肉。
<br/>　　“叮~恭喜宿主击杀三阶尸兄双生壁虎，获得积分一千点。”
<br/>　　“叮~恭喜宿主吞噬三阶尸兄双生壁虎，再生技能小幅度进行加强。”
<br/>　　连续两道系统提示的声音响起，刘哔微微一愣，只是三阶么，给予的积分竟然差不多要赶上A级喰种了。
<br/>　　这样看来，这个世界的武力值，恐怕不低。
<br/>　　他可没忘记那个迟迟没有露面的未知尸兄，那是真正能威胁到自己的存在，恐怕等级，也不会太低。
<br/>　　“我去……”
<br/>　　吃惊的看着刘哔，白小飞的表情有些震撼，虽然见过刘哔的一部分实力，可依旧有些难以置信。
<br/>　　太强大了，强大的有些梦幻，话说，这真是他的队友么。
<br/>　　捡到宝了。
<br/>　　有这样强大的庇护，不管做什么，都要更有把握。
<br/>　　就像这次，要不是刘哔出手，别说他们是否能顺利到小鹿家里，就算到了，能不能逃出生天都是个问题。
<br/>　　“走吧，怪物解决了，我们去小鹿家里看看吧。”
<br/>　　手中武器上的火焰消散，重新变为了黑不溜秋的短棍模样，刘哔轻笑一声，抱起小鹿。
<br/>　　“大哥哥好厉害啊，这么大的怪物，一下子就解决了。”
<br/>　　小鹿眼中冒着星星，胖嘟嘟的小手，拉扯着刘哔的衣领。
<br/>　　事实上，对于小鹿给他的称呼，刘哔也是有些吐槽的，虽然自己幻化的这幅模样年龄要小了一些，但看起来至少也有二十几岁的样子吧。
<br/>　　比白小飞也相差不到哪去。
<br/>　　但为什么叫白小飞叔叔，到他这里就成了大哥哥了。
<br/>　　总是有点吃亏。
<br/>　　但小鹿已经叫了这么长时间了，他也懒得纠正了，反正只是一个口头上的称呼，随便怎么叫吧。
<br/>　　他也不一定会在这个队伍里面逗留多长时间。
<br/>　　等将该做的事情做的差不多了，也就到了离开的时候了。
