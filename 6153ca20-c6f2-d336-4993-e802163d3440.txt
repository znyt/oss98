　　第297章
<br/>　　“大当家，你歇息一阵，这里交给我们。”
<br/>　　大家冲散包围圈，立刻将程大雷护在当中，几十人持刀护住了他。
<br/>　　程大雷一口气松了下来，手中的铁枪铛地一声落地，无力挥挥手。
<br/>　　“交给你们了。”
<br/>　　且看一路人分成五队人，互相配合默契，周转有度，虽然是极速冲锋，且都保持着一定秩序，急而不乱。
<br/>　　几番争斗，如同雄狮扑兔，很快将阵势压下来。赤檑一伙人集在一起，被众人围在当中。
<br/>　　赤檑也没想到这幅场面，本想仗着人多，慢慢将程大雷耗死，但一眨眼的光景，程大雷的人比他们多出十几倍。
<br/>　　“怕什么！”赤檑扬刀吼道：“只不过是一群山寨而已，我们草原上的勇士还怕了不成，给我杀！”
<br/>　　“杀！”
<br/>　　五十草原儿郎齐吼，钢刀出鞘。
<br/>　　蛤蟆寨的人开始冲锋。
<br/>　　很难称之为战斗，毕竟人数上占了绝对优势。不管这些戎族杀意多么激昂，都没起到多大作用，依旧无法改变失败的结局。
<br/>　　“杀，杀，杀！”
<br/>　　赤檑举着刀狂吼，但依旧改变不了什么。往日帝国人碰到戎族，哪怕人数占有优势，也是一打就溃，可今天这波人不一样。
<br/>　　他们见到戎族，不禁没有露出害怕的神色，反而神情很激动，有种可逮着你们的感觉。
<br/>　　很快，蛤蟆寨的人就稳住了场上形势，戎族死的死，伤的伤。已然失败，赤檑仍旧举着刀狂吼。
<br/>　　“来呀，你们来杀了我，来呀，杀呀。”
<br/>　　也不知道那个小兵手一哆嗦，一枪捅了他，赤檑似乎还想说什么，但含糊不清的话被腔子里喷出的血取代。
<br/>　　俘虏分成一波，落玉寨的墙头草分成一波，蛤蟆寨控制住整个局面，俨然成为落玉寨新的主人。
<br/>　　程大雷找个僻静屋子把身上的血污洗去，换套干净衣服，又喝了杯热茶，方才恢复些体力。
<br/>　　徐神机一波人聚在一处屋子内，当程大雷走进来时，大家看着他便笑出声来。
<br/>　　“大当家，你怎么被一个女人绑了？”
<br/>　　“什么话……什么叫被女人绑了？”程大雷争辩得红起脸，脸上青筋凸显。
<br/>　　“嘿，我们明明都看见了，如果不是我们及时到，你都要被人杀了。”
<br/>　　“这是计谋，计谋的事你们懂么，计谋……”
<br/>　　于是又换来一阵更愉悦的笑声。
<br/>　　程大雷擦擦额头的汗珠，清了清嗓子道：”我早觉得落玉寨会有问题，所以甘冒奇险，深入虎穴，当然，也多亏诸位贤弟来得及时，这件事大家都是有功的么，都是有功的。”
<br/>　　“……”徐神机等人。
<br/>　　“好啦。”程大雷也知道自己的话没什么说服力，转移话题道：“现在外面是什么情况？”
<br/>　　“人都被我们擒住了，卸了他们的兵器，现在手下兄弟正看着他们。”
<br/>　　“那个女人怎么样？”
<br/>　　“喔，就是那个将大当家擒住的女人么，喂她喝了些水，身上的腹疼药，已不碍事。”徐神机拍了拍手：“将人带上来。”
<br/>　　两个蛤蟆寨的弟兄押着五花大绑的樊梨花进来，樊梨花花容憔悴，只是脸上还带着些不屈的英气。
<br/>　　徐神机冷哼一声：“呔，见到我们大当家，还不跪下。”
<br/>　　“嗳，这是何苦来哉。”程大雷摆摆手：“还不快给樊寨主松绑。”
<br/>　　“住了吧。”樊梨花突然喝了一声：“姓程的，你何必惺惺作态，今日姑奶奶落到你手里，要杀要剐悉听尊便，姑奶奶皱皱眉头，都不算好汉。”
<br/>　　“你本来就不是好汉呐。”程大雷下意识道了一声，亲自为樊梨花松绑：“樊家姑娘，何必如此呢，我们是友非敌，我可没有害你的心事。”
<br/>　　程大雷再傻也看得出来，樊梨花无非是嘴硬而已。今日的遭遇对她来说，实在称不上愉快，先是手下背叛，最后一条命差点丢了，反而是她一直看不上的程大雷出手相救。再强悍的女人，遇到这种事心里也是一肚子委屈，如今想要的不过是给个台阶而已。
<br/>　　若是男人，程大雷绝不会惯着对方，但对于女人，尤其是樊梨花这样的美女，程大雷不介意展现一下绅士风度。
<br/>　　樊梨花松开绑绳，心中委屈泄了几分，看程大雷杵在面前的一张丑脸，也顺眼了许多。
<br/>　　“罢罢罢。”她长叹一声，道：“从今日起，这落玉寨就是程寨主的，樊梨花愿为程寨主牵马执鞍，当牛做马。”
<br/>　　嗳，程大雷松口气，到现在，自己才算是把支线任务完成。只是为何要当牛做马呢，明明以身相许也可以么，虽然一样是被骑的……
<br/>　　嘟，恭喜获得二级山贼：落玉寨。
<br/>　　程大雷忽然收到一条系统提示，他眼睛一亮，对么，没说只可以经营一家山寨，自己完全可以开分店么。
<br/>　　就想攻略游戏中，基础盘打好后，就可以向外扩张，开分基地。两个基地的采矿速度，伐木速度明显快过一个基地。
<br/>　　只是不知道这山贼之心开分基地有什么好处。
<br/>　　程大雷打开系统界面，查看落玉寨的信息。
<br/>　　山寨：落玉寨（小有名气的二级山寨）
<br/>　　拥有者：程大雷
<br/>　　人口：289
<br/>　　技能：骑术加成
<br/>　　这落玉寨位于草原上一座山谷内，本身就是养马的，因为和戎族经常接触，人人都有一身好骑术。程大雷心里想着，日后可以让徐灵儿负责这里，专门给自己养马。
<br/>　　“樊寨主，请坐。”
<br/>　　程大雷差人给樊梨花搬来一张椅子，这次收获不小，虽然冒了些风险，但也是值得的。他坐下后，整了整身上长袍，忽然喝了一声：“来人，把他们给我带上来！”
<br/>　　阴重一行叛贼押了上来，就连樊梨花都被绳索绑着，何况是他们。此刻人人鼻青脸肿，不复刚才的得意，如今都像霜打的茄子一般。
<br/>　　一见到阴重，樊梨花就咬紧牙齿，眼睛里能喷出火来。
<br/>　　程大雷示意她稍安勿躁，冷喝一声道：“阴重，事到如今，你还有什么话说？”
<br/>　　阴重态度不卑不亢，昂着头道：“我有什么说，我上不愧苍天公义，下不愧老寨主九泉之灵。”
<br/>
