　　“有劳几位相助了。”
<br/>　　顾非凡对着第一云、叶流等人拱了拱手。
<br/>　　“素闻第一副殿主、长孙副殿主还有药公子，乃是三殿最出色的天骄。”
<br/>　　“便也让顾某见识见识三位本事。”
<br/>　　顾非凡的语气，带着些许挑衅。
<br/>　　第一云三人冷笑一声，“顾非凡，你这个天藏学宫首席，本也算号人物。”
<br/>　　“现今，这点激将法般的技俩，便想激我们出手？”
<br/>　　顾非凡闻言，眉头微皱，随后淡淡一笑。
<br/>　　“我听闻，东离剑宫大比前，三位，可是在那位易霄手中败得彻底。”
<br/>　　“那易霄，端的是嚣张猖狂。”
<br/>　　“前不久，又大闹我天藏学宫。”
<br/>　　“若今日我等联手，能破了这千万年来无人能破的古帝洞府，必可压一压那易霄气焰。”
<br/>　　顾非凡，再次拱了拱手。
<br/>　　第一云三人，也再次冷笑，“顾非凡，你就这点心胸？”
<br/>　　“我等三人，败在易霄手中，心服口服。”
<br/>　　“倒是你天藏学宫天骄，似乎怨恨得很。”
<br/>　　“你们…”顾非凡的脸色，已然冰冷。
<br/>　　他乃天藏学宫首席，亦是中域首屈一指的天骄妖孽。
<br/>　　这般明显带着交好的意味，第一云三人却油盐不进，还冷嘲热讽，让他如何还能忍？
<br/>　　司空宇冷笑一声，“三位，若不想待在此的话，有本事便滚出大阵。”
<br/>　　“哼。”第一云冷哼一声，“你当我们入阵是做什么的？”
<br/>　　第一云话音刚落下，看了眼花若怜与长孙赤烈。
<br/>　　三人对视了一眼，点了点头，“救人。”
<br/>　　大阵之内，除却莫悠以外，没人能拦第一云三人。
<br/>　　但恰在此时，叶流抢先一步，拦下了三人。
<br/>　　“叶流，你做什么？”第一云皱眉问道。
<br/>　　叶流摇了摇头，“现在救不救人，已然没意义。”
<br/>　　说着，叶流看向顾非凡，“你们的目的，只是要我们入阵吧。”
<br/>　　“其他天骄，现今放走，你们根本不在乎。”
<br/>　　也就是说，即便第一云三人现今不出手，顾非凡等人也不会再为难其他天骄。
<br/>　　其他天骄，大可现今离去。
<br/>　　“不错。”顾非凡点了点头。
<br/>　　“黑云大阵，是一纯粹的防守大阵，以入阵者为阵基所维持，入阵者的元力，会化作磐石乌云，环绕四周。”
<br/>　　“入阵者越强，维持者元力越是深厚，大阵便越强。”
<br/>　　“你们已入阵，其他天骄的微末修为，已无大用，大可离去。”
<br/>　　司空宇接话道，“另外，你们也不得不帮忙维持大阵。”
<br/>　　“这些天骄，即便现今喊出‘出’字，也需要时间去触发效果。”
<br/>　　“在此之前，你们就乖乖帮忙守护大阵吧，哈哈哈哈。”
<br/>　　司空宇脸上，尽是得意之色。
<br/>　　第一云三人反应过来，脸色，顷刻难看到极点。
<br/>　　但他们别无选择。
<br/>　　无论是那些天骄要离去，还是他们众人要离去，首先要保证的，便是大阵不被如潮妖兽攻破。
<br/>　　否则，他们还未能来得及离去，便会被这些无数妖兽撕成碎片。
<br/>　　周遭如潮妖兽，已清一色达到绝世9996道的层次。
<br/>　　在场，除却莫悠以外，谁也无法单独对付。
<br/>　　“哼。”第一云等人，冷哼了一声，但还是祭出元力，维持起大阵。
<br/>　　有第一云、叶流等五人加入，黑云大阵霎时乌云更浓，威力更甚，变得牢不可破。
<br/>　　此时，主局之内，还有两位带队者在参悟禁制。
<br/>　　一是傲吟风这边，有黑云大阵守护着。
<br/>　　黑云大阵内，是五大学宫与黑云学教的弟子，当然，还有萧逸、第一云、叶流等一行人。
<br/>　　也就是说，这大阵之内，此刻几乎囊括了所有带队者。
<br/>　　此次来古帝洞府的天骄妖孽中，最出色的一批武者，皆在此。
<br/>　　而另一半，则是凌鸿那边。
<br/>　　凌鸿那边，有一种十八府少府主，以及一队跟随者天骄。
<br/>　　按理说，现今如潮妖兽的实力，已达绝世9996道，他们绝计支撑不住。
<br/>　　可此时，凌鸿一行人周遭，却有一水蓝光幕。
<br/>　　光幕，如若水波凝聚，如幻似梦，绚烂至极。
<br/>　　光幕，看似单薄，实则，坚如磐石。
<br/>　　任凭无数妖兽冲击，不过是仅让光幕产生点点涟漪，丝毫没有要溃散的迹象。
<br/>　　萧逸瞥了眼，暗暗皱眉。
<br/>　　如果他没有看错的话，那应该是水光府至宝，琉璃幻罩。
<br/>　　“莫兄。”这时，顾非凡喊了一声。
<br/>　　莫悠闻言，点了点头，“现今只能等。”
<br/>　　“18道禁制，我们已破去13道。”
<br/>　　“待凌鸿和傲吟风的禁制也破去，便达到15道。”
<br/>　　“届时，便可开始堪破最后3道禁制了。”
<br/>　　“等？”司空宇皱眉道，“不能直接开始堪破吗？”
<br/>　　司空宇的语气，略微有些急切。
<br/>　　“有黑云大阵在，我等护着这傲吟风并不成问题。”
<br/>　　“而以莫悠师兄你的实力，根本不惧这些妖兽傀儡，大可一人直接去堪破禁制。”
<br/>　　“以莫悠师兄你的天赋，参悟速度必然快于凌鸿他们。”
<br/>　　“只要莫悠师兄将剩下的3道禁制一一破去，加上凌鸿、傲吟风二人也破去禁制，18道禁制，便算悉数打破。”
<br/>　　“这主局，届时便也破了。”
<br/>　　司空宇的语气，越发激动。
<br/>　　“司空宇，你急些什么？”童烨冷声道，“莫不是眼馋主局之内的莫大机缘和重宝？”
<br/>　　“让莫悠师兄一人去堪破三道禁制？如此涉险，亏你说得出来。”
<br/>　　“你们不眼馋？”司空宇瞪了童烨一眼。
<br/>　　“好了，别吵了。”莫悠摆摆手。
<br/>　　“若能率先堪破禁制，我自会出手。”
<br/>　　“不过，古帝洞府之内，自有法则规定。”
<br/>　　“我们15支队伍进来前，触发了15道禁制；主局这里，若不先破去数量对应的15道禁制，根本无法参悟其它三道禁制。”
<br/>　　那是属于古帝洞府的考验，若不先堪破数量对应的15道禁制，根本没资格参悟剩下的三道。
<br/>　　“就快了，别急。”莫悠说了一声。
<br/>　　“呵，对，就快了，别急。”萧逸也说了一声，淡淡一笑。
<br/>　　只是，笑容显然有些莫名。
<br/>　　莫悠看了萧逸一眼，皱着眉，若有所思。
<br/>　　叶流看了萧逸一眼，嘴角咧过一道笑意。
