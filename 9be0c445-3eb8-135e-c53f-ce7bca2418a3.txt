　　这时的萧逸，手持殿主给予的令牌，前往猎妖殿宝库。
<br/>　　进入宝库，里头的满目琳琅，让得萧逸眼前一亮。
<br/>　　宝库的面积，丝毫不比裂天剑宗的宝库小，甚至是大得多。
<br/>　　拥有的各种宝物，也多得多。
<br/>　　亚灵器、灵器、元器、丹药、天材地宝、妖兽内丹、精血等等。
<br/>　　以及其它修炼之物、重宝，多不胜数。
<br/>　　当然了，毕竟是猎妖殿的宝库。
<br/>　　里头最多的，还是妖兽内丹与精血。
<br/>　　且品阶相当高。
<br/>　　最弱都是六级，几乎都是七级。
<br/>　　不错，这里并没有八级的妖兽内丹或者精血。
<br/>　　对此，萧逸早有所料。
<br/>　　八级妖兽，可是等同于极境强者的存在。
<br/>　　这等妖兽，几乎只存在于传说中。
<br/>　　它们的内丹和精血，更是珍贵无比。
<br/>　　数量肯定不多，且不会放在宝库之内。
<br/>　　萧逸思索了一下。
<br/>　　本来，完成殿主给予的考核，能在宝库内任取妖兽内丹或者精血十万之数。
<br/>　　一半，便是五万。
<br/>　　如今自己的肉体力量和武道修为，都是地元九重。
<br/>　　但，毫无疑问，就现在来说，武道修为所发挥出的力量，已经超越了肉体力量。
<br/>　　而且，武道修为，才是武者的重中之重。
<br/>　　思索半分。
<br/>　　萧逸取了五万七级妖兽内丹。
<br/>　　满意地离开了宝库。
<br/>　　五万颗七级妖兽内丹，若是放到外头。
<br/>　　其价值，几乎无法想象。
<br/>　　这时，萧逸往闭关室而去。
<br/>　　他准备开炉炼丹。
<br/>　　……
<br/>　　闭关室内。
<br/>　　萧逸取出八龙焚火炉，以及大量天材地宝。
<br/>　　还有刚才取的五万颗七级妖兽内丹。
<br/>　　嘭…嘭…
<br/>　　萧逸手中凝聚出两股火焰。
<br/>　　左手为紫炎，右手为星辰之火。
<br/>　　紫炎，向来是萧逸炼药的首选火焰。
<br/>　　其焚毁万物之效，恐怖的高温，能让炼药速度变得极其之快。
<br/>　　当然，这取决于他本身拥有几乎完美的控火手段。
<br/>　　而现在之所以凝聚出星辰之火，自然是要借星辰之火能温养并增强天材地宝药力之效。
<br/>　　两种火焰，打入八龙焚火炉。
<br/>　　随后，无数天材地宝以及妖兽内丹抛入炉中。
<br/>　　不多时。
<br/>　　一粒粒无杂质天元丹，被炼制而出。
<br/>　　虽说天元丹位列七品丹药，极难炼制。
<br/>　　但掌握了完美炼制手法的萧逸，自是炼制起来极快。
<br/>　　而且，在已经解除了禁制，位列八品炼药炉的八龙焚火炉辅助下。
<br/>　　其炼药速度，更是再度飙升。
<br/>　　两天后。
<br/>　　五万颗七级妖兽内丹，被消耗一空。
<br/>　　而炼制出的无杂质天元丹，整整一万六千余粒。
<br/>　　萧逸毫不犹豫，将一粒粒丹药吞服。
<br/>　　丹药，进入口中，瞬间化作精纯力量，传入体内。
<br/>　　而后，体内小世界中，两大武魂齐开，快速吸收着这些力量。
<br/>　　随后，精纯力量，又化作了冰火两种元力。
<br/>　　不多时，体内气泉，渐渐被填满至10成。
<br/>　　而这时，果然如萧逸所料，他的‘噩梦’发生了。
<br/>　　正常来说，气泉填满10成，便是突破，进入下一境界之时。
<br/>　　但，此刻，他并没有突破境界。
<br/>　　反倒是气泉内的元力，在不断压缩。
<br/>　　另外一点，据萧逸所知，地元境突破天元境的要求。
<br/>　　是将元力化为液态。
<br/>　　地元境的元力，是气态。
<br/>　　天元境的元力，是液态。
<br/>　　可现在，气泉内的气态元力，虽在不断压缩，却始终没有变化的迹象。
<br/>　　萧逸皱了皱眉头，随后咬了咬牙，加快了丹药的吞服速度。
<br/>　　不得不说，他现在才地元境修为。
<br/>　　而天元丹作为高于他修为一级的丹药，其效果，极其之强。
<br/>　　时间，渐渐过去。
<br/>　　随着一粒粒丹药不断被吞服。
<br/>　　体内气泉中的元力，也在不断压缩。
<br/>　　但，修为始终不见有突破的征兆。
<br/>　　待得一万粒无杂质天元丹服下，气泉中元力的压缩程度，已经达到一个相当骇人的程度。
<br/>　　再服下三千粒。
<br/>　　始终不见突破。
<br/>　　但，萧逸在内视的情况下，明显感觉到，体内元力，似乎已经压缩到一个临界点了。
<br/>　　萧逸一喜。
<br/>　　再度快速吞下丹药。
<br/>　　当再次服下数百粒无杂质天元丹时。
<br/>　　轰的一声。
<br/>　　体内气泉，所有元力，竟忽然间快速翻滚起来，仿佛成了一个闹腾的大海。
<br/>　　一股极度深寒的气息，一股炙热可怕的气息，在同一时间暴走。
<br/>　　这种感觉，萧逸再熟悉不过了。
<br/>　　“果然如此。”萧逸不禁苦笑一声。
<br/>　　数秒过后。
<br/>　　体内气泉中，猛地出现了一座冰山，以及一片火海。
<br/>　　三千三百三十三丈气泉，被完全分割。
<br/>　　二者各占一半。
<br/>　　“冰山火海，又回来了。”萧逸的脸色，有些苦涩。
<br/>　　这意味着，他将再度困于地元九重巅峰这个境界，许久。
<br/>　　并且…
<br/>　　这时，萧逸快速服下数粒无杂质天元丹。
<br/>　　果然，无杂质天元丹的精纯药力，进入小世界化为元力后。
<br/>　　悉数从冰山火海中‘溢’出来，随后被冰鸾剑吸收，用于修补那处裂缝。
<br/>　　“呼。”萧逸深呼吸一口气，平复下心头的苦涩。
<br/>　　事实上，对于冰山火海的恢复，他早有所料。
<br/>　　也并不因此而感到失望。
<br/>　　毕竟，他也很想看看，冰鸾剑真正修复之后，会强大到什么地步。
<br/>　　再者，冰山火海，本就是他的底牌之一。
<br/>　　以前，在破玄九重修为时，他能叫板天元境。
<br/>　　除了本身紫炎等强悍火焰足够强之外。
<br/>　　冰山火海作为固态力量，拥有极强的爆发力，以此作为释放火焰的源泉，也是必不可少的因素。
<br/>　　不过，自从突破地元境后，冰山火海就暂时消失了。
<br/>　　现在，地元九重巅峰，耗费了一万六千余粒无杂质天元丹，又回来了。
<br/>　　不过，此时萧逸有一个疑惑。
<br/>　　地元九重，突破至天元境，要求是气态元力，化为液态元力。
<br/>　　天元境之后的修炼，本就是以液态元力，填充气泉，增加修为境界。
<br/>　　可为何，自己的气态元力，竟是直接跳过了液态这个阶段，升华到固态了。
<br/>　　这个问题，萧逸思索了许久，并未能想明白。
<br/>　　或许，这与冰鸾剑有关。
<br/>　　萧逸没有再纠结下去。
<br/>　　当然，现在虽然元力达到固态了，比液态更为高级。
<br/>　　但却不代表着萧逸的修为，是天元境的范畴。
<br/>　　据他所知，天元境踏入后期之后，武道修炼，以及气泉，都会有着更深层次的变化。
<br/>　　这种变化，才让得天元境后期比之中期强大得多。
<br/>　　同时也是因为这种变化，才让得天元九重武者，被称之为巅峰强者。
<br/>　　当然，萧逸现在并没有兴趣想这些日后才会出现的修炼情况。
<br/>　　毕竟踏入天元境，或许会因为冰山火海的存在，再次被停滞许久。
<br/>　　更别说进入天元境后期了。
<br/>　　现在的他，尝试着这由元力而凝聚成的冰山火海的威力。
<br/>　　数秒后，一股恐怖的气势，在他身上爆发。
<br/>　　一道威力更加惊人的寒冰裂天斩于剑上凝聚。
<br/>　　“好强。”萧逸吃了一惊。
<br/>　　借助冰山火海的力量，仅仅数秒。
<br/>　　竟是比以前酝酿许久，调动数成气态元力为源，所施展的寒冰裂天斩要强许多。
<br/>　　萧逸甚至可以确定，单凭此时这一剑，足以做到重伤血成河那个层次的强者。
<br/>　　......
<br/>　　第三更。（昨天三更）。
