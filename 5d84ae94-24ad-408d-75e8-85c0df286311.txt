<br/>        “如果，婚后，你提前怀上了我们的孩子，你会打掉吗？”陈先生又问道。
<br/>
<br/>        宋云汐果断地回答道：“不会。”

<br/>        “那么，你生完孩子后，愿意当全职太太吗？”陈先生接着问道。
<br/>
<br/>        宋云汐依旧很果断地回答：“不愿意！”

<br/>

<br/>
<br/>        “谢谢。”陈先生点了下头，随即起身往收银台走去。

<br/>
<br/>        看到只有宋云汐一个人坐在这里，还有些一脸懵。

<br/>        “那我们把车开到前方的路边上商量吧！”男人和和气气地说。

<br/>        宋云汐点了下头。
<br/>

<br/>        绿灯亮起后，宋云汐开着车，跟前前方那辆军用越野车一起过了十字路口后，在路边的停车位上停了下来。

<br/>        宋云汐将车停稳，从驾驶座上下来后，先去看了看对方的车尾，然后再看了看自己的车头。
