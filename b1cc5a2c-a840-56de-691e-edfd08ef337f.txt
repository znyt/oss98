　　楚风眠瞬息之间，便是猛然之间，将浑身的力量，全部催动。

<br/>　　这战龙决，便是遇到愈发强大的对手，才会激发出真正最为可怕的力量来。

<br/>　　楚风眠的身躯，顿时猛然咆哮，直接是化为了远古战龙的模样，手持祖龙至尊剑，猛然之间一剑轰击过去。
<br/>　　从楚风眠的身躯之上，便是散发出了无数强大力量。
<br/>　　远古战龙血脉，他的九域剑术，远古杀神虚影，全部被催动凝聚起来，一股股力量都加持在了楚风眠的身躯之上。

<br/>　　楚风眠疯狂怒喝道。

<br/>　　这一剑，也是纯粹之际的力量，猛然一剑突然斩杀过去，其中蕴含的力量，也是恐怖到了极致的地步。
<br/>　　对着天空之上的一指，直接轰击过去。
<br/>　　“轰隆！”
<br/>　　两股力量，从空中疯狂的碰撞，楚风眠的剑锋，斩在这陆千刃的一指上，却是丝毫无法撼动陆千刃的这一指。
<br/>　　在这一指上，楚风眠感觉到了一种超越生死的味道，超越了一切，已经是跟武者，截然不同的力量。

<br/>　　生死台阶的第一层，碎命之境。
<br/>　　这陆千刃，如今便是踏入到了这一重境界上，踏上这碎命境，便是要粉碎自己的命运，真正的超脱一切。
<br/>　　只有踏上这一重的人物，才能够算是真正的强者，真正有资格遗留记载下去的强者。
<br/>　　楚风眠心思思考之际。
<br/>　　这一指的力量，已经是轰击在了楚风眠的身躯之上。
<br/>　　“碰！”
<br/>　　楚风眠的身躯，被直接被这一指的力量，点的破碎，楚风眠的身体，都在节节破碎这。
<br/>　　这一指的力量，恐怖至极，不单单是将楚风眠剑锋碾碎。
<br/>　　其中一指的力量，更是全部都轰击在了楚风眠的身躯上，令楚风眠的身躯，都开始破碎。
<br/>　　一道道裂痕，楚风眠的身躯如今一件灵器一般，直接破碎。
<br/>　　楚风眠的身躯，可是已经达到了极致，更是经过天晶的淬炼，可如今在这陆千刃的一指之下，还是节节破碎。
<br/>　　踏入生死台阶的老古董的实力，便是如此强大。
<br/>　　不过，就算是面对在强大的力量，楚风眠的脸色，也还是一如既往的坚毅。
<br/>　　他的身躯，开始节节破碎，但是几乎同时，楚风眠身上血脉中的战龙之力，也被全部激发出来。
<br/>　　上一次圣祭之中的建木树枝，如今其中的所有力量也都是融入到了楚风眠的战龙血脉之中。
<br/>　　在这危难时刻，一齐力量全部激发了出来。
<br/>　　顿时楚风眠的破碎的身躯，居然是还是重组，一边在这陆千刃的一指下破碎这，一边血脉的力量，也让楚风眠的身躯不断的重组。

<br/>　　楚风眠长啸一声，他的身躯，轰然之间完全重组，居然是将这一指的力量，给扛了下来。
<br/>　　“没死？”
<br/>　　“怎么可能，这小子，居然是将陆千刃老祖的一指，抵挡住了，虽然是一指力量，却也是超越生死境巅峰的力量。”
<br/>　　“看起来毫发无伤？好恐怖的恢复灵力，刚刚此人的伤势，就算不死，也是至少要重伤，可现在居然是全然恢复了过来。”
<br/>　　“这不是人了，妖兽的恢复能力，都比不上他，这是如何做到的！”
<br/>　　一尊尊武者，看着楚风眠，居然是将这一指硬抗下来，简直是不可置信。
<br/>　　这可是一尊老祖出手，要知道就算是在场的所有人联合起来，只怕都不会是这老祖的对手。
<br/>　　踏上生死台阶的人物，这已经都超越了生死境巅峰，在这样的一指下，楚风眠居然能活下来。
<br/>　　这简直是叫人无法相信。
<br/>　　尤其是刚刚，楚风眠的身躯已经是开始破碎，这样的情况下，楚风眠就算是抵挡住，重伤也是不可避免的。

<br/>　　这种恢复能力，超越了妖兽，超越了一切生物的范畴之中。
