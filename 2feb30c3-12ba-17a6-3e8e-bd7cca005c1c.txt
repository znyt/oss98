　　秦吉胜当然满意，而且是相当的满意，他并不指望能得到大唐的火药武器，那种东西不是他所能奢望的。
<br/>　　想到这里，秦吉胜搓了搓手，略有些紧张的问道：“那，那价格方面？”
<br/>　　“价格我已经在帮你争取了，不过因为钢弩在制作的成本方面比较高昂，所以单价依旧在十五贯左右，数量一万支；另外还有所配套的三棱矢，这个也不怎么便宜，一捆百支，作价两贯，而且这个数量不限。”
<br/>　　“其余粮食之类的倒是可以平价卖给你们，不过这运费需要你们来出，每斗可以给你们作价作到五十文左右。”
<br/>　　“至于说衣物，这个要根据你们的选择来定了，具体怎么样你们可以和那些商人约谈。”
<br/>　　十五贯一枝钢弩，这个价格还在秦吉胜的接受范围之内，他既然到了大唐自然也有被宰的准备，只要价格不超过二十贯，他都可以接受。
<br/>　　只是钢弩的价格虽然不高，但是箭和粮食的价格却让他有些无法接受，毕竟这两样东西都是消耗品。
<br/>　　粮食吃了之后只会变成屎，除了可以用来恶心人之外没有任何用处。
<br/>　　而箭这东西也是一样，战场之上一轮齐射就是上万支箭，按照两贯百枝的价格来计算，就等于两百贯钱随着一声‘放’，消失于无形。
<br/>　　贵，真特么贵，不过贵也得买啊，而且这还不算完，刚刚说的那些还只是一些轻武器，那些重武器他们也是需要的，比如各种攻城车，床弩之类的东西。
<br/>　　虽然他们自己也会造这个东西，可是他们自己造出来的东西往往不是那么耐用，几下功夫基本上也就报废了，不像大唐进口的东西，皮实耐用，一场仗打下来只要没有被敌人缴获，稍微把上面的血擦一擦就跟新的一样。
<br/>　　“怎么样？这个价格秦大人是否则能够接受？”王玄策见秦吉胜半晌没有说话，轻轻敲了敲椅子的扶手追问道。
<br/>　　“没问题，这个价格很合理。”秦吉胜点点头，接着深吸了一口气，试探着问道：“不过王大人，我想知道，大唐的重武器卖不卖？”
<br/>　　“你想要什么？”王玄策眯了眯眼睛，沉声问道。
<br/>　　“床弩，投石机，最好是那种可便携式的。”秦吉胜回答道。
<br/>　　王玄策看了看外面的夜空，沉默了一会儿才缓缓说道：“床弩可以卖，五百贯一架，投石机也可以卖，七百五十贯一架，不知秦大人想要多少？”
<br/>　　“越多越好。”秦吉胜这下是真的兴奋了，两只手搓的都有些发红。
<br/>　　如果能得到这些大型武器，归化人的战力无疑会成倍增加，攻破山背大兄驻守的城池指日可待。
<br/>　　但这一次王玄策并没有马上答复他，敷衍着笑了笑说道：“这样吧，我回去找人商量一下，你也不要着急，这段时间在长安好好玩上几天，看看马球比赛也是一个不错的选择。”
<br/>　　“可以，那我就等着王大人的好消息？”秦吉胜也知道心急吃不了热豆腐，连连点头。
<br/>　　而且他也看出来了，王玄策似乎也不知道床弩这东西应不应该卖，卖的话可以出多少货，之所以会答应他，应该也是看在钱的份上。
<br/>　　可钱是什么？钱不就是从地里刨出来的那些金光闪闪的东西么？这并不算什么，倭国有很多劳力，大不了让他们多刨一些也就是了。
<br/>　　……
<br/>　　是夜，王玄策离开之后，秦吉胜的房间中多出几个人来，那是他的谋士，或者说智囊，有很多他不能决定的事情，他都需要与这几个人商量。
<br/>　　寂静的夜色，昏暗的灯光下，一个老者慎重的说道：“吾王，按照您刚刚说的，臣觉得那王玄策不是很靠得住。”
<br/>　　“我知道，不过就算这样，也值得我们赌一把，如果他能搞到一千台床弩，我倒不介意按照他说的价格给他钱。”秦吉胜眯着眼睛说道。
<br/>　　“为什么？如果他是私自搞出来的呢？大唐皇帝的怒火不是我们能够承受的。”另外一个老者皱了皱眉。
<br/>　　“那有什么关系，我们只管花钱，就算是出了问题也是唐皇手下人出的问题，只要我们不少给他们钱，跟我们半毛钱关系没有。”一个年轻人口中塞着一串糖葫芦，吃的很是香甜，语气中尽是满不在乎的味道。
<br/>　　“不错，小王爷之言很有道理，我们只管拿钱，其他一概不管，他们给多少我们就要多少。”一个从来没有说过话的老头儿突然开口，表示出对青年的支持。
<br/>　　“你们说的这些都是次要的，本王关心的是大战之后，看唐皇的意思，显然是想让我们一直打下去，这次如果我们真的把山背大兄给灭了，估计接下来我们的好日子也就到头了。”秦吉胜摆手打断几个人的讨论，沉声把自己所担心的事情说了一下。
<br/>　　“那怎么办？父亲，难道我们就真的按照他的意思打下去？”青年此时已经吃光了手中的糖葫芦，将签子丢到一边，有些不情愿的说道。
<br/>　　“打是必须打，但怎么打却有说道。”秦吉胜嘴角微微一挑，露出一个诡异的笑容，然后继续说道：“我现在就在想，要不要学大唐一样，再扶植一个势力出来，等到我们剿灭了山背大兄，那个势力应该也‘成长’起来了，到时候我们就可以一直‘打’下去。”
<br/>　　秦吉胜在‘成长’和‘打’字加重了语气，目光扫过面前的几人。
<br/>　　几个老头儿和那个年轻人几乎瞬间明白了秦吉胜的意思，眼前一亮，几乎大声叫起好来。
<br/>　　不过考虑到周围的环境，他们还是忍了下来，一个个兴奋的捏紧了拳头。
<br/>　　是的，现在的归化人在倭岛上面已经取得了很大的胜利，山背大兄的势力已经被压缩到了很小的一块地域。
<br/>　　甚至可以说，如果秦吉胜对其发起全力一击的话，就算是没有大唐的这些装备，他也可以把山背大兄的势力全部消灭。
<br/>　　但是他必须考虑大唐的态度，所以才会来到大唐，一是打算摆出弱者的架式，用来换取大唐的同情；二就是打算多准备一些武器，防止万一大唐真的有一天打算对付他的时候，也不至于没有还手之力。
