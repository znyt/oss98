　　一行三人，很快准备妥当，从义庄大门走了出来，此时天色尚早，大街上面也颇为热闹。
<br/>　　因为时间尚早，三人的脚程也不算很快，顺着熙熙攘攘的大街走着，偶尔会碰见的几个熟人，都十分客气的和九叔打了声招呼。
<br/>　　看着身前笑呵呵给人回礼的九叔，刘哔舔了舔还未长成的獠牙，九叔在这里的名气之大，让他略微有些意外。
<br/>　　“我给你们说啊，任老爷是镇上有名的大户，一会你们见到他，可不要失了礼数。”走了有一段时间，九叔突然停下脚步，转过身来，对着两人仔细叮嘱。
<br/>　　闻言，文才本来满是兴奋的脸色僵硬了一瞬，动了几下嘴唇，终是没有说出什么。
<br/>　　而九叔说完，就没再理会两人，继续在前面走着，刘哔看了看有些踌躇的文才，撇了撇嘴，抬脚跟了上去。
<br/>　　因为时间太久，对于一些细节他都已经模糊了，但是大致的剧情，他却还有一些印象。
<br/>　　走了没多远，九叔伸出手来，指着前方一个装修精美的建筑道：“前面就是了，任老爷应该已经在里面等着了。”
<br/>　　点了点头，刘哔没有言语，只是默默的跟在九叔后面，恰好此时，文才紧赶慢赶的跑了过来。
<br/>　　跟着走了几步，文才有些尴尬的看了看九叔，又瞅了瞅刘哔，表情有些犹豫，看样子是想要说些什么，但是刘哔在旁边，又无法张口的样子。
<br/>　　“师父，我能不能，不和你一起去见任老爷？”最终，文才还是有些尴尬的说出这句话。
<br/>　　闻言，九叔放慢脚步，转头看了文才一眼，心中奇怪。
<br/>　　“为什么，你和任老爷有过节？”一边慢悠悠的向前走着，一边开口询问，九叔的语气有些不解。
<br/>　　看了一眼站在旁边没说话的刘哔，文才解释：“不是，我连他长什么样都不知道，不过我长这么大都还没喝过外国茶，我怕一会出洋相给师父丢脸。”
<br/>　　眉头微微一皱，九叔随即停下脚步，转身面对着文才，一本正经的说道：“难得你为师傅的面子着想，好，非常好，怕丢师傅的脸呐，你甭去了！”
<br/>　　说完，直接转身，理都没有理会文才，直接抬脚走到了前面。
<br/>　　看着呆愣愣站在原地的文才，刘哔叹了口气，杵了一下文才的胳膊：“你就没听出来吗，师父说的都是气话，还不赶快跟上？”
<br/>　　傻愣愣的应了一声，文才和刘哔一起，两人又回到了九叔身边，果然，看到文才回来，九叔只是‘哼’了一声，便也没在言语。
<br/>　　挠了挠头发，文才也不敢说话，只是乖乖的跟在后面，两人很快来到了那家店面门口。
<br/>　　“欢迎光临。”看到有客人上门，门口的侍应脸上浮现起职业化的笑容，伸手推开了店门。
<br/>　　他们这里是镇上最大的咖啡厅，服务态度自然不是别家店面可以比拟的，即使是三个人穿着老土，侍应的态度依旧很是尊敬。
<br/>　　“你好，请问订了位子没有？”随着三人走进，大堂经理迎了上来，谦卑的躬了躬身，满脸笑容的询问道。
<br/>　　咖啡厅内，装修精美，大厅内放着轻柔的音乐，虽然不是正餐时间，可还是有不少客人，他们这里常年满座，如果不提前预定，是基本找不到位置的。
<br/>　　“额，没有啊。”九叔有些茫然，对眼前的情况一时不知道该如何处理。
<br/>　　大堂经理挑了挑眉，脸上明显没有之前那样恭敬了，张了张口，刚想说些什么，却被刘哔接下来的话语给打断了。
<br/>　　“额，我们是任老爷的客人，他应该提前有订位置吧？”看到九叔脸上的尴尬，刘哔主动开口，将话头接了过来。
<br/>　　一听是任老爷的客人，大堂经理自然不敢怠慢，连忙换上一脸笑容，带三人走到了楼上。
<br/>　　跟着大堂经理上了楼，很快就见到了早在等候的任老爷，那是一个身穿米色长袍的中老年胖子，稍微有些白发，倒也算长相威严。
<br/>　　互相客套了一番，四个人纷纷落座，九叔像是想起了什么，疑惑的开口：“呃，听说令千金刚从省城回来，怎么没请她一块来呢？”
<br/>　　“唉，这个丫头啊，最近刚学会了什么化妆，整天跑去教别人，也见不到个人影。”似乎因为不满，任老爷也没有过多提及此事。
<br/>　　玄学的是，还没说几句话的功夫，任老爷突然伸出手来，指了一下楼梯口：“说什么来什么，这不就是我家丫头吗。”
<br/>　　闻言，几人都转过了头去，一个年轻女子款款走了过来。
<br/>　　转头的一瞬间，连刘哔都稍微楞了一下，那是一个打扮十分洋气的少女，粉色的连衣裙，同色的遮阳帽，脸上化着淡淡的妆容。
<br/>　　虽然因为技术的原因，少女的妆容有很多明显的瑕疵，但依旧掩盖不了那股清丽的气质，尤其是露出三分之一的酥胸，更让人浮想联翩。
<br/>　　这个女子确实漂亮，即使是放在还没爆发末世的地球，都能算是一等一的美女，但仅是如此，还不足以刘哔做出这种反应。
<br/>　　她的身上有任务！就在她刚刚上来的时候，系统就立即提醒他收到了一个新任务，而任务的内容，也是让他无比蛋疼。
<br/>　　名称：秘密（连环任务）
<br/>　　内容：一周之内，保护任发和任婷婷的生命安危。
<br/>　　等级：未知。
<br/>　　时限：七天。
<br/>　　失败惩罚：永久封闭后续任务。
<br/>　　和第一个任务不同，这次任务不但有了期限和惩罚，连任务的等级都没有明确标示出来，况且其上表明的连环任务，更是让刘哔心中犹豫。
<br/>　　看起来，这个任务也没有多难，原著中任婷婷本来就不会有什么样的事情，只有任发被僵尸袭击杀死了，只要提前做好防备，应该也出不了什么问题。
<br/>　　可是，出手守护一个本应死去的人，贸然改变剧情，这样的做法又会带来什么样的影响，这可不是游戏，不能存档，任何一个选择刘哔都不敢有丝毫大意。
