　　“你能肯定？”天尘宇闻言疑惑，他倒是没有从屠霄这个名字上考虑，毕竟，他不知道苏莫和屠霄的恩怨之深。
<br/>　　“基本上能肯定，不过，若是龙族护着苏莫，我们如何是好？”紫霄皱眉思索，这个苏莫真是有能耐，居然又和龙族攀上了关系，真是匪夷所思。
<br/>　　“紫霄兄，我会再联合雷焱神王殿、万化神殿和无双神殿的人，到时候我们联手，无论如何，要拿下苏莫。”
<br/>　　天尘宇面色郑重，继续道：“紫霄兄，我只有一个要求，若你有机会擒拿苏莫，一定交于我们天王宫，不能交于地王宫，我们愿意付出任何代价！”
<br/>　　“这个你放心，只要我得到了想要的，自然会将苏莫给你们！”紫霄点了点头，他的目标是终极剑道和终极神剑，并不是苏莫本人。

<br/>　　而他和天王宫还有些交情，和地王宫并没有什么交情，若是交出苏莫的话，自然首选天王宫。
<br/>　　“嗯，到时候，我天王宫一定会重礼酬谢！”天尘宇起身，向紫霄郑重的抱拳。
<br/>　　“嗯！”紫霄微微点了点头。
<br/>　　“紫霄兄，我还要去找雷焱神王殿和万化神殿、无双神殿之人，就不叨扰了！”天尘宇起身说道。
<br/>　　既然找到了苏莫，他就要尽量的多联系一些人，这些人都是和苏莫有仇之人，都能成为他的助力。
<br/>　　虽然他已经派人返回天王宫，但是，由于距离太远，时间上根本来不及了。
<br/>　　随后，天尘宇便带着一众天王宫之人，离开了殿中。
<br/>　　天王宫之人不多，除了天尘宇之外，还有五人。
<br/>　　无独有偶，这五人之中，有两位熟悉的面孔，这两人不是别人，正是涂赢和姜枫然。
<br/>　　这一次，跟随天尘宇而来的五人，均是天王宫的万古天骄，前来龙族观礼祖龙大典。
<br/>　　前来拜见紫霄，天尘宇也没有偷偷摸摸，而是将五人带在了身边，主要是担心地王宫之人，暗害他们五人。
<br/>　　毕竟，这五人可都是天王宫最顶级的天才，一个都不能有事。
<br/>　　涂赢早已加入天王宫多年，而姜枫然加入天王宫，也不段的时间了，在天王宫可谓是受到了重点的培养。
<br/>　　当然，以姜枫然这种天赋，无论加入哪个势力，都是会被重视的。
<br/>　　姜枫然和涂赢两人，之前并不相识，不过，相识之后，姜枫然知晓涂赢的底细，涂赢却不知晓姜枫然的底细。
<br/>　　“苏莫这次真的危险了！”姜枫然跟在天尘宇的身后，心中暗暗的想到，这要尽快将消息告知苏莫。
<br/>　　这倒是不难，他们之间可都是有龙族特制的传讯符。而且同在龙都城距离这么近，直击就可以使用。
<br/>　　……
<br/>　　在天尘宇行动之时，另一座殿宇之中，地藏神王独自一个人盘坐。
<br/>　　他的身前，恭敬的伫立着一名身材高大的弟子，此人浓眉大眼，体态雄壮，正是战乱。
<br/>　　这一次，地藏神王带了六名地王宫的天才来此观礼，因为众所周知，龙族新的祖龙，也是一位年轻天才，所以，基本上各大势力，都将麾下的天骄带来了。
<br/>　　亲眼见到祖龙的天赋，这不仅能让己方天才增长见识，还能激发斗志。
<br/>　　“云战，那天尘宇又找了雷焱神王殿之人？”地藏神王脸色沉重的问道。
<br/>　　“嗯，先找了紫霄神王，现在又去找了雷焱神王殿之人。”战乱郑重的点了点头，他心中清楚天尘宇的目的。
<br/>　　地藏神王闻言，沉默不语，面现思索之色。
<br/>　　他来到龙都城，自然也知道了那取得祖龙精血之人屠霄的情况，心中也猜测屠霄就是苏莫。
<br/>　　看来，天尘宇已经有所动作了。
<br/>　　“天尘宇有没有派人回去？”少倾，地藏神王问道。
<br/>　　“派了，派了一位天才回去！”
<br/>　　战乱点了点头，随即沉声道：“长老，我觉得，我们应该派人，将此人击杀。”
<br/>　　“不用！”地藏神王摇了摇头道：“就算是天君河亲来，也不能怎么样，龙族可不会卖天君河的面子。”
<br/>　　他并不担心，因为龙族太过强大，根本不会被天王宫左右。
<br/>　　“那我们该怎么办？”战乱好奇的问道。
<br/>　　“此事，还要从吞虚龙帝身上着手！”地藏神王眸中精光一闪，随即长身而起，大步走了出去。

<br/>　　苍穹祖龙府，密室之中，苏莫盘膝而坐，并未修炼，面无表情的望着密室的石壁。
