　　古周星，东黄城，天黄客栈。
<br/>　　苏莫再次回到了这里，回到此地之后，他本想用招来古天意的方法，将尤天韩也招到此地。
<br/>　　不过，经过从几名神武学府弟子的口中打探，他得知尤天韩并不在神武学府，而是和几名太史世家派系的弟子一起，去了一个叫做青碧城的地方。
<br/>　　尤天韩之所以去那里，是因为要购买丹药。
<br/>　　青碧城是古周星一个最大的丹药交易城池，其内有大量的以炼丹为主的势力落足，每天都有数之不尽的上好丹药，从青碧城流出。
<br/>　　苏莫打探清楚了青碧城在所在，就一路急赶了过去。
<br/>　　他已经下定主意，要将尤天韩炼化成分身了，因为对方实在是太合适了。
<br/>　　对方不仅和他有仇，修为和天赋都是很高，实乃分身的最佳人选。
<br/>　　而且，尤天韩不是和几名太史世家派系的弟子一起吗？若是有修为合适的人，正好再炼制一名分身。
<br/>　　青碧城并不远，三日之后，苏莫便到达了青碧城。
<br/>　　值得一提的是，苏莫赶路之中，一直都是催动了隐身战魂，隐藏了身形。
<br/>　　他不得不谨慎，因为古周星认识他的人实在是太多了，为了安全起见，他必须时刻谨慎。
<br/>　　青碧城，城如其名，高大的城墙呈碧青之色，仿若延绵的青色山脉。
<br/>　　进入青碧城之后，苏莫躲在一家客栈中，灵识扫荡而出，谨慎的搜寻尤天韩的踪迹。
<br/>　　青碧城并不大，没过多久他便找到了尤天韩。
<br/>　　此刻，尤天韩居然在一个拍卖场内，这个拍卖场很小，相比于大荣城的拍卖场，简直犹如蚂蚁和大象的区别。
<br/>　　拍卖场坐落在一家豪华商铺的后院，整个拍卖场中不足千人，正在进行丹药拍卖。
<br/>　　呃！
<br/>　　苏莫闷哼一声，脸色不禁一白，他的灵识刚刚进入拍卖场，探查到尤天韩的气息，还没看清尤天韩，拍卖场中便有一股浩瀚强大的灵识，瞬间将他的灵识击的溃散开来。
<br/>　　“好强的灵识！”苏莫低声自喃，那拍卖场中有强者守护，他是无法继续查探了。
<br/>　　少倾之后，他轻叹一声，用灵识注意着那家商铺，静静的等待了起来。
<br/>　　一场小型的拍卖会，也要不了多长的时间，很快尤天韩就会出来。
<br/>　　……
<br/>　　傍晚时分。
<br/>　　青月商铺门前人.流如织，很多在商铺内参加拍卖会的武者，纷纷走了出来。
<br/>　　一袭锦衣，身形挺拔的尤天韩大步的走出了商铺，其身边还跟着三名青年。
<br/>　　“尤师兄，这玉虚丹还真是贵啊，一枚就要八万下品灵晶！”其中一名身材矮小的青年，面带惊叹的说道。
<br/>　　“这还用说吗？玉虚丹可是冲击武尊境界的辅助丹药，有极大的效果！”另一名面色黝黑的黄衣青年道。
<br/>　　“尤师兄有了玉虚丹，半月之内便能达到武尊境，成为内府的弟子了！”最后一名高瘦的白袍青年，面带羡慕的说道。
<br/>　　“呵呵！
<br/>　　尤天韩闻言轻笑一声，摇头道：”没有这么快，晋升武尊境非同小可，不可贪功冒进，但是，一个月之内突破到武尊境，我还是有把握的！”
<br/>　　“提前恭喜尤师兄了！”
<br/>　　“恭喜尤师兄！”
<br/>　　“恭喜！”
<br/>　　三名青年纷纷向尤天韩抱拳道喜。
<br/>　　“哈哈！我们走吧！”
<br/>　　尤天韩开心一笑，随即带着三人离开了青碧城，向神武学府的方向飞去。
<br/>　　“尤师兄，你的伤势完全恢复了吗？”飞行之中，面色黝黑的青年向尤天韩问道。
<br/>　　“完全恢复了！”尤天韩道。
<br/>　　“呵呵，尤师兄是大难不死必有后福！”矮小的青年笑着说道道。
<br/>　　“的确是！”尤天韩点了点头，目光望着昏暗的天空，心中颇有感慨。
<br/>　　他之所以能有把握突破到武尊境界，还要多亏了苏莫。
<br/>　　本来，以他的修为情况，起码还要再沉淀个两年的时间，才能有把握冲击武尊之境。
<br/>　　但是，真圣界一役，他遭受苏莫的重创，双臂被震碎，肉身几乎快要崩溃了。
<br/>　　但是，他也因祸得福，双臂很容易便重新催生，而经过小半年的时间，他的伤势彻底的恢复之后，境界瓶颈居然松动了。
<br/>　　一场重创，让得他能提前冲击武尊之境，这真是因祸得福。
<br/>　　而为了增加成功率，他才来青碧城购买玉虚丹，有了玉虚丹之助，他有九成的把握，一举破境成功。
<br/>　　“哈哈！这么说来，我们还要感谢苏莫了！”矮小青年大笑出声，面带调侃之色。
<br/>　　“呵呵！我倒是想感谢他，只可惜他现在已经去了地狱！”尤天韩轻笑道。
<br/>　　“说起来，这苏莫辛亏是死了，若是不死真的是太可怕了！”
<br/>　　面色黝黑的青年叹道，他从来没见过像苏莫这么可怕的天才，武帝境六重修为，就能扫荡武帝境之内无敌手。
<br/>　　这种天赋和战力，唯有用‘恐怖’两个字能形容了。
<br/>　　虽然苏莫没有和姜神夜交手，但就算交手了，苏莫也是胜算极大。
<br/>　　“多行不义必自毙，这又怨得了谁呢！”尤天韩摇了摇头，在他看来，像苏莫这种狂妄无知的人，想不死都难。
<br/>　　其他三人闻言点头，对于尤天韩的话完全赞同，这完全是自作自受。
<br/>　　“你们要感谢我？”
<br/>　　就在此刻，蓦地，一道虚无缥缈的声音，突然在众人的耳边响起。
<br/>　　“嗯？”
<br/>　　尤天韩四人闻言，先是一怔，随即纷纷面色大变。
<br/>　　“苏莫？”
<br/>　　“这怎么可能？”
<br/>　　“你居然没死？”
<br/>　　四人立刻停下了身形，他们瞬间便听出了，这是苏莫的声音，绝对不会有错。
<br/>　　四人心中狂震，面色立刻惨白，难道苏莫没有死？这怎么可能？他们可是亲眼看到苏莫被巫仑大祭司一巴掌捏死的！
<br/>　　不过，四人眼眸扫视四方，发现四周方圆数万里之内，都没有半个人影。
<br/>　　“快走，分散离开！”
<br/>　　尤天韩心念急转，立刻向四人喊道，现在根本不是考虑苏莫为何没有死的时候，只有尽快逃走，才能保证安全。
<br/>　　苏莫有隐身的本事，这他们可都知道，所以虽然看不到苏莫，但他们知道苏莫肯定就在他们附近。
<br/>　　嗖！嗖！嗖！嗖！
<br/>　　尤天韩话音刚落，四人的身影，便如同四道闪电一般，撕裂长空，向四个方向急速飞驰。
