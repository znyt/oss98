陈默，陈金贵是打心眼里感谢。但是嘴上却说不出什么感谢的话，只能用自己的行动来表示，这也是他晚上起来巡逻查看的原因。<br/>　　<br/>　　陈默对于这个事情，基本上不做干涉。如果他不让金贵叔巡查的话，可能反而会打击金贵叔的积极性，还不如就让他这样子，他也能用这种方式表示感谢。<br/>　　<br/>　　“哈！金贵叔，没有关系的。就像你说的，回来后也是没有时间查看菜地等等，这不大晚上有时间了，所以就过来看看！”说着，掏出一包烟，抽~出一根香烟来递给陈金贵。<br/>　　<br/>　　“金贵叔，来抽一根！”陈默说道。<br/>　　<br/>　　“哎！好！”将手电直接夹道自己的腋下，然后结果香烟，本来想掏出火机自己点火呢，却没有想到陈默打着火机，凑到了眼前。顿时赶紧用手半捂，然后凑前点燃。<br/>　　<br/>　　他知道陈默是不抽烟的，所以就没有将口袋里的香烟拿出来。抽上烟以后，他也渐渐的话多了起来，和陈默聊了聊没在的时候，发生的一些事情。<br/>　　<br/>　　等聊了一会，陈默就和他分开了，主要是陈默回去，而陈金贵继续巡查一下葫芦谷里。<br/>　　<br/>　　在分开之前，陈默将手中的香烟，递给了陈金贵：“金贵叔，你拿着夜里抽，提神！”<br/>　　<br/>　　“哎！好好！谢谢了！”陈金贵赶紧感谢到，他也知道这是陈默的一点小小的心意，自然没有拒绝。<br/>　　<br/>　　等回到别墅里后，看时间已经是凌晨三~点多了，所以就回到了地下室的练功房里，布置了阵法以后，就开始缓缓行功。<br/>　　<br/>　　今天时间已经很晚了，什么事情也不做了，就行功当睡觉吧！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
