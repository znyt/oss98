　　　　本来两颗头就很怪异了，现在又冒出一颗头，三头摇曳，画风那叫一个惊悚呀。
<br/>　　涂小安看到双头蛇变成三头蛇，一点都不诧异，这完全是正常的套路。
<br/>　　以后这条蛇的蛇头会越冒越多，直到九颗蛇头摇曳而恐怖如斯。
<br/>　　上古九头蛇一出，敢叫天地变颜色。
<br/>　　进化还在持续，不过从三头蛇的成型，涂小安知道这场进化，也接近了收尾。
<br/>　　随着一道鬼哭狼嚎的凄厉声，整个别墅在剧烈的震动摇晃起来，一股一股恐怖惊骇的阴冷气息从地上那条三头蛇身上迸发出来，一浪高过一狼，歇斯底里一样。
<br/>　　众人纷纷的动容，感觉置身在骤雨狂风中似的，温度已经零下到几十度，这种冷，是直穿骨髓的冷，不是穿多少衣服可以抗寒。
<br/>　　涂小安眼睛越发的明亮，此时，三头蛇的体形开始从一米暴涨了起来，气息愈演愈烈，体形越来越大。
<br/>　　撕拉...
<br/>　　别墅大厅中的一面墙壁如蜘蛛网一样的蔓延开来，然后四处的波及。
<br/>　　“该死，这样下去，老牛的这房子又要成为废墟了”
<br/>　　牛妖王周身涌出一股能量，一只手按在了墙壁上，这才使得别墅的震动，慢慢的平缓了起来。
<br/>　　五米...10米...13米..。。
<br/>　　别墅的上方，彻底成了露天模式，被人掀了头盖一样，三颗蛇头仰天咆哮，气息恐怖如斯。
<br/>　　最终，三头蛇定格到了15米，一跃步入二阶精怪层次。
<br/>　　“哇哇哇...这蛇黑大爷我喜欢，太酷了”黑鸟缭绕在三头蛇的蛇躯飞来飞去，眼睛冒金光，一副作死的样子。
<br/>　　树精却被三头蛇这番样子给吓得全身树叶簌簌而下，扑通扑通扑通，既然一下子掉下了三颗灵朱果。
<br/>　　意外之喜呀。
<br/>　　三颗灵朱果一出，直接给黑鸟夺去一颗，另两颗自然是青枭跟黄金瞳的。
<br/>　　这两个小家伙，现在是准备全天候守早树精旁边，随时等待掉果实。
<br/>　　涂小安悬浮而起，飞到了别墅的上空，望着三颗蛇头相互缭绕的蛇，眼睛浑然一眯。
<br/>　　“叮...超等变异进化体三头蛇，等级二十一级，恐怖如斯的二阶邪物”
<br/>　　嘻嘻，邪物也是属于我涂小安的邪物。
<br/>　　不错不错，既然从19级直接跳到了二十一级，这就是血脉天赋强的好处了。
<br/>　　这时，三颗蛇头冲着涂小安身边环绕，三头六眼，看的人心惊胆战。
<br/>　　“主人...主人...主人...”
<br/>　　忽然，三颗蛇头同时开口，就跟有回应一样，声音是口吐人言，但是却极度的沙哑，或许是还没适应说话一样。
<br/>　　“乖...”涂小安很高兴，此蛇总算没有跟黄金瞳一样，步入二阶，还是嗷嗷叫。
<br/>　　“好啦好啦，变小吧，你那么大，这别墅可容不下你，还会引来“城管”的”
<br/>　　涂小安各自拍了拍三颗摇曳的蛇头，自己落了下来，而三头蛇也很听话，直接开始就地缩小了，变成一米的三头蛇，浑身蛇鳞片惊心动魄。
<br/>　　如今进化完成后，涂小安也不准备将它送回蛇谷去，就在自己的身边先待着。
<br/>　　蛇谷已经满足不了这条邪蛇的胃口，给扔到妖灵山脉最是合适。
<br/>　　此番进入妖灵山脉，涂小安也会将身边的青枭跟黄金瞳，都扔到妖灵山脉磨练，好好的提升实力。
<br/>　　它们的血脉都是超等，超凡脱俗，就是等级太低了点，现在没有个三阶，对涂小安来说，真的没什么用，也只能当宠物来玩玩。
<br/>　　“主人，你什么时候把老牛给进化了”
<br/>　　牛妖王眼中迸发强烈的欲望，他的实力已经很难得到提升，就算给他个几百年，也不一定可以达到妖王的层次，除非血脉进化。
<br/>　　涂小安看到这头牛几乎已经贴到自己的身上献媚，顿时一阵恶寒，很敷衍道：“牛啊，一切自有定数，等着吧”
<br/>　　牛妖王闻言顿时迷茫了，这是什么逼话？？？。
<br/>　　另一边...
<br/>　　“老三，你过来，小姐姐带你玩哦”
<br/>　　青枭老气纵横的朝着三头蛇挥了挥翅膀，都是自己人，自然要处好关系，关键还有地位问题。
<br/>　　“我是那坏蛋第一个收服的，所以我是老大，这只猫排第二，就叫二猫，你最后来，你是老三，看你是三个头，叫你三蛇吧”
<br/>　　青枭几句话下来，就将自己的地位说明白了，用先来后到来排名。
<br/>　　三蛇头摇曳着蛇头，看着漂亮绝伦的青枭，没有第一立刻发飙。
<br/>　　“嗷...”黄金瞳冲着它叫了声，算是打招呼。
<br/>　　“二猫是哑巴，还不会说话”青枭解释道。
<br/>　　黄金瞳迷了：“......”
<br/>　　三头蛇沙哑道：“我要当老大”
<br/>　　“三蛇，你要懂规矩，不然我们就不跟你玩了”青枭看到三头蛇之前进化恐怖的样子，早想明白了，要真的打，它好像有点打不过。
<br/>　　还好它早将黄金瞳给拉拢过来了。
<br/>　　为了保住自己的地位，道：“你看到那颗大树了吗，你要是听话，我就弄颗果实给你吃，吃了就会变强的哦”
<br/>　　三头蛇质疑的将六只眼睛放到树精的身上，树精全身一颤栗，树叶簌簌而下，大叫不好。
<br/>　　这只漂亮的不像话的鸟也不是什么好东西。
<br/>　　为了自己的地位，这就将它给卖了，难道天下的鸟都这么坏吗。
<br/>　　“啧啧...你一只雌孔雀还想当老大吗，将我黑大爷置于何地”黑鸟咻的一声，飞到了树精的身上，道：“告诉你们，以后跟着黑大爷我混，这颗树可是我栽，喊老大，以后分你们一颗果实吃”
<br/>　　“黑鸟，草泥马”树精看到这个逼就烦人，什么时候它成了这只鸟栽的。
<br/>　　“真丑...”青枭斜睨了这只乌鸦一眼，满眼的嫌弃，看狗屎一样的看着它，道：“死乌鸦，以后本孔雀在的时候，你不准出现，我怕你的丑会传染给我”
<br/>　　全身羽毛都是乌漆墨黑的，简直给天下的鸟丢人。
<br/>　　黑鸟差点从树精的身上掉下来，丑还会传染？我读书少你可别骗我。
<br/>　　顿时怒声道：“小孔雀，黑大爷我这叫帅，有阳刚之气，在废话，小心黑大爷我强奸你，让你当我的压寨夫人”
<br/>　　这两种鸟是一个极端，一只高贵美丽，有祥瑞之鸟的称号，一只臭名远扬，有不祥之鸟的外号，自然相互不可能处到一块去。
<br/>　　“哼...二猫，给我揍它”
