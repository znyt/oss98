　　“玄天长老，人王宫出现如此妖孽，真是可喜可贺！”
<br/>　　“恭喜！”
<br/>　　“恭喜！”
<br/>　　几位长老纷纷开口，向玄天长老道喜，如此妖孽的天才，任何势力得到，都是大幸之事。
<br/>　　不过，这只是大家表面上的客套而已，没有谁是真心贺喜。
<br/>　　比如天有为，脸色稍显难看，本来这一次，按照他的预料，天尘山得第一，十拿九稳。
<br/>　　而另一位妖孽，得第二也悬念不大，但现在，出了一个段惊天，将所有人全部镇压了。
<br/>　　“奇怪，怎会有这么逆天的妖孽？”
<br/>　　天有为有些郁闷，这个段惊天修混沌之力，有强大的剑类元神，应该是万化神剑，而那庞大的战意，只有不败战气。
<br/>　　拥有两种强大的元神，一个排在太古武魂榜第十四，一个排在第八位，着实惊人。
<br/>　　这两种元神，谁若是有任何一种，便可称为妖孽，更遑论是两种了。
<br/>　　再者，他在段惊天的身上，感受到了强大的血脉之力。
<br/>　　这股血脉之力，更加的可怕，仿佛是至尊战体，但是，明显比至尊战体更加强大。
<br/>　　而现在倒好，连古魔之眼都出来了，简直是无法想象。
<br/>　　不光是天有为长老如此，其他长老心中亦是暗暗惊讶，这个段惊天的战力，若是对战普通的武者，足以抗衡三品中级真神了。
<br/>　　一品高级的修为，拥有三品中级的战斗力，这种天赋战力，已经超越了以往那些万古天骄。
<br/>　　天有为叹了口气，这个段惊天的天赋，比天尘山的大哥，天王宫的无上神王天尘宇当年，还要厉害一些。
<br/>　　“怎么会有如此人物？”
<br/>　　天有为和一众其它王宫的长老，心中都有些疑惑，这么厉害的妖孽，未免也太夸张了。
<br/>　　……
<br/>　　平台之上，苏莫身形缓缓落了下来，目视着前方倒地的天尘山。
<br/>　　天尘山并没有受到太严重的创伤，在地上翻滚了两下，便缓缓的战了起来，收回了九龙弑天元神。
<br/>　　他的脸色极为苍白，因为精神受创，短时间内很难恢复。
<br/>　　不甘的凝视了苏莫一眼，天尘山没有说半句话，转身飞回了他所在的天尊手指之上。
<br/>　　败了就是败了，多言更显得他不甘心。
<br/>　　全场一片寂静，现在几乎是大局已定了，段惊天就是第一名。
<br/>　　这个结果，出乎了所有人的预料，若不是亲眼所见，没有人会相信。
<br/>　　平台上，苏莫环顾周围，望向剩下的人，最后目光落在了天王宫那位高瘦青年的身上。
<br/>　　“这位朋友，不知如何称呼？”苏莫朗声问道。
<br/>　　他知晓此人不凡，很可能就是涂赢，不过对方没表露身份，他也不能确定。
<br/>　　嗖！
<br/>　　高瘦青年闻言，身形一闪，便落在了平台之上。
<br/>　　“段惊天，我知道自己不是你的对手，但还是想和你切磋一招！”
<br/>　　高瘦青年并未回答苏莫的问题，面色郑重的说道。
<br/>　　“那你出手吧！”苏莫点了点头，对方是不是涂赢，和他交手就能知晓了。
<br/>　　高瘦青年没有多言，身上气势升腾，随即他手掌抬起，掌心向天。
<br/>　　霎时之间，风云突变，虚空之中，无尽的源气伴随着浩瀚的虚空之力，形成了一柄虚幻的巨刀。
<br/>　　这巨刀长达千丈，气息骇人，恐怖绝伦。
<br/>　　“斩！”
<br/>　　高瘦青年手握虚空巨刀，重重的一挥而下，犀利的巨刀，斩碎空间，速度快到巅峰，直击苏莫的脑袋。
<br/>　　这一刀虽然普通，但是极为可怕，仿佛与整个世界融为一体。
<br/>　　一刀斩下，仿佛挥动了整片世界的力量，大气磅礴，霸道无比。
<br/>　　“这……此人居然这么强大？”
<br/>　　“感觉比归无血都要强！”
<br/>　　“天啊，又是一个万古天骄！”
<br/>　　众人一见高瘦青年出手，顿时大惊之色，此人居然也隐藏了真正的实力。
<br/>　　姬月松、归无血等得到名额的人，同样是心中一惊。
<br/>　　一个段惊天，已经彻底改变了战局，居然又出一个万古天骄，这太不可思议了。
<br/>　　“果然是你！”苏莫见此，眉头一挑，此人果然就是涂赢。
<br/>　　这一招，和他当初与涂赢交手之时，对方施展的那一招一模一样，只不过现在威力强大了十几倍。
<br/>　　而且，从现在高瘦青年施展这一招，苏莫也瞬间明了了。
<br/>　　对方之所以施展这一招，不仅是要和他切磋，更是在告知他身份。
<br/>　　对方在告诉他，其就是涂赢。
<br/>　　轰隆隆！
<br/>　　虚空巨刀斩下，磅礴的威势掀起空间爆炸，巨大的力量，仿佛太古神山镇压而下。
<br/>　　“碎！”
<br/>　　一声轻喝，苏莫再次出剑，无匹的剑气逆斩苍穹，狠狠的斩在了虚空巨刀之上。
<br/>　　轰！
<br/>　　一声爆响，虚空巨刀炸裂，化为无尽的虚空之力，迸射八方。
<br/>　　“我认输！”
<br/>　　一招结束，高瘦青年朗声说道，苏莫战胜天尘山之前，他是很想和苏莫一分高下的。
<br/>　　毕竟，当初一战，他不比苏莫弱，现在想要看看两人之间有没有差距。
<br/>　　但是，在看到苏莫战胜天尘山之后，他就没有了这个心思。
<br/>　　因为他知道，自己依旧落后了一步。
<br/>　　苏莫点了点头，虽然知道了对方就是涂赢，但是，既然对方隐藏身份，不愿意和他正常交流，他也没有揭穿。
<br/>　　只不过，他心中有些疑惑，涂赢为何会在天王宫？又为何隐藏身份呢？
<br/>　　随即，涂赢转身飞回了天尊手指之上。
<br/>　　“还有谁来赐教？”苏莫没有多想，环视剩下的几人，继续问道，他已经击败了四人，还有五人需要击败。
<br/>　　不过，剩下的无人，包括姬月松在内，个个面面相觑，却是无人敢上台。
<br/>　　“你们都要认输？”苏莫见众人没有反应，朗声问道。
<br/>　　“这……”
<br/>　　几名妖孽有些迟疑，若是不战上一场就直接认输，实在有失颜面。
<br/>　　但是，若是上台一战，无异于自取其辱。
<br/>　　他们都是妖孽，基本上年轻一代之中，从来没有败过的，更没有认输过，现在不禁有些犯难了。
<br/>　　仿佛战也不是，不战也不是，处于进退两难之地。
<br/>　　“诸位，给你们一个机会，要么集体认输，要么联手与我一战！”苏莫目视几人，朗声开口道，声震全场。
<br/>　　既然要表现给玄天神王看，他自然不会藏拙，要极近展现自己。
