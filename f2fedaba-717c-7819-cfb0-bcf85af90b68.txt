<br/>　　这太史轩也太能胡扯了吧！
<br/>　　苏莫正欲张口否决，一道太史轩的意念传音，直接钻进了他的脑海中。
<br/>　　“苏莫，我太史世家的廖云剑，是你杀的吧！你若是加入我太史世家，我们不予追究，否则就是与我太史世家为敌，你要考虑清楚了！”

<br/>　　只见太史轩面带如沐微笑，给人温文尔雅的感觉，但话语却是赤.裸.裸的威胁！
<br/>　　看来对方已经知道了，廖云剑死于他的手中。
<br/>　　“我并没有答应加入太史世家一脉！”目光凝视着太史轩，苏莫一字一句的说道。
<br/>　　他本来就对太史轩没有任何的好感，现在对方居然出言威胁，真是不知所谓！
<br/>　　这更加大了他的反感。
<br/>　　苏莫本就是不畏强权的人，太史轩想压他，根本不会有用。
<br/>　　太史轩闻言，顿时脸色一黑，眸光寒光闪烁。
<br/>　　苏莫拒绝居然了他，而且还是当着众人的面，否决了他的话，这是在打他的脸！
<br/>　　一股怒火，在太史轩心中缓缓升腾。
<br/>　　“哼！太史轩，你可真是有出息，简直是谎话连篇！”
<br/>　　身材高壮的景雷不屑的冷哼一声，满脸鄙夷之色，然后对苏莫道：“苏莫，加入我们赤霞洞天一脉吧！保准你不会后悔！”

<br/>　　“我们星象圣门才是最强大的宗门，苏莫，你莫要错失良机！”

<br/>　　苏莫看的暗暗皱眉，他目前不想加入任何派系。

<br/>　　“诸位，你们十三大派系，都各有各的优点，我暂时也拿不定主意，日后会好好思考，再做决定！”
<br/>　　苏莫眸光扫视众人，微微抱拳，继续道：“各位请回吧！不送！”
<br/>　　言罢，苏莫转身回了阁楼之中，嘭一声关上了阁楼的大门。
<br/>　　他既然不想加入某个派系，也懒得和众人虚与蛇尾。
<br/>　　众人见此，个个面现无奈之色，所有人都来争抢，自然很难让苏莫做出决定。

<br/>　　随即，众人纷纷返回，此事不会就这么结束的，有机会他们还是要私下拉拢。
<br/>　　“不识抬举！”太史轩目中寒光一闪，冷笑一声，便也转身离开。

<br/>　　二个时辰！
