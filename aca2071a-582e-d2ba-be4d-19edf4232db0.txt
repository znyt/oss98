　　　　“这条三阶虺蛟我让给你，不与你争了，只要今晚你能放我一马”

<br/>　　绝美无双的她这声音清朗，冷冷传出，犹如水激寒冰，风动碎玉，置之死地而后生。

<br/>　　紫金巨蟒蛇瞳出现布满了无数的冰刃，每一道冰刃射到人的身上，都能瞬间将一个人冻结，也能瞬间一箭穿心，夺人性命。

<br/>　　紫金巨蟒的巍峨身躯一寸寸被冻结，眼看就要形成一具完整的冰雕。
<br/>　　“还是一样的套路”
<br/>　　涂小安冷笑一声，浑身一震，准备将冻结的冰震碎。
<br/>　　可霎时间，他微微一愣，这些冰的厚度很高，将他三十米的身躯严严实实的包住，密不透风。
<br/>　　一时半刻，居然挣脱不了。
<br/>　　“就在现在，跑...”
<br/>　　冷霜莲步生风，似浮光掠影般抽身而退，她跟这条变态到逆天的蛇精，没有任何的胜算，必须立刻回到温城，回到妖精管理局。
<br/>　　那条三阶虺蛟...她眸光贪婪，却不得不放弃。
<br/>　　命要紧，等她会到妖怪管理局，这个仇必须要报。
<br/>　　她要让这条蛇精上天无路入地无门，被整个人类修武者通缉。
