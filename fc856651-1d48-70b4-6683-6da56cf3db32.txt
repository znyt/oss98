　　“不，他交给我来。”
<br/>　　布克尔特从一名武装毒贩手中拿过一把MK17自动步枪（SCAR-H），将快慢机拨到了半自动模式，用安装在战术导轨上的ACog战斗瞄准镜瞄准了正朝着海滩边跑去的毒贩，缓缓扣动了扳机。
<br/>　　因为这把MK17自动步枪的枪口上安装了消音器的关系，武器只是发出一声沉闷的射击声，枪口也没有明显的上扬，7.62mm口径的子弹飞旋出枪膛，带着尖锐的破空声飞向那个还在狂奔的毒贩。
<br/>　　“噗嗤！”
<br/>　　子弹从逃跑毒贩的后脑勺贯入。
<br/>　　在子弹击中毒贩后脑勺的瞬间，毒贩的身体明显的向前踉跄了一下。
<br/>　　弹头击中目标后所产生的空腔效应将逃跑毒贩的大脑搅成了一团浆糊；继续保持着高速运动的子弹从毒贩的前额位置穿出，掀飞一大块骨头，血浆喷溅而出，毒贩脸皮就像是被人硬生生从面骨上剥离下来一样，上半部分跟他的面骨分离开来，挂在他的鼻子处，毒贩又向前迈出几步，随后一头栽倒在地上，从他身体中涌出的血液将他脑袋下那片土地染成暗红色……
<br/>　　一刀，一颗子弹，这就是布克尔特对待触犯了他底线的人的办法。普提查一言不发的站在旁边，看了眼倒在地上，脖子上那道狰狞口子中还在往外涌着鲜血的阿扎普，何尝不清楚，布克尔特这次出手杀掉自己的两个手下，是对自己的一次警告？警告自己不要触犯他的底线，否则两具尸体就是自己的下场。
<br/>　　布克尔特放下手中的MK17自动步枪还给那名武装毒贩，迈步走出了这间充满了血腥味的屋子，地面上那颗还带着温度的子弹壳，冒出缕缕青烟，从弹壳中飘散出来的火药味，被那浓烈刺鼻的血腥味彻底掩盖住……
<br/>　　布克尔特在诊所外等了有一个多小时，olivia和村子里那个女医生才从诊所中走出来。
<br/>　　“她的情况怎么样？”布克尔特对olivia问道。
<br/>　　“她的性命是保住了。”
<br/>　　还没等到olivia开口，站在olivia身边的乌克兰女医生就抢先开口回答道：“但是她肚子里的孩子没了，这次流产可能会对她以后的生育功能造成影响，最好还是把她送到正规的医院去，这里的条件太差，而且药品也不全。”
<br/>　　布克尔特看向olivia，olivia冲他点了点头，说道：“这次多亏了她，如果不是她的帮助，我一个人恐怕很难处理好，你可以让我在战场上完成一次截肢手术，但是让我为一个女人进行引产，我并没有这方面的经验。”
<br/>　　“她是谁？”布克尔特对普提查问道。
<br/>　　“娜塔莉娅。”
<br/>　　女医生扯掉自己头发上的发圈，将有些凌乱的金色长发整理后，重新用发圈扎好，很从容的看着布克尔特，眼睛中没有任何的害怕。
<br/>　　“娜塔莉娅？”布克尔特盯着娜塔莉娅看了一会儿，说道：“你这次帮了我的忙，我可以答应你一个条件，如果你想要离开这里，这是不可能的。你可以选择跟我离开，因为我需要一个女医生。”
<br/>　　“既然你已经帮我做出了选择，还需要我再多说什么？”
<br/>　　娜塔莉娅依旧保持着之前的从容，从一开始，她就没奢望着能获得自由，从这个渔村里离开；比起留在这个村子里不仅要做那些毒贩的医生，还要供那些男人玩乐，娜塔莉娅情愿跟着布克尔特离开这里。
<br/>　　“我可以跟着你离开，我也可以按照你的要求照顾好这个女人。”娜塔莉娅竖起食指，对布克尔特说道：“但是我有一个要求，除非我愿意，否则你和你的手下不能强行跟我发生关系。”
<br/>　　“这个我可以向你保证，绝对不会发生你担心的事情，这是我的底线。”布克尔特向娜塔莉娅保证道：“任何触犯我底线的人，只有死路一条。”
<br/>　　布克尔特没有去问普提查的意见，因为根本不需要，就算普提查心中有再多的不情愿，也无法违抗布克尔特。
<br/>　　……
<br/>　　经过四个多小时，长途飞行了三千多公里，战术中队搭乘的A330客机降落在T国南部的一个机场。
<br/>　　下了飞机，换上T国当地的电话卡，罗昊跟葛铭取得了联系，问清楚葛铭位置后，战术中队的十几个人立刻朝着航站楼的停车场走去。
<br/>　　亮粉色、翠绿色，颜色鲜艳到扎眼的出租车停靠在停车场车道边，几个司机嘴里叼着烟，正在用T国语交谈着什么，看到战术中队十几个人走出航站楼，立刻站起身来，但是看到罗昊他们这些人根本就没有打车的想法，又重新蹲了下来，抽着他们嘴里那并不好抽的烟。
<br/>　　葛铭这次是开着一辆旅游大巴车来的，当罗昊再三确认了车牌，确认没有弄错后，才是爬上了旅游大巴。
<br/>　　葛铭穿着一件花衬衫、戴着一副大墨镜、穿着一条大裤衩，脖子上还挂着根金链子。就葛铭现在这个打扮实在是很难跟龙隐部队特勤小队队长的身份相联系起来。
<br/>　　这就是特勤小队强于战术中队和战备中队的地方，他们每个人都是伪装的高手，他们在一个国家生活，他们就能彻底融入到这个国家中去，不仅是语言上，更是各种生活习惯和谈吐方式上。
<br/>　　“葛队长，你这打扮‘够潮’的啊。”
<br/>　　等到旅游大巴车的车门关上后，罗昊对打着方向盘，把车驶出停车位的葛铭说道：“我差点都没认出来。”
<br/>　　葛铭露出被烟熏得有些泛黄的牙齿，嘿嘿一笑，说道：“如果随便就能被人认出来，那就是我们的失败。”
<br/>　　“绑架者的身份确定了吗？”
<br/>　　车子驶入高速公路后，罗昊开始向葛铭了解情况。
<br/>　　说到正事，葛铭的表情也是变得严肃起来，说道：“通过我们在T国的情报系统，我们也是得到了一些情况。”
<br/>　　葛铭从遮阳板中抽出一张照片，递给罗昊说道：“根据对酒店监控录像中所拍到人的脸部轮廓分析，基本可以确定，进入王权酒店对佟千雪实施绑架的人就是照片上的人。”
