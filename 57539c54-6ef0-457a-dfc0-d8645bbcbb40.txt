　　皇甫擎现在无比的后悔，后悔将大三宝术传于苏莫。
<br/>　　若是苏莫没有大三宝术，现在哪里会有实力和宗主一战！
<br/>　　当年，他偶得大三宝术，视若珍宝，也就传给了几名弟子。
<br/>　　本以为能轻易掌控几名弟子，没想到却是出了苏莫这个变数，现在一切都已经无法挽回了。
<br/>　　先如今，皇甫擎只希望，宗主能击杀苏莫，彻底解决这个祸端。
<br/>　　“杀！”
<br/>　　在皇甫擎思索之间，苏莫和巫森已经再次交起了手。
<br/>　　只见苏莫身形如剑，直接飞临了巫森头顶，帝皇剑裹挟亿万钧之力，重重的斩击而下。
<br/>　　沸腾的玄力、强悍的气力，让得帝皇剑锋芒绝世，威势滔天。
<br/>　　一剑斩下，整个天地都被分开，恐怖的杀伐之气，逆冲九天。
<br/>　　“滚！”
<br/>　　巫森大喝，手爪一扫，锋锐无匹的利爪如同疾电，瞬间和帝皇剑碰撞在了一起。
<br/>　　铛！
<br/>　　刺耳的金铁交鸣之声，响彻在天地之间，利爪和帝皇剑相击，猛烈的劲气直接炸开，如同狂风扫荡。
<br/>　　嗖！嗖！
<br/>　　苏莫和巫森同时暴退，强大的反震之力，让两人均是暴退了数百里。
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　而后，两人如同两颗飞驰的流星，不断的碰撞，杀招迭出。
<br/>　　轰轰轰！！
<br/>　　爆响不断，巨响震天，两人不停的碰撞，几乎是呼吸之间，两人便硬撼了上百次，每一次碰撞都毁灭一方天地。
<br/>　　一时间，两人谁也奈何不了谁，再次呈平分秋色之局。
<br/>　　“巫森，你的实力不过如此，也该结束了！”
<br/>　　又是一击碰撞之后，苏莫朗声高喝，他已经摸清了对方的具体实力，不想再生变故，准备彻底解决对方了。
<br/>　　对方的实力，也就差不多如此了，就算还有其它底牌，也不可能强到哪里去了！
<br/>　　而后，苏莫身形划破长空，再次向对方暴冲而去。
<br/>　　“哦？”巫森闻言眼眸一眯，难道苏莫还有什么底牌？
<br/>　　不过，就算是一般的底牌，能再增加一些实力，也不可能轻易打败他。
<br/>　　嗖！
<br/>　　苏莫身形如同一阵幻影，瞬息间便临近了巫森万丈之内。
<br/>　　“死吧！”苏莫怒喝一声，眸中杀机凌厉万分，他身上的气息再次暴涨，更加的狂暴，如妖似魔。
<br/>　　他再次的抽取了体内两成的精气神，所以实力再次暴涨。
<br/>　　不仅如此，下一刻，震撼人心的一幕出现了，苏莫背后虚影不断的升腾，足足十种武魂齐齐出现。
<br/>　　这十种武魂，分别是大日武魂、幽冥魔瞳武魂和八柄剑武魂。
<br/>　　大日武魂对于苏莫的实力增幅，自然不言而喻，八柄剑武魂和苏莫修炼的剑道契合，能最大程度的增加他的攻击力。
<br/>　　这八柄剑武魂，虽然等级都不是太高，但全部都是天级武魂，也不算太低。
<br/>　　而幽冥魔瞳武魂，虽然不能增肌他的攻击力，但却能看穿他人的弱点，看清对手的薄弱之处。
<br/>　　这一刻，苏莫展现了从未动用过的本领，他以前很少动用多武魂的力量，也就当初在天月国之时，动用过五种武魂。
<br/>　　以及和战乱一战之时，动用了三种武魂。
<br/>　　但现在，他却是动用了十种武魂的力量，这是从未有过的，他就是要一击绝杀巫森。
<br/>　　当然，苏莫吞噬的武魂数以百万计，他能动用的武魂远不止这个数，但动用的武魂越多，对于他的精神控制力要求越高。
<br/>　　他不能分出太多的心神，因为他还要控制本命灵剑阵。
<br/>　　就算是一次催动十种武魂，他已经感觉非常的吃力了，再多就会适得其反了。
<br/>　　十大武魂齐现，浩荡的锋锐的之气如狂涛骇浪，席卷天地，形成无数的虚幻剑影，方圆数千里之内的空间立刻支离破碎。
<br/>　　大日武魂的至热至阳之力，同样充斥方圆千里。
<br/>　　几乎是瞬息之间，方圆千里之内，仿佛化为了一片九幽死地。
<br/>　　“星辰陨！”怒喝震天，苏莫出手了，居高临下，一剑挥出。
<br/>　　刹那之间，恐怖浩大的三色剑气挥洒长空，贯穿天地，无匹的锋芒，恐怖的剑威，逆冲天穹，浩荡不休。
<br/>　　惊天剑气从九天之上急速斩下，粉碎了空间，切开了虚无，裹挟毁天灭地之威，向巫森当头斩去。
<br/>　　不仅如此，在苏莫出剑的同时，早已被他收回的二十四柄本命灵剑，再次出击。
<br/>　　二十四柄本命灵剑，顷刻间化为二十四星宿剑阵，向巫森展开了绝杀。
<br/>　　这是苏莫必杀的一击、绝杀的一击，誓要在这一击之下，斩杀巫森，不给对方任何翻盘的机会。
<br/>　　苏莫几乎没有多少保留了，大三宝术抽取了六成的精气神，动用了十种武魂，以及帝级上品武技大苍穹剑法、二十四星宿剑阵。
<br/>　　他相信，在他这一次绝杀之下，整个苍穹世界，没有人能挡的住，即便是武帝境的巫森，也不可能抵挡的住。
<br/>　　以前在苏莫弱小之时，他不敢在人前展示逆天的武魂天赋，但现在，他已经站在了苍穹世界的巅峰，没有了任何的顾忌
<br/>　　这一刻，时间范若静止，天地间仿佛完全失声，巫森瞪大了眼眸，犹如石化了一般，一时间被眼前惊人的一幕搞懵了！
<br/>　　不仅是巫森，所有观战之人，都看到了苏莫的变化。
<br/>　　所有人均是张大了嘴巴，眼眸外突，仿佛要瞪出眼眶一般，不可思议的看着眼前的一幕。
<br/>　　十种武魂？
<br/>　　这……可能吗？
<br/>　　所有的人均是完全石化了，思维彻底停滞了，因为他们看到了这世间最为惊人、最为恐怖、最为不可能的一幕。
<br/>　　在苍穹大陆的历史上，双生武魂的逆天天才稀少无比，万年难出一人。
<br/>　　战乱便是这种万年难得一见的天才，拥有双生武魂，所以才能成为万尊榜第一人。
<br/>　　而苏莫在与战乱一战之时，展露了三生武魂，已经让无数人为之震憾，因为三生武魂的天才，在苍穹世界有记载的历史上，根本没有出现过。
<br/>　　但现在可好，简直是吓死人不偿命，苏莫居然拥有十种武魂！
<br/>　　这超越了所有人的想像！颠覆了所有人的认知！
<br/>　　再一次的刷新了武魂天赋的记录！
