        “你一个人逛，我不放心。”傅昼景也没意识到大哥傅夜沉已经接听了电话。
<br/>
<br/>        童筱颖仍旧坚持自己的想法：“这里是大学，挺安全的。你放心好了，你先去忙。不用管我！”
<br/>
<br/>        “筱颍，你是在生我气吗？”傅昼景凝眉反问。
<br/>
<br/>        童筱颖一脸懵然：“我没有。”
<br/>
<br/>        “没有的话，那你为什么不听从我的安排。”傅昼景严肃道。
<br/>
<br/>        童筱颖怔了怔，不得不低沉地应了声：“那我听你的安排便是。”
<br/>
<br/>        傅昼景看了看童筱颖后，垂下眸子，这才发现自己的手机正在和大哥傅夜沉处于通话状态。
<br/>
<br/>        “喂，大哥。”傅昼景将手机举到耳边，唤了一声。
<br/>
<br/>        童筱颖这才闭上嘴，不再吭声。
<br/>
<br/>        “你找我有事？”傅夜沉淡淡地问。
<br/>
<br/>        傅昼景回答道：“嗯，想麻烦大哥，派个司机来临海大学的正门口接筱颖回傅家大庄园。晚上，我公司里还有事情要去办，得马上回滨江城一趟。不方便送筱颖回去。”
<br/>
<br/>        “好，我派个司机过去接筱颖回来。”傅夜沉应道。
<br/>
<br/>        傅昼景微微一笑：“谢谢大哥了！”
<br/>
<br/>        电话挂断后，傅昼景见童筱颖仍旧一脸不开心，便抬起手来，在她的头顶揉了揉，然后扣住她的后脑勺，倾着身子，在她的眉心上吻了吻。
<br/>
<br/>        “好啦！我的筱颍不要不开心！等我忙完了，回来我一定补偿你！”傅昼景温柔地哄道。
<br/>
<br/>        童筱颖强颜欢笑地点了点头。
<br/>
<br/>        从傅昼景的车上提着自己的行李箱下来后，童筱颖站在了临海大学的校门口，看着傅昼景开着车子扬长而去。
<br/>
<br/>        现在虽然临近放暑假的时间，但是临海大学的校门口，仍旧人来人往有很多大学生路过。
<br/>
<br/>        童筱颖拖着行李箱，在一旁花坛前的长木椅上坐了下来。
<br/>
<br/>        可她没坐多久，耳畔便响起了一个久违了的声音。
<br/>
<br/>        “诺筱颖？！”
<br/>
<br/>        听到这声呼唤，童筱颖抬起头来，循声望去。
<br/>
<br/>        只见一个戴着眼镜，留着短碎，身上穿着浅蓝色条纹衬衫的男生朝她走了过来。
<br/>
<br/>        临海城四季如夏，男生身上那件条纹衬衫，她记得是在他俩分手前，她花了一千块钱，在品牌专柜里买下来，特意送给他的生日礼物。
<br/>
<br/>        没想到，这分手后，他竟然还有脸穿着她给他买的衣服。
<br/>
<br/>        一看到何明旭，童筱颖本能地从长椅上起身，拖着行李箱欲准备离开，却被何明旭伸手给拦住了去路。
<br/>
<br/>        “怎么？跟我玩欲擒故纵的把戏吗？拖着行李箱，特意坐在我大学的校门口来等我，好不容易等到我，又想着用离开的方式，来试探我吗？”何明旭一脸嘚瑟着，上上下下将童筱颖打量了一番。
<br/>
<br/>        童筱颖今天穿着的是傅昼景昨天送给她的那件水粉色真丝雪纺连衣裙，虽然脚上还是穿着白色的平底鞋，但是她扎着马尾，额前的斜刘海俏皮可爱，身上散发着青春灵动的气息。
<br/>
<br/>        或许是因为是傅昼景挑选的缘故，比起以前，她穿衣的品味，要提高了不少。
<br/>
<br/>        而且，最主要的是何明旭很识货，一眼就看出筱颖身上这件水粉色真丝雪纺连衣裙是限量版。
<br/>
<br/>        因为，他的那位富家女友林若琴也想买，但是没买到，还跟他抱怨过。
<br/>
<br/>        “你还真是一如既往的不要脸。”童筱颖嗤之以鼻，打算绕路离开。
<br/>
<br/>        何明旭顿时气恼了起来，伸手便抓住了童筱颖的手腕，将她往自己跟前拉动，恼火地质问道：“不要脸的女人是你才对吧！怎么，到底是人靠衣装佛靠金装。把自己打扮得比以前更漂亮了，然后跑到我这儿来显摆吗？诺筱颖，我知道，你还喜欢着我。但是，我要告诉你。你不要再对我有任何幻想了！”
<br/>
<br/>        “麻烦你，放手。”童筱颖深吸了口气，冷冷地说道。
<br/>
<br/>        她不打算跟这家伙解释，因为也没必要跟他解释。
<br/>
<br/>        以前，她还以为他何明旭是她的初恋，现在想想，他在她心里其实什么也不是。
<br/>
<br/>        当初，她看上他，只不过是在他的身上，看到了小卫的影子。
<br/>
<br/>        所以，现在他俩算是扯平了吧！
<br/>
<br/>        “诺筱颖，你这是用什么口气跟我说话！”何明旭一时间受不了童筱颖对自己冷言冷语，心里突然间有点儿不甘心起来。
<br/>
<br/>        以前的那个筱颖，对他可谓是百依百顺，她勤工俭学，能赚多少钱，就给他多少钱。
<br/>
<br/>        就连宿舍里那三个舍友，都羡慕他找了一个超级温柔体贴的女朋友。
<br/>
<br/>        而现在，她诺筱颖是用什么口气在跟他说话？
