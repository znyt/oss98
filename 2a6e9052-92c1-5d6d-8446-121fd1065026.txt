　　第294章
<br/>　　樊梨花强撑着上马，腹痛如绞，汗落如雨，可手依旧紧紧握着刀柄。
<br/>　　“落玉寨的好兄弟就无人了么？”
<br/>　　樊梨花马上喝了一声。
<br/>　　“寨主，我们兄弟都在！”
<br/>　　“和这帮混蛋拼了！”
<br/>　　樊梨花并不是没有人，依旧有五十几人护在他左右，有男有女，即便是落玉寨的女人，也都有一身好武艺。
<br/>　　阴万良无奈的摇摇头，道：“寨主，你为何不能顾全大局，为山寨的数百位兄弟考虑考虑。”
<br/>　　“好无耻的话，难道你们这帮叛徒还有理了。”
<br/>　　“婚姻大事，父母之命，老寨主死得早，我是老寨主的兄弟，你的婚姻大事就由我做主。”阴重道：“我让你嫁，你就要嫁。”
<br/>　　“嘿，我爹可没你这样厚脸皮的兄弟，你不用给自己脸上抹金。”
<br/>　　“你个小妮子懂什么，我这都是为了你好。”阴重摊开手掌：“也是为山寨三百多弟兄好。”
<br/>　　樊梨花目光看过去，见许多人挤在一起，脸上神色变幻，既没有站在阴万良那里，也没有站出来支持自己，这些人都是墙头草，只要谁能在今日获胜，他们就会听谁的。
<br/>　　看来阴重父子拉拢的人并不多，如果自己身体无恙，倒也不惧他们。可是今日……悄悄下毒，看来阴重父子已经谋划良久，这毒也不致命，却令自己腹内阵阵绞痛，险些刀都握不住。
<br/>　　“还与她废话什么，直接动手抢了。”赤檑此刻已经有些烦躁。
<br/>　　“寨主，对不住了。”
<br/>　　阴万良手持铁枪拍马而来，樊梨花冷笑：“凭你还差得远呢。”
<br/>　　梨花刀甩开，漫天梨花飞落，只是一个回合，就将阴万良击落马下。再进攻，就能取阴万良性命，可无奈何腹内像揣了一根匕首，浑身力气用不上。
<br/>　　等樊梨花压住体内阵痛，就只看自己的手下已经和赤檑的人，阴万良的人厮杀起来。
<br/>　　对方人多，己方人手，瞬间便被压在下风，接连有几个兄弟毙命。
<br/>　　“阴万良，今日我必杀你。”樊梨花咬紧贝齿，杀意绽放。
<br/>　　可此刻的她也陷入戎族的包围圈中，赤檑一马当下，用双刀会她。本来樊梨花本事并不惧对方，可身上的毒却削减了她的实力，十成本事只能使出五六成而已。
<br/>　　如此，二人便僵持不下，最后樊梨花身边围了七八个人。一柄长刀上下舞，叮当声不绝于耳。
<br/>　　“杀呀，杀！”
<br/>　　阴万良没料到樊梨花如此了得，中了毒还能撑这么久。他不敢再向樊梨花出手，选择攻击仍旧忠于樊梨花的弟兄。
<br/>　　“阴万良，你这个畜生，竟敢背叛寨主！”一名大汉吼着：“你给我去死吧。”
<br/>　　此人名叫马丹霞，和阴万良一样，在山寨中长大。他本是被人遗弃在草原上的孤儿，由樊锐收养，所以对樊锐父女忠心耿耿。
<br/>　　一柄铁枪，扎入他的胸膛。枪的主人是阴万良，他将铁枪拔出，脸上露出不忍之色。
<br/>　　“马兄弟，对不住了，我也是为了山寨的弟兄能有口饭吃。”
<br/>　　樊梨花仍陷在众人中间，挣脱不去重围。与她交手的赤檑此刻阵阵心惊，只听说这女人了得，却没料到她的实力真就如此恐怖。
<br/>　　只见她一身白衣，此刻被染得血红，双手握刀，在马上抡开。即便身体中毒，以多敌少，但却不露下风，反而越打越勇，七八个人难以近身。
<br/>　　无边落玉萧萧，碎琼梨花纷纷。
<br/>　　刀光绽放如梨花，身影翻动胜玉琼，身上白袍沾染的血滴越来越多，一袭白袍由白转红。
<br/>　　“这女人怎么这么猛！”赤檑大叫：“难道不知道给她多下点药。”
<br/>　　“已经是双倍剂量，可以放倒马的。”
<br/>　　“呔，樊梨花，你不怕死，这些兄弟的命你也不要了么！”
<br/>　　阴万良突然喝了一声，樊梨花扭过头来，看到忠于自己的手下已经死伤大半，仍活着的也被擒住，刀压在他们脖子上。
<br/>　　“阴万良，你……”
<br/>　　趁着她愣神的光景，赤檑的刀压在她肩头。
<br/>　　“樊寨主，还不束手就擒么？”赤檑冷笑着。
<br/>　　樊梨花咬紧牙关，最后却是无可奈何的将刀放下。而她的身体，其实早已撑不下去，一口气泄了，身体立刻跌落马下。
<br/>　　“来人，绑了。”赤檑道。
<br/>　　立刻有人上前，将樊梨花五花大绑。战斗过后，落玉寨死了几十名弟兄，地上倒着一具具尸体，空气中弥漫着血腥味。
<br/>　　“这是何苦来哉，这是何苦来哉。”阴重痛心疾首：“都是山寨兄弟，为何要拔刀相向，寨主，你若早日明悟，何止让这么多兄弟丢了性命。”
<br/>　　樊梨花已痛得说不出话来，此刻只能用愤恨的眼神盯着阴万重父子。
<br/>　　阴万良又冲着落玉寨其他人道：“寨主嫁到西颜部，对我们来说也是好事。以后两家联手，定有我们的吃穿。今日死去的弟兄，都是为这件事而死，我们绝不可忘了他们。”
<br/>　　其他这些人战斗时没有插手，倒也不是支持阴万良父子，只碍于对方势大，不敢动手。到此刻，他们更加不敢说什么了。
<br/>　　“你们的事你们自己处理，樊寨主我就先带走了。”赤檑骑在马上笑道。
<br/>　　程大雷心急如焚，他刚才一直没闲着，努力在撬钉子。也不是没有进展，他已经撬出一颗钉子，可想要突破木笼还是做不到。
<br/>　　眼看樊梨花就要被戎族带走，程大雷怎能看着这一幕发生。但此刻陷入木笼中的他，便是有天大的力气也使不上。
<br/>　　“哈，这才是人生难预料，不想团圆在今朝，回首繁华如梦渺，残生一线付惊涛。”
<br/>　　赤檑正押着樊梨花往外走，不想一个声音突然响起。众人都循声音望去，只见骡马围着的木笼中，一落魄汉子坐在其中，口中慵懒的唱着。
<br/>　　樊梨花看到这一幕，心中叹口气：这下你可得意了，可我被抓走，你怕就要送到莫鸣米那里。
<br/>　　赤檑皱起眉头：“你是什么人呐？”
<br/>　　“若问我的名要从女娲造人说起，若问我的姓，则要讲到开天辟地。不过你问我是什么人么？”程大雷笑笑：“我可以告诉你，我在这里，你不能带走她？”
<br/>
