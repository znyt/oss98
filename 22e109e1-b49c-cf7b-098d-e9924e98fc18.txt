　　高空之上，苏莫急速而行。
<br/>　　不用去猜，他也能知道，自己必然会遭到追踪。
<br/>　　毕竟，终极神剑乃是神剑山，终极神王殿的山门，他把别人的山门都取走了，别人不可能放过他。
<br/>　　“不行，以我的速度，很难逃过神王级别强者的追踪！”
<br/>　　苏莫心中暗道，随即，他再次改变气息和容颜体貌，并且极度收敛气息，并变换了飞行的方向。
<br/>　　改变气息容貌，苏莫不单单是用的天幻神丹，连千幻元神也动用了，可谓是双管齐下。
<br/>　　他兜了一个圈子，不向远处逃窜，反而向终极神王殿所在的方向而去。
<br/>　　这样更容易逃脱，毕竟，别人轻易不会想到，他没有逃走，反而去了终极神殿的方向。
<br/>　　不怕一万，就怕万一，他一定要最大限度的保证安全。
<br/>　　嗖！
<br/>　　苏莫的身形，如同一道流光一般，眨眼间便消失不见了荒原之上。
<br/>　　少倾之后，流光再现，终极神王率领着紫霄神王等人，来到了此地。
<br/>　　终极神王停留下来，紫霄等人亦是停了下来。
<br/>　　“气息居然消失了！”
<br/>　　身材高壮的大汉毁灭剑王皱眉说道，适才，他们一直追寻着虚空中残留的气息，但是到了这里，那股气息居然消失不见了。
<br/>　　其他人尽皆皱眉，神念笼罩八方，查找苏莫的踪迹。
<br/>　　神王境的强者，神念极强，轻易可以笼罩方圆万里，如同终极神王这种高级神王，神念可以笼罩三万里以上。
<br/>　　但是，三万里之内，并没有那秦陨的气息。
<br/>　　虚空中残留的气息，本就很是淡薄，而且是无数气息混杂在一起，非常的杂乱。
<br/>　　“怎么会？”终极神王面色阴沉，若是他们几人，连一个小小的武者都抓不到，那简直是个笑话。
<br/>　　“殿主，此地秦陨的气息中断了，但是，却有另外一股很是淡薄的武者气息！”风雪神王沉声道。
<br/>　　终极神王面无表情，并未回答，他同样感知到了这股气息。
<br/>　　但是，这股气息，与适才秦陨的气息完全不同，不能确定是不是与秦陨有关。
<br/>　　“这样吧，未免此人逃走，我们分散追踪！”
<br/>　　略一沉吟，终极神王沉声说道，话音一落，他便身形一闪，顺着那股留下的气息，快速的追击而去。
<br/>　　而紫霄神王三人，点了点头，便分别追向另外三个方向。
<br/>　　……
<br/>　　高空之上，苏莫速度如电，苏莫快到极致。
<br/>　　“不知道有没有神王境的强者追来？”
<br/>　　一边急速飞行，苏莫一边心中暗暗猜测，不过，宁可信其有不可信其无。
<br/>　　“怎么样脱身呢？”苏莫心念电转，他必须想一个稳妥的方法，神王级别的强者，感知力无比的敏锐，速度肯定比他快的多。
<br/>　　“有了！”
<br/>　　蓦地，苏莫脑中闪过一缕灵光，手掌一番，掌中出现数十只噬冥虫。
<br/>　　噬冥虫便是当初组成飞虫大剑的飞虫，只不过，当初飞虫大剑彻底毁灭，大多数高等级的噬冥虫都死了，他剩下的并不多。
<br/>　　苏莫手掌源力涌动，一瞬间，数十只噬冥虫身体上，都缭绕了他的气息。
<br/>　　然后，所有噬冥虫四散而开，分散飞向八方。
<br/>　　而苏莫再次改变了方向，急速离去，靠这些噬冥虫可以混肴感知。
<br/>　　任凭追踪之人多强，总不可能从这么多气息之中，辨别出哪个是他的真身吧！
<br/>　　很快，苏莫便再次消失不见。
<br/>　　几个呼吸的时间过后，终极神王到来，身形停留在了适才苏莫释放噬冥虫之地。
<br/>　　“嗯？”终极神王眉头紧皱了起来，在他的感知之中，几乎是四面八方，都有秦陨的气息。
<br/>　　他明白，这是秦陨动用了某种混肴感知之法，想要瞒天过海。
<br/>　　随即，终极神王闭上了眼眸，神念铺天盖地，感知力弥漫八方，仔细的感知。
<br/>　　“噬冥虫！”
<br/>　　片刻之后，终极神王睁开了眼眸，在他的感知之下，有数十股气息之中，蕴含极为淡薄的噬冥虫气息。
<br/>　　虽然这气息很淡，但是，却是难以瞒过他的感知。
<br/>　　“难道此子是魔族之人？”终极神王心中一冷，噬冥虫乃是古魔的体内的寄生之物，一般只有魔族之人才有。
<br/>　　不过，他很快就否决这个猜测，因为魔族之人，不可能修炼混沌源力。
<br/>　　没有耽搁时间，终极神王再次锁定了苏莫的方向，快速的追了过去。
<br/>　　……
<br/>　　苏莫一路疾行，不时的释放出一些噬冥虫，并且不时的改变方向。
<br/>　　飞行了小半个时辰，他也没有没有被任何人追上，心中稍稍的松了口气。
<br/>　　“城池！”
<br/>　　就在此刻，苏莫眼眸一亮，因为，他在视线的尽头，前方出现了一座城池。
<br/>　　这座城池不大，但是，对于他来说，却是能有极大的帮助。
<br/>　　苏莫心中大喜，身形如电，立刻便进入了城池之中。
<br/>　　进入城池之中，苏莫没有做任何的停留，南门进北门出，并且再次改变了气息和体貌。
<br/>　　这城池之中，武者很多，每时每刻都有人外出，正是给予了他逃离的机会。
<br/>　　离开城池之后，苏莫径直的向轮回神朝的方向而去。
<br/>　　……
<br/>　　苏莫走后，半刻钟之后，终极神王来到了这座小心的城池。
<br/>　　身形伫立在城池的上空，终极神王面色阴沉的可怕，因为，他追丢了。
<br/>　　之前，因为苏莫不时的释放裹挟他气息的噬冥虫，每一次都让他耽搁不少时间。
<br/>　　但是，总算是一路辨别真伪，追到了这座城池。
<br/>　　但是，这座城池虽然不大，也有近百万的武者，各种气息铺天盖地，非常的混乱。
<br/>　　所以，他彻底的无法锁定苏莫的气息了，也就无法在追击了。
<br/>　　“混账！”
<br/>　　愤怒的终极神王，一声暴喝，震动苍穹，城池都剧烈的震颤了起来。
<br/>　　虽然他不擅长追踪，但是，连一个小小的蝼蚁，都能在他手中逃脱，他如何不怒？
<br/>　　“神王！”
<br/>　　“终极神王！”
<br/>　　“神王息怒！”
<br/>　　城池之中，近百万武者，被暴怒的终极神王吓的面无人色，更有不少人直接下跪，向天穹之上的终极神王跪拜。
<br/>　　这里是终极剑域，终极神王可谓是终极剑域的主宰，所有人均是心惊肉跳。
<br/>　　阴沉着脸，少倾之后，终极神王身形一闪，向终极神王返回而去。
<br/>　　既然无法追踪，他也不会耗在此地，此刻当务之急，一是重建山门，二是调查出秦陨的具体背景。
<br/>　　不过，此时此刻，终极神王心中已经有了一个猜测。
<br/>　　当初，天尘宇来找过他，拖他留意一个叫苏莫的人，并且告知了苏莫的大概情况。
<br/>　　现在，他基本能够断定，这个秦陨很可能就是苏莫，当然，具体秦陨是什么情况，他还需要回去了解一番。
