　　第552章
<br/>　　“山水经么……”
<br/>　　夫子口中说出这三个字后，大厅内明显分成两派。书院宋寒之等人的表情变得十分难看，像自己心爱的女人被山贼抢走，又住了一年半载似的。
<br/>　　而马猛这样的江湖高手，却是一头雾水：山水经么，是什么东东？
<br/>　　程大雷也属于江湖一派，山水经这三个字，他还是第一次听见。
<br/>　　而刘发财很显然也没听过这三个字。楚云生此刻也在大厅内，程大雷冲他招招手，楚云生乖巧过来。
<br/>　　“山水经……什么东西？”
<br/>　　楚云生果不愧是有文化的人，缓缓道：“书院收集天下藏书，自己也著书立传。书院曾派出弟子，考察帝国山河地理。前后花费十年时间，出动上百名弟子，成一本地理志。帝国十三州一百单八城都在这本山水经中。”
<br/>　　“呦，不过就是一份地图么，丢了就丢了吧。”刘发财不屑道。
<br/>　　场上和刘发财同样想法的人不在少数，诸多江湖人表情都松弛下来。
<br/>　　“我当什么呢，原来就是份地图，应该不太要紧吧。”
<br/>　　“想书院多少藏书，他们偏偏偷了一份地图，看来这伙贼人没什么眼光啊。”
<br/>　　……
<br/>　　夫子脸上没有什么表情，静坐不动。宋寒之等人却很难掩饰不耐烦了：和这群没文化的江湖人当真是没办法沟通。
<br/>　　程大雷这次没有和这群江湖人同流合污，他还是能明白山水经的重要性。他本就是个十分重视地图的人，若能提前掌握帝国地理，知道何处适合骑兵冲锋，何处适合屯兵防守，何处可以埋伏设置陷阱，那么未战便多一半胜算。
<br/>　　关键就是看书院出品的这本山河经，能精细到什么程度。
<br/>　　那么问题来了，书院呕心沥血，煞费苦心琢磨这本书究竟是什么目的。
<br/>　　“那个……”程大雷率先问出口：“有副本么？”
<br/>　　夫子叹了口气：“山河经成书之后，便一直搁在藏书楼中，因为此书极其重要，即便在书院，也只有寥寥几人有资格翻阅这本书。”
<br/>　　“那么……”程大雷再问：“原始资料什么的有么？”
<br/>　　“书成之日，所有收集的资料都被我亲手毁了。”
<br/>　　“这样啊，写书的人呢，该不会也杀了吧？”程大雷。
<br/>　　“呃……”夫子不得不把注意力放在程大雷身上，这小子打破砂锅问到底，明显对山河经很感兴趣，如果有副本会怎样，你再去偷一次么。
<br/>　　程大雷并不是没有这个打算，他已打定主意今日要夜探藏书楼。万一夫子撒谎呢，说实话，对于读书人的人品，他并不是很信得过。
<br/>　　过了片刻，又一人走进大厅，他便是书院守门的张六。当年他也是书院巨贼，面对在做的江湖豪强也是一点不惧的。
<br/>　　“张六昨夜与贼人动过手，张六，你来说说，贼人的功夫是什么路数？”夫子。
<br/>　　张六笔直的站着，一副谁也不看在眼里的表情。他开口道：“为首是个黑袍大汉，相貌我倒是没看清楚，不过他的功夫大开大合，偏重强攻，看上去不像帝国路数。”
<br/>　　这推测与程大雷猜测一模一样的。根据张六所说，目前便有几种可能，一种是各路诸侯手下的高手，为了隐藏身份，所以没有用本门功夫；其二，对方可能来自海上，琅琊城来来往往的客商是多的，要么是异域的客商，要么就是海上盘踞的大盗。第三种可能，对方可能是戎族。
<br/>　　“还有一件事，昨夜我与他们交手，应该伤了他们中一人。”张六道。
<br/>　　“当真？”宋寒之眼睛一亮。
<br/>　　张六不是很愿意搭理他，冷哼一声道：“能在我追魂棍下逃脱的，怕满天下也没有几人。”
<br/>　　张六一直是这个秉性，宋寒之也不以为意。他道：“昨夜城门就关了，因为出了这件事，城门现在还未开，而且贼人还负伤，很大可能，他们还没有离开琅琊城。”
<br/>　　夫子点点头：“要向诸位赔罪，因为山水经十分重要，若落到外族手中，后患无穷。所以在贼人没有找到前，琅琊城的进出都被封锁了。还要留诸位在琅琊城多住几日，当然一切所需都由我书院照顾。”
<br/>　　“好，原该如此。”程大雷大声道：“能在书院垂听圣人教诲，是我们求之不得的造化，大家说是不是啊。”
<br/>　　众人目光都看着他，心道这娃是个逗比吧。
<br/>　　谁不知道，夫子这话有软禁的意思。往更深里说，就是和盗书事件有关，即便没有亲自参与，说不得也会里外勾结。
<br/>　　大家虚伪的应承着，说着些『是，是』之类的话，然而心里都有些不痛快。
<br/>　　程大雷心里是愿意留在琅琊城的，他对帮书院抓贼没什么兴趣，但对山水经却极为有兴趣。这山水经对蛤蟆城来说，也有很大的用处。
<br/>　　当然，程大雷没忘记另外一件事，待大厅的人散去后，他逮到宋寒之。
<br/>　　“宋先生，请问一下，华氏一族的人到了么？”
<br/>　　“他们在今早到的，暂且安排在后山居住。欧公子是现在就要见他们么？”
<br/>　　“不，不，出了这样的事，还是先等一等吧。”
<br/>　　如何和华氏一族的人见面，程大雷还没有想好，总不能自己说句话，人家就随着自己当山贼去吧。这件事，还需好好想想。
<br/>　　反正，现在城门关着，他们也走不了。等解决掉盗书的事，再解决华氏一族的事也不迟。
<br/>　　如今，整座琅琊城都被封锁，进出都被禁止了。不单单是城门，就是客商来往的码头也被封锁，也就是在琅琊城，书院才有这样的势力。
<br/>　　城内，一座荒废的义庄中，出现几个黑衣人。
<br/>　　其中一人趴在棺材板上，后背的衣服用刀隔开，背心出现一片黑淤，看上去甚为可怖。
<br/>　　黑袍大汉冷着一张脸，问：“她的情况怎样？”
<br/>　　“柔劲入肺腑，外表看不出什么，但五脏六腑都受了重伤。想要救好她，只有开刀将淤血放出来。”手下答道。
<br/>　　“那为何还不动手？”
<br/>　　“我们……都没有这样的本事。”
<br/>(本章完)
