　　“呵呵！没有这么夸张吧？”苏莫谦虚的笑了笑，他虽然对于吞噬战魂有很大的自信，但还没狂妄到横扫环宇，独断万古的地步。
<br/>　　这话说的未免太不切实际了！
<br/>　　“一点不夸张！”
<br/>　　阎魔鬼圣声音郑重了起来，凝神道：“汝吞噬战魂可以掠夺他人的战魂和血脉，这就是最逆天的存在；虽然受限于精神力，汝一次性能动用的战魂不多，但依旧逆天到了极致；而且若是汝能吞噬数种、乃至数十种特殊体质的血脉，汝的天赋将超越万古，无人能出其右。”
<br/>　　“嗯！这个时间不会太久！”苏莫点了点头，战魂多得是，但有特殊体质的人却是少之又少。
<br/>　　“苏莫，吾给汝一个忠告，将来若是遇到强大的战魂或是特殊体质，尤其是太古武魂榜和太古异体榜上，那些排名前列的战魂和特殊体质，一定不能错过，全部吞噬为己用！”阎魔鬼圣郑重的告诫道。
<br/>　　“嗯！”苏莫点头，若是敌人有这种战魂和特殊体质，他当然要掠夺过来，但若是朋友的话，那就另当别论了。
<br/>　　随即，阎魔鬼圣便陷入了沉寂之中，不再开口。
<br/>　　苏莫轻叹一口气，随即低头沉思了起来，少倾之后，他手掌一番，便取出了一枚玉简。
<br/>　　这枚玉简，正是记录了太古异体榜的玉简。
<br/>　　在巫尘的记忆中，巫人王的特殊体质好像叫什么‘天玄武体’，苏莫要查看一番，这到底是什么体质。
<br/>　　毕竟，他现在吞噬了对方八成的血脉力量，也拥有了这种体质，总要搞清楚有什么作用吧！
<br/>　　稍稍查看了一番，苏莫便找到了天玄武体，此体质排在太古异体榜第七百四十九名。
<br/>　　而根据太古异体榜上的介绍，这种体质能增加玄力的威力，只要催动这种体质的血脉力量，无形中就能让玄力的威力增加近五成。
<br/>　　此体质不受修为的限制，无论是什么修为境界，都能增加五成的实力。
<br/>　　“不错！”苏莫暗暗点头，虽然这种体质没有多逆天，修为较低之时没有太大的作用，但增加近五成的攻击力，在武帝境之上的武者身上却是很恐怖。
<br/>　　只不过，他并没有将巫人王的血脉力量吞噬干净，只是吞噬了八成而已，威力会稍稍减弱。
<br/>　　还有就是，他现在天玄武体和大日神体融合，也不清楚到底变成了什么体质。
<br/>　　终之，不管变成了什么体质，都是有百利而无一害。
<br/>　　“嗯？”
<br/>　　就在此时，苏莫眉头一挑，因为有人来到了他的住所门前，此人正是巫穹。
<br/>　　在巫族之内，苏莫一直都很小心，灵识感知时刻注意着外界的动向。
<br/>　　因此，巫穹一来到他的住处，他便发现了。
<br/>　　随即，苏莫吞噬战魂立刻进入分身之中，然后收起了本体，便走出了密室。
<br/>　　“你有何事？”
<br/>　　打开大门，苏莫淡淡的望着对方，冷冷的问道。
<br/>　　“那个……巫尘，我来是向你道歉的！”巫穹有些忐忑，面色不自然的望着苏莫。
<br/>　　他此行是要向巫尘道歉，以化解两人之间的过节。
<br/>　　毕竟，现在巫尘成为了少司，身份地位远不是他可比的了。
<br/>　　为了免遭报复，巫穹只能硬着头皮来道歉了。
<br/>　　“道歉？”苏莫闻言一怔，随即打量了对方一番，眸中闪过一缕微不可查的寒光，对方居然来了，那就不要走了。
<br/>　　沉吟了片刻，苏莫点了点头，道：“进来再谈吧！”
<br/>　　言罢，苏莫直接向屋内走去。
<br/>　　巫穹不敢怠慢，急忙跟了上去。
<br/>　　很快，两人来到了一处偏殿之中，分宾主落座。
<br/>　　巫尘的住处，是一个院落，客厅、偏殿、练功房等应有尽有。
<br/>　　“巫尘，之前少司选拔之时，是我错了，说了一些不对的言语，还望你莫怪！”巫穹刚一坐下，便深吸了一口气，沉声说道。
<br/>　　“嗯，我不怪你，多谢你给我送资源了！”苏莫淡淡的点了点头。
<br/>　　“送资源？”巫穹闻言一怔，满脸疑惑之色，他什么时候给对方送资源了？
<br/>　　“嗯！你就是资源！”苏莫如是的点了点头，话音未落，他身上陡然喷涌出一股淡紫色雾气，瞬间将巫穹笼罩。
<br/>　　下一刻，巫穹目光呆滞，身躯石化了。
<br/>　　时间缓缓流逝，约莫过了十个呼吸之后，巫穹的身体直直的从座位上倒了下去，再无半点生命气息。
<br/>　　“是你自己找死，怪不得我了！”苏莫走到了巫穹的身边，一脸的冷漠。
<br/>　　别说对方招惹过他，就算无冤无仇，他也是不会放对方离开。
<br/>　　苏莫对于巫族之人，只要能杀，就绝不放过。
<br/>　　他刚才将对方拉进了梦魔空间之中，然后折磨了一个时辰，直接将对方的精神摧残的崩溃了。
<br/>　　然后，他便将对方的精神意志彻底杀死了。
<br/>　　本来他以为对方不会真正的死亡，毕竟只是梦境，但没想到对方居然真的死了。
<br/>　　这倒也让他微微意外，看来他对于梦魔战魂的玄妙之处，还是了解的不够透彻。
<br/>　　苏莫心中感慨，这梦魔战魂在他的手中，有他强大的精神力支持，真是无往而不利，实在恐怖，简直是杀人于无形。
<br/>　　比在巫魔的身上，不知道强大了多少倍。
<br/>　　这种梦魔战魂并没有出现在太古武魂榜上，其上虽然有跟梦魔战魂类似的战魂，但并不相同。
<br/>　　他现在甚至有些后悔，后悔没有在梦魔空间中直接杀死巫人王，若是如此的话，他能借机留下对方的尸体，然后完全吞噬干净。
<br/>　　叹了口气，苏莫微微催动了吞噬战魂，顿时将巫穹没有意识的战魂吸了出来，而后直接吞噬了。
<br/>　　巫穹的战魂，居然是一棵树，较为少见，但也不是什么逆天战魂。
<br/>　　吞噬了巫穹的战魂之后，苏莫收取了对方的储物戒，然后便将对方的肉身直接丢进了储物戒中。
<br/>　　这具肉身就是资源，其内武帝境四重的神丹，更是不同凡响。
<br/>　　随后，苏莫便开始了修炼。
<br/>　　一夜无话，第二日天明，苏莫便离开了住处，向天冥圣城的宗府而去。
<br/>　　他根本没有什么好准备的，直接就可以住进宗府。
<br/>　　一路上，苏莫心中有些微微的期待，不知道青璇有没有可能在宗府之内？
