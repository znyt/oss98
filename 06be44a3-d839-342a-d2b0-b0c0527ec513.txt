　　不知道过去了多久，双眸紧闭的苏莫，睁开了眼眸。
<br/>　　因为他已经停了下来，明虚星河已经到了。
<br/>　　抬眼看向四周，这是一片名亮的空间，依旧是真圣碑内的空间，和古灵星河真圣碑内的空间一般无二。
<br/>　　同样的通道，通道之外同样的九彩旗杆，在疯狂的吸收灵气。
<br/>　　这一切的一切，差点让苏莫以为自己又回到了古灵星河。
<br/>　　“这到底是什么鬼东西？”
<br/>　　目光穿过空间通道，落在那通天侧地的九彩旗杆之上，苏莫呢喃自语。
<br/>　　真圣碑应该是万界山所布置，为了从各地招收弟子，但这九彩旗杆，难道也是万界山所布置？
<br/>　　苏莫想不明白，便也没有多想，少倾之后，他走出了真圣碑。
<br/>　　真圣碑之外，依旧是一片广阔的虚无空间，巨大的真圣碑屹立在虚空之上，依旧和古灵星河的真圣界一样。
<br/>　　苏莫抬头看去，真圣碑上有数十个名字，想必是明虚星河在真圣碑上留名的天才。
<br/>　　随即，苏莫掐起法诀，开启了这处真圣界的空间，大步的走了出去。
<br/>　　走出真圣界，苏莫出现在了一处荒原上，目光扫视四方，方圆数万里之内，没有半个人影。
<br/>　　“好淡薄的灵气！”轻吸一口气，苏莫呢喃自语，此地的灵气，浓郁程度不足古周星的百分之一，也就比以前的苍穹世界稍好一点。
<br/>　　“青璇，你在太阴族吗？”叹了口气，苏莫身形冲天而起，向远处飞去。
<br/>　　来到了明虚星河，苏莫已经迫不及待的想要见到青璇了，多年不见，心中的思念如潮水般涌现。
<br/>　　只是，青璇被人抹除了记忆，怕是现在都不认识他了。
<br/>　　飞行之中，苏莫发现这颗星辰，基本上已经接近半废了，大地荒芜，灵气与其它生命星辰相比，实在是太淡薄了。
<br/>　　“难道有真圣碑和那九彩旗幡的星辰，都是这般模样？”苏莫心中惊疑，古灵星河真圣界所在的星辰，也是半废星辰。
<br/>　　这颗星辰虽然比古灵星河的要好一些，但也没有好太多。
<br/>　　这让苏莫心中疑惑，这些星辰接近废弃，不会都是万界山干的吧！
<br/>　　思索之间，苏莫的视线中，数万里之外出现了一座城池。
<br/>　　此城非常的庞大，占地方圆千里，城墙高达百丈不止，雄伟壮阔。
<br/>　　不过，随着临近，苏莫发现此城之人非常的少，整个城池之中，不足十万人。
<br/>　　很快，苏莫便走进了城中，此城名为‘大燕’城，城中一片萧条的景象，大街上冷冷清清，并没有多少人。
<br/>　　“这位兄台，这里为何如此萧条？”苏莫拦住了一名身形微胖、面容略显憨厚的黄衣青年，抱拳问道。
<br/>　　“你不是我们昌玉星人士？”黄衣青年闻言疑惑的打量着苏莫，这种人尽皆知的事情居然都不知道！
<br/>　　“不是！”苏莫点头。
<br/>　　“我们昌玉星以前相当繁华，但不知道从什么时候开始，灵气逐渐的减弱，如今沦为半废星辰，强大的武者都走了，留在这里的都是修为较低的武者！”
<br/>　　黄衣青年沉声说道，他虽然看不出苏莫的修为，但也能感觉到苏莫比他强大百倍，不敢不回答。
<br/>　　“都走了？那此星上有星空传送大阵吧？”苏莫沉声问道。
<br/>　　“有！就在北方的天山之巅！”
<br/>　　黄衣青年点了点头，随即叹道：“不过，星空传送大阵被外界的大势力把持，使用星空传送大阵的费用相当昂贵，一般人根本出不起，不然我们这些普通的武者，也早已离开了！”
<br/>　　“什么价格？”苏莫问道。
<br/>　　“五千下品灵晶！”黄衣青年说道，其面上充满了苦涩，他才真罡境的修为，这个价格是他不敢相像的存在。
<br/>　　不过，他有信心能在三十年之内，存够五千下品灵晶，然后离开这里。
<br/>　　苏莫点了点头，略一沉吟，道：“你知道太阴族的所在之地吗？”
<br/>　　“太阴族？”黄衣青年闻言，面色古怪了起来，再次上下的打量了一番苏莫。
<br/>　　苏莫见此心有疑惑，自己这个问题有哪里不对吗？
<br/>　　“如实告诉我，我可以送你一场造化！”苏莫沉声说道。
<br/>　　“造化？”
<br/>　　黄衣青年闻言一怔，但他也没有多想，说道：“太阴族是明虚星河的霸主，威震整个明虚星河，无人不知无人不晓，其族所在之地，就在太阴星，据此遥远无比！”
<br/>　　“那你知道通过哪些星辰，能到太阴星吗？”苏莫继续问道。
<br/>　　“呵呵，你真是问对人了，我对去往太阴星的路线非常清楚！”黄衣青年笑着说道，他以前的梦想就是能去明虚星河的武道圣地太阴星，所以对此有过详细的了解。
<br/>　　同时，他也暗暗惊讶，面前的青年询问去往太阴星的路线，看来是要去太阴星了。
<br/>　　“是吗？还望告知！”苏莫面上也是露出了笑容。
<br/>　　“去往太阴星，首先要传送到天坛星，然后……！”随即，黄衣青年将去往太阴星的路线，详细的告知了苏莫。
<br/>　　“多谢了！”
<br/>　　听完了黄衣青年之言，苏莫向对方抱拳道谢，随即手掌一翻，掌中出现了一个储物戒：“这个给你！”
<br/>　　不由分说，将储物戒塞到黄衣青年的手中，苏莫身形拔天而起，瞬间消失在了北方天际。
<br/>　　“好快……！”
<br/>　　黄衣青年神情微怔，他还从来没见过这么快的速度，随即，他看向手中的储物戒，意念探入了其中中，面色顿时凝固了。
<br/>　　嘶！
<br/>　　少倾之后，黄衣青年口中倒吸了一口凉气，因为储物戒内，赫然就有着五千下品灵晶。
<br/>　　“贵人！今日真是出门遇贵人啊！”黄衣青年激动的身体都颤抖了起来，有了这五千下品灵晶，他就能离开这个半废星辰，到外界追寻他的梦想。
<br/>　　“这是什么？”
<br/>　　而后，他发现储物戒中居然还有一枚玉简，立刻取了出来。
<br/>　　不看不知道，一看吓一跳，这枚玉简之内，居然记录了一门帝级功法！
<br/>　　“造化……大造化啊！”黄衣青年激动不已，他终于知道苏莫所言不虚，真的给了他一场造化。
<br/>　　而后，黄衣青年立刻收起了玉简和储物戒，匆匆忙忙的返回了住处。
<br/>(本章完)
