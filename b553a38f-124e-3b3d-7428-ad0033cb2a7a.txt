　　“你说的是实力而已！根本无法代表低贱和高贵！”
<br/>　　苏莫满脸不屑，讥讽一笑，道：“若是实力能代表高贵，你巫族就是垃圾，同阶之中无人能打败我？”
<br/>　　苏莫自信无比，完全无惧大祭司，言语中尽是藐视。
<br/>　　大祭司闻听此言，眸中犀利的寒光一闪而逝，不过，他懒得与苏莫辩论，而是转头看向了府主。
<br/>　　“杜府主，苏莫的话你也听到了，他承认杀死我族百余万族人，现在老夫要将他带走，你没有意见吧？”大祭司向府主说道。
<br/>　　府主沉默不语，眸中闪过沉吟之色。
<br/>　　苏莫见此，心中立刻紧张了起来，对方的态度将决定他的命运。
<br/>　　少倾之后，府主瞥了一眼苏莫，又看了看一脸平淡、默不作声的的火元圣师，便摇了摇头，道：“巫仑大祭司，苏莫杀你巫族百万族人，完全是为了报仇，我辈武修有仇报仇，有怨报怨，此乃人之常情，这并没有什么过错吧！”
<br/>　　苏莫闻言顿时松了口气，看来这个府主，还是会维护他的，这让他有了一些底气。
<br/>　　而大祭司，闻听此言却是立刻双眸微眯了下来，冷冷的道：“杜府主，你这是要庇护苏莫？”
<br/>　　对方庇护苏莫，大祭司早有预料，所以并没有什么意外。
<br/>　　“并不是庇护，只不过我神武学府的弟子，并没有什么过错，在下身为府主，自然要保护府中弟子的安全！”杜府主摇头道，言语滴水不漏，让人无法反驳。
<br/>　　“杜府主，为了一个小小的武帝境弟子，与我巫族为敌，你考虑清楚了！”大祭司淡漠的说道，其言语随平静异常，但威胁之意不言而喻。
<br/>　　他没有针对火元圣师，目光直视杜府主，只要杜府主松口了，他便能带走苏莫。
<br/>　　杜府主闻言眉头微皱，他自然不想为了一个弟子，而彻底得罪巫族，但这不是一个普通的弟子！
<br/>　　此子天赋绝世，万古罕见，最主要的还是火元圣师看重的人。
<br/>　　火元圣师是他曾经的导师，对他恩重如山，两千年前他还是神武学府弟子之时，多受火元圣师的指点。
<br/>　　沉吟了少倾之后，杜府主微微一笑，道：“巫仑大祭司，此事何必说的如此严重！这样吧，在下愿意拿出几件重宝，算是为苏莫赔罪，你看这月可好？”
<br/>　　杜府主心中无奈，此事若是能用补偿来化解，那就是再好不过了。
<br/>　　若是无法化解，他也不能为了苏莫而与巫族为敌，毕竟，巫族主脉极度强大。
<br/>　　“杜府主，此事只有苏莫的鲜血方能彻底化解！”大祭司沉声说道，声音冰冷，直接堵死了任何补偿的可能。
<br/>　　杜府主犯难了，他转头看向火元圣师。
<br/>　　一脸平淡的火元圣师，感受到府主的目光，随即向大祭司道：“巫仑大祭司，苏莫是神武学府的弟子，亦是老夫看重的弟子，无论如何不会让你带走！”
<br/>　　“是吗？”
<br/>　　大祭司闻言缓缓站了起来，凌厉的眼眸直视火元圣师，道：“火元圣师，你是老一辈武圣，别怪我没提醒你，别为了一个苏莫，而赔上了一切！”
<br/>　　大祭司言语带上了戾气，阴森而冷冽。
<br/>　　“巫仑大祭司，你是在威胁老夫吗？”火元圣师问道。
<br/>　　“算不上威胁，只是给你的忠告，神武学府虽强，但在我巫族主脉眼中不值一提！”
<br/>　　大祭司摇头说道，他也并不想双方闹僵。若能和平解决此事，当然最好不过了。
<br/>　　他也并不是危言耸听，神武学府的确是很强，但无论是神武学府，还是其背后的十三大势力，都没有虚神境大能。
<br/>　　就算神武学府武圣境强者再多，面对虚神境的大能，也不过是蝼蚁而已。
<br/>　　火元圣师闻言，沧桑的眼眸中精光一闪而逝，他自然知道巫族的强大。
<br/>　　不过，他知道对方不过是虚张声势而已。
<br/>　　那些虚神境的大能，遨游太虚，游历在无边无际的宇宙之中，万年都难得一见。
<br/>　　巫族的虚神境强者，怎么可能为了此等小事，回来与神武学府开战。
<br/>　　当然，火元圣师也不想与巫族死磕，他略一沉吟，道：“巫仑大祭司，今日老夫不会让你带走苏莫，你若是要强行带走，凭你一个人的实力也做不到。”
<br/>　　“是吗？”
<br/>　　大祭司眼眸微眯，阴冷的道：“火元圣师，你们神武学府是要与我巫族彻底为敌了？”
<br/>　　“不！”
<br/>　　火元圣师摇了摇头，道：“今日虽不能让你带走苏莫，但会给你巫族一个报仇的机会！”
<br/>　　“什么机会？”大祭司淡漠的问道。
<br/>　　“苏莫和你巫族之仇，是从杀死你族近千名天才子弟而起，小辈的事还是交给小辈解决吧！”
<br/>　　火元圣师叹了口气，随即继续道：“真圣界即将开启，苏莫会前往真圣界，你族若想报仇，那就让后辈天才在真圣界的争锋中战胜苏莫吧！”
<br/>　　“哦？”
<br/>　　大祭司闻言微微皱眉，他没想到火元圣师居然是这个想法，略作沉吟，他点了点头，道：“也好，那就让小辈在真圣界解决，但苏莫若是不去，你等别怪我无情！”
<br/>　　大祭司没有反对此事，其一，火元圣师如此强硬，今日仅凭他的实力，是不可能带走苏莫了。
<br/>　　其二，真圣界开启只有区区的两个多月了，转瞬即到，不在乎多等两个多月时间。
<br/>　　其三，谅火元圣师和神武学府也不敢耍什么花样，不然的话，他们承担不起后果。
<br/>　　苏莫静静的听着火元圣师和大祭司的谈话，心中微凛，又是真圣界，看来真圣界之行有大麻烦了！
<br/>　　不过，再大的麻烦他也不怕，这是他的机会，总比现在被对方带走要好吧！
<br/>　　“好，既然巫仑大祭司同意在真圣界解决，那老夫倒是想问大祭司一句，若是你族天才无法战胜苏莫，报不了仇，又当如何？”火元圣师淡淡的问道。
<br/>　　从头到尾，火元圣师都是面色平淡，仿佛一切尽在掌控。
<br/>　　“报不了仇？”
<br/>　　大祭司闻言，打量了苏莫一眼，随即冷笑一声，道：“你认为这可能吗？”
<br/>　　苏莫的修为他尽收眼底，武帝境五重修为而已，就算天赋是万古妖孽，那有如何？
<br/>　　真圣界开启，可是所有武帝境的天才都能入内，巫族之中可不是只有支脉有天才，主脉中天才无数，修为达到武帝境九重，战力和天赋超越巫人王的人不再少数。
<br/>　　苏莫就算战力再如何逆天，能跨越四个境界战胜巫族天才？能跨越四个境界战胜众多的巫族天才，这显然不可能？
<br/>　　所以，巫族的后辈天才，想在真圣界杀死苏莫，简直是易如反掌。
