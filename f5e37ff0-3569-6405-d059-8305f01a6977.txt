　　第218章
<br/>　　究竟是谁在造谣？源头自然是高高在上的明帝陛下，消息是从太监口中传出的。
<br/>　　从太监口中得知，明帝对牛三斤赞不绝口，说他如椽之笔，直击军中痼疾。这让李天乐对牛三斤产生好奇，据自己听说，这牛三斤是个大字不识的莽夫呐。
<br/>　　如今所有人的目光都落在程大雷身上，只见他弓腰回头，姿势其实极其尴尬。
<br/>　　“哈，大皇子果然慧眼如炬，牛校尉的名声我也略有耳闻，说实话初见牛校尉时便被他吓了一跳。”
<br/>　　宋天函温和笑道，目光扫过百里胜时闪过一道杀意。捧你有时候就是为了踩你，捧得多高，便能踩得越狠。
<br/>　　“喔，当真如此么，那今日恰逢盛会，牛校尉正好也让大家领教领教大才。”
<br/>　　百里胜心里泛起一阵苦涩，这牛三斤能有什么才华，从西北边关偏僻地方来，能够认字就已经不错了。宋天函这不是让牛三斤展现才华，是让他丢人呐。
<br/>　　“这次是宴请参加武科举的士子，牛校尉是本次的考官，怕是上场不合适么。况且牛校尉是武将，对于诗词之道是不懂的。”
<br/>　　“唉，常忠伯此言差诶，都是帝国人才，本无任何区别。况且难道武将就不需通诗书了，依我看不对。况且大皇子都说了，牛校尉胸怀山河，腹藏锦绣，却也不要太过谦，说不得，牛校尉心中已有了文章呢。”宋天函笑看着程大雷：“我说的对不对啊？”
<br/>　　对你个麻麻批！
<br/>　　程大雷脸上长着长须，为了避免被人认出来，他还将脸描黑了。这模样实在不像读圣贤书的样子，而宋天函也看出来了，程大雷现在极其紧张，汗水都从脸上渗了出来，这肯定是做不出诗，羞愧得无地自容。
<br/>　　这便是宋天函猜错了，程大雷的确很紧张，但不是因为做不出诗，而是担心被一支桃认出来。此刻这姑娘笑盈盈的看着自己，也不知有没有被认出，认出倒是无所谓，可千万不要乱说话。
<br/>　　但大多人和宋天函的想法是一样的，文官团体趾高气扬，武将垂头叹气，不过在诗词之道上被文官血虐，也不是第一次，他们依旧被虐习惯了。
<br/>　　“战场杀敌，也用不着诗词吧。”柳轻名忽然道。
<br/>　　“可是你们也没有杀敌呐。”屈九江抿抿嘴。
<br/>　　哗！
<br/>　　一言激起众怒，多少武将恨不得起身拔刀，一刀砍了这姓屈的。而大家之所以愤怒，原因是……他说的是对的。
<br/>　　军人的尊严自然不需要在诗词上争，关键是在战场上他们实在没有拿得出手的战绩，如此被文官团体看轻鄙视，却也只能憋屈的忍着。
<br/>　　“呔！屈九江，你在说什么！”李天乐忽然喝道。
<br/>　　直呼其名，便是李天乐动了真怒，俗话说打人不打脸，骂人不揭短，你一句话直接捅进人家的心窝子，非要把文官和武将的矛盾搬到台面上来么。
<br/>　　“胜败乃兵家常事，帝国军人征战沙场，马革裹尸，不是为了杀敌是为了什么？”李天乐不得不帮武将找回一点颜面，他轻嘘一口气：“帝国战神三日后就要到了，这些话你敢当着他面说么？”
<br/>　　百里胜一帮人瞬间抬起头来，是呐，我们不是没有胜仗啊，幽州那一仗难道打得不漂亮么。虽然那一仗不是我们打的，但大家都是帝国军人呐。
<br/>　　某种程度上，杨龙停已经成为帝国军人的精神象征，他们现在还能直起腰杆都是靠着杨龙停撑着。
<br/>　　没有人注意到，程大雷弯着的腰已经挺得笔直，低头间将眼中的杀机掩过，这一刻，程大雷竟发现自己的手腕在轻轻颤抖，这不是因为恐惧，而是因为兴奋。
<br/>　　等了一年，等的不就是这一刻么。
<br/>　　大家都因杨龙停的到来而兴奋，杨龙停不仅在武将团体中极有权威，在文官中也有很多人崇拜他。
<br/>　　“战神大人三日后就要到了么，不知能否请大皇子帮我引见。”
<br/>　　“若能得到战神大人教诲，当真是朝闻道夕死可矣。”
<br/>　　没有人注意到，程大雷已经穿过众人，从大殿门口走到大殿最里，来到摆着文房四宝的书案前。
<br/>　　有人注意到他的行动，下意识道：他要干嘛？
<br/>　　百里胜心中一惊：不好，别让他丢人。
<br/>　　而宋天函则是面带笑意：“看来牛校尉已经是有诗在心头，不书不快了。”
<br/>　　程大雷执笔，饱蘸浓墨，一手将宣纸铺开，他握笔的姿势自然又引得一阵嘲笑，可起笔的第一句，就令所有人惊讶得合不拢嘴。
<br/>　　『少年十五二十时，步行夺得胡马骑』
<br/>　　……
<br/>　　『一身转战三千里，一剑曾当百万师』
<br/>　　呼吸声越来越重，在场上百人此刻竟已无人说话，只闻上百人整齐的呼吸声。
<br/>　　『路旁时卖故侯瓜，门前学种先生柳』
<br/>　　……
<br/>　　『贺兰山下阵如云，羽檄交驰日夕闻』
<br/>　　……
<br/>　　『试拂铁衣如雪色，聊持宝剑动星文』
<br/>　　……
<br/>　　『莫嫌旧日云中守，犹堪一战取功勋』
<br/>　　鸦雀无声，诸人看看纸上的字，再看看程大雷，皆同时摇摇头。
<br/>　　李天乐伸出手，用僵硬的声音道：“牛校尉还真是，还真是……”
<br/>　　百里胜摇摇头，心道：你别看我，我也没想到啊。
<br/>　　宋天函脸上的表情僵硬住，想说什么，却接连几次欲言又止，最后尴尬的摇摇头，什么都说不出。
<br/>　　“此诗，可惊天下呐。”李天乐长叹一声，算是给出了这首诗的评价。
<br/>　　“不，不对。”屈九江忽然抬起头：“这诗不对，这诗不是你写的。”
<br/>　　所有人都被屈九江的话震惊，不知他为何会如此说。
<br/>　　屈九江又读一遍，心中更加确定，道：“请问牛校尉，这黄须儿是何人，贺兰山又在何地，而这先生柳又是出自那个典故，在下才疏学浅，从未听闻，还请牛校尉讲解一番。”
<br/>　　哪里不对？用典不对，两个世界有相似的地方，但更多地方是不同的。有些诗中提到的典故，这个世界从未出现过。刚才初读大家都不觉得如何，此刻细细咀嚼，才发现诗的味道不对。
<br/>　　程大雷看着屈九江，见他语气诚恳，目光灼灼，但嘴角却挂着微笑，一副我拆穿你喽，我拆穿你喽的得意感。
<br/>　　“我解你马勒戈壁。”
<br/>　　程大雷转身离开。
