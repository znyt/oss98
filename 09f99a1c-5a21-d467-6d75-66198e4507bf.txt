　　“据说，要进入太阴山，得有他们四个帮忙，需要他们做什么？”
<br/>　　“不知道，我是真的一点都不知道。”杨宫梓想了一下道，“我也问过道风，他不告诉我。”
<br/>　　林三生插了一句：“青牛说的是李浩然？他不是去了修罗界？”
<br/>　　“他的金刚琢给了道风，其中有他斩下的魂魄，八成也是能用的。”
<br/>　　叶少阳心中牢牢记下了这件事，打算见到道风时问问他，又想，怪不得好久没见到岳恒和孙映月了，原来是被保护起来了，喃喃道：“这么看来，道风其实应该是有所布局的。”
<br/>　　林三生道：“谁都别小看道风，他不光能打，在谋略上不输给你我。”
<br/>　　叶少阳笑道：“跟你比就行了，我不行。”
<br/>　　“不是我捧你，之前若是换我在绝境中，我不一定能够冷静下来，想到那样的妙计。”林三生在心中分析了一下他们三人在谋略上的不同，叶少阳大局观不行，基本上只有遇到事情才会动脑子，否则懒得去想太多，运筹帷幄的能力不如他，但在战斗方面是个天才，也是急才，总能在危难关头找出翻转局面的法子。
<br/>　　道风是个狂人，指挥打仗不如自己，但有战略眼光，懂布局，能隐忍，不拘小节。而他自己，是个十足的军师，是为战争而生。
<br/>　　在这方面，与他能力最接近的，他所知道的就一个人，那人在空界天弃山。
<br/>　　这时候冲和子带了几个人过来，见面先问候了叶少阳，告诉他们，金刚三藏已经回来了，在小楼上等着他们，想要跟他们见一见。
<br/>　　“有没有想跟我一起去的？”叶少阳环顾左右，问道。他想着见金刚三藏的话，人去太多不太好，也没什么意思，他们跟金刚三藏本来也不认识，自己过去，主要还是问一些私事。
<br/>　　“我去拜见一下。”四宝说道。他是和尚，对金刚三藏这样的前辈还是要拜访一下的。
<br/>　　林三生也要去。其余人则都摇头了。
<br/>　　“那你们待会直接去海眼，我们在那边集合，直接回去。”
<br/>　　叶少阳跟着冲和子走，穿过一片茅庐，来到林中的一片空地上，矗立着一座二层小楼，通体都是木柴打造的，看着很简陋。有两个小沙弥在楼下的院子前面把守，冲和子走过去，两人都行礼。
<br/>　　冲和子跟他们也没多说，直接带着叶少阳三人上楼，进去之后，是一个经房，门口有珠帘垂着，能听到里面有人在说话。冲和子就让他们等一下。
<br/>　　“大师，证道之路，果真如此艰难？”
<br/>　　叶少阳一下就听出是慕寒的声音。
<br/>　　接着是金刚三藏的声音传来：“难与不难，且看各人，若不开悟，修炼得再圆满，也如空中求索，不得法门，一遭悟得机缘，便如鱼跃龙门，一飞冲天，总归还是一个悟字。”
<br/>　　慕寒叹了口气，道：“我何尝不知道这个道理，我从五岁开始修道，家叔父一来为了保护我，不让我与法术界接触，二来也是为了让我静心修行，我修炼了十八年，无欲无求，忍耐寂寞，才有今天，放眼人间，也算是数一数二的少年天才，可如今修为再难进步，证道的边都摸不到，如今法术界四面临敌，危在旦夕，我有心带领人间法术界崛起，但我的实力……与强敌实在相去甚远，不免有些心急，还请大师指点一二。”
<br/>　　叶少阳在门外听着，心中隐有所感。
<br/>　　停了有一会，金刚三藏的声音才想起来，“你方才说，你的实力，在人间数一数二……是数一还是数二？”
<br/>　　“这……有什么关系吗？”
<br/>　　“数一还是数二，对你却很重要，是不是？”
<br/>　　慕寒沉默良久，道：“是。”
<br/>　　接着金刚三藏说道：“我这有一片孽镜碎片，为我炼化，可照人生忧患，指点迷津，你却照一照，便有谒语显像。”
<br/>　　等了一会，慕寒说道：“不争，这是何意？”
<br/>　　“就是不争。”
<br/>　　“这……是让我不要去争，争什么？”
<br/>　　金刚三藏笑道：“张天师，争什么，不是该问你自己吗？”
<br/>　　慕寒沉默半晌，道：“不争，便能证道？”
<br/>　　“正是。”
<br/>　　“就这么简单？”
<br/>　　“就这么简单。”随即又补充道：“简单的事，往往却又是最难的，世人皆知贪色损性，贪财昧心，贪名毁格，贪虚忘诚，凡此种种，真正能戒得的，又有几人，就如你，你明知不该去争，但能做到不去争吗？”
<br/>　　过了半晌，慕寒沉声说道：“多谢大师指点，只是……不争的话，岂不是一点机会都没有了。”
<br/>　　金刚三藏站了起来，道：“你们道家的老子有句话：夫唯不争，天下莫能与之争，争与不争之间，便大有文章，张天师，请了。”
<br/>　　慕寒道了谢，懵懵懂懂地走出来，一抬头看到叶少阳三人，愣了一下，笑了笑，下楼去了。
<br/>　　“这慕寒，凭他也想证道。”四宝撇撇嘴，“不是我瞧不起他。”
<br/>　　里头响起金刚三藏的声音：“诸位请进。”
<br/>　　叶少阳三人进去，见经室内装饰十分简单，甚至可以说简陋，地上铺着一张草席，上面放着一章香案，有一套粗瓷茶具，一只香炉，有烟雾从里面飘出，带着一股奇异的香味，有点像藏红花。
<br/>　　金刚三藏坐在蒲团上，指着对面，让三人坐下。
<br/>　　四宝跪在地上，行了个大礼，金刚三藏也坦然受了。
<br/>　　叶少阳道：“大师之前是在与什么人斗法，半天不见。”
<br/>　　“我在海底与右君斗法，双方不得脱身，后海眼法阵已成，他便走了。”
<br/>　　原来是右君，那就怪不得了。没想到这家伙也来了。想到这也是感慨不已，幸亏有金刚三藏镇守，不然光是一个右君，怕就能打乱他们的计划，由此也能看出太阴山对海眼的重视，幸亏完成了任务，不然阴司真的要大乱了。
<br/>　　三人对金刚三藏表达了谢意，然后金刚三藏拿起茶壶，为他们倒茶。
<br/>　　茶水淡红色，叶少阳尝了一口，有某种花香，精神为之一振。
