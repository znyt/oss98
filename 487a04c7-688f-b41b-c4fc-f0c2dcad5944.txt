
<br/>　　<br/>　　    刘珺瑶被警方的人领走了，这个绑架案既简单也复杂，简单是因为从报案到破案一共没到一小时就结束了，复杂是因为里面涉及到了沈君，一位袍哥大佬，警方也知道如果抓着不放的话，没什么意义也很麻烦。<br/>　　<br/>　　    在询问过当事人的意见之后，刘珺瑶也没有追究的意思，到这就算是拉倒了。<br/>　　<br/>　　    刘珺瑶上了警车，惊魂未定的心才放了下来，赶紧拿出手机还没等拨出去呢，胡胡的电话就进来了。<br/>　　<br/>　　    “你没什么事吧？”<br/>　　<br/>　　    “没有，是你报警了吧？哎呀呀，吓死我了，幸亏你是和我一起回来的”<br/>　　<br/>　　    “一会我到警局门口等你……”<br/>　　<br/>　　    仓库里面，沈君抽着烟来回的踱着步子，刘珺瑶虽然被带走了，但他也不是一无所获，至少知道了下手的人叫胡胡，在沈沉鱼这件事上，刘珺瑶是主谋那这个胡胡就是帮凶，是拿刀的刽子手，要不是他给了刘珺瑶那个吊坠的话，恐怕沈沉鱼的魂也不会差点就被带走了。<br/>　　<br/>　　    沈君琢磨了一会，就给陈重打了个电话过去：“小鱼儿是没事了，幕后的人也被追问出来了，不过有点麻烦的是，对方明显会点什么，跟咱们不是同一类人，我怕直接对他下手的话会有些问题……你说，我再请那个王惊蛰出手，行不行？”<br/>　　<br/>　　    “你没品出来么，这是个无欲无求不愿意揽事的年轻人，他上次被我请过来，是因为欠我个人情才不得不来的，你想再找他的话那就难了，人家肯定不愿意鸟你，多一事不如少一事，你找他明显是要让他干得罪人的事，你说人家能愿意么？”陈重在电话里说道。<br/>　　<br/>　　    “好像还真是这么回事”<br/>　　<br/>　　    “呵呵，我之前想套上他，差点都想卖女儿了，你要是想请他的话，想想看有没有什么能打动人的筹码吧”<br/>　　<br/>　　    “你让我卖妹子啊？”沈君无语的说道。<br/>　　<br/>　　    “哈哈，关键是王惊蛰好像还不近女色，你卖了也未必能行”陈重挺中肯的评价了一句，又接着说道：“其实，我觉得吧，没有什么人是真正的无欲无求的，就看你能不能找准他的需求点了，乞讨的人需要钱，升官的人要关系，他们这种人会需要什么，你就琢磨一下，找到能打动他的点我觉得就行了”<br/>　　<br/>　　    “行吧，我想想的……”沈君挂了电话，就看是研究上王惊蛰了。<br/>　　<br/>　　    刘珺瑶坐着警车到了分局的门口。<br/>　　<br/>　　    “刘小姐，确定不报案了？”<br/>　　<br/>　　    刘珺瑶鄙夷的说道：“这个世界上，总有那么一小撮人是享受特权的，人民警察也不一定是为人民的”<br/>　　<br/>　　    带队的警察皱眉说道：“你不打算追究，我们自然不会抓着不放”<br/>　　<br/>　　    “呵呵，你们就是抓了沈君又能怎么样？抓完不还是得放么，袍哥大佬是什么人我还是知道的，所以啊，干脆咱们就别麻烦了，我也不想给自己找麻烦，就这样吧警察叔叔……”<br/>　　<br/>　　    刘珺瑶从警车上下来，就看见了胡胡的奔驰小跑停在了路边，她走过去上了车。<br/>　　<br/>　　    胡胡说道：“没什么事吧？”<br/>　　<br/>　　    “你要是不报警，那我就真的有事了，差一点被人给轮了”刘珺瑶一点都不怀疑，沈君说到做到的真实性。<br/>　　<br/>　　    “哎，有点失算了，我都没想到沈君的身边会有高人，能一下子就看透了沈沉鱼是被人给接了灵媒，没想到他们袍哥里还有这种高人呢”胡胡扭头问道：“你招了是不是？”<br/>　　<br/>　　    刘珺瑶说道：“你觉得，我在他们手里，有不招的可能么？女人落在男人手中，他们有一百种方式让我开口，而我只能选择问什么说什么，你埋汰我？”<br/>　　<br/>　　    “不是埋怨，也是无奈啊，算了不说这个了，么的，面对沈君有点让我犯愁了”<br/>　　<br/>　　    “那怎么办，不然我们离开川中吧？”刘珺瑶有些后怕了，今天的经历说实话挺险的，要不是正好胡胡跟她是一起回来的，那今天是什么后果，可就真不好说了。<br/>　　<br/>　　    “往哪走？在蜀地我还有不少买卖呢，根本就扔不下，我人走了可以，难道生意都不要了？我也不是怕了沈君，就是不想跟他闹得太僵了，太麻烦……”胡胡不怵沈君，但在蜀地跟一位袍哥大佬死磕的话，注定得是要劳神伤财的，从哪个角度来讲都不合适。<br/>　　<br/>　　    “实在不行，我找个人说和一下吧，看看能不能把这一页给掀过去，大不了我姿态就放低一点”胡胡寻思了半天，最后惆怅的说道。<br/>　　<br/>　　    一晃两天过去了，沈君给王惊蛰打了个电话。<br/>　　<br/>　　    “王老弟，方便么？小鱼儿这边没什么事了，方便的话咱们见个面吃个饭，我聊表一下感谢之情”似乎知道王惊蛰会推了他的邀请，沈君又接着说道：“你可千万不要拒绝我，不然我直接就堵到你们家门前去，除非你打算一直连门都不开，那我就没辙了”<br/>　　<br/>　　    “那我多不好意思啊……”王惊蛰无语的说道。<br/>　　<br/>　　    晚间的时候，沈君让人过来接上了王惊蛰，司机开车带着他驶出了市区，来到了郊区附近的一个农家院，他到的时候，沈君已经在门口等着了。<br/>　　<br/>　　    “这个地方挺僻静的，我平时没事的时候，就喜欢过来摘一些山野菜钓钓鱼，然后下厨做几样小菜，约上两三个朋友来打发时间，城市里太喧嚣和浮躁了，偶尔偷得浮生半日闲，这日子才不会过得那么糟心”沈君跟晚景这边走边说道。<br/>　　<br/>　　    “你还会做菜呢？”王惊蛰讶异的问道。<br/>　　<br/>　　    沈君笑了：“还是有两把刷子的，我和小鱼儿的父母去世得早，她算是被我手把手拉扯大的，吃喝拉撒睡都是我照顾，做菜算是基本的生存技能”<br/>　　<br/>　　    “那我是真没看出来，您还有挺居家的一面呢”<br/>　　<br/>　　    “哈哈，我除了是袍哥的老大，同时也是个会生活的男人”沈君跟王惊蛰来到一间包房的时候，桌子上已经摆上了几样小菜：“喝点吧，有一坛上好的竹叶青，年头挺长了，咱们把酒言欢少喝一些”<br/>　　<br/>　　    王惊蛰随后跟沈君边吃边聊，两人说的都是一些闲话，就跟两个久违的朋友聊家常似的，一直吃到后半程的时候，沈君才把正题给抛了出来。
