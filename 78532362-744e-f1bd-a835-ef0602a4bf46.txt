　　眼下这个车间内的很多东西都是来自于白忧的那座私人基地，而楚涵第一次和高少辉去过一趟拿走了天照3000后，一向不浪费任何好处的楚涵，当然又抽空折返。
<br/>　　并且那一次没有外人在场，楚涵直接将那里所有的东西都一次性放进了次元空间带走！
<br/>　　巧合的是白忧似乎是个机械迷，那里存放的很多东西被楚涵带回来后，又经过了和上的改良，最终成为了狼牙好几个部门的重要设施。
<br/>　　卢宏盛的锻造部之所以生产制造的速度这么快，更是与这些机械息息相关！
<br/>　　这时候的高少辉才明白为何白忧会暴怒之下毁了整座岛，原来那座岛上早已空无一物，都特么被楚涵拿空了！
<br/>　　原本对于得到了天照3000而沾沾自喜的他，顿时心情郁闷的彻底，与楚涵比起来他这点小偷小拿算啥啊？
<br/>　　楚涵简直就是强取豪夺！
<br/>　　不过一想到这些东西都被楚涵悄悄转移到了狼牙，白忧还一无所知，高少辉心里又平衡了起来。
<br/>　　他找到正满头大汗工作的卢宏盛，小心翼翼的将手中那个包裹的严严实实的东西递出：“大个头，帮我打造个武器呗？”
<br/>　　卢宏盛看了他一眼，擦着汗挥手道：“贵公子别来捣乱，我这忙着呢。”
<br/>　　虽说狼牙内部的人都清楚高少辉和楚涵关系不一般，但这时候正是紧急的大战当下，虽说狼牙战团的那八万新兵都有武器，但那些都是别的基地提供的资源，比起狼牙自己生产制作的来说太过于寻常。
<br/>　　所以哪怕是最普通的刀剑，卢宏盛也让整个锻造部都加急赶制中，他知道这是使用者最多的两种武器。
<br/>　　看到卢宏盛一口回绝，高少辉急了，道：“我说正经的，不是捣乱，你不帮我制造武器，我怎么杀丧尸，跟李必疯一样徒手撕？”
<br/>　　卢宏盛上下看了高少辉一眼，那单薄的身板很真和徒手撕丧尸不搭界。
<br/>　　并不聪明也不善于交际的卢宏盛想了想，最终还是接过了高少辉递来的东西：“那好吧，你想要什么样的武器？”
<br/>　　“短刀！”高少辉双眼发亮。
<br/>　　卢宏盛点着头将那团包裹打开，剥开了一层又一层后，最终只看到一颗蚕豆大小的明黄色物体。
<br/>　　“就这么点？哪够打造短刀啊！”卢宏盛锻造的武器材料，很多都来自于考核石壁的特殊金属，所以在见到不明物的时候并没有惊，只想到了材料不足这一点。
<br/>　　高少辉笑着摸了摸鼻子：“够了，你只需要将这个石头淬炼在短刀外层就行。”
<br/>　　卢宏盛愣住：“这不是金属？是矿石？”
<br/>　　高少辉点头，眼神带着一丝神秘的光芒：“千月石。”
<br/>　　卢宏盛并不知道千月石的特殊之处，只是从后方的陈列架中取出一柄短刀，道：“如果是淬炼的话，短刀取现成就就行，这把你看行不行？别小看它外表普通，我打造它的时候可是用了特殊金属，韧性和锋利程度是普通短刀的好几倍，最重要的是它很坚固！”
<br/>　　“没问题。”高少辉一点不挑。
<br/>　　“那好。”卢宏盛当即就将短刀架在锻造炉上：“只是淬炼在表层那速度很快，你走之前来我这取。”
<br/>　　搞定了武器问题，高少辉长久等待的不爽心情一扫而空，跟着江左心情颇好的重新回到和上的科技部。
<br/>　　这时候和上已经从密室里出来，只见他衣衫褴褛的过分，和另外两名科技部的成员一起趴在桌子上大口吃着饭，那样子好像十天半个月没吃到东西一样。
<br/>　　“我去，哥，你这打扮够潇洒！”高少辉与和上也称得上是不吵不相识，这会儿已经颇为熟悉。
<br/>　　和上看都没看他一样，道：“东西在桌上自己拿。”
<br/>　　高少辉将目光转移，顿时看到了一个类似于老式音响的玩意儿。
<br/>　　“就这东西？”高少辉的眼神里分明带着失望：“为了这竟然让我专门跑一趟，有这功夫我都能个杀好几万头丧尸了！”
<br/>　　和上瞥了他一眼：“你能杀几万丧尸，但你能杀的了几百万几千万不？狼牙战团多你一个不多少你一个不少，赶紧带着东西回去。”
<br/>　　高少辉摸了摸鼻子：“行吧，我再去看下他们物资的准备如何。”
<br/>　　第一批送物资的直升机队已经从南署基地返回，速度相当快，毕竟他们几乎就是扔下东西就头也不回的离开，一秒钟都不想在那处比地域还恐怖的地方待着。
<br/>　　听完这几名飞行员回来后结结巴巴的转述，段氏基地的几名高层都陷入了长久的沉默之中，他们有想到那种程度的尸潮场面有多壮观，也能料到南署基地就像是孤岛一样被淹没。
<br/>　　但他们没想到的是狼牙战团那九万人在这场战斗中的状况，竟然与一千多万的丧尸厮杀了整整两天两夜没停！
<br/>　　南署基地那么大的地方，这时候却每一处角落都被挤满，哪怕是屋顶也同样如此。
<br/>　　不是成片丧尸群的呼啸而来，就是大·片堆放在一起的丧尸尸体，亦或是抱团紧凑用着最原始手段攻击的人类。
<br/>　　到处都是战斗场面，也处处充满了丧尸的嘶吼与人类队伍的厮杀战斗声，喧闹噪杂不足以形容待在那里的感受，而是精神极度压抑中的备受折磨！
<br/>　　高少辉之前的三言两语，根本不足以形容那场战役的哪怕百分之一，要知道段氏基地出动的直升机队飞行员，也是见过不少大场面的，更是在段氏基地参与过好几次不同程度的尸潮，就连他们都吓成那个样子，可想而知此时的南署基地有多惊心动魄。
<br/>　　鲁初雪此时正在紧急准备物资，同时这回知道确切战况的她，再一次派人将消息传递出去。
<br/>　　“中将，是食物多一些还是军火多一些？”一名后勤人员开口询问，显然对物资的分配很为难。
<br/>　　鲁初雪想要开口的话顿了顿，南署基地的战役打成那个样子，什么物资都属于紧缺状态。
<br/>　　“食物，还有水。”鲁初雪下令道：“狼牙的射击队成员一共才五百个，九万人都是在用冷兵器作战，要那么多枪支弹药没用。”
<br/>　　“明白！”
