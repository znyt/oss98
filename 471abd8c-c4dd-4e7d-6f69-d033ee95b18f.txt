　　苏莫看到了蛮荒山脉的圣山，顿时心中一喜。
<br/>　　他可是清楚的知晓，蛮荒圣山中，居住着三只准帝级别的蛮荒大妖。
<br/>　　其一是远古战象，其二是八翼飞龙蛇，还有一只苏莫没有见过，不知道是什么异种！
<br/>　　但他可以肯定的是，这三只蛮荒大妖，实力绝对恐怖至极。
<br/>　　苏莫心中闪过一道电光，他或许可以借助这三只大妖的力量脱身。
<br/>　　不过，现在二魔老的攻击已经来临，他根本来及多想其他。
<br/>　　“给我破！”
<br/>　　苏莫立刻转身，手中帝皇剑光芒耀眼，疯狂的斩击，瞬间斩出数十剑。
<br/>　　数十道剑气铺天盖地，形成浩大的剑气风暴，向袭来的枪芒迎击而去。
<br/>　　于此同时，苏莫亦是动用了本命灵剑，道道剑光闪烁，九柄灵剑瞬间形成了九绝剑阵。
<br/>　　九柄灵剑齐齐一斩，彼此气机相连，斩出了一张细密的剑网，紧随剑气风暴之后，迎向袭来的数十道枪芒。
<br/>　　这一次，苏莫由于有了防备，不会再像之前那般毫无抵抗之力了。
<br/>　　下一息！
<br/>　　剑气风暴率先和数十道枪芒狠狠的碰撞在了一起，一阵惊天动地的巨响，整动了小半个蛮荒山脉。
<br/>　　轰隆隆！！
<br/>　　巨响连连，剑气风暴被数十道枪芒刺的千疮百孔，立刻崩溃爆炸，化为漫天剑雨，密密麻麻。
<br/>　　数十道枪芒所向披靡，突破了剑气风暴的阻挡，又与本命灵剑阵击出的大网相击在了一起。
<br/>　　轰！轰！轰！！
<br/>　　又是一连串巨响，细密的剑气大网崩溃，数十道枪芒也崩碎了一半，浩荡的冲击波席卷八方。
<br/>　　下方的蛮荒山脉，遭到冲击波的扫荡，无数参天古树灰飞烟灭，一座座大山被夷为了平地。
<br/>　　咻咻咻！！
<br/>　　剩下的十余道枪芒，威势稍减，但依旧急速向苏莫袭杀而去。
<br/>　　“不好！”
<br/>　　苏莫见此，心中暗道不好，他身上玄力涌动，形成厚厚的护体玄力。
<br/>　　而他手中帝皇剑，急速舞动，化为了幻影，舞出了一层层剑幕，抵御枪芒的攻击。
<br/>　　轰轰轰！！
<br/>　　十数道枪芒，尽数击在了剑幕之上，剑幕层层的奔溃炸裂，四散湮灭。
<br/>　　几乎是瞬息之间，剑幕便被彻底破开，残余的枪芒尽数击在了苏莫的护体玄力上。
<br/>　　嘭嘭嘭！！
<br/>　　一阵沉闷的声响传出，苏莫全身剧震，身形再次倒飞了出去。
<br/>　　好在枪芒的攻击力，都被他削弱到了极致，就算洞穿了他的护体玄力，也无法再对他造成多大伤害了。
<br/>　　不过，即便如此，强烈的攻击，也震的他全身气血上涌，本就受伤的身体，伤势更重了两分。
<br/>　　喉头一甜，苏莫的嘴角留下了一缕金色的血液。
<br/>　　咻！
<br/>　　苏莫丝毫不敢逗留，一击之后，便急速向蛮荒圣山飞去。
<br/>　　“不错！但只是垂死挣扎而已！”二魔老见苏莫再次挡住了他的攻击，微微有些意外。
<br/>　　因为这一次，苏莫并不是靠帝级内甲，而是靠的自身实力。
<br/>　　不过，即便如此，苏莫也是必死无疑。
<br/>　　抬起脚步，二魔老继续向苏莫追了过去。
<br/>　　看到远处的蛮荒圣山，二魔老眉头微皱，他自然知道蛮荒圣山中居住着谁。
<br/>　　不过，二魔老也没太过在意，因为蛮荒圣山中那三位存在，和苍穹世界的人族，一向是井水不犯河水。
<br/>　　他们途径此地，只要不屠杀蛮荒山脉中的生灵，就不会有什么事！
<br/>　　咻！
<br/>　　苏莫速度极快，瞬息之间便飞临了蛮荒圣山。
<br/>　　蛮荒圣山占地方圆数千里，通天彻底，仿若支撑天地的天柱。
<br/>　　不过，苏莫却是没有心思感叹蛮荒圣山的雄伟，直接身形一转，向圣山上一个巨大的洞穴飞了过去。
<br/>　　圣山之上，各种妖兽奇多，俱是瞪大了眼眸，仅仅的盯着着苏莫等人。
<br/>　　这些妖兽级别都不低，甚至还有不少相当于武皇强者的七级妖兽。
<br/>　　它们自然能看出苏莫等人的强大，所以并没有出手阻拦。
<br/>　　“什么？”
<br/>　　二魔老见到苏莫飞向圣山的洞穴，面色微变，他没想到苏莫居然要进入圣山之中。
<br/>　　难道苏莫不知道圣山中有什么？
<br/>　　亦或是知道，想进入圣山需求一线生机？
<br/>　　二魔老没有去拦苏莫，因为已经来不及了，他立刻停下了脚步。
<br/>　　在他看来，苏莫若是进入圣山，那也不用他出手了，同样是必死无疑。
<br/>　　因为蛮荒圣山有规矩，任何敢闯入圣山的人族，均是杀无赦！
<br/>　　苏莫身形如剑，急速冲向圣山上的那处洞穴，那洞穴正是当初他走过的通向圣山内部的通道。
<br/>　　不过，苏莫刚刚飞到洞穴的入口处，却是猛然停了下来。
<br/>　　因为他的前方，洞穴之中，走出了三道身影。
<br/>　　嗒！嗒！嗒！
<br/>　　清脆的脚步声响起，三道身影走到了洞穴的入口处。
<br/>　　这三道身影，乃是二男一女。
<br/>　　那两位男子，其中一人是一名身高达三米、全身肌肉虬扎的光头大汉。
<br/>　　另一人则是身形消瘦，面无表情的黑衣中年人。
<br/>　　这两人，正是苏莫当初见过的远古战象，和他的二哥八翼飞龙蛇。
<br/>　　至于另外一名女子，则是一名豆蔻年华的小少女，头上扎着两根羊角辫。
<br/>　　此女有一双晶亮的眼眸，明净清澈，灿若繁星，只不过其眼瞳为碧绿色，给人一种妖异之感。
<br/>　　而此女的面容，白皙粉嫩，仿佛粉雕玉琢，容颜可爱俏丽，活脱脱的一个小萝.莉。
<br/>　　“嘻嘻！”
<br/>　　此刻，这名少女正打量着苏莫，面带欣喜的笑容，让苏莫颇为奇怪。
<br/>　　不过，此时的苏莫可没有心情理会这名少女，他立刻向光头大汉和黑衣中年人躬身一拜。
<br/>　　“拜见两位前辈！”苏莫沉声说道。
<br/>　　此时，光头大汉和黑衣中年，亦是在打量着苏莫，眸中露出惊奇之色。
<br/>　　“小子，你是怎么修炼的？”光头大汉率先开口了，声如闷雷，凭空炸响。
<br/>　　呃~~
<br/>　　苏莫闻言一怔，怎么修炼的？就是这样修炼的啊！
<br/>　　“前辈，怎么了？”苏莫疑惑的问道。
<br/>　　现如今，苏莫面对光头大汉两人，已经没有了当初的那种忐忑了。
<br/>　　这也是因为他的实力达到了一定地步，今日，就算这两人不帮他，他也没有太多的担忧。
<br/>　　凭他的底牌，二魔老想杀他，也不是那么容易。
