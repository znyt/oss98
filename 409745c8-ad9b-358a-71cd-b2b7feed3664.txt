    当公国长老利安德尔·拉尔夫正式宣布65号地下求生所探索任务结束时，整个兰花镇都欢腾起来。<br/>    <br/>    纵眼望去，大街上熙熙攘攘，两侧摆满了地摊。各个家族的士兵掺杂在一起，有说有笑，热闹不凡。<br/>    <br/>    相比刚来的时候，士兵们变得友善多了。那时候，他们面临着巨大的压力，生死未知，心情普遍焦躁、不安，又不适应边缘镇落后的环境，经常和当地民众发生冲突。<br/>    <br/>    现在归家在即，所有人的心情都变得好起来，就连平时不苟言笑的军官，脸上也洋溢着笑容。<br/>    <br/>    欢快的气氛很快就感染了镇上的人们，他们纷纷走上大街，一边欢送即将离开的士兵，一边向他们兜售零食香烟、果汁糕点，还有各类纪念品。水站的流民也不例外，三三两两地跑到大街上售卖土特产。<br/>    <br/>    当天，镇长召集镇上所有官员，在镇长府设下宴席，为门阀世家的贵客们送行。陈兴也在行列之中。下午刚过两点，他就带着叶阳白柳来到镇大街，准备前往镇长府赴宴。<br/>    <br/>    “卖桑果干咯~”<br/>    <br/>    走过一段路，陈兴忽然听见一个似曾相识的声音。<br/>    <br/>    “叔叔，您要桑果干吗？这是我爷爷去年秋天摘的，晒了两个月，很甜的~”<br/>    <br/>    嫩稚的声音里透着几分怯懦。循声看去，一名穿着破旧花格布衣衫的小女孩正在对一名路过的士兵兜售桑果干。小女孩的眼睛又圆又大，水汪汪的，十分可爱。<br/>    <br/>    陈兴很快就认出来了，这是水站的一对爷孙，新年的时候他们曾经有过一面之缘。在陈兴的印象中，小女孩十分乖巧懂事，老人也淳朴善良。<br/>    <br/>    此刻，老人正坐在一旁的小板凳上，腿脚似乎有些不灵便，看着小女孩的眼睛里，充满了慈爱。<br/>    <br/>    “小妹妹，一袋子要多少钱？”“三个铜板。”“喏。”士兵摸出钱袋，取了三个铜板。<br/>    <br/>    “谢谢叔叔。”小女孩接过钱，一边递过装着桑果干的小袋子，一边嘴甜地说道，“叔叔是个大好人，以后一定会平平安安的。”<br/>    <br/>    “呵呵，谢谢啊。”士兵傻傻地笑了，接过袋子，由衷地感谢了一句。扛枪的汉子，不知哪天就战死异乡。听到小女孩的祝福，眼角竟然有些微湿。<br/>    <br/>    “爷爷，我们又赚了三个铜板！”士兵走后，小女孩欢快地扑进老人怀里，“等我们赚够钱，就给你去看病。”<br/>    <br/>    “爷爷没病，爷爷的身体好着呢。”“可是，可是你的脚……”“就是崴到了，过几天就好了。”“呜呜呜，爷爷，你一定要好好的，好好的……”说着说着，小女孩的眼睛里闪出了泪花。<br/>    <br/>    “宝贝乖，爷爷没事，爷爷会一直陪着你，等你长大，等你嫁人……”老人拍着小女孩的背，安慰道。<br/>    <br/>    “我才不嫁人呢~”小女孩的脸红了一下，娇嗔道，“爷爷真坏。”<br/>    <br/>    看着相依为命爷孙俩，陈兴深深地吸了口气，愈发地感到肩上的担子沉重。他来自于地球的华国，深受农耕社会价值观的影响。身为一方父母官，就应该保一方平安，让领民们安居乐业。<br/>    <br/>    “让开让开，快让开！”<br/>    <br/>    思绪之间，后方忽然传来粗暴的呼喝声。转头看出，人群迅速向两边分开。一个壮汉走在路中间，昂首阔步，仿佛这大街是他家的后花园。<br/>    <br/>    大汉的身后是一位衣着华贵的少年。背着双手，三角眼凶光外露，令人感到害怕。所到之处，人们纷纷垂下头，不敢触其锋芒。少年的背后跟着一名精瘦的老者，黑袍鼓荡，目光如炬。再往后，是数十名穿着护身甲的格斗兵，牛高马大，气势十足，灰色的胸甲上喷涂着大大的“炀”字。<br/>    <br/>    来人正是灰角城炀家二少爷和他的仆从们。<br/>    <br/>    “哟，这小女孩挺可爱的。”<br/>    <br/>    路过小女孩和老人的桑果摊时，炀智勇忽然停了下来。三角眼瞪着稚气未脱的小女孩，似乎被对方的天真可爱吸引了，又或许是想表示一下亲民，他的脸上露出微笑，尽管看起来很生硬。<br/>    <br/>    他在地摊前蹲下来，朝小女孩招了招手，“来，告诉哥哥，今年多大了？”<br/>    <br/>    小女孩似乎有些害怕，躲在老人的怀里，不敢过去。<br/>    <br/>    一时间，场面变得有些尴尬。<br/>    <br/>    遭到冷遇的炀智勇脸色微变，眯着三角眼，安静得可怕。那样子，仿佛一头随时暴起的狼狗。他出身高贵，平时根本不屑于理会这些贱民。今天心情好，想施舍些恩惠，却没想到对方如此不识抬举。<br/>    <br/>    贱民就是贱民，永远只能用拳头和鞭子对付。<br/>    <br/>    炀智勇的样子，看得周围的人一阵心悸，不禁为小女孩和老人捏了把汗。<br/>    <br/>    所幸的是，炀智勇并没有发难。沉默了一会儿，重新露出笑容，指着地摊上紫黑色的果干问道：<br/>    <br/>    “这是什么东西？”<br/>    <br/>    “是，是桑果干，爷，爷爷摘的……”小女孩抬起头，怯生生地回答。<br/>    <br/>    “多少钱？”炀智勇问道。<br/>    <br/>    “不，不用，大人您尝，不，不用钱……”老人抱着孙女，脸色发白，连说话都结巴了。<br/>    <br/>    老人和小女孩惶恐不安的样子，不但没有激起炀智勇的同情，反而让他觉得丢了面子，脸色愈发地阴沉起来。他没有再说话，随手抓起几颗桑果干，仰头放进嘴里。<br/>    <br/>    可刚咀嚼几下，他就脸色骤变，“噗”的一口，将果干吐出来。<br/>    <br/>    “什么烂东西，还有沙子！”他大吼起来，像头狂吠的恶犬，“竟敢骗小爷吃土，不想活了吗！”<br/>    <br/>    仿佛泄愤一般，炀智勇起身一阵乱踢，将桑果干踢得到处都是。好端端的一摊子东西，就这样被糟蹋了。老人低着头，紧紧地抱着孙女，不敢言语。<br/>    <br/>    这些桑果干本就是在荒原里采集的，又在热砂中晒干，带些沙土并不奇怪。只是炀智勇养尊处优，从未吃过底层的食物，这才不小心磕了牙。<br/>    <br/>    看见自己的领民被欺负，陈兴胸中血气翻涌，正想上前阻止，却被身后的人拉住了。<br/>    <br/>    “你现在上去，情况只会更糟糕。”叶阳白柳低声劝道。炀二少生性乖戾，喜怒无常。陈兴贸然上去干涉，只会激起更大的矛盾。<br/>    <br/>    陈兴楞了一下，想起对方的性格，只怕越帮越忙，只好作罢。<br/>    <br/>    果然，炀智勇发泄完，整了整衣领，仰起头，朝镇长府方向走去。<br/>    <br/>    陈兴见状，终于松了口气。<br/>    <br/>    可就在大伙以为事情就这么过去的时候，小女孩忽然挣脱老人的手，在地上抓起一把桑果干，砸向炀智勇！<br/>    <br/>    桑果干落在肩上，留下了几个黑点。炀智勇停下脚步，缓缓转过头。一时间，空气好似凝固了一般，四周静得连针掉地的声音都能听见。<br/>    <br/>    “坏蛋，大坏蛋！”小姑娘脸上挂着泪痕，倔强地喊着。稚嫩的声音刺破了死寂，仿佛天地间唯一的声音。<br/>    <br/>    在平常人的眼里，这些桑果干或许值不了几个钱，但却是她爷爷顶着烈日，冒着生命危险一颗颗摘回来的。她还指望着靠它们换些钱回来，给爷爷治病。可现在，全都没了。<br/>    <br/>    “贱——民，找——死！”炀智勇面露狰狞，牙缝里挤出几个字来，上前两步，抡起手臂，朝小女孩脸上扇去。<br/>    <br/>    在他眼中，这些贱民比蝼蚁都不如。他大发慈悲，想表示一下善意。却没想到，对方不仅不领情，还对他不敬，骗他吃土，现在还拿脏东西扔他。他这身衣服是在巨蜥城有名的裁缝店定制的，每一根丝线都是金币，是为了今天的宴会特地穿的，却被这些贱民弄脏了！<br/>    <br/>    他怒不可恕，掌风刚猛凌厉，让人毫不怀疑，这一下若是扇实了，整张脸都要变形。<br/>    <br/>    小女孩瞪大着眼睛，似乎被吓着了，呆在原地，眼睁睁地看着巴掌扇来。千钧一发之际，老人扑了过来，用身体护住孙女。<br/>    <br/>    炀智勇一击落空，脸色变得更加阴狠，横起一脚，抽向老人怀中的孩子。<br/>    <br/>    脚劲十足，刚猛霸道，老人为了保护孙女，无处可躲，只能用身体抵挡。只听见“噗”的一声，老人喷出一大口血，横飞出去，仿佛断了线的风筝，一头扎在地上。<br/>    <br/>    打完老人，炀智勇还不肯罢休，对着小女孩又是一巴掌！<br/>    <br/>    他卯足了劲，毫不留情。小女孩本能地闭上眼睛，脸侧向一边，等待着命运的降临。<br/>    <br/>    可过了好几秒，预想中的巴掌并没有落下。她眼睛睁开一条缝，看见了水站的叔叔。恶少的手腕，正被叔叔牢牢地抓住。<br/>    <br/>    最后的一刻，陈兴忍无可忍，发动三重奏，阻止了炀智勇的暴行。<br/>    <br/>    <br/>
