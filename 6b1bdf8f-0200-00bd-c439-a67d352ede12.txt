　　可燕皇也不会给葵海王这个机会。
<br/>　　他手中带着一枚戒指闪烁，当即化为了一道星河，向着葵海王镇压而去。
<br/>　　这一枚戒指，便是燕皇在九玄仙尊洞府之中得到的机缘。
<br/>　　虽然不是真正的九玄戒，但毕竟也是九玄仙尊亲手炼制之物，位于天级灵器的巅峰。
<br/>　　星河落下，葵海王的身躯便是被镇压在星河之中，力量被不断磨灭，终于意识消除，这所有的精血，也被燕皇当即吞噬。
<br/>　　碧螺老祖，五行刀祖，葵海王三尊九劫古帝。
<br/>　　如今全部陨落在这密室之中。
<br/>　　虽然在九域剧变开始，为了争夺宝藏，已经是有数位九劫古帝，死在了九域之中了。
<br/>　　但如今三位九劫古帝一同陨落，也是一件惊天动地的大事，用不了多久就会传遍整个九域。
<br/>　　“走。”
<br/>　　楚风眠看了一眼燕皇直接开口道。
<br/>　　两道身影闪烁，顿时通过光幕，离开了这洞府之中。
<br/>　　每过多久，又是一名九劫古帝到来，他观察这周围的血腥味，眉头一皱。
<br/>　　“碧螺老祖发出了呼唤，他人哪里去了？”
<br/>　　这一尊九劫古帝，观察了一下，突然间手心之中凝聚灵力，向着前方打了过去。
<br/>　　这一股灵力，在这一尊九劫古帝的面前，当场开始凝聚，随即化为了一道水镜。
<br/>　　这水镜之中，模糊无比，但是正好隐隐约约足以看到，是有人杀了碧螺老祖的那一幕。
<br/>　　“这，此人是谁？怎么如此之强，碧螺老祖此人是被此人杀了？”
<br/>　　这一尊九劫古帝，看着水镜之中碧螺老祖被轰杀的景象，有些不可置信的说道。
<br/>　　碧螺老祖可也算是一尊成名已久的九劫古帝了，绝不是什么弱者，现在居然这样，就被轰杀了。
<br/>　　而接下来的场景，更是让他目瞪口呆，不止是碧螺老祖，在下一刻，五行刀祖也被轰杀了。
<br/>　　这一尊九劫古帝，他看得出来，这五行刀祖在之前，就已经是被人逼入到了绝境指中，身受重伤。
<br/>　　接下来葵海王死的一幕，也显现了出来。
<br/>　　只是杀了这三人的身影。
<br/>　　“居然是燕皇？”
<br/>　　这九劫古帝不由的眉头一皱，杀了葵海王的人，他认的出来，就是燕皇。
<br/>　　燕皇虽然不属于三大势力，但是平日里跟三大势力的关系也还算不错，没想到这一次燕皇居然是出手杀了葵海王。
<br/>　　而至于杀了碧螺老祖跟五行刀祖的人，他一时间都认不出来。
<br/>　　“走，先把消息传递了出去！此人施展这龙族武技，跟圣龙秘境肯定脱不开关系！”
<br/>　　这一尊九劫古帝沉声说道，旋即化为了一道遁光，也从光幕之中离开了。
<br/>　　另一边，赤红大陆之中，在一处山谷之内，楚风眠跟燕皇，盘膝而坐。
<br/>　　在这山谷之内，已经是被楚风眠跟燕皇联手布置出了无数禁制，以至于其他强者，根本无法察觉出这山谷之内还有人。
<br/>　　至于楚风眠跟燕皇，则是目光紧闭，他们二人身上的气息，都在一点点的增长。
<br/>　　楚风眠在炼化这那五行刀祖的精血，而燕皇，也同样是在炼化这葵海王的精血。
<br/>　　九劫古帝的精血，都是完美的宝物，足以提升实力之用。
<br/>　　五行刀祖的精血之中的力量，比起碧螺老祖精血之中蕴含的力量还要庞大，楚风眠缓缓炼化。
<br/>　　大量的灵力涌入到了楚风眠的身躯之中。
<br/>　　旋即在楚风眠的背后，远古战龙虚影显现了出来，如同一条活着的远古战龙一般，张开嘴，不断的吞噬这其中的灵力。
<br/>　　一股股灵力，都被战龙精血所吞噬，一点点的蜕变这，这五行刀祖的精血也全部炼化的一刻，楚风眠身躯之中的龙皇精血，也足足是达到了一千一百滴。
<br/>　　“不错，这两个人的精血，几乎抵得上那武祖精血的十分之一了。”
<br/>　　楚风眠缓缓的抬起头，眼神之中露出几分喜色。
<br/>　　他现在只要是多多将身躯之中的远古战龙精血，转化为龙皇精血，那楚风眠的实力也将不断的提升。
<br/>　　现在楚风眠的境界还只是八劫古帝，如果在找一个机会，突破到九劫古帝的话，那楚风眠的实力也将会再进一步。
<br/>　　倒时候面对九劫古帝，纵然是以一敌三，以一敌四，楚风眠都足以横扫。
<br/>　　楚风眠睁开眼的一刻，另一边的燕皇也已经是将葵海王的精血全部吞噬完，他身上的气息也变得强横了许多。
<br/>　　燕皇是刚刚踏入到九劫古帝，正是需要巩固境界，这葵海王的精血，正好是大补之物，让他足以巩固境界之用。
<br/>　　“没想到楚兄的实力，居然如此厉害，这一次我也算是欠了楚兄一个人情了。”
<br/>　　燕皇睁开双眼，看了一眼楚风眠，笑着开口道。
<br/>　　他这一次，杀了葵海王，得到了巨大的好处。
<br/>　　杀一尊九劫古帝得到的好处，不亚于是在这赤红大陆之中拼死拼活的得到一处宝藏所得到的好处。
<br/>　　这一次，如果不是楚风眠先将三人已经是逼入到了绝境，燕皇想要杀了葵海王，也没有那么容易。
<br/>　　毕竟燕皇虽然有着一枚九玄戒，但是他还是刚刚突破的九劫古帝，在力量上，比起葵海王还要逊色几分。
<br/>　　凭借九玄戒，最多打平，想要杀了葵海王，还是因为葵海王已经是被逼入到了绝境，无比虚弱，才给了他可乘之机。
<br/>　　“不算什么人情，这一次燕皇不来，只怕是我也杀不了其他二人。”
<br/>　　楚风眠听到燕皇的话，笑了笑道。
<br/>　　这一次燕皇敢来的，也是无比及时，要不是这燕皇赶到的话，也许是这三人，只怕是就逃出去了。
<br/>　　到时候楚风眠这里一点好处都捞不到。
<br/>　　虽然燕皇看似没有出多少力，但是却也是压垮碧螺老祖三人的最后一根稻草。
<br/>　　不过这一趟燕皇也是没有白来，杀了葵海王，得到了葵海王的一身精血，还有他空戒之中的宝藏，也算是大赚一笔了。
<br/>　　当然楚风眠的收获更大，那五行刀祖还有碧螺老祖的精血，以及是他们二人的空戒，现在都落入到了楚风眠的手中。
