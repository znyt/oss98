出斧头帮千军万马的感觉来了，震得人耳朵嗡嗡直响。<br/>　　<br/>　　    茅小草脑门上冒出三道冷汗，捂着脸说道：“艾玛，太能装了”<br/>　　<br/>　　    王惊蛰呵呵一笑，表情低调而又装逼，迈步就朝着茅小草走去，然后伸手抓着她的手腕，说道：“我们走了”<br/>　　<br/>　　    以后多年，小草姑娘和他每当回忆起这件事的时候，她就跟王惊蛰说道：“你都不知道，你那个时候简直都傻透了，自以为自己横刀立马纵横沙场呢？其实你就跟个傻缺脑残似的”<br/>　　<br/>　　    王惊蛰惊愕的说道：“这意思是，成为我人生中的污点了呗？”<br/>　　<br/>　　    “你算是洗刷不掉了·······”<br/>　　<br/>　　    茅小草被王惊蛰拉着快速的朝着远处走去，一帮人被他唬的一愣一愣的眼睁睁的看着两人离去了。<br/>　　<br/>　　    特别是反应木讷，慢半拍的秋成子看见人都走没影了，才拍着脑袋说道：“我的灯呢？”<br/>　　<br/>　　    茅小草被王惊蛰拉着，挪蹭着小碎步，昂着脑袋看着他的后脑勺问道：“你怎么来长安了？”<br/>　　<br/>　　    “别说话，感受下这片刻的宁静，你看这夜色什么的，美不美？”
