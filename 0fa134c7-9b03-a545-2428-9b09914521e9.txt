　　众人震惊了，因为苏莫在大规模的吞噬源气，那被剑山锋锐之气绞碎的源石，其中的源气几乎被瞬间吞噬一空。
<br/>　　当然，众人之所以震惊，倒不是这种能力逆天，毕竟，拥有吞噬能力的元神，或者是功法等等，也并不是非常罕见。
<br/>　　众人震惊的是，这明显是元神之力，而苏莫之前展现了万化神剑和不败战气这两种超强元神，居然还有第三种元神。
<br/>　　“这是什么元神？”
<br/>　　“好强的威力！”
<br/>　　“难道是吞天圣魂，或者是吞天蟒元神？”
<br/>　　众人纷纷出声，各自猜测，苏莫的元神没有释放而出，众人暂时无法得知。
<br/>　　“这股力量……！”拓跋流明双眸微眯了下来，他在苏莫的身上，感受到了一丝熟悉的气息。
<br/>　　这种气息，和他的二叔拓跋青云有些类似，但又不太相同。
<br/>　　“秦陨……青云！”
<br/>　　拓跋流明低声自喃，这秦陨的名讳，居然和他二叔的名字同音，这让他愕然不已。
<br/>　　拓跋流明疑惑了起来，难道这个秦陨，和他的二叔有关系？
<br/>　　这应该不太可能，他的二叔，已经几千年没有离开过地王宫了，一直在闭关。
<br/>　　“他的元神，到底是什么元神？”拓跋流明心中思忖，他感觉苏莫的元神，或许和他的二叔有些联系，但是，又有些想不通。
<br/>　　至于苏莫是轮回分魂，他倒是没有联想到此处。
<br/>　　吞噬了大量源气的苏莫，身上气息浮动，立刻便开始冲击三品境界。
<br/>　　苏莫的修为，早已达到二品高级的极致，距离三品修为也就是半步之遥。
<br/>　　所以，他才有信心，在此地冲击三品境界。
<br/>　　轰隆隆！！
<br/>　　苏莫无论是体外还是体内，都是爆响连连，体外是剑山的锋锐之气，不停的想要破开他的防御，体内是混沌源气，如同大河奔流，川流不息。
<br/>　　可以说，此刻的苏莫，面临的是双重压力，内外的压迫，驱使他突破境界瓶颈。
<br/>　　但是，即便他天赋绝世，并且修为也到了顶点，修为的突破也不是简单之事。
<br/>　　所有人都凝视着苏莫，因为，此时此刻，苏莫身在最前方，距离山巅最近。
<br/>　　而且，他的天赋超越了所有人。
<br/>　　众人心中惊疑不定，不知道苏莫能否突破境界，若是不突破还好说，一旦突破了修为，说不定真有登上山巅的可能。
<br/>　　时间流逝，约莫半刻钟之后，源石的源气，便被苏莫炼化一空。
<br/>　　他身上的气势，如狂风呼吸，气息不断的鼓荡，显然已经到了突破修为的关键时刻。
<br/>　　唰！
<br/>　　苏莫意念一动，再次取出了十万左右的五行源石，这是均是中品源石，源气庞大无比。
<br/>　　众多的源石，乍一出现，便被剑山锋锐之气绞碎，所有的源气便流泻而出，疯狂的涌动。
<br/>　　在苏莫的吞噬之力下，所有源气便疯狂的向他体内涌去，几乎是呼吸之间，便全部涌入了他的体内。
<br/>　　这一批中品源石的源气，进入了苏莫体内之后，他的气息变得非常可怕，如同体内蛰伏者一头凶猛的太古神兽。
<br/>　　浑厚如海的五行源气，在苏莫的体内浩荡，飞速的炼化为混沌源力。
<br/>　　苏莫的气息不断的暴涨，隐隐要破开眸中桎梏，进入全新的层次。
<br/>　　“他……居然真的快突破了！”
<br/>　　“已经近在咫尺了！”
<br/>　　“只需半个时辰，他应该就能突破了！”
<br/>　　下方的众天才见此，顿时心中震动，因为，他们明显的能看出来，苏莫的身上，出现了修为瓶颈即将被冲开的征兆。
<br/>　　很显然，要不了多久的时间，苏莫将达到三品低级的境界。
<br/>　　众人面色各异，因为到了那个时候，眼前的秦陨，将有希望登上剑山之巅。
<br/>　　当然，只是有希望而已，能不能登上剑山之巅还是未知数。
<br/>　　但是，即便是有一丝可能，也让众人心中复杂万千。
<br/>　　毕竟，自万道剑冢出现以来，也就那么三两人有希望登上剑山之巅，但无一例外，全部都失败了。
<br/>　　轰隆隆！
<br/>　　苏莫身上的气息肆虐，加之剑山的锋锐之气不停的攻击，在他的体外形成了混乱的气流，虚空崩溃，劲力席卷。
<br/>　　众人所料的不错，苏莫的确是突破在即，而且，所用的时间，比之众人的预料还要短。
<br/>　　仅仅是半刻钟出头的时间，苏莫身躯一震，全身的气息猛然暴涨，迈入了一个全新的层次。
<br/>　　三品低级！
<br/>　　这一刻，苏莫终于突破了境界瓶颈，达到了三品低级的地步。
<br/>　　苏莫的周身，崩溃的混沌源力，溃散的各种气流，包括一些剑山的锋锐之气，都被他吞噬进入了体内。
<br/>　　少倾，苏莫的气息，缓缓回归平静。
<br/>　　在他踏入三品之时，明显的感觉到身下剑山的锋锐之气增强了一截。
<br/>　　不过，剑山增强的锋锐之气，远远不如他增强的实力。
<br/>　　所以，这一刻，他的防御再次彻底恢复，周身浑厚的混沌源力急速凝实，变得稳如泰山。
<br/>　　所有人均是面色惊疑不定，对于古剑山、拓跋流明、云沐等人，是不希望苏莫突破修为的。
<br/>　　毕竟，若是苏莫突破了修为，就有可能登上剑山之巅。
<br/>　　他们，根本不希望苏莫登上剑山之巅，毕竟，登上了剑山之巅，就有可能得到终极剑道的秘密。
<br/>　　终极剑道的秘密，关系甚大，即便他们得不到，也不想让苏莫得到。
<br/>　　“哈哈，秦兄果然厉害，远超万古天骄！”
<br/>　　公孙嘉见此，顿时朗笑一声，他倒是没有什么不喜，他有自知之明，自己是根本不可能登上剑山之巅。
<br/>　　相反，他甚至希望苏莫得上剑山之巅，解开终极剑道的秘密，让他一窥究竟。
<br/>　　韩生面色凝重，他本应该高兴，毕竟眼前的秦陨是他终极神王殿的弟子。
<br/>　　但是，他却是知道，秦陨肯定不会这么简单。
<br/>　　一个天赋如此妖孽的人物，修炼混沌神帝的正宗道统，会是终极神王殿一个名不经传的普通弟子吗?
<br/>　　轰隆！
<br/>　　就在此刻，在所有人的注视之中，苏莫再次前进了，又踏出了一步。
<br/>　　一步一丈，苏莫踏入了距离剑山之巅三丈之地。
<br/>　　这一刻，在外界的终极剑域，终极神王殿的山门，那耸如苍穹的神剑山，再次光华大放，更甚之前十倍，耀眼的金色光华照亮了整个终极剑域。
