
<br/>　　<br/>　　    几分钟之后，写字楼的保安下来到地下停车场，看了下车子的状况后说道：“陈小姐，车子以前要是没什么毛病的话，那可能是电瓶亏电了”<br/>　　<br/>　　    “有什么办法，能让它赶紧有电呢？”<br/>　　<br/>　　    保安说道：“找保险公司来或者修理厂来接一下电，然后再开几十公里就行了”<br/>　　<br/>　　    “啪”陈臣把车钥匙扔给保安，说道：“我赶时间，还是打车吧，你找人来接电吧”<br/>　　<br/>　　    “好的·····”<br/>　　<br/>　　    陈臣拎着包走出地下停车场，边走边从包里拿出电话，拨了出去：“喂？储汐颜，我的车子没电了，可能得玩一会到你那了”<br/>　　<br/>　　    “没事，我们还有两个姐们儿也没到呢，你慢点哈”<br/>　　<br/>　　    陈臣来到路边，拦了几辆车都没有停的，这个点得算是晚高峰了，出租车基本都没有空客的，等了半天之后，陈臣看见前面有个公交站台，走过去扫了眼后发现有一趟公交车，跟她要去的目的地正好一趟线。<br/>　　<br/>　　    “本小姐都多少年没坐过公交了······”陈臣仰着脑袋感叹了一句后忽然之间就顿住了，她冥冥中觉得这句话听着好像有点耳熟。<br/>　　<br/>　　    “唰，唰”陈臣眨了眨眼睛，疑惑的嘀咕了一句：“错觉？”<br/>　　<br/>　　    片刻后，连续两辆公交驶入站台，前面一辆车里上下了好几个人，唯独后面那一辆车门打开后没人上去，陈臣看了眼公交上面的车号，正好是自己要坐的那一辆，就赶紧走了过去。<br/>　　<br/>　　    “叮咚”车门关上，陈臣从包里找出两枚硬币投了进去，抬起头往座位那边走过去的时候，看着略显空荡荡的车厢就有点发愣了。<br/>　　<br/>　　    公交车里很空，只有靠右侧那一边坐了几个女人，都面无表情的，要么望着车窗外，要么就是低头看着手机，陈臣的心里忽然有点直敲鼓，她隐约觉得这一幕似乎有点熟悉，但却实在想不起来熟悉在哪了。<br/>　　<br/>　　    走到左侧车边，靠窗的位置她坐了下来，但心头却始终都不太静，一直都七上八下的。<br/>　　<br/>　　    公交车开始报站点，陈臣抱着包舔了舔干裂的嘴唇：“我怎么有点小紧张了呢······”<br/>　　<br/>　　    “唰”陈臣突然转过头，看向右边那一排座位坐着的几个女人，忽然之间她回忆起了，几天前曾经听到过的一句话，那是个陌生的男子跟她说的。<br/>　　<br/>　　    “如果，你这两天要是坐公交的话，假如上车后发现车里正好坐了六个女人，你千万不要坐上去”<br/>　　<br/>　　    “我都不记得自己多少年没坐过公交了，以后估计也没这个可能了”<br/>　　<br/>　　    陈臣头皮瞬间就麻了，鸡皮疙瘩掉了一地，她总恍惚的觉得今天这一幕似乎非常熟悉，于是终于想起来熟悉在哪了。<br/>　　<br/>　　    那是几天前，在自己家院中的水池边上，一个陌生男子跟她讲过。<br/>　　<br/>　　    “咕嘟”陈臣咽了口唾沫，在心里告诉自己：“巧合，一定是巧合”<br/>　　<br/>　　    “叮咚”公交车这时缓缓驶入站台：“各位乘客请前门上车，后门下车，下车时请注意安全·······”<br/>　　<br/>　　    车门开了后，仍然一个上来的乘客都没有，司机按了下按钮后面的车门就关上了。<br/>　　<br/>　　    陈臣望着关闭的车门，还有那一侧坐着的六个女子，坐在座位最后面的一个看不太清楚脸庞的女子这时也抬起来头，陈臣忽然发觉对方的笑容似乎有点那么阴霾？<br/>　　<br/>　　    就是那种似乎很阴森的笑容。<br/>　　<br/>　　    她突然紧张的就跑了过去，伸手拍着车门说道：“不好意思师傅，我要下车，下车，麻烦帮忙再开下车门，抱歉，刚才没注意到”<br/>　　<br/>　　    在车门关上，公交车马上要驶离站台的这一刻，陈臣最终抵不住内心的躁动，跑到后面要下车了。<br/>　　<br/>　　    车门打开后，陈臣就从车上下来，站到站台上气喘吁吁的捏着小拳头：“个瓜娃子的，哎呀呀，这是犯什么邪病了呢，怎么还信了一个疯子的疯言疯语了”<br/>　　<br/>　　    下车之后，陈臣都觉得自己的做法挺不可思议的，有点可笑，她差不多还有三站地就要到地方了，但中途却突然又下来了。<br/>　　<br/>　　    “嘎吱······”突然间，前方路口那边传来一声急促的刹车声。<br/>　　<br/>　　    陈臣下意识的转过头，就看见了令她无比惊恐的一幕，那辆刚刚驶离站台的公交汽车，在经过一个路口的时候，一辆渣土车突然快速的开了过来。<br/>　　<br/>　　    “砰”渣土车的车头直接怼在了公交车左边的出身，然后瞬间就被顶的侧翻了，还往前滑行了一段距离，地上都被擦出了火星子。<br/>　　<br/>　　    陈臣茫然，不解，又惊恐的看着撞车的一幕，她还没有下车之前，所坐的那个位置正好就是渣土车车头撞击的地方。<br/>　　<br/>　　    陈臣无助，惊慌的蹲在了地上，耳中嗡嗡直响，附近奔跑喊叫的人群，还有前方鸣笛的动静，她仿佛全都没有听到，脑子里就一直回荡着一个声音。<br/>　　<br/>　　    “如果，你这两天要是坐公交的话，假如上车后发现车里正好坐了六个女人，你千万不要坐上去······”<br/>　　<br/>　　    今天她的车子恰巧就坏了，她要去的地方正要可以乘坐公交车只有几站路就能到了，车上更巧的是正好坐了六个女人。<br/>　　<br/>　　    然后，公交车被撞了。<br/>　　<br/>　　    “怎，怎么回事，这，这是怎么了······”陈臣抓着头发无助的喃喃的嘀咕了几句。<br/>　　<br/>　　    半个多小时后，一辆七系宝马开了过来，储汐颜打开车门，说道：“陈臣你怎么了？”<br/>　　<br/>　　    “咣当”陈臣关上车门，看见旁边放着一瓶水后，抓起来拧开瓶盖“咕嘟，咕嘟”的就喝了起来。<br/>　　<br/>　　    “不就是撞车了么，有什么大惊小怪的，你不是从车上下来了么？”<br/>　　<br/>　　    “如果我要是没有下来呢？还有，明明就只差三站地我就要到和你们约会的地方了，你知道我为什么会突然下车么？”<br/>　　<br/>　　    储汐颜不解的问道：“对啊，马上就要到了，你怎么就下来了呢”<br/>　　<br/>　　    “几天前，我碰到过一个人，他和我说的，如果我坐上一辆公交车的话，车里如果有六个女子，他告诉我不要坐上去”宝马这时开到前方出事的路口，陈臣指着还翻到在地上的公交说道：“就是这一辆，而且我上去的时候里面恰好就真的坐了六个女人”<br/>　　<br/>　　    储汐颜顿时懵了：“不，不可能吧？或者是太巧了？”<br/>　　<br/>　　    “不说了，不说了，送我回家，今天我没心情出去了······”
