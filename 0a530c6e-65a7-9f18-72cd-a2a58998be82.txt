　　　　涂小安突然觉得自己是何等的幸运，死过一次的人，还能活在这人世间。
<br/>　　纵然是蛇躯，但现在也化为人形，哪怕这具虚构的身体是假的。
<br/>　　他也充满了幸运，无比的幸运。
<br/>　　陈高阳住在这偌大的四合院，可却从来不曾睁开见过四合院一眼。
<br/>　　涂小安想看陈高阳现在的所想，可什么画面都提取不到。
<br/>　　正当涂小安的意识力要退出陈高阳的脑海之时，一道甜如浸蜜，让人倍感舒适，心旷神怡的声音骤然响起。
<br/>　　“高阳哥哥，你怎么又自己找东西了，人家不是说了，你想要什么东西跟我说，海棠帮你找”
<br/>　　那道声音柔声细气，悦耳动听，让人听了，便永远忘不了。
<br/>　　陈高阳的声音响起：“我没想找什么，只是这房子被你收拾的，什么顺序都乱了，什么东西在哪，我也不知道，我想要都摸一摸，然后就能记住”
<br/>　　那柔声细气的声音，像是带着一丝不满：“高阳哥哥，这辈子海棠就是你的眼睛，你的手，你的脚，你不需要记”
<br/>　　“海棠，我是一个天生的瞎子，你真的愿意这辈子都跟着我这个瞎子吗”
<br/>　　她含此轻吐：“梧桐相待老，鸳鸯会双死，高阳哥哥不嫌弃我就好”
<br/>　　嫌弃？
<br/>　　陈高阳闻言，将头摇的跟一个拨浪鼓似的。
<br/>　　老天拿走了他的眼睛，却送了一位天使伴他。
<br/>　　涂小安看不到画面，只有一声声的男女对话，一些日常，一些甜蜜。
<br/>　　“高阳哥哥，我想问你一个问题”
<br/>　　“你问”
<br/>　　“如果眼睛跟我，你选哪样”
<br/>　　这个问题就好像是你老妈跟我同时掉水里了，你先救谁。
<br/>　　陈高阳不假思索的说：“自然选眼睛了，我的眼睛好了，我就能看见你，我能看见你，我就可以照顾你，而不是你来照顾我”
<br/>　　“高阳哥哥，你不要选眼睛好吗，我怕你看到我，就不要我了”
<br/>　　“傻瓜，我怎么会不要你”
<br/>　　“你的眼睛如果有一天能看见了，那么便是海棠消失的那一天了”
<br/>　　站在四合院中的陈高阳回忆起这句话，浑身颤栗，呢喃自语：“海棠，我的眼睛并没有好，你为什么就消失了”
<br/>　　涂小安将意识力散去，睁开眼，看着陈高阳：“又想你女朋友了是吧”
<br/>　　“嗯”他苦笑的点了点头。
<br/>　　涂小安看着四合院，随意道：“你这个家那么多房间，你平时跟你女朋友是睡在一起，还是分开睡呀”
<br/>　　陈高阳猛然身子一僵，墨镜下的脸不禁的微微晕红起来，他没想到涂小安冷不丁的问那么私人的问题。
<br/>　　吞吞吐吐的说：“我们虽然是男女朋友了，但还没结婚，自然是分开睡”
<br/>　　“那她睡那个房间呀”
<br/>　　陈高阳道：“你问什么做什么”
<br/>　　涂小安笑了笑：“我看你对自己女朋友一往情深，有点被你感动，所以想帮你找找线索”
<br/>　　“她...她应该是住那个房间”陈高阳举起拐杖，朝着一件房间指去。
<br/>　　“那我能进那个房间看看嘛”
<br/>　　陈高阳连忙点了点头，有个人愿意帮他，他自然高兴。
<br/>　　对于一个瞎子，找任何东西都困难，更别说去找一个活生生消失的人。
<br/>　　涂小安走过去，推门而入，霎时间，他的眉头紧促起来，惊讶道：“好浓郁的觉醒类生物气息”
<br/>　　这就好比一些电影中的道士，大喊：好强烈的妖气。
<br/>　　“你说什么”
<br/>　　陈高阳没听清楚的问。
<br/>　　“哦，没什么”
<br/>　　涂小安在房间里仔细的看了看，找了找，心中大致有点数了，问：“你女朋友确定是住在这件房间吗”
<br/>　　“这个自然，我虽然瞎，但这点不会记错”
<br/>　　“那为什么在这个房间中，找不到一件女孩子的衣物，甚至是日用品”
<br/>　　陈高阳诧异道：“没有吗，什么都没有吗”
<br/>　　倏然，陈高阳摇摇欲坠，想到了一个可怕的可能，痛苦的说：“看来她并不是出什么意外，想必一定是她对我这个瞎子厌了倦了，所以收拾东西，悄悄的离开了”
<br/>　　不告而别。
<br/>　　也只有这种可能了，不然海棠住的房间，怎么可能什么东西都没有。
<br/>　　陈高阳一直在自欺欺人，海棠消失后，他就担心她是不是出了什么意外，是不是遇到什么坏人了。
<br/>　　但另外有一个更为现实的可能性他一直不愿意去想。
<br/>　　现在事实摆在面前，他不去想都不行。
<br/>　　试问，一位女人怎么可能愿意一辈子照顾一个死瞎子。
<br/>　　她的声音是那么的动人，长的也一定是非常的美丽。
<br/>　　为什么要跟他这个瞎子厮守终生。
<br/>　　“到也不是什么都没有”
<br/>　　此时，涂小安的手中捏着一缕动物的毛发。
<br/>　　“有什么”陈高阳连忙紧张的问。
<br/>　　涂小安思忖了下：“就女孩子的头发而已”
<br/>　　他失望了。
<br/>　　“海棠，就算你要离开了，为什么要不告而别，我们可以好聚好散的，我不会耽误你的幸福的”
<br/>　　此时，陈高阳变的悲恸欲绝，已经彻底被悲伤占据，他低下头，颤抖的双肩，墨镜的遮掩下，看不到眼，但却看到了一丝热泪滚烫的流淌下来。
<br/>　　“你先别难过，或许她有什么难言之隐”
<br/>　　涂小安拍了拍他的肩膀，安慰人这种事情，他还真不怎么会。
<br/>　　更况且还是安慰一个男人。
<br/>　　陈高阳伤心欲绝，他认准了自己的女朋友其实是悄悄的收拾东西不告而别。
<br/>　　十分钟后，涂小安待他稍稍冷静，才问：“你这个四合院中，养过什么宠物吗”
<br/>　　“宠物？”陈高阳还沉寂在痛苦中，心不在焉道：“没有，我一个瞎子还养什么宠物”
<br/>　　“到是以前我奶奶还活着时，这院中养着一只猫”
<br/>　　“猫？”涂小安的眼睛猛然一亮：“那猫长什么样子”
<br/>　　这句话刚说出，涂小安就打了自己的大嘴巴，问一个天生的眼瞎的人这个问题，好像有点脱了裤子放屁，多此一举。
<br/>　　“那只猫后来呢，是死了，还是怎么了”
<br/>　　陈高阳不太明白对方为什么突然会问这个问题，但还是回答了：“五年前，我奶奶去世，那只猫也就在也没有出现过了”
<br/>　　“那你跟你女朋友是什么时候认识的”
<br/>　　涂小安心中已经有一个答案了，他现在要做的就是慢慢的证实自己心中的答案是真还是假。
