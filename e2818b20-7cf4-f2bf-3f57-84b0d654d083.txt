　　当天晚上，叶少阳带着大伙开始布阵——倒不是布置什么阵法，几天的时间，根本来不及布置什么阵法，而且茅山自有风水大阵，什么阵法也没这个厉害。
<br/>　　他们要布置一个口袋阵，利用地理环境，给他们来个包圆，让他们钻进口袋，然后尽数消灭。这就是道风的战略，不打什么防御战，要来就来个歼灭战，一个活的不留。
<br/>　　布置了一个晚上，大伙都找到了自己的位置，然后叶少阳开始了茅山的风水大阵，一旦有外来的邪物入侵，钟声就会响，大家这才放心地在住下来——叶少阳把他们都安顿在了藏经阁的宝塔中，等候战争来临。
<br/>　　这些事情，苏钦章大部分插不了手，只能帮着做一些小事，早上，他安排弟子蒸了十笼包子，一大锅稀饭，送到叶少阳等几个法师的房间里。
<br/>　　大年初二。
<br/>　　叶少阳一边吃着包子，一边听着山下隐约传来的鞭炮声，正在发呆，突然道风推门进来，看到他在吃饭，就在床上坐下来。
<br/>　　“有事？”叶少阳招呼他，“你要不要吃点包子，这是我当年跟师父去杭州的时候，无意中得到的配方，师父特别爱吃，回山就经常让我做，我后来传给苏钦章了，正宗的杭州小笼包我跟你说。”
<br/>　　道风虽然不是人类，但他已斩尸证道，重塑灵身，人间的东西也是能吃的。
<br/>　　听他这么说，道风也过来，捏了一个包子塞进嘴里。
<br/>　　“好吃吧？”叶少阳看他的表情就知道，得意笑道：“我跟小苏说了，这配料房子要一直传下去，当做我茅山的传统，将来万一没香油钱了，至少还能卖包子挣钱不是。”
<br/>　　“你心情好像不错。”道风不看他，淡淡说道，又伸手拿了一个包子。
<br/>　　叶少阳耸了耸肩，“假装不错吧，总不能死气腾腾的，影响士气。”
<br/>　　他内心很怕，这即将到来的一战，很可能比之前海眼之战还要可怕，这一点，光从道风把风之谷骨干都调来便也可看出大概。上一战折损了秋莹，他到现在还没走出伤痛，“茅山守卫战”还没结束，结局未可知不说，万一再有兄弟发生意外……他真的不敢多想，但自己又是老大，只能做出一副乐呵呵的样子，鼓舞大家的士气。
<br/>　　道风这才看了他一眼，道：“我知道你在想什么，我也有点担心。其实，你想要知道结果，倒是很容易……”
<br/>　　叶少阳愣了一下，方才明白他这句话的意思，叹了口气道：“我不敢去做，如果看到结果，那一切都不一样了。”
<br/>　　道风说的是山海印，通过山海印穿越到未来——几天之后，便可以预见到这一战的结果，但是……如果终究有人要死，那怎么办？让他远离战斗？
<br/>　　改写历史，会产生不可预测的后果，甚至历史没有改写、反而会产生各种难以掌控的连锁事件。
<br/>　　这是使用山海印的第一大忌。
<br/>　　可如果真的知道某个兄弟要死，自己又不可能干看着不管。这才是真正的矛盾核心。
<br/>　　做一件事情，一定不能提前知道结果，否则可能会因此变成另一件事。
<br/>　　“我不会这么做的。”
<br/>　　道风赞许地点点头，刚才他那么说，也不过是试探。
<br/>　　“你快点吃，你吃完了我带你去一个地方。”
<br/>　　“什么地方？”
<br/>　　“现在说出来，你就没兴趣吃饭了。”
<br/>　　叶少阳一愣，干脆不问，继续吃饭，不过加快了速度，一口气吃了十个包子，一大碗白米稀饭，抹了抹嘴，满怀期待又有些紧张地抬头望着道风。
<br/>　　“快说吧，去哪。”
<br/>　　“你……想不想去见师父？”
<br/>　　叶少阳内心颤了一下，傻傻地看着他，试图通过他的表情看出他为什么要这么问。
<br/>　　道风脸上虽然没表情，但是接下来的一句话，却让叶少阳燃起了一身热血：“我带你去找他。”
<br/>　　“真的？”
<br/>　　几秒钟后，叶少阳猛地从椅子上站起来，又觉得有点不现实，皱眉说道：“去哪里找啊，他不是去了修罗界吗。”
<br/>　　“是啊，我们就去修罗界。”
<br/>　　修罗界……
<br/>　　道风没再多说什么，从房门走了出去。叶少阳赶紧跟上去。
<br/>　　道风脚步飞快，叶少阳只好一路跟着，不觉中来到了藏经阁。
<br/>　　大伙都在藏经阁里坐着聊天，看道风和叶少阳相继进来，都站了起来，问他们干什么。
<br/>　　叶少阳没工夫解释，让他们该干啥干啥，然后尾随着道风一口气爬到顶楼。
<br/>　　藏经阁是一座宝塔，但是不高，没有龙虎山的镇妖塔看上去恢弘，一共只有九层，最高一层，是一个空荡荡的房间，供奉着一块巨大的乌黑的石头。
<br/>　　这石头有一个人那么高，形状就像是一座假山，耸立在房间中心。
<br/>　　叶少阳当然看过这石头，小时候就好奇它是做什么用的，师父只说这是一块天外飞石，早年就被他师父的师父给放在这座塔上，他也不知道是干啥用的，后来叶少阳尝试了几次，对这块石头作法，发现什么法师落在这块石头上，都不会传导出去，也不会反弹，而是完全吸收。
<br/>　　不管施加在石头上的法力有多强，都会被石头奇怪地吸收，转瞬即逝，不留一点痕迹。
<br/>　　他有一次还想把石头切开，锯子都找好了，结果被青云子发现，打了一顿，那是师父为数不多的一次发怒，告诫他，不管这块石头是什么，既然祖师爷给放在宝塔顶上，那一定是有原因的，绝不能损毁。
<br/>　　叶少阳就不敢再它的主意了，后来上了中学，学到陨石，叶少阳就怀疑这石头会不会是一块陨石……
<br/>　　他实在没想到，道风居然把他带到这里来，一颗心顿时激动起来，甚至忘了此行的目的，问道风：“你是不是知道这石头是什么？”
<br/>　　“它没有名字，我只知道，这是一块来自异界的石头，这样的石头，人间只有三块，一块在泰山，一块在龙虎山，还有一块就在茅山。”
<br/>　　道风继续说，“通过它，人类可以穿越虚空，进入修罗界。这也是活人想要进入修罗界的唯一的常规途径。”
<br/>(本章完)
