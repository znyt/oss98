　　“丧尸王，精神攻击？！”白羽大惊之下脚步猛地一顿，停在原地呆傻起来，口中更是下意识的念念有词：“怎么可能？怎么可能？怎么可能？”
<br/>　　白允儿停下脚步回头看着他，眼神出乎意料的淡定：“这不是很明显么？”
<br/>　　“明显什么？不明显！这究竟是怎么回事啊！”白羽情绪很激动，但随即又想起了什么似得猛然一愣，眼神变化个不停道：“等等，等等，让我捋一捋……”
<br/>　　“两个月前类尸木天去了南方埋骨之地，一个月后丧尸王诞生，类尸本身就有着天然优势，也就是说这个丧尸王的身份很有可能就是木天，也就是异种王木叶的弟弟。”
<br/>　　“丧尸王诞生的一个月后就爆发了尸潮，第一批尸潮攻击了南署基地，与狼牙战团展开了大战。”
<br/>　　“楚涵为了解决这批尸潮中的丧尸小头目，亲自追击，只是在击杀了丧尸小头目之后，忽然原因不明的昏迷，至今未醒。”
<br/>　　“他的体内能量一切正常，什么状况都没有就好像是在睡觉，更准确的说是像个植物人。”
<br/>　　“植物人，莫名昏迷，事发地点周围没有出现异常……显然就是精神攻击。”
<br/>　　“楚涵已经是八阶巅峰的新人类，能让他一瞬间就昏迷不醒的精神攻击，当下除了最强大的丧尸王，亦或是十阶的神秘家族并有着精神攻击能力的人，其他都可以排除。”
<br/>　　“可是据我所知四大神秘家族之中只有肖家有过此类人，但不巧肖家已经灭族了，而肖七的能力不是这个，其余还活着的三大家族成员，并没有任何一位有精神攻击能力，所以让楚涵昏迷的强大者，只可能是异种王。”
<br/>　　白羽一层层的分析着，紧接着愣愣出声：“小姐姐好厉害，那个高漫秋也好厉害，你们竟然一下子就知道了，我却需要提醒，还逐步分析了这么久才得出结论。”
<br/>　　白允儿微微一笑：“我和高漫秋并非根据信息分析出的这一结论，而是一开始就有此猜测，只是缺少所能应证这一想法的证据。”
<br/>　　“所以那四点爆料就是证据，又恰好吻合这一猜测。”白羽满是惊叹的开口：“白忧一心想解决楚涵这点我知道，而丧失王木天一开始并非自由身，是白忧命令异种王木叶将其放逐到南方，正常情况来看丧尸王引发的尸潮，它本身也不可能出现在第一批尸潮中前来，所以丧尸王攻击楚涵必然是由白忧指使。”
<br/>　　说着白羽皱眉：“但我想不明白的是，白忧凭什么能指使丧尸王？这完全不可能啊！”
<br/>　　“你忘记了异种王？那可是丧尸王的亲·哥哥，在他们还是人类的时候……”白允儿唇角扬了扬，露出一个充满嘲讽的笑容：“白忧想杀楚涵的心已经变成了执念，却又不能自己动手，无论什么代价都愿意付出，只为了让楚涵死。”
<br/>　　白羽眨了眨眼睛：“我还是不明白。”
<br/>　　白允儿垂下眼眸，道：“他怎么让我失忆的，就可以怎么让丧尸王恢复记忆。”
<br/>　　白羽瞳孔剧烈收缩，震惊道：“你猜到白忧是怎么做到的了？是什么？”
<br/>　　白允儿眼底出现了一抹带着寒意的笑容，惊心动魄：“白家上上下下这么多人，还没有人知道白忧解开双基因锁后，他的能力是什么吧？”
<br/>　　轰！
<br/>　　白羽心神剧烈震荡，声音拼命在颤抖：“莫，莫，莫非？是他的，能力？！”
<br/>　　白允儿抬头看向上方明亮的天空，声音说不出的悠扬：“除此之外，这世上还有其他的什么东西，能做到这一点么？”
<br/>　　白羽愣住，反复思索后摇头：“没有，只能是能力。”
<br/>　　“但还有一点，我想不通，也很奇怪。”忽的白允儿皱起眉。
<br/>　　她道：“丧尸王都已经来到这处边境地带，貌似也与白忧达成了某种共识，既然对楚涵动了手，以它身为丧尸王的战力及能力，想杀死楚涵应该只是举手之间，为何只是给了楚涵一个昏迷不醒的精神攻击，甚至还是不会造成任何损伤的精神攻击，这不合逻辑。”
<br/>　　白羽也愣住，呐呐点头：“是啊，为啥？”
<br/>　　白允儿叹口气，摇摇头后抬脚继续往前走。
<br/>　　白羽再次追上，好奇问道：“但是小姐姐你怎么猜出高漫秋去过狼牙，见过了楚涵？我们偷听到的她和段江伟对话，完全没有暴露过这一点啊！”
<br/>　　白允儿回答道：“因为她隐约说出了楚涵不在，段江伟便可以在这里施展才华的话，如果不是亲眼见过楚涵的状态，如何确认楚涵在接下来的尸潮大战中不会出现？”
<br/>　　“小姐姐果然好聪明！”白羽眼中异彩涟涟，紧接着又察觉到不对，问道：“可是楚涵这下子变成了植物人，醒不过来，小姐姐你怎么好像一点不担心？你不喜欢他了吗？”
<br/>　　“你知道什么是喜欢？什么是不喜欢？”白允儿好笑的摇摇头，随后眼角的笑意带上了明亮：“还有谁说，楚涵醒不过来？”
<br/>　　白羽愣住：“他那个状态，根本就是大脑深度沉睡的状况啊！怎么醒？难道你想找到丧尸王，让它把楚涵唤醒？”
<br/>　　白允儿对白羽的夸张情绪有些无奈，道：“有精神类能力的又不是只有丧尸王。”
<br/>　　白羽有些生气：“我说小姐姐，你是想找个有精神类能力的新人类救楚涵？你别开玩笑了，楚涵可是被丧尸王的攻击导致昏迷的，这种程度的攻击哪个新人类能解？十阶的吗？十阶巅峰的吗？”
<br/>　　“这种人根本不存在！一出生就被摁死了！怎么可能给他发展的机会？”
<br/>　　“精神类的能力虽然列在神之狙击之下，并不是最强，但却是唯一能攻击到神秘家族成员的存在，因为他们的攻击完全是无形的，超脱了血脉等级的限制，白家都发出诛杀令了，精神类强化者一旦出现就立即击毙。”
<br/>　　“不存在么？”白允儿幽幽一笑：“我怎么恰好认识一个呢，还曾经找上门，对我敌意很大。”
