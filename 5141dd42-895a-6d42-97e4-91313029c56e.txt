　　苏莫离开客栈之后，便离开了那座城池，一路向南飞去。
<br/>　　因为在他之前听到的消息之中，太阴族的所在之地，就在南方。
<br/>　　太阴族，和巫族完全不同。
<br/>　　巫族之人的血脉和普通人族有异，有独立的族印，能与普通人族区分开来。
<br/>　　而太阴族无法区分，因为太阴族就是人族，只不过是特殊的群体而已。
<br/>　　据闻，太阴族的祖先，是一种拥有太阴之体的人类，后来其和自己的子嗣创造了太阴族。
<br/>　　经过无数岁月的发展，太阴族才成长到了如今的地步。
<br/>　　太阴之体，乃是太古异体榜排名第六十三的体质，万古难出，比纯阴之体、玄阴之体、阴煞之体，这三种阴属性的体质，不知道强大了多少倍。
<br/>　　当然，在太阴之体不出的年代，这三种阴属性体质，就是太阴族道统最好的传入。
<br/>　　而太阴族之人，大多都是修炼阴寒属性的武学，这是他们族中自古以来的道统。
<br/>　　太阴族为了壮大，从整个明虚星河，甚至是其它星河之中，不断的吸收适合阴寒属性功法的天才。
<br/>　　如此，太阴族才能长盛不衰，成为整个宇宙中都有名的种族。
<br/>　　可能巫仑大祭司，正是知道这一点，所以才会把宏青璇交给太阴族，想必从太阴族手中得到了巨大的好处。
<br/>　　当然，或许是巫仑大祭司不想留下后患，又或者是太阴族不想让宏青璇拥有过去，能真正的融入太阴族之中，才抹去她的记忆。
<br/>　　苏莫不断的向南方飞行，太阴族的所在之地，在天阴山脉，他要前去查探一番。
<br/>　　苏莫的速度极快，可以说是瞬息数千里，但这太阴星非常的大，他距离天阴山脉的距离，亦是非常的遥远。
<br/>　　随着不断的飞行，约莫过了七日之后，苏莫发现周围的环境渐渐的变了。
<br/>　　空气中阴气极重，气温都降到了冰点，大地之上也渐渐的荒芜了起来。
<br/>　　这不是普通的寒冷，而是阴气造成的阴寒，让人灵魂都能感觉到的寒冷。
<br/>　　当然，这种寒冷还不至于对人造成伤害。
<br/>　　苏莫一路疾行，又过了一日之后，远远的，他看到了所谓的天阴山脉。
<br/>　　只见在他视线的尽头，出现了一片庞大的黑色山脉，犹如盘旋的巨龙，占地方圆数万里，蛰伏在大地之上。
<br/>　　山脉高大、雄伟，山峰连绵起伏。
<br/>　　整个山脉之上，阴气冲天，让得九天之上风云变化，但又不会让人感觉到山脉阴森。
<br/>　　“这就是太阴族的所在之地吗？”苏莫呢喃自语，极力催动幽冥魔瞳武魂，看向天阴山脉。
<br/>　　不过，如今的幽冥魔瞳武魂，已经没有了任何作用。
<br/>　　因为这只是武魂，而不是战魂，而且级别较低，对于他这个境界的武者而言，已经失去了作用。
<br/>　　随着不断的飞近，苏莫发现天阴山脉之上，并没有任何的建筑，而且连一个人影也没有。
<br/>　　“怎么回事？”苏莫心中疑惑，继续飞近，少倾之后，他终于知道了原因。
<br/>　　原来在天阴山脉的中央，居然有着一座庞大的城池。
<br/>　　天阴山脉，呈环形围拢，中央有一片很大的地域，坐落着一座浩瀚的古城。
<br/>　　古城庞大，城中建筑连绵不绝，此城并没有城墙，四周那高达数千丈的山脉岩壁，便形成了天然的城墙。
<br/>　　在距离天阴山脉，还有近万里距离之时，苏莫停下了身形。
<br/>　　他没有靠近，因为他知道，自己不可能进入太阴族之中，还可能被当成闯入者，遭遇危险。
<br/>　　想要找到青璇，甚至是带走青璇，必须要有详细的计划，不然只会死在这里。
<br/>　　“青璇，你在这里吗？”苏莫凝望着远处浩瀚的古城，压下心中冲进太阴族的冲动，呢喃自语。
<br/>　　不时的有太阴族之人从古城内飞出，不过，倒是没有人理会苏莫。
<br/>　　因为太阴族在明虚星河名气太响，是无数武者心中的武道圣地，平常来到天阴山脉观望的武者，不在少数。
<br/>　　“必须先弄清楚那云悠悠是不是青璇！”少倾之后，苏莫深深的吸了口气，这才是当务之急。
<br/>　　随即，他身形一闪，向另一个方向飞去。
<br/>　　在他的前方，之前有一名从太阴族内飞出的武者，修为不是很高，苏莫顺着对方离去的方向，跟了上去。
<br/>　　苏莫飞的不是很快，没有急速的追上去，只是感受着对方留下的气息，远远的吊在对方的后方。
<br/>　　此地距离太阴族太近，苏莫不得不谨慎，免得被太阴族的强者发现，而遭遇大麻烦。
<br/>　　飞行了几个时辰，远离了天阴山脉不知多少万里之后，苏莫加快了速度，向那名太阴族的武者追了上去。
<br/>　　很快，那名太阴族的武者，便出现在了他的视线之中，此人身穿紫袍，身材高瘦，修为只有武帝境七重境界。
<br/>　　咻！
<br/>　　苏莫身形如电，撕裂长空，虚空中发出刺耳的破空之声。
<br/>　　“嗯？”
<br/>　　飞行之中的紫袍青年，顿时发现了急速追上来的苏莫，立刻停了下来。
<br/>　　唰！
<br/>　　呼吸之间，苏莫来到了紫袍青年的身前。
<br/>　　“阁下为何追我？”紫袍青年打量着苏莫，皱眉问道。
<br/>　　“这位兄弟，在下想请教几个问题，还请告知！”苏莫向对方抱了抱拳说道。
<br/>　　“什么问题？”紫袍青年冷声说道，突然被人追着请教问题，让他心中阴霾了下来。
<br/>　　“你应该是太阴族之人吧？不知你有没有见过云悠悠？”苏莫沉声问道。
<br/>　　紫袍青年闻言，眸中精光一闪而过，冷冷的道：“见过又怎样？没见过又怎样？”
<br/>　　“我想知道她的容貌，具体是何模样？”苏莫没有在乎对方的敌意，继续问道。
<br/>　　“呵呵！”
<br/>　　紫袍青年闻言，顿时冷笑了起来，随即满脸不屑的说道：“又是一个癞蛤蟆想吃天鹅肉的人，我太阴族的神女，也是你一个小小的武帝境八重武者能觊觎的？”
<br/>　　紫袍青年心中非常不屑，云悠悠乃是族中的神女，无数顶级妖孽心仪的对象，只有真正的妖孽才有资格追求。
<br/>　　而眼前的家伙，一个武帝境八重的武者，有觊觎之心倒也罢了，居然还来找他打听云悠悠的容貌，真是不知所谓。
