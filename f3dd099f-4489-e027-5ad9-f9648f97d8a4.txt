>　　“好，你那里的鲈鱼、虹鳟鱼、鳗鱼、桂鱼，这四种鱼我按照每斤两百元的价格进货，其他鱼类我按照100元的价格进货，你看怎么样？”<br/>　　<br/>　　李瑞没有等陈默给他报价，而是自己开了价格。他知道陈默报价也不会多高，虽然市场上虹鳟鱼卖的比较高，也不过是100左右一斤。其他鲈鱼和鳗鱼之类的，也都在100以下，不会超过这个价格。<br/>　　<br/>　　但是他不是奸商，陈默哪里的东西他可是有体会的，都非常的好。那么只要东西好，卖出去价格高，饕餮们还就认这个。<br/>　　<br/>　　所以自己绝对能赚钱，那么自己兄弟还不能赚一点啊？因此他才将这个价格定到200元。<br/>　　<br/>　　“你怎么不问问我定制的价格，就报出这个价格，你也不怕亏本啊？”陈默笑着说道，李瑞的报价还真就比他心中所想的价格要高一些，所以他想了一下也就能明白李瑞的心思。<br/>　　<br/>　　“哈哈！不用问，只要比你心中的价格高一些就成！要是低了，你就哥给个价格，我按照你给的价格进货。”<br/>　　<br/>　　李瑞大气的说道。<br/>　　<br/>　　“不用，你给的价格非常好！就按照你说的走！”<br/>　　<br/>　　陈默心中本来定制的是150元每斤，但是能听到200元每斤，那么他当然按照高的走了。<br/>　　<br/>　　“你这个价格行不行，要知道进货就200元，加上运输和存储等等，下来进价可就高了，再加上人员工资和房费等等，价格就高了！”<br/>　　<br/>　　“哈哈！兄弟，你是没有开过餐厅，所以一些事情在你看来非常的高，但是对于一些吃货来说则非常的便宜。就算是一盘普通的红烧鲤鱼，只要是从你那里进的货，那么我就敢标价标到480元一盘！更何况是其他鱼呢！”<br/>　　<br/>　　陈默听了，只能说出两个字：“奸商！”<br/>　　<br/>　　这看着各个行业都简单，好像跨行也是非常简单，不过一旦进入并做起来，就会发现里面好多弯弯道道，要是没有个心眼，迟早要完！<br/>　　<br/>　　所以只要李瑞能这么自信的说话，那么就按照他说的做就成。<br/>　　<br/>　　“好，就按照你说的办。你明天来拉鱼！不过要自己准备水产运输车！”<br/>　　<br/>　　陈默这里没有什么运输车，也就让李瑞他自己准备。而且这么大的一个连锁餐饮集团，如果连个水产运输车都没有的话，那还真是要骂娘了。<br/>　　<br/>　　两个人就进货量和进货品种讨论了一下，这才挂断电话。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
