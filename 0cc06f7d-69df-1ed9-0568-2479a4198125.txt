夏夕绾冲了好久的澡，等身体的燥热全部退去了，她才开门出来。
<br/>
<br/>        房间里，陆寒霆已经在隔壁冲好澡了，身体穿了一件黑色丝绸的睡衣，利落的短发沾着湿漉漉的水雾，年轻英俊矜贵。
<br/>
<br/>        现在他坐在沙发上，两条大长腿优雅的叠加在一起，正垂眸看着手里的一份文件，他修长的两指里夹了一根香烟正蹙眉抽着。
<br/>
<br/>        夏夕绾还是第一次见他办公的时候，他边抽烟边看文件的样子那股上位者的冷厉气场很重。
<br/>
<br/>        这时陆寒霆抬眸，在烟雾缭绕后将目光落在了她的身上，“肚子饿么，过来吃点东西。”
<br/>
<br/>        他指了指身边的位置。
<br/>
<br/>        夏夕绾见茶几上已经放了一碗小米粥，还有几样精致的点心和爽口的小菜，很明显都是刚准备的。
<br/>
<br/>        她一直在枢密研究院里工作，就在酒吧里喝了一杯酒，还没有吃晚饭，现在肚子真的饿了。
<br/>
<br/>        没想到他早已经考虑到了这些，还为她准备了可口的晚餐。
<br/>
<br/>        夏夕绾走了过去，坐在他的身边，“陆先生，你不吃么？”
<br/>
<br/>        陆寒霆在烟灰缸里叩了叩烟灰，“我吃过了。”
<br/>
<br/>        “哦。”
<br/>
<br/>        夏夕绾不打扰他工作了，开始吃晚饭。
<br/>
<br/>        很快她就发现烟灰缸里落了一根又一根的烟蒂，身边的男人抽烟抽得很急，有些酗烟。
<br/>
<br/>        以前他也抽过，不过不像今晚这么酗。
<br/>
<br/>        陆寒霆觉得身体有点不适，算算时间，他已经有很久没有发病了，这些天她一直住在叶翎的醉玉欢里，还在枢密研究院里工作，他连见她的次数都屈指可数，更别说抱着她睡觉了。
<br/>
<br/>        她走了多少天，他就有多少天没有入眠了。
<br/>
<br/>        那种一直潜藏压抑在身体里的危险因子开始复苏和觉醒，指间落下的猩红烟灰都带了一层阴郁的戾气。
<br/>
<br/>        但是下一秒他指尖里的香烟突然被抽走了，一只小手伸了过来，把一样东西塞到了他的嘴巴里。
<br/>
<br/>        陆寒霆嚼了一下，是一个干奶块。
<br/>
<br/>        干奶块被咬干，那股香甜的奶味都蔓延进了口腔里。
<br/>
<br/>        陆寒霆侧眸，看着身边的女孩。
<br/>
<br/>        夏夕绾也仰着绝色的小脸望着他，她腻白的肌肤刚沐浴过，如羊脂般明媚剔透，“陆先生，不要抽烟了，抽烟有害健康，干奶块不好吃么？”
<br/>
<br/>        陆寒霆滚了滚喉结，大手直接扣上了她的后脑勺，垂眸就吻上了她的红唇。
<br/>
<br/>        她身上那股清甜的少女体香一直都是他所喜欢的，迷恋的，他的睡眠障碍没有转好的迹象，只能在她身边才能平息入眠。
<br/>
<br/>        他还不知道她身上究竟有什么魔力，但她离开醉玉欢的时候，他也在心里算了日子，看自己可以离开她多久。
<br/>
<br/>        很不幸，前后不超过七天的时间，他又要发病了。
<br/>
<br/>        陆寒霆吻着她，她的唇瓣柔嫩芬芳，他一口咬了上去。
<br/>
<br/>        嘶。
<br/>
<br/>        夏夕绾吃痛，两只小手拽紧了他胸前的睡衣，但是没有将他给推开。
<br/>
<br/>        如果她猜的没错，他又发病了。
<br/>
<br/>        距离上一次发病已经很久了，久到她竟然开始忽略了他的病情。
<br/>
<br/>        这种病一旦发作，他体内的狰狞暴戾性格就要跳出来了，他会格外的痛苦。
<br/>
<br/>        夏夕绾不敢将他推开，如果他觉得这样好受一点的话，那她配合。
<br/>
<br/>        她抬起小手，主动的去搂抱他的脖子。
<br/>
<br/>        陆寒霆察觉到了怀里女孩的乖软，腥甜的血液蔓延进口腔，这股腥气令他眼角一红，当即用力的去允她被咬破的伤口。
<br/>
<br/>        吸她的血。
