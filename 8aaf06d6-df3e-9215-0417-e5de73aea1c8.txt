那个人吓得往地上一瘫，谁也没有想到好好的一个寿宴竟然搞出人命来了出事了，夏…夏夕绾死了！
<br/>
<br/>        什么？
<br/>
<br/>        大家已经都围了过来，他们看着躺在地面上的夏夕绾，女孩闭着眼，脸色苍白的毫无血色，也毫无生机。
<br/>
<br/>        她…她真的死了？
<br/>
<br/>        今天这个party夏夕绾可以说是主角了，她先是得到老太君的宠爱，后又被陆家两兄弟抢夺，刚才在舞池里她惊鸿一舞，更迷倒了多少双眼睛，就是这样一条鲜活而璀璨的生命，现在却死气沉沉的躺在了地面上。
<br/>
<br/>        大家唏嘘不已。
<br/>
<br/>        陆寒霆没什么表情，短发湿漉漉的往下滴着水，那些冰冷的水珠顺着他的俊颊往下不停的滴落，他抿着泛白的唇线开始对夏夕绾做心肺复苏。
<br/>
<br/>        两只大手叠加了放在夏夕绾的心口上，他一下接一下的往下按压，他垂着的俊美眼睑十分专注，好像世界的纷纷扰扰都已经退去，他就是这样专业而机械的给夏夕绾做着心肺复苏。
<br/>
<br/>        如果夏夕绾没有醒的话，他会一直这样做下去，直到永远。
<br/>
<br/>        很快，几分钟都过去了，夏夕绾毫无反应。
<br/>
<br/>        有人上前，忍不住的劝道，“陆…陆总，我看夏小姐已经死了，不要再做这些无用功了，夏小姐不会醒了。”
<br/>
<br/>        陆寒霆一把将那个人推开，他狭长的眼眶里布满了可怕的红血丝，就连那深邃的瞳仁都变成了血红色，他目光狠戾而狰狞的盯着那个人，“滚！她没有死！她不会死的！”
<br/>
<br/>        所有人都被这样的陆寒霆给吓到了，现在的陆寒霆，睚眦欲裂。
<br/>
<br/>        陆寒霆没有放弃，他伸手捏开了夏夕绾的嘴巴，然后将自己温凉的薄唇覆上去，给她做人工呼吸。
<br/>
<br/>        “夏夕绾，快点把眼睛睁开，以前你不肯张开嘴巴给我吻，还咬我，现在怎么样了，我可以尽情欺负你了！“
<br/>
<br/>        陆寒霆捏开她的小嘴，来了一个深吻，她嘴巴里像吃了什么蜜糖，香香甜甜的，让人一吻就上瘾。
<br/>
<br/>        陆寒霆狠狠的吻了她一口，继续给她做人工复苏，他现在这个样子阴鹜又偏执，自顾自的跟夏夕绾说话，更像是魔怔了。
<br/>
<br/>        “夏夕绾，你不要再装了，这一次你又想跟我玩什么把戏，你就喜欢惹事，整个人锋芒毕露，璀璨又耀眼，这样的你让多少人眼红嫉妒，你却一点都不知道收敛！”
<br/>
<br/>        “你在我面前永远凶巴巴的，但是在别人面前跟个弱鸡似的，上一次被几个男人给绑走了，这一次又是溺水，如果你不行的话就明说，你可以躲到我的身后，我可以给你遮风挡雨！”
<br/>
<br/>        “夏夕绾，快点醒一醒，你别想摆脱我，我说过我一定要得到你的，如果你真的死了，那我就建一座冰窖，把你藏在水晶棺材里，这样你就永远是我的！。”
<br/>
<br/>        大家听得毛骨悚然，他们都知道了，陆寒霆这是爱上夏夕绾了，他这是爱惨了夏夕绾！
<br/>
<br/>        夏夕绾做了一个好长的梦，她一个人在大雪里迷了路，浑身好冷啊。
