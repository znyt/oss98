她空洞的澄眸在这么一瞬间有了聚焦，刚才黑暗的世界迅速恢复了光明。
<br/>
<br/>        夏夕绾低眸看，手里的那根烟花滋滋的绽放，五彩六色的花火像是璀璨的流星雨，闪耀了她整个世界。
<br/>
<br/>        可是她没有心情看烟花，而是站在原地四处眺望了一下，刚才，她感觉…陆寒霆来了，他就在她的身边！
<br/>
<br/>        这个烟花也是他给的。
<br/>
<br/>        但是看了一圈，她并没有看到那道高大英挺的身躯。
<br/>
<br/>        这时一个小朋友将一根烟花递了过来，“大姐姐，你还要玩吗？”夏夕绾的心直接坠了下去，哦，原来他没有来。
<br/>
<br/>        她还在期待什么？
<br/>
<br/>        夏夕绾觉得自己真可笑。
<br/>
<br/>        ……
<br/>
<br/>        今晚夏夕绾没有回醉玉欢，而是找了一家酒店开了一个房间，走了一天的路，她先泡了一个牛奶玫瑰花瓣澡。
<br/>
<br/>        飞机票是明天下午的，今晚是她在海城的最后一晚。
<br/>
<br/>        泡过澡夏夕绾拿出手机，给双双发了一条微信双双，你现在能不能将那株曼珠罗花送给我，我在天州一号酒店。
<br/>
<br/>        双双知道她要离开这里抱着她哭了很久，还扬言要去帝都找她玩，她的微信刚发过去，双双的回信就来了没有问题，绾绾，等我。
<br/>
<br/>        夏夕绾将潮湿的头发吹干，很快“叮铃”的声音就响起了，有人来了。
<br/>
<br/>        双双速度好快啊。
<br/>
<br/>        夏夕绾走过去打开了房间门，“双双，你怎么这么…快…陆子羡，你怎么来了？”
<br/>
<br/>        门外不是双双，而是陆子羡。
<br/>
<br/>        陆子羡手里拎着一个袋子，清寒的黑眸落在她的小脸上看着。
<br/>
<br/>        夏夕绾这才意识到自己洗过澡没有戴面纱，她巴掌大的小脸被泡澡的热气蒸蕴出了健康色泽的粉红，绝丽的五官跟她那双澄眸一样纤尘，小小的樱桃红唇娇艳欲滴，身上是酒店的白色浴袍，沾着湿漉漉香气的女孩顾盼流转之间皆是动人勾魂的神色。
<br/>
<br/>        陆子羡是第二次看她的真容，他伸手，指腹抚上了她的小脸。
<br/>
<br/>        陆寒霆就站在不远处看着，他看着门里门外的那对俊男美女，陆子羡动情的去抚她的小脸，夏夕绾吓了一跳，一双澄眸如小鹿乱撞般闪躲开，直接想关门。
<br/>
<br/>        陆寒霆就在想，昨天那个小女佣也有过小鹿乱撞的神色，但她就不是夏夕绾，不会让他有一种想将人丢在柔软的大床上狠狠咬上一口的冲动。
<br/>
<br/>        这时陆子羡的手抵在了门板上，霸道的抵着，不让关。
<br/>
<br/>        夏夕绾抬眸之间有些错愕，这时陆子羡俊美颀长的身躯已经进了房间，将房门给关上了。
<br/>
<br/>        陆寒霆眼睁睁的看着那扇房间门在他眼前关上了，他看到的最后一幕是走进去的陆子羡用背关上门，夏夕绾向后退了一步，黑漉漉的水眸在昏黄的灯光里看着陆子羡，画面十分引人遐想。
<br/>
<br/>        “总裁，这是你的房卡。”这时严毅走过来将房卡递过去。
<br/>
<br/>        陆寒霆没有伸手接，他只是掀了掀薄唇，嗓音低哑森然，“你说，他们两个人会在房间里干什么？”
