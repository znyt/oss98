叶翎拿了2，众人已经沸腾了，都等着看好戏。
<br/>
<br/>        要知道叶翎是海城第一美人，从小就在聚光里长大，现在又是炙手可热的四小花旦，不知道被多少男人惦记着，她来参加真心话大冒险本来就很有意思。
<br/>
<br/>        这个公子哥一直就喜欢叶翎，他换女朋友的速度就像是换衣服，不过每一任女朋友都是照着叶翎的影子去找的。
<br/>
<br/>        现在他迫不及待的问叶翎第一次还在不在？
<br/>
<br/>        “孙少，虽然说这是真心话大冒险，但是你这个问题也太毒了吧？”有人笑着起哄。
<br/>
<br/>        公子哥孙少贪婪的看着对面叶翎那明媚动人的小脸，“叶美人，刚才我们可是有言在先要遵循游戏规则的，所以这个真心话你一定要回答哦。”
<br/>
<br/>        “孙少，你实在太坏了哈哈哈。”
<br/>
<br/>        牌桌上的男人都在起哄，都在等着叶翎的回答。
<br/>
<br/>        夏妍妍和霍璇看着叶翎被这些男人给包围了，这些男人其实都好奇叶翎的第一次还在不在，只不过孙少问了出来。
<br/>
<br/>        虽然夏妍妍和霍璇很讨厌叶翎总是被这样的簇拥包围，但是关于叶翎的第一次…她们还是很期待叶翎自己说出来的。
<br/>
<br/>        叶翎没有第一次了。
<br/>
<br/>        第一次给了顾夜瑾！
<br/>
<br/>        她们倒是想看一看这对名义上的兄妹撕碎了所有的伪装，如何应对世俗的目光和流言蜚语。
<br/>
<br/>        夏妍妍抬眸看向一边的顾夜瑾，顾夜瑾俊美的面上并没有什么情绪波澜，他只是从香烟包里抽出了一根香烟叼在薄唇上，然后用打火机点燃，烟雾弥漫迅速模糊了他的俊颜。
<br/>
<br/>        他将香烟夹在手指里幽幽的吐出了一口烟雾，那清寒的目光也落在了对面的叶翎身上。
<br/>
<br/>        霍西泽小声道，“夜瑾哥，要不…我出面干预一下？”
<br/>
<br/>        顾夜瑾和叶翎的事情，几个好兄弟还是知道的。
<br/>
<br/>        霍西泽比任何人都清楚顾夜瑾是怎么变态的宠着叶翎这个妹妹，叶翎从小就生的粉雕玉琢，上学了就开始接触男同学，记得有一次同学聚会，一个喝醉的男同学强行抱住了叶翎，顾夜瑾赶去的时候就亲自动了手，差点没将人给弄废了。
<br/>
<br/>        顾夜瑾作为顾家的少主，俊美如玉，斯文清润，很少有动怒的时候，但是叶翎可以让他轻易的失控。
<br/>
<br/>        后来的每一次同学聚会顾夜瑾都会跟去，他一身精良的黑色西装坐在角落里的沙发上，边等叶翎边批阅文件。
<br/>
<br/>        叶翎的同学跟他说话，他会很有耐心的回答，最后会让秘书提前结账，俊美强大的顾夜瑾气场是那些女孩子所招架不住的，大家往往会面红耳赤的偷瞄他。
<br/>
<br/>        后来所有人都知道叶翎有一个哥哥，叫顾夜瑾。
<br/>
<br/>        顾夜瑾宠叶翎，整个四九城都知道。
<br/>
<br/>        “叶翎，大家都在等你回答呢，你究竟还有没有第一次了啊，既然来玩真心话大冒险你不会玩不起吧？”这时霍璇嫉恨的出声道。
<br/>
<br/>        叶翎挑起勾人的水眸，看了霍璇一眼，“不是我玩不起，而是你等不起，好，我现在就回答，我的第一次…”
