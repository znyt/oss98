　　“苏兄，你此去定要万分小心！”韩盈面色凝重的告诫道。
<br/>　　因距离过远，苏莫和聂乾两人，感受不到天冥圣城和九大卫城之内巫族的强大，但韩盈却能靠战魂聆听得到。
<br/>　　苏莫闻言点了点头，道：“聂兄、韩姑娘，你们两人就隐藏在这座青山附近，我去找目标夺舍！”
<br/>　　“好！你小心点！”韩盈两人点了点头。
<br/>　　随即，苏莫身形一展，便向远处飞去。
<br/>　　他离去的方向，正是天冥圣城相反的方向。
<br/>　　夺舍巫族之人非同小可，可不能被巫族强者发现了，距离天冥圣城太近的话，难免不被巫族的武圣强者察觉。
<br/>　　所以，苏莫要尽量远离天冥圣城之后再行动。
<br/>　　嗖！
<br/>　　苏莫身形划破天际，不多时便远离了天冥圣城百万里之遥，这个时候他身形停了下来。
<br/>　　此刻，他深处在一片山脉的上空，准备在此地截杀落单的巫族之人。
<br/>　　随即，苏莫身形降临而下，落在了茂密的山林之中，静静等待了起来。
<br/>　　只要从他方圆三万里之内路过的人，都逃不过他的耳目。
<br/>　　夺舍的目标，也是要仔细选择，首先必须是居住在黑耀城的巫族之人，其次修为不能太高，也不能太低。
<br/>　　他所在的方向，正对着远处的黑耀城，所以，从此地路过的巫族的人，有很大的可能是黑耀城之人。
<br/>　　嗖！嗖！
<br/>　　苏莫刚刚隐藏在山林中片刻，便有两名巫族之人从山脉上空飞过。
<br/>　　不过，他并没有对这两人动手，修为太低了，才武皇境八重修为。
<br/>　　时间缓缓流逝，苏莫在山林中等待了三个时辰，依旧没有出手。
<br/>　　不是他不想出手，实在是没有合适的目标，要么是修为太高的巫族人，他根本不敢动手。
<br/>　　要么是修为太低，不适合夺舍。
<br/>　　偶尔发现合适的目标，还并不是落单之人，而是和一大批巫族之人同行，这更加不能出手了。
<br/>　　苏莫不由的叹了口气，想夺舍都这么难！
<br/>　　嗖！
<br/>　　就在此刻，一道刺耳的破空声，从远处传来。
<br/>　　苏莫急忙抬头看去，便看到万余里外，一名高大的巫族青年急速而来。
<br/>　　这名巫族青年飞行的方向，正是黑耀城。
<br/>　　苏莫顿时眼眸一亮，此人武帝境一重巅峰修为，虽然修为有些低了，但勉强还算合适。
<br/>　　他目光瞬间便锁定了此人，立刻将自身气息收敛到了极致，准备做到一击擒拿。
<br/>　　嗖！
<br/>　　呼吸之间，这名高大的巫族青年飞到了山脉上空，距离苏莫也只有百余里而已。
<br/>　　这名巫族青年警惕心并不高，此地距离天冥圣城和黑耀城不是太远，是巫族核心之地，对于巫族来说安全无比。
<br/>　　“下来！”
<br/>　　苏莫瞬间出手了，大手一探，一只玄力大手，直接向对方凌空抓了过去。
<br/>　　与此同时，吞噬战魂也催动了起来，直接将对方笼罩。
<br/>　　“是谁？”高大的巫族青年顿时大惊失色，不过，面对苏莫的擒拿，他没有半分反抗之力。
<br/>　　吞噬战魂的威力何等之大，连赵城和郭剑青这等武帝境四重的武者都很难反抗，更别说武帝境一重之人了。
<br/>　　虽说吞噬战魂没有全力催动，但威力依旧恐怖无比。
<br/>　　几乎是瞬息之间，这名高大的巫族青年便被苏莫擒拿，拉到了身边。
<br/>　　“低贱的人族，你居然敢偷袭我，真是找死！”高大的巫族青年见到苏莫，顿时厉喝出声，他没想到偷袭他之人，居然是卑微的人族之人。
<br/>　　“死到临头，居然还敢嚣张！”苏莫冷笑一声，手掌直接按在了对方的天灵盖上。
<br/>　　高大的巫族青年顿时全身剧震，面现呆滞之色。
<br/>　　搜魂术！
<br/>　　苏莫动用了搜魂术，在查看此人的记忆，他当然不能直接夺舍，要查看清楚此人的身份，才会进行夺舍。
<br/>　　少倾之后，苏莫松开了手掌，巫族青年的身躯顿时瘫倒在了地上，双眼翻白，面色呆楞。
<br/>　　苏莫叹了口气，此人并不是黑耀城之人，而是另一座卫城黑金城之人。
<br/>　　他若是夺舍了此人，虽然有了巫族的身份，但在黑耀城中还是有些不便。
<br/>　　沉吟了片刻，苏莫放弃了夺舍此人，收取了对方的储物戒，然后掌中火属性玄力吞吐，将对方焚化成了灰烬。
<br/>　　不过，虽然没有夺舍此人，但搜了对方的记忆，苏莫对于天冥星巫族支脉，了解的更加的清楚了。
<br/>　　天冥星巫族之中，最强的人乃是支族的族长，麾下有四大祭司，每一位都是武圣强者。
<br/>　　而从此人的记忆之中，苏莫查探到，巫娴居然是大祭司的曾孙女，身份非常的高。
<br/>　　而自从一年前，苍穹世界试炼结束之后，巫娴被擒拿，巫族天才陨落九百余人，整个天冥星巫族一脉，举族震动。
<br/>　　为了追查苏莫和秦不死的下落，巫族出动了大量的强者，到现在依旧在查探。
<br/>　　而且，由于是柴管事放走了苏莫，天冥星巫族一脉，险些和苍云宗开战。
<br/>　　苍云宗，又被称为苍云圣宗，正是神武学府背后的十三大势力之一。
<br/>　　而柴管事不仅是炼器师公会的炼器师，还是苍云圣宗的长老。
<br/>　　苏莫心中有些感慨，虽然他没有亲历此事，但也能想得到，因为放走了他，柴管事估计也不好过。
<br/>　　还有就是，以巫族的能量，估计要不了多久，巫族就会追查到神武学府。
<br/>　　毕竟，苍云圣宗是神武学府背后的势力之一，柴管事以前又是神武学府的弟子，这根线巫族肯定会追查到。
<br/>　　这些，对于苏莫来说，并没有什么意外，他早已知道巫族不会放过他。
<br/>　　但是，最让他感到愤怒的是，苍穹世界所有的人，因为此事，全部陪葬了。
<br/>　　半年之前，巫族大祭司派出了几位高阶尊者，将苍穹世界这座位面小世界直接打爆了。
<br/>　　苍穹世界彻底毁灭，其内无尽的生灵，全部都灰飞烟灭了。
<br/>　　“真是狠毒！”苏莫双拳紧握，脸色阴沉如水，苍穹世界的人类，何止千亿，居然就这样全部死了。
<br/>　　在巫族之人的眼中，苍穹世界的人，根本就是低贱的土著，命如蝼蚁。
<br/>　　片刻之后，苏莫才压下了心中的情绪波动，这个仇他迟早会报，现在还是先完成眼前的目标再说。
<br/>　　而后，苏莫隐藏身形，继续等待了起来。
