<br/>				青龙嗖的一声回到轩辕墨泽的身体内，伴随着它的消失，空气中的那股强大上古威压也散去。<br/>　　<br/>　　陌尘远远的就看到那半空中的一幕了，见那山峰之巅，凤九和轩辕墨泽两人并立着，他当即御剑上前。<br/>　　<br/>　　当看到不仅是吞云身上有伤，就连凤九身上也有伤时，目光一缩：“是什么人伤的你们？”<br/>　　<br/>　　“回去再说。”凤九的声音一出，轩辕墨泽便搂着她御剑往回而去。<br/>　　<br/>　　陌尘看了眼受伤的吞云，便带了它一程，一同跟在凤九他们的身后一并离开。<br/>　　<br/>　　凤府里，越等越焦急的关习凛几人站在屋顶上往远处看着，他们来回的在屋顶走着，仿佛时间是那样的慢。<br/>　　<br/>　　直到，看到那远处的几抹身影往这边而来时，他喊了一声：“回来了！他们回来了！”<br/>　　<br/>　　轩辕墨泽身上的是黑色衣袍，在夜色中本就不显眼，因此，他身上有没伤旁人看不到，但，凤九的是红色的衣裙，虽说鲜血染上了红裙看不出血的颜色，但那被划开的衣裙上出现的刀剑裂痕却是众人看得见的。<br/>　　<br/>　　“小九受伤了！”<br/>　　<br/>　　关习凛心头一突，看到他们进了院子后连忙从屋顶跃了下来，快步来到他们的面前，见凤九身上大小伤不少，连忙回头唤着：“阿箐，快，帮小九先将伤口处理一下。”<br/>　　<br/>　　叶箐也快步上前，来到凤九的身边扶着她，道：“我先帮你处理下伤口吧！”<br/>　　<br/>　　凤九还没开口，轩辕墨泽便道：“不用，我来就好了。”声音一落，便带着她进了房间。<br/>　　<br/>　　看着两人进了房，关习凛便问着：“陌尘，你可知是什么人对他们出手？怎么小九还会伤成那样？”以她仙帝的实力修为，居然还会被人所伤？难道那人的实力比她还强不成？<br/>　　<br/>　　众人的目光此时也落在陌尘的身上，他们没有去，不知情况，但陌尘多多少少应该知道一点。<br/>　　<br/>　　“我到时那些人已经被墨泽所杀，只知道，那些人的实力很强，其中也有一名是仙帝级别的。”他的声音微顿，略一沉思着：“我猜测，这些人可能不是这上游大陆的人。”<br/>　　<br/>　　“不是上游大陆的人？”众人一怔，有些奇怪：“怎么说？”他们主子也没得罪什么别的地方的人啊！若不是上游大陆的人，又是哪来的人？<br/>　　<br/>　　“这大游大陆虽说占地甚广，但有多少仙帝级别的强者也是为众人所知的，更何况，今晚袭击他们的人除了十几名仙圣级别的修士之外，还有一名仙帝，可见这股对他们出手的势力是极为强大的。”<br/>　　<br/>　　听到这话，众人倒抽了口气：“十几名仙圣级别的修士？”<br/>　　<br/>　　有一名仙帝就已经很让人不可思议了，可竟然还有十几名仙圣？<br/>　　<br/>　　要知道，这仙圣可不是大白菜啊！这上游大陆有多少仙圣强者也是屈指可数，难怪，难怪他说这些人极有可能不是这上游大陆的人，除去他们主子手底下的强者之外，这整个上游大陆的人找遍，估计也就十几个仙圣。<br/>
