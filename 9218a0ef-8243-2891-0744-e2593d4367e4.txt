　　“祝英！”看到这名俊朗的银甲青年，任云都顿时双眸微眯。
<br/>　　此人乃是祝家天才人物，自幼随军征战，战功赫赫，年纪轻轻便已经成为了‘尉将’，比他高一个级别。
<br/>　　所谓的尉将，全称为尉将军，是将军的一种，但级别并不是太高。
<br/>　　“原来是任副尉，不知你们所来何事？”祝英目光落在任云都的身上，微笑着问道。
<br/>　　显然，祝英认识任云都。
<br/>　　“我等奉八皇子之命，来此执行任务，这是兵府的文书！”任云都说道，随即他手掌一番，手中出现了一块兽皮文书。
<br/>　　祝英也没有检查文书，点了点头，道：“将军有令，命你们去西方三十余万里外的环山矿脉镇守，协助那里的守卫，保护矿脉的安全！”
<br/>　　“什么？”任云都闻言脸色一沉，他们都是神武学府的天才，个个修为高达武帝之境。
<br/>　　尤其是他，修为高达武帝境九重巅峰，战力直追一般的尊者。
<br/>　　而祝隆将军，居然派他们去镇守小小的矿脉。
<br/>　　镇守矿脉，不单单是镇守那么简单，还是苦力，需要开采矿脉。
<br/>　　这祝隆，是把他们当苦力使了吗？
<br/>　　其他神武学府的弟子，个个脸色一沉，他们是来参战的。
<br/>　　若是有战事，他们才会出手，没有战事的话，平常就安心修炼。
<br/>　　现在倒好，居然被安排成了苦力。
<br/>　　“怎么？你敢违抗军令不成？”祝英面色一冷，冷漠的问道。
<br/>　　“不敢！”任云都虽然心中愤怒，但也不敢公然违抗军令，因为他们既然被兵府派到了此地，就归属祝隆将军统领。
<br/>　　“不敢最好，这是将军的口谕，你们去吧！”祝英点了点头，随即手臂一抛，一块玉符便从城楼上飞射而下。
<br/>　　任云都接过玉符，稍一查看，其内确实是祝隆将军留下的话语，命他率领众人去镇守环山矿脉。
<br/>　　“我们走！”任云都没有多言，招呼苏莫等人一声，便飞身而起，向西方飞了过去，很快便消失在了天际。
<br/>　　“尉将军，这些人，难道就这么让他们去镇守环山矿脉？”一名高大的黑甲士兵，疑惑的向祝英问道。
<br/>　　“当然不会！”
<br/>　　祝英摇了摇头，面带冷笑道：“这些人可都是八皇子古天意的爪牙，你可知八皇子这些年，为何经常派人进入军中参战？”
<br/>　　“是为了培养自己的势力吧？”黑甲士兵说道。
<br/>　　“不错！这些人以后离开了神武学府，都会进入军中，身居要职，从而壮大八皇子的势力！”
<br/>　　祝英徐徐说道，言罢，他眸光一冷，继续道：“不过，八皇子与三皇子和大皇子相比，还是太弱了，他们岂会纵容八皇子壮大！”
<br/>　　“原来如此！”黑甲士兵恍然，随即面上也露出了一丝笑意，看来三皇子殿下，是要对八皇子动手了。
<br/>　　而首要的任务，便是从斩掉八皇子的爪牙开始。
<br/>　　……
<br/>　　环山矿脉，距离黑山要塞不远，处在黑色的山脉深处，因为处于几座大山的中央，故名环山矿脉。
<br/>　　没过多久的时间，苏莫等人便来到了环山矿脉。
<br/>　　从高空望去，只见幽深的山谷中没有半个人影，仿佛是一片荒芜之地。
<br/>　　但众人都能感受到一股股强大的气息，隐藏在山谷之中。
<br/>　　“我们下去！”任云都吩咐一声，随即率领众人降落而下，落在了山谷之中。
<br/>　　“什么人？”
<br/>　　众人乍一落进山谷，顿时响起一声暴喝，森然的杀机铺天盖地。
<br/>　　嗖嗖嗖！！
<br/>　　随即，从很多山洞之中，立刻飞掠出数百名黑甲士兵，将众人团团包围了起来。
<br/>　　这些士兵大多都是武王境和武皇境修为，武帝境修为的不足五十人。
<br/>　　而为首的一人是一名青瘦的中年将军，身穿古铜色战甲，身上弥漫着尊者的气息。
<br/>　　“我等奉祝隆将军之令，前来协助镇守环山矿脉！”任云都向中年将军抱拳说道。
<br/>　　“可有手谕！”中年将军打量了众人一番，冷冷的问道。
<br/>　　任云都闻言，立刻将祝英给予的玉符，递给了中年将军。
<br/>　　中年将军查看了一番玉符，点了点头，随即向任云都抱拳，沉声道：“原来是任副尉，失礼了！”
<br/>　　中年将军虽然是武尊境武者，但也是副尉之职，和任云都平级。
<br/>　　不过，中年将军虽然是在道歉，但面上并无丝毫歉意。
<br/>　　“客气了！”任云都同样向对方抱了抱拳。
<br/>　　“你等既然来了环山矿脉，以后就要听本将之命，可有异议？”中年将军问道。
<br/>　　“没有！”众人摇头。
<br/>　　“那好，你们先在山谷中各自开凿栖身之所，然后负责此地的警戒和守卫任务，一月一轮换，下月开始开采矿脉！”中年将军吩咐道。
<br/>　　“是！”任云都无奈的点了点头，既然来了此地，他们也只能奉命行事。
<br/>　　只不过，这次历练任务，怕是得不到什么好处了。
<br/>　　好在这次的历练时间不长，他向兵府申请的时限是半年，半年一过立马打道回府。
<br/>　　随后，众人纷纷行动了起来，在山谷中开凿出大量的山洞用于栖身。
<br/>　　接下来的时间中，众人就真的变成了守卫，每日镇守在山谷中。
<br/>　　不仅如此，众人还被分成了十个小分队，经常在山谷附近巡视。
<br/>　　这环山矿脉，就在山谷的大地之下，连接着大地龙脉，出产大量的极品灵石，其中偶尔还能诞生出下品灵晶，非同小可。
<br/>　　所以，此地遭到陀罗古国和血瞳族之人的窥视，时常过来袭扰，不得不防。
<br/>　　时间飞快的流逝，眨眼间变过去了一个月，这一个月间相安无事。
<br/>　　而苏莫，在这一个月的时间中，也非常顺利的突破到了武帝境五重的境界。
<br/>　　沉淀够了，境界瓶颈便自然而然的松动了，突破境界轻而易举，远比他强行破镜，要来得更好。
<br/>　　这一日，苏莫等人全部被中年将军召集了起来。
<br/>　　“接下来，你等下矿开采灵石一个月，开采灵石期间，会有人监督，任何人若是敢中饱私囊，杀无赦！”中年将军对众人吩咐道，声音冷漠至极。
