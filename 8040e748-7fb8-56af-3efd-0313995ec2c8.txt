　　“啊！”
<br/>　　苏莫口中痛呼一声，他只感觉双眸一阵火热之感，仿佛眼眶里被灌进了开水。
<br/>　　不过，好在他感觉这灼热之感并没有破坏他的眼瞳，而是在改变着他的眼瞳。
<br/>　　这个过程时间不短，足足过去了半刻钟，苏莫双眸的灼热之感才慢慢的消失。
<br/>　　但是，他却有些犯愁了！
<br/>　　因为他现在，依旧是一双血红色的眼眸，眸中血光爆射，仿佛是择人而噬的凶兽，极为吓人，这幅模样未免太怪异了！
<br/>　　“这血脉之力，应该能隐藏吧！”苏莫呢喃自语，随即沟通眼瞳中的血脉之力，慢慢加以压制。
<br/>　　片刻之后，他终于将血脉之力压制在了眼瞳深处，眸中恢复了清明。
<br/>　　“不知血瞳族之人，是如何施展血脉瞳术？”苏莫面现思索之色，自从见识过了血瞳族的瞳术神通，他便也很是心动。
<br/>　　这种瞳术非常不凡，可以说一道目光就能杀人，而且还非常的强大。
<br/>　　这种血脉瞳术，应该是血脉眼瞳，连接体内血气和玄力，凝聚而成的攻击。
<br/>　　但是，没有具体的方法，苏莫就算知道原理，也是施展不出来。
<br/>　　随后，苏莫将斩杀血瞳族之人得到的储物解查看了一番，并没有在其内找到血脉瞳术的施展之法。
<br/>　　“算了，以后若是有机会，搜魂一名血瞳族之人！”良久之后，苏莫叹了口气。
<br/>　　接下来的时间中，他便安稳的住在谢府，偶尔和谢彬谈论武道，平静的度过了半个月的时间。
<br/>　　这半个月的时间，他主要是疗伤，上次被血瞳族老者的血脉神剑击伤，他伤势一直没有恢复。
<br/>　　清晨，朝阳初升，将云雾映成了橘黄色，霎是美丽。
<br/>　　谢彬和苏莫联袂走出了谢府，今日的苏莫，穿了一袭蓝衣，显得年轻而有朝气。
<br/>　　“苏兄，我见你气色不错，伤势恢复的怎么样了？”谢彬边走边问道。
<br/>　　“差不多痊愈了！”苏莫微笑着说道。
<br/>　　“那就好，今日就是拍卖会的日子，希望我们都能淘到好东西！”谢彬哈哈大笑道。
<br/>　　“希望吧！”苏莫点了点头，两人边走边聊。
<br/>　　“苏兄，我这几年攒了大量的财富，若是等会你手头拮据，尽管和我说便是！”谢彬豪爽的说道。
<br/>　　“没问题！”苏莫笑道，不过，他不相信对方的财富，能比的上他。
<br/>　　两人一路闲聊，没过多久，便来到了两座城中之城的前方。
<br/>　　其中一座城中之城，门前聚集了大量的武帝境武者，均是在排队进城。
<br/>　　“还有两座内城？”苏莫停下了脚步，面带疑惑的问道。
<br/>　　“哈哈！苏兄，这并不是内城，而是拍卖会场！”
<br/>　　谢彬闻言大笑，解释道：“这两座会场一座是武帝境的拍卖会场，一座是武尊境拍卖会场！”
<br/>　　“哦？”苏莫闻言惊讶，随即灵识一探，顿时恍然。
<br/>　　这的确是两座拍卖会场，但却是露天的拍卖会场，建造的如同武斗场一般。
<br/>　　这种拍卖会场，并没有什么贵宾包间，只有普通的坐席。
<br/>　　但这种拍卖会场有一个优点，那就是大，巨大无比，每一个会场都占地方圆百里，能容纳的人数，简直是恐怖。
<br/>　　此刻，其中一座拍卖会场已经人山人海，全部都是武帝境的武者，另一座武尊级的拍卖会场，人数则是少的多，只有万余人的样子。
<br/>　　不过，苏莫发现，他的灵识只能在会场上方看到其内的情景，却听不到半点声音。
<br/>　　而且，一股无形的力量，在阻挡他的灵识窥探，看来整个会场被强大的阵法包裹了。
<br/>　　“这么大规模的拍卖会，如此多的人争夺，并不是好事！“苏莫无奈的摇了摇头。
<br/>　　“并不能这样说，参加拍卖会的人虽多，但拍卖的宝物同样是多不胜数！”
<br/>　　谢冰摇了摇头，继续道：“我们大荣城的拍卖会，每次拍卖最少都要历时三日，而今年真圣界要开启了，是十年一次的盛会，最少都要历时七、八日！”
<br/>　　“恐怖！”苏莫闻言咂舌，一个拍卖会居然举行七、八日，他还是第一次听闻。
<br/>　　随即，他有些疑惑的问道：“你刚才说的真圣界，那是什么地方？”
<br/>　　“不会吧？苏兄，你连真圣界都不知道？”谢彬闻言一怔，随即一脸夸张的望着苏莫。
<br/>　　真圣界在古灵星河，几乎是无人不知，无人不晓，苏莫居然不知道？
<br/>　　“呃~~的确不知道！”苏莫摸了摸鼻子，苦笑道。
<br/>　　“真圣界是一处小世界，但却不是自然生成的小世界，而是无数岁月之前，我们古灵星河一位远古大能所建的小世界！”谢彬解释道。
<br/>　　“有什么不凡吗？”苏莫问道。
<br/>　　“当然不凡，真圣界有一块真圣碑，只有在真圣碑上留名，才能得到真圣之气，将来晋升武圣境的希望才大！”谢彬解释道。
<br/>　　“真圣之气？这么厉害！”苏莫闻言惊讶，能让人增加晋升武圣的希望，这的确是天大的好处吧。
<br/>　　武圣，这可是数之不尽的武者，都梦寐以求的境界啊！
<br/>　　在古灵星河，武圣境的超级强者非常的少，比如巫族天冥星一脉，只有四位武圣境强者。
<br/>　　而像皇祖圣朝这种大势力，不过是只有两位武圣，可见武圣境是多么让人渴望而不可及的境界。
<br/>　　而这真圣之气，居然能增加晋升武圣的希望，可见其多么的珍贵！
<br/>　　苏莫眼眸闪亮，这真圣界，说不得他也要去一趟了。
<br/>　　“对，就是真圣之气，非常玄妙，就算是真正的武圣，都没有这种真圣之气！”谢彬满脸唏嘘的说道，他对于真圣之气，是做梦都想得到，但是以他的天赋，根本不可能得到。
<br/>　　没有真圣之气，将来想要晋升武圣之境，对于他来说根本不可能！
<br/>　　虽然他的天赋很好，但修为到了武帝境之后，修炼速度已经大降，将来到了武尊境之后，提升的会更慢，甚至停懈不前。
<br/>　　武尊境修为以后，再想要提升修为，已经不单单资源的问题了，需要超强的天赋、强大的气运，以及莫大的机遇。
<br/>　　而他，若是能得到真圣之气，天赋就能够加强，将来才有一丝晋升武圣的希望。
<br/>　　“此事，我们回去之后再详谈！”苏莫沉声说道，他对于这种真圣之气，也是非常的感兴趣。
<br/>　　“好！”谢彬点了点头。
<br/>　　随即，两人便准备向拍卖场的大门走去。
<br/>　　“连真圣界和真圣之气都不知道，你可真是土狍子！”
<br/>　　就在此时，一名黄衣青年从苏莫身边走过，口中发出一声嗤笑。
