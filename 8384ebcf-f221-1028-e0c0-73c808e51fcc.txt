　　这一日，从终极神王殿，传出了一个消息。
<br/>　　这个消息的内容便是，秦陨的爱人，已经被终极神王找到，会在一年之内彻底灭杀。
<br/>　　若是想让此女活命，唯有秦陨亲来终极神王殿，归还终极神剑和终极剑道。
<br/>　　当然，若是秦陨归还终极神剑和终极剑道，那是对终极神王殿有大恩，终极神王殿会有丰厚的报酬。
<br/>　　至于秦陨的爱人，是不是和紫霄有关系，终极神王殿并未告知。
<br/>　　此消息一出，立刻引起了神域无数人的议论，终极神王这是要靠秦陨的爱人，吸引秦陨现身吗？
<br/>　　还是说，终极神王要以此，来瞥开终极神王殿和秦陨的关系？
<br/>　　与此同时，地藏神王、雷火神王、赤云神王，以及天尘宇，又再次返回了终极神王殿。
<br/>　　但是，四人具体和终极神王谈论了什么，就不得而知了。
<br/>　　……
<br/>　　“终极神王！”
<br/>　　密室之中，苏莫睁开了眼眸，眸中闪过犀利的光芒。
<br/>　　他通过分身，第一时间便得知了这个消息，分身体内，有他一部分神魂，两人的心神感应是极为强烈的。
<br/>　　苏莫微眯着眼眸，这一刻，他想立刻前去终极神王殿。
<br/>　　但是，他并没有因为急躁而丧失冷静，此事诡异。
<br/>　　有可能是终极神王殿，想要故意引他现身。
<br/>　　但是，万一此事是真的呢？
<br/>　　芊寻月在紫霄的身上，真的很有可能被终极神王得到，万一此事为真，他不去相救的话，真的就完了。
<br/>　　“紫霄！”
<br/>　　苏莫双拳紧握了起来，心中的杀机无比的浓郁，若不是因为此人，事情也不会走到这一步，也不会让他如此步步艰难。
<br/>　　他不能现身，一旦现身就完了，因为，他秦陨这个身份，肯定被帝一魂知晓了。
<br/>　　一旦现身，必死无疑。
<br/>　　“还有十个月的时间！”苏莫深深的吸了口气，从这个消息传出，到被他的分身得知，过去了两个月的时间。
<br/>　　“终极神王殿，别怪我无情了！”
<br/>　　苏莫面上，厉芒闪动，立刻离开了密室，动身前往轮回神朝。
<br/>　　与此同时，他也命令分身，前去轮回神朝。
<br/>　　月余之后，苏莫进入了轮回神朝境内，并且，来到了距离人王宫，不是太远的地方。
<br/>　　他立刻给予清雅传讯，告知了接下来的计划，让其与人德神王安排。
<br/>　　苏莫躲藏在人王城之外，一片密林之中，遥遥的注视着人王城的东西。
<br/>　　他不敢进入人王宫，毕竟，人王城和人王宫之中，不知道有多少天王宫的探子。
<br/>　　在他传递完消息之时，分身也进入了轮回神朝境内，向天王宫和地王宫而去。
<br/>　　苏莫静静的等待，约莫大半个时辰之后，在他的视线之中，主要到人王城之中，有五位强者火速离开。
<br/>　　这五人，为首之中，正是人德宫主，另一位，则是其子玄天神王。
<br/>　　至于另外三位，其中一位是佝偻着身子的老妪。
<br/>　　不过，这位老妪虽然年迈，并且骨瘦如柴，但是身上的气息，并不比玄天神王差。
<br/>　　显然，这位老妪，也是一位中级神王。
<br/>　　另外两位，则都是中年男子，一位高大壮硕，一位高高瘦瘦，两人的气息不如老妪，显然实力稍弱。
<br/>　　五位神王级别的强者，离开了人王城，身形刺破天穹，如同雷电划破长空，直接向着终极剑域的方向而去。
<br/>　　苏莫见此，少倾之后，便也跟了上去，只不过，他的速度远远比不上五位神王。
<br/>　　时间流逝，大半个月之后，苏莫进入了天池神朝境内，他再次变换了气息和容貌。
<br/>　　在这大半个月之中，天池神朝算是颇为平静，但是轮回神朝却是发生了惊天动地的大事。
<br/>　　因为，天王宫的天尘宇的弟弟天尘山死了，作为顶级妖孽，接近万古天骄的天才，天尘山的死，让得整个天王宫陷入了巨大的悲愤之中。
<br/>　　随后，在一番调查之后，杀死天尘山的凶手，直至地王宫。
<br/>　　原来，在当日，天尘山和天王宫的几名弟子，在天浑虚界所在的八卫城，碰到了一群地王宫的弟子。
<br/>　　一个叫风然的天王宫弟子，言语得罪了地王宫一位叫闻天的弟子，最终导致两方人马大战。
<br/>　　天王宫弟子所在的一方，由于人手较少，总体实力也远远低于地王宫，最终被地王宫之人杀的片甲不留。
<br/>　　天尘山当场惨死，神形俱灭，被轰杀成了渣渣。
<br/>　　不仅如此，天王宫除了天尘山之外，还有四名天才弟子，同样陨落。
<br/>　　此事，在轮回神朝掀起巨大的轰动，天王宫为之震怒，立刻命地王宫交出凶手。
<br/>　　但是，地王宫直接拒绝了，因为，那一场大战，地王宫共有十三人参加，个个都是天才，其中有几人更是顶尖的妖孽。
<br/>　　如此多的天才和妖孽，地王宫自然不可能拱手送出，毕竟，送出就等于将这十几人杀死了。
<br/>　　作为顶级大势力，别说是十几位妖孽天才，就算只有一位天才，地王宫都不可能送出，不然以后还如何立足？
<br/>　　地王宫的态度，彻底的激怒了天王宫。
<br/>　　据悉，现如今，天王宫在紧急回招弟子和强者，并且给地王宫下了最后的通牒，十日之内，地王宫若是不交出那十几名凶手，那天王宫将打开杀戒。
<br/>　　至此，整个轮回神朝，气氛变得非常紧张了起来。
<br/>　　因为帝一魂和拓跋青云，地王宫和天王宫，一直在明争暗斗。
<br/>　　难道，现如今，两宫之间要提前开战了吗？
<br/>　　整个轮回神朝，都变的沉重了起来，帝一魂和拓跋青云之间，会提前交手吗？
<br/>　　“还有十天！”
<br/>　　飞驰在苍穹之上，苏莫心中暗道，通过自己的分身，他轻易的便得知了轮回神朝的局势。
<br/>　　这一切都是他安排的，姜枫然和闻人天都、战乱三人，算是做的很好，成功的挑起了两宫的仇恨。
<br/>　　不过，现如今，他也不清楚，姜枫然三人怎么样？
<br/>　　毕竟，三人修为都很低，很容易出事。
<br/>　　但是，三人有目的策划，应该都会顾忌自身的安全，第一时间撤退的。
<br/>　　“天王宫和地王宫交恶，应该不会有人再在外面追查我了吧？天尘宇应该也会立刻回天王宫。”
<br/>　　苏莫心中沉吟，帝一魂这边，他暂时不用担心了。
<br/>　　天王宫和地王宫，就算彻底的开战，也不会很快分出胜负的，毕竟，两宫的实力差不多。
<br/>　　那么，他主要需要对付的，就是终极神王殿了，这还要靠人王宫的力量。
<br/>　　他之所以，让天王宫和地王宫乱起来，就是给人王宫腾出空间。
<br/>　　不然的话，人王宫一旦出手，帝一魂和天王宫一定会插手。
<br/>　　不过，仅凭人王宫之力，对付终极神王殿稍稍有些不足。
<br/>　　虽然人王宫有十大长老，加上人德宫主在内，就是十一位神王，实力远超终极神王殿，但是人王宫没有倾全宫之力，也只是出动了五人而已。
<br/>　　五人，面的终极神王殿，有明显的优势，但是，没有压到性的优势。
<br/>　　苏莫心中不断的思索，主要是，他不了解人德神王和终极神王的实力，所以，心中也没有底。
<br/>　　不过，就算人王宫无法压迫终极神王殿低头，他还有办法。
<br/>　　“不管是真是假，这一次，一定要救出芊寻月！”苏莫咬了咬牙，心中杀机无比的浓郁。
<br/>　　不管终极神王殿传出的消息，是真还是假，对他来说都是一样的。
<br/>　　若是真的，芊寻月就在终极神王的手中，若是假的，那么芊寻月就还在紫霄的手中。
