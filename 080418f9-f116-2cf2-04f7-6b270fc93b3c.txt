　　第486章
<br/>　　有使者飞马将消息递到蛤蟆军大营，再飞马返回黄沙城，来来去去，靠快马商议交接的具体细节。
<br/>　　最后双方敲定，地点选在蛤蟆军大营和黄沙城两地之间，时间是五更时分，双方各派出三十六人，交接林少羽的遗体。
<br/>　　连夜交接，时间这么急，也是出乎程大雷的预料，不过想一想，也能明白其中关键。帝国军已经断粮，再拖下去，怕是会发生大规模哗变。
<br/>　　到五更时分，双方人马都已到了。场上点着篝火，将一片荒地照得亮若白昼。
<br/>　　程大雷骑着黑牛，身后跟着银眸、秦蛮、赵子龙等三十余人。
<br/>　　帝国军一方的代表是宋伯康，他与程大雷算是认识，至于李乐天、崔相、尉迟离等人，都未在今日出现。
<br/>　　“程当家，一向未会，别来无恙。”宋伯康。
<br/>　　“好啦，好啦。”程大雷摆摆手：“一切都谈妥了，咱们快点开始，快点结束，别说这样废话了。”
<br/>　　宋伯康被噎得够呛，不过他因为了解程大雷的为人，倒也不觉得如何。至少，程大雷没有抡着大斧子来砍自己，宋伯康已经觉得很幸运了。
<br/>　　他拍拍手，有手下抬着一具棺材走来，林少羽的遗体都在棺材内，帝国军已经简单的处理过了，至少不能让他尸骨不全。至于林少羽的武器和盔甲，也同样在棺材内。
<br/>　　程大雷命人看过，至于他自己，却是没有勇气再去想起那段不好的记忆。确认帝国军没有耍什么诡计后，程大雷的人将棺材接了过来。
<br/>　　宋伯康立在那里，见蛤蟆军抬着棺材走了，直到他们的背影消失在远处的夜幕中，才长长的松了口气。
<br/>　　他还生怕程大雷设下什么圈套，要将自己一网打尽。
<br/>　　“走啦，走啦。”
<br/>　　宋伯康挥挥手，带领手下麻溜快的返回黄沙城。
<br/>　　程大雷抬着林少羽的棺材返回大营，此刻天近黎明，东方已有些亮光。
<br/>　　等在营地的徐神机等人凑了过来，道：“大当家，少羽的后事怎么办，是安葬在蛤蟆城，还是选择在少羽的祖籍？”
<br/>　　“不用这么麻烦，就在这里埋了。”
<br/>　　“那我们要先准备一下，选一风水宝地，这里什么都缺，有些需要的东西怕是准备不及。”
<br/>　　“说了不用麻烦，随便挖个坑埋掉就可以，人已经死了，没必要整那些有的没的。”
<br/>　　程大雷不是特别在意这些，他择了几个人，自己在营地外找了个地方，命人掘土为坟，将林少羽的棺材埋入其中。程大雷只是在上面浇了一坛酒，连碑都没有立，等着冬去春来，草原上长满野草，这块地方和其他地方其实也没有区别。
<br/>　　程大雷默默在坟前站了一会儿，就回到了营地。路上难免又想起曾经的事，当初林少羽忽然离去，想要找个地方静下心，好好想想。那时程大雷便觉得，如果能有一山清水秀之地终老一生，对林少羽来说，未必不是好事。
<br/>　　然而，他想来想去，想着举起了大旗，最后落得如此下场，终究，还是令程大雷心中有几分不开心的。
<br/>　　营地正在准备撤退事宜，收拾物资，其实也无所谓什么物资，大部分物资都是从帝国军哪里抢来的。
<br/>　　每人随身带上干粮，至于带不走的辎重粮草，全部就地焚烧。
<br/>　　一时火光冲天，黄沙城的坛子一直盯着蛤蟆军的动静，将这一幕飞速报到黄沙城，知道这个消息的李乐天、崔相诸人心底都气得一直骂娘。
<br/>　　帝国军已经断粮了，数万人一天吃不到饭，就会发生许多无法预料的事。在这种情况下，程大雷竟将大家视若性命的粮食一把火烧了。
<br/>　　在天亮时分，蛤蟆军开始全军撤退。三千龙字旗探路，秦蛮的蛮字军断后，开始赶路后，神行技能便开始发动，用了十天时间，大家便从并州赶到京州。
<br/>　　这一日，到了京州的白马坡，程大雷下令大军停下。
<br/>　　“大当家，是有什么事么？”徐神机问：“难道是要趁机打下长安？”
<br/>　　程大雷白了他一眼，不是很愿意搭理他。他从身上逃出一份地图，冲着周围地形比对了一番，最后选择一个点。
<br/>　　“挖！”
<br/>　　全军一起动手，用手中的武器挖掘，一直挖了三四米，在空地上挖出一个大坑，结果还是一无所获。
<br/>　　程大雷有些困惑，难道说选错地方了，还是林少羽临死之前给自己开了一个小小的玩笑，照道理来说，他没有这样的幽默感呐。
<br/>　　“大当家，挖到了，挖到了。”
<br/>　　正这个时候，一个手下突然喊了起来，程大雷忙走过去，见手下手里捧着一尊白玉佛。
<br/>　　玉润润如同姑娘的脸蛋一般，捏一把能捏出水来。蛤蟆城的这些人，程大雷、徐神机、刘悲、关鱼……总的来说，是没见过什么世面的。唯一一个出身不错的林少羽，最近还死了。
<br/>　　他们那里见过这个，一个个瞪得眼睛都直了，不敢相信世间真有如此宝物。
<br/>　　“好啦，好啦。”程大雷摆摆手：“凡挖到宝贝的，都去刘悲处登记，现在所有人都给我小心些，不要在用兵器挖了，都用手刨。”
<br/>　　随着第一件白玉佛出土，接下来挖出的宝物越来越多，金银玉器自不在话下，一米长的珊瑚树，玲珑金塔，翡翠玉像……每一件都价值万金，每一件都巧夺天工。相比之下，第一个挖到的白玉佛，竟也不算什么了。
<br/>　　刘悲将这些宝物登记在册，徐神机在一旁协助，然后再选几个人抵消徐神机带来的阻力。
<br/>　　这颇费了一番时间，一直忙碌了两天，最后程大雷让手下将运输这些宝物的任务，分别交给秦蛮、关鱼等五支队伍。若有遗失损坏，拿他们五人是问，五人在拿下面的营长是问，营长再拿队长……就这样，一直具体到个人。
<br/>　　因为要携带宝物，接下来的行军速度自然被拖慢了，这样是没有办法的事。从京州到凉州，一直走了一个月时间，众人才算是回到了老巢。
<br/>(本章完)
