　　帮不帮到另说。
<br/>　　“为什么对我那么好”涂小安细细想来，落子烟就宛如他背后的女人，一直帮着他，跟一个贤内助似的。
<br/>　　落子烟忽然有点心慌意乱，这种质问，让她不知如何回，柔声的说：“这里不是说话的地方，等去蛇宫了再说行吗”
<br/>　　好吗，行吗，都在征求涂小安的意见，端端小女人的架势。
<br/>　　涂小安有点懵逼，有点不适应。
<br/>　　“蛇宫？”
<br/>　　他只听说过龙王的海底龙宫，没想到蛇族还有蛇宫的存在，这到是有点意思。
<br/>　　“好！”
<br/>　　“等我解决了那只偷袭的鼠辈，我就随你去蛇宫看一看究竟”
<br/>　　堂堂灰妖大王，五大深山妖王，被涂小安一口一个鼠辈的叫。
<br/>　　落子烟松了口气，生怕他没完没了的问。
<br/>　　忽然...
<br/>　　“你是不是暗恋我啊，让我去蛇宫，莫非是想跟我表白吗”涂小安没羞没躁的说了一句，落子烟的态度转变，让他的优越感有点爆棚。
<br/>　　“滚！”
<br/>　　落子烟丢去一个大大的白烟，嗔声了一句，白绫下的容颜却晕红了起来。
<br/>　　几百年了吧，她都不知道自己脸红是一个什么样子。
<br/>　　“你等我，我送你一样东西”
<br/>　　“你要干嘛”她问。
<br/>　　涂小安没说，无视下方两大种族间的生死厮杀，飞身朝着猴族的后方掠去。
<br/>　　之前的那株花妖一直不愿意臣服涂小安，被涂小安的天火慢慢的焚烧着，此时已经彻底的显化原形，变成了一株含苞待放的百合花，在充满血腥的空气中随风摇曳。
<br/>　　其实天火早就被涂小安消散掉了，不然别说百合花了，就算一片花瓣都不会存在。
<br/>　　这花妖被焚烧的不但原形毕露，而且还元气大伤，短期之内是绝对无法化形了。
<br/>　　但最少它还活着。
<br/>　　不得不说这朵花妖原形的百合花是非常的美丽，洁白无暇，阵阵香气，轻淡轻淡，清香宜人。
<br/>　　看上去跟一株的百合花并没有太大的不同，只是更加高雅绝伦了一点。
<br/>　　“蛇妖，你想干嘛，杀人不过头点地，你别欺人太甚”
<br/>　　百花姑娘的声音传了出来，因为他看到涂小安不怀好意的朝着它走去，可它现在的状态糟糕极了，真真就是一朵普通的花朵。
<br/>　　“借你一用”
<br/>　　用？怎么用？她都变成一朵花了，你还想糟蹋我不成？
<br/>　　涂小安嘻嘻一笑，伸出白皙的手，直接将上方绽放的百合花给粗暴的摘了下来，放到鼻尖一闻，心旷神怡，清香宜人。
<br/>　　“蛇妖，要杀就杀，休要辱本姑娘”
<br/>　　这一闻，让百花姑娘感觉自己被侵犯了一样。
<br/>　　“乖，安分点，不然我的手一捏，你就往生极乐了”
<br/>　　如今的花妖可是非常的脆弱，说是奄奄一息也差不多。
<br/>　　花妖被吓住了，还是惜命的。
<br/>　　涂小安捧着美丽的百合花笑嘻嘻的准备回去，可忽然，他脚步一滞，朝着一个方向看去，脸色冰冷了下来：“差点把你这只丑蜈蚣给忘记了”
<br/>　　蜈蚣大妖自从被涂小安的蛇毒笼罩，巨大的体形被侵袭的越来越小，现在早已不见当初的巨大狰狞，只是足足数米的大小了。
<br/>　　虽然一条数米大的蜈蚣同样是吓人的，但是跟之前一比，简直不值一提。
<br/>　　而且由始至终，蜈蚣大妖都无法挣脱那股琉璃般的蛇毒笼罩，体内已经被蛇毒侵蚀的岌岌可危，生命急转直下。
<br/>　　“我认输，饶我一命，我不过是奉了灰妖大王的命令杀你，冤有头债有主，你找它算账就是”
<br/>　　数米大的蜈蚣在地面痛苦的翻滚，度日如年，它强大的生命力迅速的流失。
<br/>　　啪...！
<br/>　　涂小安一脚踩在蜈蚣的身体上，就跟踩一只臭虫一样。
<br/>　　“不好意思，我的心胸不是很大，你既然出来杀我，就要做好被我杀的准备”
<br/>　　蜈蚣大妖闻言顿时慌了：“别杀我，我可以认你为主，只要你放过我”
<br/>　　“认我为主？”涂小安倏然一愣，这好像还是头一次有生物主动说认他为主的。
<br/>　　而且还是深山大妖级别，让他有点措手不及，因为由始至终，涂小安还真没想过收这丑陋的蜈蚣。
<br/>　　其实这蜈蚣大妖的实力是很强的，在妖灵山脉也派的上号，只是他跟涂小安打，根本还没开始，就被蛇毒笼罩。
<br/>　　无法对抗蛇毒的侵袭，也就一身本领无法施展了。
<br/>　　这跟猿猴王的下场差不多，明明有着强大的实力，可愣是没有发挥的余地。
<br/>　　也怪它自己作死，偏偏在涂小安这条证道的蛇王面前用毒。
<br/>　　这不是在关公面前耍大刀吗。
<br/>　　“对，对，我认你为主，以后对你这条蛇唯命是从”蜈蚣大妖感觉自己继续被这股笼罩的蛇毒侵蚀，死亡也不过数个小时的事情了。
<br/>　　救命稻草一定要抓住。
<br/>　　什么尊严，什么自由，在生命的面前，都不值一提，对着这种活了不知道多少年的妖灵来说，一切都太不容易，岂能轻易的死。
<br/>　　世界多美好。
<br/>　　“不好意思，我不喜欢主动送上门倒贴的”涂小安愣了半晌，却摇头拒绝了。
<br/>　　这一拒绝，让被一只脚踩住的蜈蚣一脸的懵逼，感觉天都要塌陷了下来。
<br/>　　它堂堂深山大妖主动臣服，还被拒绝？？？
<br/>　　这什么情况，这条蛇脑子有病？
<br/>　　在之前，它还亲眼看见这条蛇妖没羞没臊的要收服花妖，它难道还不如一朵花？
<br/>　　“你太丑了，不符合我收服战宠的原则”
<br/>　　蜈蚣闻言又扎心了，欺人太甚，倒贴都不要，嫌弃自己主动送上门，嫌弃自己长的丑，这都什么鬼。
<br/>　　你的战宠收服过去是准备开宠物店的？
<br/>　　要知道它可是深山大妖，而且有着几乎伪妖王的战斗力，真正站着妖灵山脉顶峰的大妖，可却在这条蛇的面前，变得一无是处了一样。
<br/>　　涂小安一脸的傲娇，现在此一时彼一时，他可是拥有整个蛇族的男人，就在刚才，就有十几位蛇妖灵跪在自己的面前顶礼膜拜。
<br/>　　他的家底如今可是非常的浑厚，要你一只丑陋的蜈蚣干嘛，相比之下，涂小安更喜欢蜈蚣的本命金丹。
<br/>　　“看你主动犯贱的份上，本蛇王也不在折磨你了，让你死的痛快点，去吧，下辈子投个好胎”
<br/>　　主动臣服，被说成主动犯贱，蜈蚣承受了一万点暴击。
