　　“你师父让我转告你，他飞升去空界了，你将来若是寻他，可去空界找他。”
<br/>　　林三生一怔：“空界？哪个时代的空界？”
<br/>　　“自然是你们那个时代。”徐福有些严肃地说道，“不管你们怎么穿越，你时刻记住自己来自哪个时空，那才是真正属于你的世界。”
<br/>　　“你师父因为这件事协助我，获罪于阴司，已不可能去阴司轮回，东皇钟出，他心愿已了，他本就是阐教宗师，而今空界与尸族大战，他去了自有用处，况且通玄也在空界，你师父要去捉拿他。”
<br/>　　林三生点了点头，拱手道：“谢老祖告知。”
<br/>　　“你而今，过得如何？”徐福又问。
<br/>　　“很好，”林三生笑道，“感谢老祖帮忙，让我找回婉儿。”
<br/>　　徐福道：“可你眉宇之间又一抹愁色，却是为何？”
<br/>　　“这……”林三生耸了耸肩，道：“我不说老祖你也知道。”
<br/>　　徐福沉声说道：“并非我不愿让你带她去见少阳他们，鬼有天知，她若去，轮回逆转，会产生诸多不可确定之灾难。”
<br/>　　林三生点头说道：“这我知道，这也是我自己的决定。”
<br/>　　如果自己带着婉儿回去，再跟叶少阳一起回2016年，那个世界有婉儿转世的金莹，实际上……金莹才是自己真正寻找的爱人，虽然不会见面，但是想到真正的婉儿跟自己在一个世界生活，而自己……却跟另一个婉儿一起生活，这种感觉，会让他产生错乱，这也是他要求徐福带他来到这里隐居的目的。
<br/>　　他虽然不知道这是什么时代，但一定是在自己出生之前，虽然说自己和婉儿的鬼魂，在这个时代也是存在的，但是身份不一样，还没有产生交集，这个时候的婉儿，还可以说不是婉儿。这一想法，让林三生心中的错乱感觉会减少一些。
<br/>　　再说，万一让婉儿到时候知道了真相，她很可能会去找金莹，到时候，这两个不同时空的同一个人见面，会发生什么事情，林三生也不敢想。
<br/>　　因此，他也知道万万不能带婉儿回去。
<br/>　　无论怎么样，这都是自己的选择。林三生低头默然，说道：“不说这个了。老祖，九尾天狐怎么样了？”
<br/>　　“她还在东皇钟里。”
<br/>　　“我知道，我是说……”
<br/>　　林三生话没说完，就看见徐福从衣襟里摸出一口金色的小钟，登时怔住：“东皇钟，变成这么小了？”
<br/>　　“东皇钟是上古神器，上镇山岳，下镇妖邪，自然可大可小。”
<br/>　　“那……小九怎么样了？”
<br/>　　“她用自己妖血开启了东皇钟，帮东皇钟认叶少阳为主，自己也成为这口钟的一部分了。”
<br/>　　林三生傻傻地看着他手中的东皇钟，喃喃道：“真的没救了？”
<br/>　　“魂魄破碎，但神念还在钟里，只是出于混沌状态。”
<br/>　　“可以聚魂啊，找……对了，找道风，建文帝和小马，就是他帮忙聚会重生的！”
<br/>　　徐福摇了摇头。“魂魄碎为精魄，要数百年才能重生，道风那法术，也只是加速了这个过程，同时保留神念而已，九尾天狐一身魂力俱被东皇钟吸收，如何能这么聚得？”
<br/>　　林三生看着他，说道：“不过听你这么说，似乎办法是有的。”
<br/>　　“办法是有，得先找到千年生的太岁，太岁为天地灵气所生，非草非肉，可用来重塑肉身……”
<br/>　　林三生愣了一下道：“那赶紧找啊。”
<br/>　　徐福看了他一眼道：“你以为是好找的？”
<br/>　　“这……”
<br/>　　“太岁之生，为民鱼肉，历朝历代，只有天灾，百姓食不果腹时才会出现，必然为民发现，哪有长生千年的机会？”
<br/>　　林三生知道他还有今后话，于是也不吭声，继续等着。
<br/>　　徐福继续说道：“我最近穿越乾隆年间，在典籍中找到资料……”因为听林三生他们说多了“穿越”这个词，感觉挺贴切的，也就用上了。
<br/>　　林三生忍不住打断他问道：“为什么要到乾隆年间找资料？从四库全书里找？
<br/>　　乾隆名为修书，实为毁书，所修的四库全书，是收集了历朝历代不少经史子集，不过也毁了很多，尤其是篡改了很多明朝的历史。
<br/>　　林三生从古墓里出来，立刻找书了解清史，发现里面记载的很多所谓的明朝前期的历史，简直就是扯淡，作为一个大明朝的忠实粉丝，气的快岔气，在他混到阴司当官之后，还特意找关系查了乾隆死后的情况，得知乾隆因为篡改历史，大兴文字狱，犯下大罪，本来应该下地狱受苦，不过因为在位之初也做了一些好事，于是贬为普通鬼役，终日劳作，不得赦免。
<br/>　　因此当徐福提到乾隆，林三生立刻按捺不住地反问。
<br/>　　徐福道：“你当我是找四库全书？我是去找乾隆毁书之前那些经史子集，从中寻找线索。”
<br/>　　林三生一听就恍然了，乾隆修书，毁了很多书，在他之后再想找就找不到了，但是如果选取的时间太早，就容易错过之后的时间，所以他去了乾隆修书之前那段时间，可能性当然是最大的。
<br/>　　“我在一本书上看到，在北宋初年，有一人在昆仑山之上发现一株太岁，朝廷邀诸多方士前去查验，得知为吸取昆仑山天真地秀长成，已有千年，本为山洞密封所长，不能见风，只因山体运转，这才显露出来，发现之后，不日便枯萎了，宋太祖只能命人摘取，炼制成药丸服用……”
<br/>　　林三生听他说到最后，耸了耸肩说道：“不是宋朝时候就被炼药了吗，那还有什么用？”
<br/>　　说完之后，突然想到什么，一拍脑门，“对啊！你可以直接穿越到宋朝，在太岁被发现之前，把它取走！”
<br/>　　徐福不做声，表示默认。
<br/>　　“徐老祖，你说这……你有山海印在手，这简直就是无视任何规则，历朝历代，你想要什么就能拿到什么啊，这听上去简直……”林三生找不到词语来形容，如果叶少阳等人在，立刻会想到一个词：BUG。
