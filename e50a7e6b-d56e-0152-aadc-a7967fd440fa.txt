　　这一次严君来到九域之中，便是想要收取宝藏，他身边的这两名严家长老，可都算是严家长老之中的强者了。
<br/>　　死在这二人合击之术之下的九劫古帝，都有着不少。
<br/>　　哪怕是严君做梦，都想不到短短的几天时间之内，楚风眠的成长会如此可怕。
<br/>　　旁边的慕家二人，也是脸色苍白无比的盯着楚风眠。
<br/>　　他们本来也觉得，楚风眠是送上门的肥肉，所以才愿意联合严君，一同来对付楚风眠。
<br/>　　可如今，楚风眠哪里是什么肥肉，这是一条真龙，一头雄狮，他们才是任人宰割的肥肉。
<br/>　　“少主，我拖延住此人，你逃出去，然后让我们慕家的强者到来，诛杀此人！”
<br/>　　那慕家老者，看向楚风眠，眼神之中露出几分坚韧之色，向着楚风眠突然出手，一掌轰击而来。
<br/>　　那慕家的慕九，也是毫不犹豫，急忙转头，化为了一道遁光，一溜烟的向着远方逃去，打算逃走。
<br/>　　“想逃？就凭你一个老家伙，也想要拦住我？”
<br/>　　楚风眠看到那慕家的九劫古帝出手，冷笑一声，突然收起剑锋，而是徒手之间，一掌向着那慕家老者轰击而去。
<br/>　　“始祖第二式！”
<br/>　　轰隆隆！
<br/>　　随着楚风眠的血脉越来越精粹，越来越接近始祖天龙，他所施展出的始祖九式，威力也更加的恐怖。
<br/>　　始祖九式，也血脉之力催动，却是精粹，高贵的龙族血脉，这始祖九式的威力也是更加的强横。
<br/>　　哪怕同样是其中的第二式，这一招已经是跟之前楚风眠打出的，截然不同。
<br/>　　一招轰击而去，漫天灵力凝聚，群龙起舞，仿佛天下之间，尽是龙族，如同回到了曾经群龙称霸的时代之中。
<br/>　　而楚风眠，便是这群龙的皇者，群龙的主宰。
<br/>　　群龙之力，皆是凝聚在楚风眠的手心之上，一招轰击而去。
<br/>　　慕家老者掌心之力，被当即打破，同时这群龙之力，更是向着那慕家老者轰击而去。
<br/>　　“八卦神佑！”
<br/>　　慕家老者看到群龙之力的降临，急忙举起了手中的八卦盾牌，将灵力统统注入到了其中。
<br/>　　这八卦盾牌之上，无数纹路浮现而出，化为了一道八卦大阵，守护住他的身躯。
<br/>　　“区区一件灵器，也想要抵挡住我的力量？给我破！”
<br/>　　楚风眠冷喝一声，龙吟咆哮，群龙之力轰击在这八卦大阵之上。
<br/>　　咔嚓一声！
<br/>　　八卦大阵，在这群龙之力的面前，被猛然轰破。
<br/>　　那八卦盾牌，都是被一下子打飞了出去。
<br/>　　旋即被楚风眠出手一爪，这八卦盾牌，就落入到了楚风眠的手心之中。
<br/>　　这八卦盾牌，足以抵挡住楚风眠的剑锋，也是难得一见的宝物，而且楚风眠看的出来，这慕家老者，还无法发挥出这八卦盾牌的全力。
<br/>　　这种宝物，楚风眠当然不会放过，他将这八卦盾牌抓在手心之中，一股灵力注入其中。
<br/>　　顿时这八卦盾牌之中，本来属于这慕家老者的烙印，被当即抹除，变成了属于楚风眠的烙印。
<br/>　　“噗！”
<br/>　　失去了八卦盾牌，这慕家老者根本无法承受群龙之力的轰击，身躯当即是被碾碎。
<br/>　　他的精血，同样是被楚风眠收取到了吞天秘术之中去。
<br/>　　而与此同时，另一边。
<br/>　　那慕家的慕九，催动遁光，已经是快要逃出这一处平原之中。
<br/>　　他刚刚回头，正巧是看到楚风眠一招轰杀了那慕家老者的一幕，让他浑身战栗起来，无比恐惧。
<br/>　　“此人绝对是妖孽，我们阎罗星域，哪怕是在妖孽的天才，也没有此人如此恐怖，此人的身上不知道是有着多少的机缘。”
<br/>　　“我慕家的强者，就快赶来九域了，我要是可以杀了他，得到了这些机缘，阎罗星域之中，谁可以跟我争锋！”
<br/>　　慕九充满恐惧的眼神之中，露出了几分贪婪的兴奋之色。
<br/>　　他只要是能够逃出去，就可以寻找慕家的强者，来围攻楚风眠，杀了楚风眠，楚风眠身上的一切奇遇，都将属于他。
<br/>　　“天龙变！封锁！”
<br/>　　就在这慕九，即将逃出这一片平原的一刻。
<br/>　　突然之间，一声长啸，响彻在这平原之中。
<br/>　　那慕家慕九的身形，突然停止在了空中，他的遁光，就在空中逐渐的消失。
<br/>　　周围的空间之力，凝聚到了极致，化为了层层牢笼，将他封锁在其中，任由是这慕九如何催动遁光，也无法在进一步。
<br/>　　“还想要去寻找家族强者，来对付我？可惜你今日没有这个机会了，给我滚回来！”
<br/>　　突然之间，从慕九的身后，楚风眠的声音响彻而起。
<br/>　　这一道声音，如同死神之音。
<br/>　　与此同时，在这慕九的身边，一道无形大手，笼罩在他的身上，要将他拉扯回去。
<br/>　　“金蝉脱壳！”
<br/>　　慕九将身上的灵力全部爆发而出，从他的身上，浮现出了一道金光，这一道金光，似乎是想要挣脱天龙变的封锁，逃脱升天。
<br/>　　“哼！”
<br/>　　一声冷哼传递而来。
<br/>　　只见那无形大手猛然一捏，一把抓了过去，根根手指，刺入到了慕九身边的金光之中，将这金光一把捏碎。
<br/>　　轰！
<br/>　　慕九的身躯，也被这无形大手捏在了手心之中，旋即下一刻，被直接拉扯了回去，重新回到了平原之中，就在楚风眠的面前，跪了下去。
<br/>　　“楚风眠，楚风眠你不能杀我，你可知道我慕家之中，强者如云，我慕家足以跟圣龙秘境平起平坐，你不过是大罗天星域圣龙秘境的一名弟子，我可是慕家少主，你敢杀了我，圣龙秘境也护不住你！”
<br/>　　看着楚风眠眼神之中的杀意，慕九急忙尖声吼道。
<br/>　　“是吗？慕家，我倒是要看看慕家谁敢来给你报仇，来一个我就杀一个，正好我还需要古帝的精血，来巩固我的境界！”
<br/>　　今日楚风眠已经是杀了一位慕家的九劫古帝了，哪怕是放了慕九，这慕家也不会放过楚风眠。
<br/>　　倒不如将他们一齐杀了，更何况这慕九，还想要妄图去请慕家的强者，来对付楚风眠，这是他自己找死。
<br/>　　“死！”
<br/>　　楚风眠手心一拍，慕九的身躯，当即被碾碎。
