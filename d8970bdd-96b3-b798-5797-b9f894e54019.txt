
<br/>　　<br/>　　    老人死了的话，都讲究小辈和晚辈给养老送终，这是古来的人之常情，也是孝之大道，自然没人养老也没有人给送终的老人，死后就得有些怨念了，同时你给老人安慰地送走了她泉下有知的话，也肯定会保佑你的，要不为啥现在烧纸祭奠的时候，都会念叨几句死者收了钱在阴间买点想买的东西，同时也保佑我怎么怎么地的呢。<br/>　　<br/>　　    “去你们殡仪馆，能把她的骨灰盒取出来吧？”<br/>　　<br/>　　    陈放说道：“可以，这老人存放骨灰盒的钥匙就在我们单位里放着呢，我去了就能给拿出来”<br/>　　<br/>　　    “回去，今晚我就给你操办下”<br/>　　<br/>　　    陈放连忙感激涕零地说道：“那谢谢您了，事后……”<br/>　　<br/>　　    王惊蛰摆了摆手，催促道：“事后给我钱就行了，不用客气”<br/>　　<br/>　　    两人当即就往回走，路上在车里的时候，王惊蛰问他能否找到毛笔，朱砂还有贡香什么的，陈放说他们单位里就有这些东西，专门卖给来火化死者家属带来的阴阳先生，殡仪馆有一整套的营销流程，专门做的就是死人的生意。<br/>　　<br/>　　    “大哥，我能问一下你为什么要主动帮我么？我记得那天是我先撞了你的，然后你就说我会出事了”陈放想问王惊蛰这个问题半天了，他觉得这个雷锋同志出现的让他挺措手不及，有点凭空掉下个救命菩萨的意思。<br/>　　<br/>　　    王惊蛰皱了皱眉，才说了一句：“你就当我是闲的吧”<br/>　　<br/>　　    王惊蛰说的这一句话，看起来很敷衍，但这里还真有一半的意思确实是闲着的，至于另外一半，则是因为向缺临走前跟他说过的那四个字。<br/>　　<br/>　　    “不惧苍天”<br/>　　<br/>　　    赊刀人到了王惊蛰一辈，他总是刻意地回避自己赊刀人的身份，如无必要的话从来都不会主动摆摊赊刀，为的就是减少身上所沾的因果，然后不至于在面对三十岁的那个诅咒时因果太多而产生负面的影响。<br/>　　<br/>　　    在秦村，向缺的一番指点过后，就让王惊蛰的心思豁然通透了，与其刻意躲避也解决不了，那还不如直面迎刃而上了，这就像打太极一样，他强由他强，清风拂山岗，王惊蛰觉得他干脆就直视赊刀这门职业得了，所以这才有了在城隍庙他主动揽事的那一幕。<br/>　　<br/>　　    回到殡仪馆，王惊蛰让陈放把老太太的骨灰盒给取了出来，然后又找来毛笔，朱砂等东西，在殡仪馆一处角落的工地上，把骨灰盒放在地上，又点上了贡香。<br/>　　<br/>　　    “敬四方天地，敬鬼神，三清聚顶通神明……”王惊蛰高举贡香过头顶拜了鬼神然后插在地上，手拿毛笔蘸着朱砂，在一块白布上写着：“慧元江边玩，金刚列两边，千里魂灵至，急急如律令”<br/>　　<br/>　　    一道招魂幡画完，平地刮起一股阴风来，老太太死后没有入阴曹地府，直接召唤她的亡魂过来还是挺简单的。<br/>　　<br/>　　    “这老太太是因为死的太凄凉了，心里有怨念才导致她不愿意去阴曹地府的，正好那天你去拉的她尸体，就把你给缠上了，待会我把她给叫出来后，你按我说的去做就行了”<br/>　　<br/>　　    陈放一听脸就白了，懵懂地问道：“叫出来，把什么叫出来啊？”<br/>　　<br/>　　    “当然是把她的魂给叫过来啊，鬼缠人基本都是有原因的，这老太太的原因就在于没有人给她养老送终，所以呢问题挺好解决的，一会她出来之后，你答应她找到她的儿女然后为她烧点纸就行了，老太太看到后人为她烧纸祭奠，自然就会去往阴间，以后也没缠着你的道理了”王惊蛰看着阴风吹起后，一道鬼影若隐若现地出来，就说道：“她快来了，记住了，你承诺她的时候一定要心诚点，千万不要敷衍，你但凡有一点假情假意她能都感觉得到，而且你答应的若是没有办到，她以后可就彻底跟着你，再也不会离开了”<br/>　　<br/>　　    陈放顿时懵了，急促地说道：“不是，这不对劲啊，你想老太太要是有儿女的话那人家不就能给她送终了么，何至于一个人死在养老院里都没人管了啊，我找不到她儿女，或者她家人早就死了，那怎么办？”<br/>　　<br/>　　    王惊蛰皱眉说道：“我让你这么办，就肯定有这么办的道理，我看过老太太的鬼脸面相，她不是绝后的面必然还有亲人在世，而且就是她的儿女，家家都有本难念的经，她肯定是多年前跟儿女闹了别扭然后才老死不相往来的，但老太太临死前还记挂着这个事呢，所以死后才心有不甘的，估计就是想再见自己儿女一面”<br/>　　<br/>　　    “唰”这时骨灰盒旁边一道鬼影突然闪现，王惊蛰捅咕了陈放一下，他咽了口唾沫，深吸了口气后诚挚地说道：“老人家我是那天拉运你尸体的，也是我送你来火化的，但没想到从那以后你就缠上了我，有人说是你留在阳间不愿意走，就是想再看自己的儿女一面，您看这样行不行？我找到您的儿女，让他们给你烧纸祭奠，您老就好好的去阴间吧，不要在缠着我不放了”<br/>　　<br/>　　    陈放话一说完，那老太太的影子就清晰了不少，脸上顿时露出一抹哀怨的神情，然后朝着陈放点了点脑袋。<br/>　　<br/>　　    王惊蛰看见老太太应承了，就伸手摆了下招魂幡，念出一句招魂咒把老人又给送走了。<br/>　　<br/>　　    “记住你刚才说的话，一定要找到这老太太的儿女……”王惊蛰慎重地叮嘱道。<br/>　　<br/>　　    “明白，明白，我肯定找”陈放搓着手，有点担忧的问道：“大哥，我这就没事了？”<br/>　　<br/>　　    “她家人把纸烧了，你就没事了，还有个事你记着，想办法把自己的名字改了，你的八字本来就轻这个放字不适合你，你找个会起名的先生，跟他说你命里缺金五行差土，他就知道该怎么为你起名了”<br/>　　<br/>　　    王惊蛰跟陈放交代了两句就告辞了，陈放松了口气，抹了把头上的冷汗嘀咕道：“总算是能安心了吧”<br/>　　<br/>　　    几分钟后，陈放把老太太的骨灰盒送了回去，然后去停车场取车。<br/>　　<br/>　　    “滴滴”陈放按了下车钥匙，车灯一闪，他忽然隐约间看见车灯前有一道影子闪了过去，他茫然一愣然后顺着车灯望去，就看见他平时开着的那辆运尸车就停在自己车的对面。<br/>　　<br/>　　    看见灵车，陈放突然间又莫名地心慌起来：“这他么的，怎么又慌了？”
