夏夕绾走在回廊里，只要想到刚才在牌桌上陆寒霆又让她叫爸爸又让她叫老公的样子，她一张小脸就像是煮熟的大虾，十分的羞恼。
<br/>
<br/>        这时她脚步一顿，因为前方走来了一道高大英挺的身躯，陆寒霆来了。
<br/>
<br/>        他怎么来了？
<br/>
<br/>        夏夕绾转身就跑。
<br/>
<br/>        但是跑了两步，陆寒霆就从后面追了上来，他探出健臂箍住了她盈盈一握的纤腰直接将她扯怀里，“你跑什么？”
<br/>
<br/>        夏夕绾哼了一声，“就是不想理你，怎么，夏妍妍的烟还没有买回来么？”
<br/>
<br/>        陆寒霆勾起薄唇，英俊的眉眼里荡漾出愉悦的笑意，“让我美男计的是你，现在小心眼的怎么又是你，让她买包烟不过是将她支走，陆太太你连这个醋都吃？”
<br/>
<br/>        夏夕绾转过身，小脸凑到他的身上，到处的嗅了嗅，像个小狗一样。
<br/>
<br/>        陆寒霆推开了一个房间门，将她抱进去，“嗅什么呢？”
<br/>
<br/>        “我看你身上有没有女人香，夏妍妍那样贴着你，我怕某些人假公济私，假戏真做。”夏夕绾仰着澄亮的眸子，嘟着粉腮望着他，这样子就像是一个娇蛮撒娇的小女友。
<br/>
<br/>        陆寒霆有点被愉悦到了，“家里有一只小母老虎，借我一个胆子我都不敢。”
<br/>
<br/>        “好吧，暂且相信你，陆先生，你可以放我走了，待会儿夏妍妍就要来找你了。”夏夕绾推了推他。
<br/>
<br/>        但是陆寒霆纹丝不动，他高大挺拔的身躯将她堵在怀里，“不让走，陆太太，叫声老公给我听一听。”
<br/>
<br/>        “…不要！”夏夕绾小脸爆红。
<br/>
<br/>        “真的不要？”陆寒霆的大手落在她的腰窝上轻轻的捏了两下。
<br/>
<br/>        “好痒…陆先生，放开我…”
<br/>
<br/>        “是不是害羞，没人笑话你，就叫给我听，叫老公。”
<br/>
<br/>        “就不要~”
<br/>
<br/>        夏夕绾笑着挣扎，两个人抵在墙角里闹成一团。
<br/>
<br/>        这时外面传来了夏妍妍焦急的呼唤声，“陆总！你们看到陆总了么？”
<br/>
<br/>        夏妍妍真的找来了。
<br/>
<br/>        这一次夏夕绾认真的推了推身上的男人，提醒道，“陆先生！”
<br/>
<br/>        陆寒霆将沉重的身体压在她纤柔的身上，薄唇用力的吻了一遍她散发着香气的长发，嗓音沙哑的呢喃道，“好吵，我能让她消失么？”
<br/>
<br/>        “不行！让夏妍妍消失太便宜她了！”夏夕绾拒绝。
<br/>
<br/>        陆寒霆缓缓松开了她，突然出声道，“我在这里。”
<br/>
<br/>        外面的夏妍妍听到声音迅速走了进来，“陆总，你是在这里么？”
<br/>
<br/>        夏夕绾吓得瞳仁一缩，这是个单间，中间有一个古风的屏风挡着，现在她和陆寒霆在屏风后面，夏妍妍进来并不能看到里面。
<br/>
<br/>        但是她还是吓得屏住了呼吸，毕竟在同一个房间里，很容易暴露，“陆先生，你疯啦？”
<br/>
<br/>        陆寒霆看着她，“我看这个夏妍妍对我虎视眈眈的，十分觊觎我的男色，我可不能保证她会对我做些什么，现在让你这个小醋坛子在这里看着，免得再一个人吃醋。”
