　　天穹之上，一行十数人急速飞行。
<br/>　　这一行人，自然便是巫尘和巫玛、巫幽兰等人。
<br/>　　“巫尘少司，我们查探了三个消失的小镇，都没有任何线索，现在怎么办？”一名巫族天才向巫尘问道。
<br/>　　他们查询了三个小镇，辗转十数日，没有查到丝毫的线索。
<br/>　　“继续前往下一个消失的小镇查看吧！”巫尘淡淡的说道。
<br/>　　“巫尘少司，貌似这个方向不是通向下一个小镇的方向吧？”巫幽兰略带疑惑的问道。
<br/>　　“这个方向好像有不少的小镇，走这里碰碰运气，说不定能碰到某个小镇出事！”巫尘沉声说道。
<br/>　　巫幽兰闻言，和旁边的巫玛对视了一眼，两人眸中闪过一丝异色。
<br/>　　不过，两人也没有多言，临行之前，大祭司向他们意念传音，吩咐两人注意一番巫尘是否有反常的行为。
<br/>　　虽然大祭司只是随意吩咐了一句，但两人也是格外上心，一路上都在暗暗注意着巫尘。
<br/>　　不过，巫尘并无任何反常行为。
<br/>　　一众人一路疾行，又过了几日之后，路过了一处偏僻之地。
<br/>　　“你们看那里，好像和之前查看消失的小镇差不多！”
<br/>　　这时，一名巫族天才突然手指前方，惊呼出声。
<br/>　　众人闻声望去，便见到前方一处山脚下，有一处不毛之地。
<br/>　　此地蕴含着强烈的死气与怨气，和之前查看的那些消失的小镇，一般无二。
<br/>　　“我们去看看！”巫尘朗声说道，随即身形加速，率先向前方飞了过去。
<br/>　　其他亦是急忙跟上巫尘，一起飞了过去。
<br/>　　很快，众人便飞到了山脚下，个个面色凝重的打量着眼前的一切。
<br/>　　这的确是一个小镇的遗址，只不过建筑基本没有了，地面全部呈暗黑色，是沾染了大量的血液所形成。
<br/>　　“大家仔细查看一番！”巫尘朗声喝道，随即众人在这不大的小镇遗址周围查看了起来。
<br/>　　“南方有打斗的痕迹，破坏的很严重！”
<br/>　　“北边也有，好像是有人在追杀留下的痕迹！”
<br/>　　不多时，众人便将小镇查探完毕，此镇和其它消失的小镇一般无二，但是南北两个方向，均有严重的打斗痕迹。
<br/>　　“巫尘少司，现在我们怎么办？”巫玛问道，毕竟巫尘的身份高，一切都要听从巫尘的安排。
<br/>　　“南北两个方向都有严重的打斗痕迹，说明有人向这两个方向逃走了！”
<br/>　　巫尘面现沉吟之色，继续道：“这样吧！巫玛、巫幽兰，你们两人实力较强，你们带几个人向南方查探，我带几个人向北方查探，我们分头行动！”
<br/>　　巫玛和巫幽兰两人闻言眉头微皱，他们要监察巫尘的异动，可不好分开。
<br/>　　“这样不妥吧！我们一旦分开了就实力大减，若是遇到敌人怕是有危险！”巫玛摇了摇头说道。
<br/>　　“你是少司还是我是少司！”
<br/>　　巫尘面色一沉，随即冷声道：“就这么定了，若是没有查到线索，我们就返回这小镇回合！”
<br/>　　言罢，巫尘立刻挑了五名巫族天才，招呼一声，便向北方飞行而去。
<br/>　　“这……！”巫玛见此顿时无奈，随即只好和巫幽兰一起，带着剩下的七、八名巫族天才，向南方飞去。
<br/>　　一众人向南方飞去，一路上都有不少打斗的痕迹，他们顺着打斗的痕迹不断的前进，不多时便远离了那座小镇数十万里，与巫尘几人彻底分开了。
<br/>　　“幽兰，你说我们和巫尘分开行动，不会有事吧？”一边飞行，巫玛一边向身边的巫幽兰问道。
<br/>　　“应该不会，他身边还有其人呢，实力都不弱！”巫幽兰略作沉吟，便摇了摇头。
<br/>　　大祭司并未说巫尘有什么问题，只是让他们多加留意而已。
<br/>　　现在查到了小镇消失的一点线索，他们分开行动，也在情理之中。
<br/>　　“嗯！”巫玛闻言点了点头，便不再多想。
<br/>　　几人继续前进，又前进了片刻功夫，地面上的打斗痕迹彻底消失了。
<br/>　　“痕迹没有了，巫玛大哥、幽兰姐，我们怎么办？”一名巫族天才问道。
<br/>　　“我们在周围查探一番吧！若是没有什么发现，就原路返回！”巫玛说道。
<br/>　　“嗯！”众人点了点头，随即便准备仔细查探周围。
<br/>　　就在此时，所有人均是猛然转头，看向后方。
<br/>　　只见后方百余里处，大地之上有一名青年，正在冷冷的望着他们。
<br/>　　这名青年身穿灰白色长袍，面目普通，眸光冷漠。
<br/>　　“此人是谁？”
<br/>　　“他怎么会在这里？”
<br/>　　“是普通人族，不是我族之人！”
<br/>　　众人见到这名青年，顿时眸光一凝，此人或许与小镇消失有关。
<br/>　　“拿下他！”巫玛轻喝一声，毫不犹豫，立刻下达了命令。
<br/>　　唰唰唰！！
<br/>　　霎时之间，巫玛等一众巫族天才，瞬间爆发出最快的速度，将这名青年包围了起来。
<br/>　　“你是何人？”巫玛直视青年的双眼，冷冷的问道。
<br/>　　“我在等你们！”青年微微一笑，露出一口雪白的牙齿。
<br/>　　青年不是别人，正是苏莫。
<br/>　　他要斩杀巫玛和巫幽兰，自然不能让大祭司发现，所以只能让两人和分身分开。
<br/>　　现在此地距分身百万里都不止，就凭大祭司附于少司令牌上的一丝意念，根本不可能注意到这么远的地方。
<br/>　　所以，他现在足够的安全，可以放心的出手了。
<br/>　　“等我们？”巫玛等巫族天才闻言，均是眼眸微眯，不过他们仔细打量了苏莫一番，发现苏莫只是武帝境四重的修为，便放心了。
<br/>　　他们几人之中，修为最低的都有武帝境四重，巫玛和巫幽兰更是武帝境六重境界，而且他们都是天才，自然不会将苏莫放在眼里。
<br/>　　“快说，你是什么身份？为何等我们？”一名巫族天才厉声喝道。
<br/>　　“等你们，是为了杀你们！”
<br/>　　苏莫森然一笑，话音一落，他身上陡然升起了一团淡紫色雾气，瞬间将巫玛和巫幽兰等几名巫族天才笼罩。
<br/>　　众巫族天才大惊失色，不过，他们还来不及有任何动作，面色便瞬间凝固，被拉进了梦魔空间之中。
<br/>　　苏莫没有直接用玄力攻击，而是率先动用了梦魔战魂。
