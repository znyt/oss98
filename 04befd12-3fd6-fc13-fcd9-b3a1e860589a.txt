　　她也是跟着叶少阳一起来到这个世界的，不过她清醒过来的时候，谁都没有看见，身边只有徐福一个人。她立刻就质问徐福崔颖在什么地方……她之所以留在阴阳司，最初的目的就是为了找徐福问这件事。
<br/>　　徐福倒也没含糊，带着她去找到了崔颖……先去找到白起，然后一起去了西域的那座古墓，找到了崔颖……原来在利用完凤兮之后，白起觉得崔颖生前毕竟是修为高深的法师，就算死后，魂力也十分强大，因此将他禁锢在一只金锁里，金锁用来锁在一口棺材上，自然就成了一个机关，在打开金锁的瞬间，已经被禁锢神识的崔颖会立刻凭着本能对入侵者展开进攻。
<br/>　　“当时那口棺材是在北面的地下河中，与石棺相对，那里看上去是一条古墓的出路，其实是个陷阱，如果走到尽头，必然会发现这口棺材……只是当时老大没有走到那里，自然也就没有发现。”凤兮说道，“当时，徐福就带我去见了白起，一起到古墓里放出了崔郎的魂魄……”
<br/>　　叶少阳听到这里，猛然想起一个问题，插嘴说道：“等下，你说徐福带你去见了白起？哪个时代的白起？”
<br/>　　“就是我们那个时代，被关押在阴司里的白起。”凤兮也想起了什么，说道，“对了老大，徐福告诉我一件事，白起是他跟道风一起从昭狱里救出来的……”
<br/>　　什么！
<br/>　　叶少阳震惊，皱眉问道：“什么时候的事？”
<br/>　　“就是在我们来到这个时代之后吧。具体如何做到的，徐福没有告诉我。”凤兮甜蜜地看了崔颖一眼，道，“徐福只是让我和崔颖相见，然后……把我们也送到这个时代，告诉我们，跟你总有一天会相见……我也问过他，什么时候把你送回去，徐福的回答只有一句话……”
<br/>　　叶少阳立刻激动起来，听她往下说。
<br/>　　“他说，如果你做得到自己该做的事，就可以回去……”
<br/>　　“什么事？”
<br/>　　凤兮摇摇头，“他没有说什么事，只说到时候你自己会知道的。”
<br/>　　自己会知道的……这个关子卖的。叶少阳恨得牙根痒痒。
<br/>　　“如果这件事我做不到呢？”
<br/>　　“那……你就回不去了。”
<br/>　　叶少阳深吸了一口气，无奈地苦笑起来。他不知道徐福为什么要搞的这么神秘，一点头绪没有，不过根据凤兮的经历，至少确认了一点就是：徐福有能力带他回到自己的时代，而且也不排斥这么做。至少，自己总算是有点了希望。
<br/>　　“那你们……”叶少阳环顾四周，“这一切是怎么来的，还有，你们现在的肉身是怎么回事？”
<br/>　　从进来开始，叶少阳一眼就看穿面前这对夫妻都是借尸还魂。
<br/>　　“他们是五色土塑形，形成肉身的。”包子突然插了一句。
<br/>　　凤兮和崔颖顿时有些吃惊，转头看向包子。
<br/>　　“我能用幻象迷人，对于生灵的本尊形态，能够一眼看穿。”包子冲叶少阳挤眉弄眼地卖弄起来。
<br/>　　火眼金睛啊卧槽！
<br/>　　叶少阳有些震惊地看着包子，这种特殊能力……还真是牛比，将来能用到的地方肯定不少。至于他所说的五色土塑形，叶少阳大致是知道的。
<br/>　　用五色土塑形，模拟人的肉身，这原本是一种道门法术，据说重塑的肉身就像真实的人一样，普通人绝对看不出来区别，不过最重要的不是这个：用很多材料，都可以塑造出同样的肉身，但是灵魂或者无法附着，或者附着之后操控不便……只有五色土，通灵效果极好，因此成为重塑人身的最佳材料。
<br/>　　不过这门法术因为违反了天地大道，在法术界也是禁术，名声大，但是现实中运用的机会极少，叶少阳也只是听说过而已，在21世纪，像很多古代的法术一样，这种用五色土塑造人形的法术也失传了，连叶少阳也只是听说过而已。
<br/>　　因此，叶少阳怀疑他们的身体，是徐福帮忙重塑的。一问之下，果然是这样。
<br/>　　“我们来到这里，也是徐福特意寻找的结果，在这里，我叫刘珊，他叫沈文，这是一对可怜的小夫妻，几年前结婚去欧洲度蜜月，在法国游船的时候，不小心翻船丧命……尸体在很远的地方被找到，但因为是在异国他乡，当时没有能验证出他们的身份，只能作罢。
<br/>　　而他们的父母，用尽了各种方法，也是无法找到他们的下落，当他们已经死了……我们被重塑成这对夫妻的模样，找了个借口，从海外归来……事情就是这样。然后呢，这个刘珊的父亲，是这一带的督军……权势很大。”
<br/>　　“督军啊……那不就是军阀吗？”叶少阳有点明白了。这本来就是个军阀混战的时代，什么县长市长的，在军阀的枪杆子面前都不好使。
<br/>　　凤兮道：“没错，之前那位县长，就是我父亲一手提拔上来的，因此才会亲自去接你们，因为我得到消息之后告诉他，你是我的贵客，他便不敢怠慢。”
<br/>　　原来如此……
<br/>　　叶少阳总算明白了一切，望着凤兮，沉吟道：“这么说，你跟翠英是冒充了督军的女儿，但你们实际上并不是，有句话我不当说，这是不是有点利用别人的感情？”
<br/>　　凤兮笑道：“一来，我们能够安抚他们的丧女之痛，二来……刘珊的父母，是一对六世夫妻，在我跟崔郎的那一世，他们便是我的父母……因此这也不算是利用了，我也当他们是父母一样看待的。”
<br/>　　叶少阳这才恍然大悟，仔细一想，徐福送他们到这户人家冒充儿女，看中的应该不是权势之类，而是他们之前在前世的亲缘关系，毕竟他们是凤兮最后一世为人时期的父母，这种感情还是不一样的。
<br/>　　“那这个刘珊……为什么跟你长得一样？”叶少阳只剩下这个疑问。
<br/>　　凤兮笑了笑，道：“老大，这是我为了见你，特意变化出的模样，他们看到的我，可不是这个样子。”凤兮说着，在脸上抹了一把，就跟变脸似的，立刻就变化成了另一张脸。崔颖也跟着变化。
