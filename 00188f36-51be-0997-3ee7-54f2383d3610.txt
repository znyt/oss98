　　白萱猛然站了起来。
<br/>　　从她身上之中，玄妙到极点的剑意，涌现了出来。
<br/>　　这剑意中，乃是百道剑气融为一体，其中最为显赫的，便是其中通天剑气的力量。
<br/>　　“这通天剑气的力量，居然是被全部吞噬了？其中九成九的力量，都被吞噬？”
<br/>　　李长老看着白萱的变化，也是极为惊讶。
<br/>　　这种外来剑气的力量，能够吞噬其中三成，便是不易，吞噬四成，五成，都是难得一见的剑道天才了。
<br/>　　至于像是如今这样，白萱居然是吞噬了通天剑气其中九成九的力量，则根本就是一件不可思议的事情。
<br/>　　“恭喜师姐，剑道小成了。”
<br/>　　楚风眠看着白萱，笑着开口道。
<br/>　　对待朋友，一直以来楚风眠都不会吝啬，这白萱帮了楚风眠不少，这通天剑气送与白萱，楚风眠也不会觉得有什么。
<br/>　　“我的剑道，终于是凝练完成了。”
<br/>　　白萱站起身来，感觉着身上的剑意，脸色也是大喜。
<br/>　　她的剑道终点，那是万道剑气合一，现在的白萱，只是融合了百道剑气，只能算是略有小成。
<br/>　　但是就算是小成，也终于是凝聚完成，九十八道跟一百道，虽然相差极少，但是却是截然不同的境界。
<br/>　　现在的白萱，论起实力，比起刚刚都要强大无数倍。
<br/>　　只怕是实力，甚至都要超越洪狱，落长恨等人，回到北邙学院中，内门中的真正第一人了。
<br/>　　“还要感谢你的相助啊。”
<br/>　　白萱看着楚风眠，眼神中也露出几分若有所思。
<br/>　　李长老不请吃，但是白萱自己清楚，她的力量足以提升如此之多，将通天剑气其中的九成灵力都给吞噬，可不是因为她自己的原因。
<br/>　　而是楚风眠。
<br/>　　她所缺少的最后一道剑气，也是楚风眠补全的。
<br/>　　楚风眠打出的最后一道剑气，其中虽然没有蕴含多么霸道的力量，但是却是有着一种足以融合一切的力量。
<br/>　　将她身上的其他的九十九道剑气，完美的融合在了一起，才令她的剑道小成。
<br/>　　可以说这一切，都是归功于楚风眠。
<br/>　　只是白萱却是并没有点破这一点，楚风眠身上的秘密的确众多，若是暴露出去也是极为麻烦。
<br/>　　楚风眠不愿意去说，白萱也不会去主动问。
<br/>　　白萱突破，身上的剑意，也从包厢中，流露了出去。
<br/>　　“天字十号包厢之中，有人突破了！”
<br/>　　“好锋利的剑意，这就是通天剑气的力量吗？”
<br/>　　“通天剑气，就这样被人凝练了，只怕是任何一个巅峰圣者，凝练了这通天剑气，都会成为神力境中的无敌人物吧。”
<br/>　　一道道声音响起，有嫉妒，但是更多的都是羡慕。
<br/>　　通天剑气这等东西，的确不管是谁得到，都不是他们可以有机会沾染的。
<br/>　　“可恶，通天剑气居然被北域的土著炼化了！”
<br/>　　古林感觉到其中的剑意，自然也清楚，通天剑气是被人炼化了。
<br/>　　“通天剑尊可是超越了生死境的大能，他留下的剑气，居然被北域的土著炼化了，简直是暴遣天物！”
<br/>　　这通天剑气，本是属于古林的，可是楚风眠的杀出，却是从古林的手中，硬生生的抢走了。
<br/>　　“古师兄不用担心，通天剑气炼化了，等我们杀了此人，也可以得到这通天剑气。”
<br/>　　一尊荒古门弟子，劝解道。
<br/>　　“哼。”
<br/>　　古林负气坐在王座上，眼神中尽是怒火，他的脑海中，已经在无数次的计划，如何要彻底斩杀了楚风眠才行。
<br/>　　通天剑气，算是一个插曲。
<br/>　　接下来拍卖的无数宝物之中，没有任何一件，楚风眠看得上眼。
<br/>　　这些宝物，许多都是来自远古时代，出自古洞府中，价值极高。
<br/>　　但是对楚风眠来说无用的话，他也不会浪费财富去购买，楚风眠的财富虽然众多，但是也不会肆意挥霍。
<br/>　　不一会，这拍卖行，便是接近末尾，只剩下最后三件宝物。
<br/>　　众人没有因为漫长的时间，磨灭的兴趣，反而是接近尾声，一个个都是极为激动起来。
<br/>　　因为众人都知晓，这最后三件宝物，才是最后的压头戏。
<br/>　　“第三十七件宝物。”
<br/>　　灵尊真人，在看台上，几尊武者，联手抬着一道巨大无比的东西，放在了台上。
<br/>　　这东西，足足是有着数十米长宽，一眼望去，其中却是造型诡异。
<br/>　　似乎隐隐约约看起来，像是一种生物的手掌。
<br/>　　“这是，龙爪？龙皇龙爪？”
<br/>　　楚风眠看到这东西的一刻，眼神猛然一变。
<br/>　　从这东西上，散发出来的淡淡龙威，瞒不过楚风眠的双眼，这眼前的东西，居然是一只龙爪，一直实力达到龙皇地步的龙爪。
<br/>　　“此物，是傲恒龙皇的一枚龙爪！”
<br/>　　灵尊真人在看台上，缓缓的开口道。
<br/>　　“此物是出自这一次的古洞府中，傲恒龙皇是上古龙族中的佼佼者，足以冲击龙帝的人物，虽然陨落万年，但是他的一枚龙爪，其中还蕴含着恐怖至极的力量，甚至是蕴含血脉，谁若是可以得到，便是意味着可以得到龙皇血脉！”
<br/>　　灵尊真人的话一出，顿时在会场中，掀起了轩然大波。
<br/>　　“龙皇龙爪！”
<br/>　　“傲恒龙皇，这可是万年前赫赫有名的大人物，龙皇之首，近乎龙帝的存在！”
<br/>　　“龙族血脉，还是龙皇血脉，谁要是可以得到，好处只怕是无穷无尽！””
<br/>　　“我今日就算是拼劲家财，将一切财富都拿出来，也要得到这龙爪！”
<br/>　　一道道灵识，在互相交谈着。
<br/>　　楚风眠感觉到，在他身边的天字包厢中的人，都在议论纷纷起来。
<br/>　　龙皇龙爪。
<br/>　　通天拍卖会，不愧是虚空海中最大的拍卖会，这最后三件拍品，第一件，居然都是这种东西。
<br/>　　“好东西！傲恒龙皇，可是接近龙帝的存在，虽然不能跟颜古龙帝抗衡，但是也是龙族中的绝代霸者。”
<br/>　　楚风眠看向那龙皇龙爪，眼神中也是露出几分兴奋之色。
<br/>　　“这龙爪中蕴含的力量并不少，我的手中，正好有着那涅槃龙蛋，可以拿这龙皇龙爪跟涅槃龙蛋，同时圣祭，有着事半功倍的效果！”
