        简单的说几句感言，不会耽误读者老爷你们太多时间。
<br/>
<br/>        为了方便，第一版愿心暂时和第二版愿心合二为一，以第一人称口吻叙述。
<br/>
<br/>        首先说实话这本书被签约是一件让我很意外的事情。
<br/>
<br/>        它其实是我在写《千年王座》时候意外的产物。
<br/>
<br/>        当时只是想简单的写点什么脑袋里突然蹦出来的吐槽点，自己搞笑一下。
<br/>
<br/>        所以当时我写了三章就在也没写。
<br/>
<br/>        这也是时隔了四个月的原因
<br/>
<br/>        老实说，开始写的时候，除了开头吐槽我什么没想，什么大纲全都没想，就是写着乐呵的
<br/>
<br/>        因为我把全部的心血、感动和热血投入到了王座里了。
<br/>
<br/>        《千年王座》
<br/>
<br/>        为了这本书，我的处女作，我真的是绞尽脑汁的设定了一个庞大瑰丽的玄幻世界观，种种别人没有的剧情，虽然最后还是失败了
<br/>
<br/>        那本书真的见证了我整整一年的努力，我的成长，我的不知道有没有的进步，
<br/>
<br/>        从零开始，一步步艰辛，我现在看着都觉的自己不一定能再坚持下来了，哈哈，有兴趣的读者可以去看看那本书，看看愿心当时从三十万零读者坚持到888888字的一路悲剧（但我仍然认为倾注了我当时全部心血的王座非常好看）
<br/>
<br/>        从王座成长过来的愿心到现在，再说这本书，
<br/>
<br/>        从得知签约开始，我就开始补全这本书的一切，剧情、设定、后续人物、主角的心理之类的
<br/>
<br/>        我敢保证我写的故事绝不套路、也绝不小白，这点文笔的自信，我还是有的
<br/>
<br/>        《都市夜战魔法少男》
<br/>
<br/>        这个当时我恶搞瞎起的名字，我真的没想到现在能再封推上看见它。
<br/>
<br/>        所以我想明白了，我要为这本书付出我所能付出的努力
<br/>
<br/>        这个故事我一定要写的精彩！写的漂亮！
<br/>
<br/>        我仍然是那个厌恶老版套路的愿心不变，我愿意用我全部的点子去写好这本书，就像曾经的王座一样。
<br/>
<br/>        二次元的故事轻松搞笑写的很有意思，夹杂着的热血和感动也更加贴近现实，或许真的是想写王座的时候书友说的，我不适合写玄幻，该去二次元....
<br/>
<br/>        这个故事轻松、搞笑、有热血、也有感动、
<br/>
<br/>        方然也只是个平凡人，
<br/>
<br/>        而我这次想写的就是这么一个平凡人的故事！
<br/>
<br/>        他会和很多书的主人公都不一样，因为他只是个平凡人，其实和枫然一样，枫然也只是活了千年的平凡人而已。
<br/>
<br/>        《都市夜战魔法少男》
<br/>
<br/>        请你期待、之后的故事。
