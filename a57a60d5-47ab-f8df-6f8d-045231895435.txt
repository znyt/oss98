　　密室之中，苏莫全身气息鼓荡，滚滚混沌之力涌动，他正在冲击三品高级的修为。
<br/>　　他的修为，早已达到了三品中级的极限，破境只在旦夕之间。
<br/>　　大量的五行源石，被苏莫直接吞噬，凝练成五行源力，壮大圣丹，洗條全身。
<br/>　　冲击境界的时间并不长，因为苏莫服下了天材地宝，对冲击境界瓶颈，有极大的裨益。
<br/>　　在吞噬了大量的源石之后，约莫两个时辰左右，苏莫全身一震，气息大涨，终于再次破境，达到了三品高级的地步。
<br/>　　良久之后，苏莫炼化了体内所有的五行源气，气息缓缓收敛，便睁开了眼眸。
<br/>　　他并没有任何的欣喜之色，因为，即便是突破了修为境界，依旧远远不够，还无法改变他现在的窘境。
<br/>　　“不知道帝一魂和拓跋青云如何了？”
<br/>　　苏莫低声自喃，随即，他立刻取出传讯石符，给予人德宫主传讯，询问情况。
<br/>　　不多时，石符震颤，人德宫主回消息了。
<br/>　　“苏莫，上次帝一魂和拓跋青云交手了，看情况，帝一魂略胜两分，不过，总体而言算是在伯仲之间！”苏莫意念探入石符，其内传出人德宫主的意念之音。
<br/>　　苏莫闻言，心中沉思，这一点在他的预料之中，并没有什么意外。
<br/>　　“前辈，最后的结果呢，天王宫和地王宫有什么大动作？”苏莫再次传讯询问。
<br/>　　“并没有什么动作，不过，过几日帝一魂会来人王宫见老夫！”人德宫主的消息再次传来。
<br/>　　“他去见你？有什么目的？”苏莫眉头微皱，再次传讯询问。
<br/>　　虽然是询问，但是，不用猜他也能明白，帝一魂的目的。
<br/>　　其目的不外乎两种，一是打探他的所在，二是拉拢人王宫。
<br/>　　苏莫心中有了一丝警惕，他现在人不在轮回神朝，没有亲眼见到具体情况，若是人德宫主倒戈，那他就危险了。
<br/>　　倒不是他不信任人德宫主，而是事关生死存亡，他不得不留下警惕之心。
<br/>　　“目的不言而喻，是想要拉拢老夫，不过，你不用担心，既然老夫答应了忙你，自然不会出尔反尔，此次，借此机会，老夫会好好谈谈帝一魂的口风！”
<br/>　　人德宫主的消息，再次通过传讯符传来，先是安慰了一番苏莫，再说了一下自己的打算。
<br/>　　“多谢前辈了，有什么情况，随时可以给我传讯！”苏莫传讯道。
<br/>　　“嗯，你尽快修炼，将修为提升起来吧！”人德宫主传讯道。
<br/>　　少倾，苏莫结束了和人德宫主的交流，面色沉默，他不会完全信任人德宫主。
<br/>　　虽然对方帮了他很多，但是，他还是要保留着警惕之心。
<br/>　　不过，他目前并未告知对方，自己身在天龙神朝龙都城之中，只是不清楚，对方能不能通过传讯石符，找到自己的位置。
<br/>　　“怎么办呢？”苏莫面色阴晴不定，若是传讯符能锁定他的位置，那此传讯符留在身上，无异于一颗定时炸弹。
<br/>　　但是，此符是他和人德宫主交流的重要宝物，珍贵无比，仅此一件，又不能丢弃。
<br/>　　沉吟了一番，他将传讯的石符，丢进了自己的吞噬元神的漩涡之中。
<br/>　　他的吞噬漩涡之内，与世隔绝，与外界完全没有丝毫的联系，石符放在其内，人德神王应该就无法通过石符，来锁定他的位置了。
<br/>　　“龙都城，也不是久留之地啊！”苏莫长叹一声，声音中充满了无奈。
<br/>　　不过，他现在倒是不急着离开龙都城，先巩固一番自己的修为再说。
<br/>　　随后，苏莫闭上了眼眸，继续修炼了起来，巩固刚刚突破的修为，凝练体内的混沌源力。
<br/>　　就这样，又过了三日，密室之外，响起了沉闷的敲门声。
<br/>　　苏莫闻声，结束了修炼，长身而起，打开了密室的大门。
<br/>　　大门之外，一身金色龙袍的龙腾，精神熠熠，显得意气风发。
<br/>　　“苏莫，有大喜之事？”龙腾笑着说道。
<br/>　　“哦，什么喜事？”苏莫好奇，难道是对方在龙族，得到了什么重要的身份不成？
<br/>　　“龙帝要见你和司空炎？”龙腾沉声说道。
<br/>　　“龙帝？”苏莫闻言一怔，随即立刻讶然，所谓的龙帝，便是天龙神朝目前的掌权者，吞虚龙帝。
<br/>　　传闻，吞虚龙帝，乃是洪荒祖龙之子，也是目前至强者消失的神域中，最强者之一。
<br/>　　“龙帝为何要见我们？”苏莫沉声问道，他要问清楚其原因，毕竟，面见龙族掌权者，这可不是小事。
<br/>　　“上次奴仆争夺宝物一事，你和司空炎的表现，在龙族内几乎人尽皆知了，此事龙帝自然也听闻了！”
<br/>　　龙腾面色郑重，继续道：“前几日我在龙都城中，闯过了一个极为艰难的考验之地，算是再次名声大震；因为此事，龙帝接见了我，然后也提了一句你和司空炎；我便顺势向他介绍了你们两人。”
<br/>　　“你如何说的？”苏莫沉声问道。
<br/>　　“我告诉他，你们和我自幼相识，与任何大势力都没有干系，能为龙族所用！”龙腾沉声说道。
<br/>　　苏莫闻言点了点头，他明白龙腾的意思了，对方是想要龙帝看重他们，加以重用，从而在龙族拥有一定的地位，而不是一直做一个奴仆。
<br/>　　“龙腾，天龙神朝和轮回神朝之间，关系怎么样？”苏莫再次问道，他想知道龙族和轮回神朝各大王宫之间的关系，这关乎到日后，龙族能否成为他的助力。
<br/>　　“关系一般，龙族和各大神朝之间，基本上是井水不犯河水！”龙腾道。
<br/>　　“嗯，那我们走吧！”苏莫点了点头，既然如此，这龙帝他有必要一见。
<br/>　　“好，我们去喊司空炎！”龙腾道，随后，两人前去司空炎闭关的密室，告知对方此事。
<br/>　　而后，三人一起，联袂离开了府邸。
<br/>　　龙帝的住所，在龙都城的中央部位，是一座非常辽阔的府邸。
<br/>　　府邸占地千顷，巨大的府门之前，摆放着两尊巨大的神龙石雕，栩栩如生，霸气森然。
<br/>　　这府邸，名为吞虚龙府，门前有八名龙族护卫，个个都是六七品的修为，非常的强大。
<br/>　　因为有龙帝的手谕，龙腾带着苏莫和司空炎，畅行无阻的进入了龙帝府，来到了一座恢弘大气的宫殿之前。
<br/>　　宫殿大门敞开，站在殿外，苏莫便能看到，大殿中高高的主座之上，端坐着一位魁梧的老者。
