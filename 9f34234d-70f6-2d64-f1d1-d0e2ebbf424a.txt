　　第195章
<br/>　　大军攻破城门，长驱直入直抵山谷谷底。
<br/>　　事实上，他们已取得这场战争的胜利，虽然不知怎的，让山谷的人逃走了，无法让大家大杀特杀，但这也不算什么，毕竟大家的目的是山谷内埋藏的宝物，杀人只是顺便的事情。而只要抓住程大雷，大家一样有机会泄愤。
<br/>　　“放箭，放箭！”
<br/>　　眼看程大雷已经爬上山壁，大家怎能让他逃了，大王子急忙号令士兵放箭。
<br/>　　此时徐神机等人已经将火药从山头丢了下来，轰隆隆炸开一片人。秦蛮扯着绳索，拼尽全力将程大雷往上拉。
<br/>　　“喂，小心点，丢到我身上啦。”程大雷扯着嗓子大骂。
<br/>　　借着火药的威力，程大雷在极短的时间内被拽上山谷，此时弓箭已射不到这样的高度，众人只能无奈的看着程大雷离开。
<br/>　　程大雷跌坐在地上呼哧呼哧喘气，忽然站起来，冲下面挤出个鬼脸。
<br/>　　“哈，我出来了，没想到吧。”
<br/>　　这个人怎么这么讨厌，明明已经逃走了，还要故意炫耀。
<br/>　　“姓程的，任你机关算尽，这山谷已经被我们占了，你究竟有什么好得意的。”大王子说这话时，悄悄示意手下，派出一队人去抓程大雷，让他这样走了，还真有些不甘心。
<br/>　　程大雷冲他挤眉弄眼，带着手下麻溜快的撤了，他可不想落在戎族手中，那将会是什么下场，其实想也明白。
<br/>　　大王子心中也知道，程大雷对地形熟悉，他化整为零，往大山里一钻，想抓到他也并不容易。自己派出人，所做的也不过是无用功而已。
<br/>　　但自己毕竟把这里占了，不是么？
<br/>　　他站在点将台上，俯视着整座山谷，不得不说，程大雷选了一个极好的地方。此地易守难攻，若不是程大雷最后放弃，真想攻下这座山寨，怕是要付出很大代价。
<br/>　　山谷内被程大雷建设得不错，房屋井然有序，仓库内还存储着大量兵器，马圈内各色战马有数百匹，这些都是程大雷无法带走的。人可以翻山，马也可以翻山么……
<br/>　　“倒是个屯兵的好所在呐。”
<br/>　　大王子心中暗想，此地如果驻扎一支精兵，进可以攻打幽州各地，退，可以据险而守，此地究竟能挡住多少人，程大雷已经证明过了。
<br/>　　得此一谷，可以控制幽州大半，嗨，可惜帝国的军官尸位素餐，这样的军事要地却不加重视，若是派下一支精兵，自己怎么可能长驱直入，如入无人之地。
<br/>　　本来大王子想要毁掉这里的，可是看到此地模样后，却打消了这个打算。这里完全可以发挥出更大的作用。
<br/>　　如今，自己已经是这里的主人了，不是么？
<br/>　　他得意洋洋的行走在山谷中，用脚步丈量着山谷的土地，如一个农场主早晨醒来的第一件事，就是巡视自己的牧场。
<br/>　　“这是刚开垦的农田么，不错，不错，地看上去很肥呐。”
<br/>　　“这房子修得不错，不得不说，南人建房是有一套的，不过，这房子却是给我们建的，哈哈。”
<br/>　　“来，让我看看仓库，南人打造兵器可比部落的铁匠强。”
<br/>　　大王子所到之处，到处洋溢着一种欢快愉悦的气氛。所有人都迫不及待的涌进山谷，前赴后继，谁也不想落后一步。
<br/>　　无它，大家在山谷外憋了太久了。从开始攻打蛤蟆寨距今，屈指算来，已经有三四个月之久。这三四个月，大家过的是怎样的日子，风吹日晒，忍受大山内闷热潮湿的环境自不必说。每天做梦都想着杀进山谷，用一千种法子折磨那只可恶的癞蛤蟆。
<br/>　　但每天醒来，大家悲催的发现，自己做不到。
<br/>　　这种无能为力的感觉长久围绕大家。
<br/>　　今天，我们终于做到了，虽然没有擒住那程大雷是有些遗憾，可大家毕竟进入了这里，这座大家想了好久的山谷。
<br/>　　这里的天是那么蓝，水是那么清，空气是那样的新鲜……一切都显得那样美好，令人迷恋。
<br/>　　有人趴在地上，咀嚼山谷中的青草，有人痛哭流涕，哭着哭着竟笑出声来，有人在山谷内狂奔。
<br/>　　他们在疯狂，他们在呼喊，他们疯狂呼喊是因为他们在庆祝。
<br/>　　他们是胜利者，有资格去庆祝。
<br/>　　夜来，山谷内灯火通明，到处点满篝火，一场庆功会正在此地举行。
<br/>　　而北蛮部的九位王子此时已聚在一起，将一份地图拿出来，细心与山谷的地形比对，寻找三十年前藏宝的位置。
<br/>　　轰！
<br/>　　此时爆炸声突然响起，众人心里已经有了阴影，听到这声音心里咯噔一下，人人均想：那恶魔又回来了。
<br/>　　九人跑到外面，只见火药是从山壁上丢下来的。程大雷一行人点着火把立在悬崖上，程大雷看着下面人目瞪口呆的表情放声大笑。
<br/>　　“哈，我又回来了，惊不惊喜，意不意外！”
<br/>　　“狗贼，你还敢回来，我正愁杀不了你。”大王子喝道。
<br/>　　“呔！”程大雷突然收敛笑容：“尔等化外蛮夷，披毛戴角的畜类，见我帝国繁华，不识学礼以通教化，反而妄动不义之师，杀我百姓，占我土地。”
<br/>　　“天道循环，报应不爽，我乃是南极仙翁传人，与太白金星八拜之交，与天蓬元帅同门的蛤蟆仙人，我大徒弟宇智波佐助，二徒弟漩涡鸣人……今日本仙人替天行道，天不罚你，我来罚你。”
<br/>　　山谷内众人一头雾水，大王子喊道：“你胡说八道什么！”
<br/>　　“我知道，这些话你们心里也不信，那我就稍稍给你们证明一下。”程大雷忽然抿嘴一笑，右手高高举起，火把在空中转了个圈：“二师兄，借我一挂天河呐。”
<br/>　　黑暗中，关鱼抿抿嘴，心里不满道：谁是你二师兄。
<br/>　　“掘堤”
<br/>　　守着瀑布的几百兵丁一起动手，三两下就将堤坝掘开，忍了半个多月的河道早就迫不及待，堤坝一旦掘开，立刻呼啸而下。
<br/>　　轰隆隆声响，宛若万马奔腾，千雷炸响，大河之水从天而降。
<br/>　　“他，他真的借来一道天河！”
<br/>
