　　第669章
<br/>　　宋伯康对这件事的策略是严防死守，禁止谣言的传播。但他不明白，谣言如山洪，宜疏不宜堵。
<br/>　　有时候谣言可以传播，并不是因为人们相信，而是因为人们需要。
<br/>　　当然，这背后的道理，宋伯康是不会去想的。他只是用强硬的手腕，准备将谣言扼杀在摇篮中。
<br/>　　的确抓了一批人，杀了一批人，城里再没人敢公然议论这件事。但委实说来，这样做收效不大，人们不再公然议论，可他们会悄悄议论。宋伯康控制得住凉州城，但他控制住得浩甲、金城、武威三座城么？
<br/>　　只是宋伯康看不到而已，他看不到，便以为有些事情不存在。
<br/>　　这些事，只是宋伯康每天在做的诸多事中的一件，他更多的精力，还是放在备战上。另外一桩事，则是关注京州那边的消息。
<br/>　　自己求援的人已经去了，相府那边会不会给自己想要的东西。
<br/>　　这一日，宋伯康杵在王府的门口，一身锦衣华服，身边立着凉州城的文武官员，人人表情严肃。
<br/>　　一匹快马奔来，那小校急急忙忙将消息禀告给宋伯康：“来了，来了。”
<br/>　　宋伯康顿时一激灵，整个人变得精神起来，他左右看看，是想看自己手下文官武将中有没有偷懒的。
<br/>　　整条街道上都有站岗的士兵，一直从城门到王府。普通人根本没资格上街，大家只能躲在屋子里远远张望着，心里琢磨着：来的究竟是谁，竟会值得宋伯康摆出如此大的阵仗。
<br/>　　远远望着，大家好奇的人终于出现，他们从城门而来，前面是凉州兵开道，在其后是一辆低调却不失豪华的马车，在之后是马车随行的亲卫。
<br/>　　大家都在好奇车上的是谁，直到马车在王府门前停下，宋伯康降阶相迎，来到马车前。
<br/>　　“郡主，凉州宋伯康率领凉州文武参见。”
<br/>　　车内传出一个女人的声音，她用清冷的声音道：“来是做事而来，就省了这些繁文缛节，我们府中说话。”
<br/>　　对方根本没有走下马车的意思，宋伯康竟然也没有半点不满的意思，甚至亲自指挥马车入府。中间有些明白来者是谁，但也有不明白的，稀里糊涂不明所以然。
<br/>　　……
<br/>　　蛤蟆城，城主府。
<br/>　　程大雷窝在书房内，听罢阿喜汇报的消息后，轻轻叹了一声：“原来是她。”
<br/>　　书房内还有刘悲与徐神机几人，刘悲好奇问道：“大当家竟然认得她？”
<br/>　　“认识自然谈不上认识啦，但我如何也在长安城混过，也听过她的名字。”程大雷叹道：“拿出她来对付我，崔相大人还蛮看得起我的。”
<br/>　　一旁的阿喜也轻轻点头。他和程大雷都曾经在长安城做过地头蛇，对于长安城有名气的人，还是听说过的。
<br/>　　但刘悲和徐神机都不曾有这样的经历，二人仍旧稀里糊涂，不知程大雷口中这位是何方神圣。
<br/>　　见二人不解，程大雷开口道：“这位郡主算起来大概和我同龄，她是崔相最小的女儿，崔相一大把年纪，老来得女还是蛮不容易的一件事，对其自然视若掌上明珠。以崔相的权势地位，为她争取一个郡主的封号不算什么大事。”
<br/>　　徐神机困惑不解：“那为何崔相舍得将她送到这个地方？”
<br/>　　“你莫要着急，听我说完。”程大雷道：“崔相权势熏天，那怕他家的一个仆人也是可以在长安城横着走的人物，何况是崔相的小女儿。不过她出名并不是因为此，长安城有权有势的人还是多的。”
<br/>　　“那是因为什么？”刘悲也忍不住问。
<br/>　　“原因有三。”程大雷娓娓道来：“一是她的辈分，许多人甘愿给崔相当牛做马，做个干孙子什么的，所以不管大小，都习惯唤她一声小姑姑。其二就是她的脑袋，传说此女三岁读书，诗词歌赋无有不通，医卜星相无有不知，还曾化名去书院读过一年书，最后搞得整座书院已无人能传授她东西，如此，震惊帝国。”
<br/>　　“喔，原来是个才女呐。”徐神机恍然大悟。
<br/>　　“嗯，是个才女。据说此人在征伐攻略上很有见地，被称作神算子，许多事崔相都要与其商议。”程大雷道：“不过，单单有才，也不会有太大名气。男人对一个女人，最关注的还是……”
<br/>　　“美貌？”刘悲道。
<br/>　　程大雷点点头：“是个美女呐，此人名叫崔白玉，有好事着称她为无暇美人。传说她从来以黑纱遮面，原因就是太过美丽，走在街头常常引起混乱。关于她有许多传说，传说她天生异香，常常引得蝴蝶围绕，也有人说她的洗脸水泼在池中，池中荷花竟然朵朵鲜红，曾经有南方的蛮国的国王提亲，愿意用一座城池做聘礼，崔相都不曾答应。于是，她又有倾城美人之名，当初在长安，她与李婉儿被誉为帝国双姝，明珠有华，白玉无瑕。”
<br/>　　“小姑姑，神算子，无暇美人……”刘悲叹道：“果然来头不小。”
<br/>　　“当然了，传说只是传说，也是听过的人多，见过的少。李婉儿大家都见过，也不过是一般么。”
<br/>　　说到这里，程大雷忽然有些失神。话说不知李婉儿在东海如何，不过随在李行哉身边，应该也不用自己担心。
<br/>　　帝国是不缺美人的，程大雷也见过几个。但说某个人能引得万人空巷，倾国倾城，程大雷也不大相信。
<br/>　　但毕竟心中有些好奇：这娃究竟什么模样。
<br/>　　这恐怕也是帝国许多人，尤其是男人的共同想法。
<br/>　　目前，程大雷还没想到破局的法子，仍旧在筹划准备的阶段。苏樱与樊梨花已被接了回来，程大雷不大去后院，一是的确很忙，二来是因为有些做贼心虚。
<br/>　　自己和柳芷的事不知被苏樱看穿没有，也不知她会如何反应。
<br/>　　许多时候，很多事是不需要说话的。程大雷做贼心虚，苏樱只是瞟一眼，便能推测出什么。
<br/>　　其实苏樱刚回来，便提了柳芷的身份，给她安排单独的院落，有丫鬟伺候。还与她单独聊过，言谈话语间点了点她。
<br/>(本章完)
