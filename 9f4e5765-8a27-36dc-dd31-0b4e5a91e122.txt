　　“我了个擦，这么长的名字？”
<br/>　　林峰顿时呆住了。
<br/>　　叮！
<br/>　　恭喜宿主成功领悟天上地下唯我独尊八荒六合神功第一重！
<br/>　　“这就领悟了？有毛用？”
<br/>　　林峰握了下拳头一拳就是轰出。
<br/>　　砰的一下。
<br/>　　一股拳风呼啸而出居然直接把墙壁给打了个窟窿。
<br/>　　“我去，拳风？我都可以打出拳风来了？”
<br/>　　林峰震惊了。
<br/>　　要想打出拳风，不仅仅需要力道，还需要发力的配合，这就和剑啸一个道理，同一把剑，有的人刺出去，剑会呼啸，而有的人刺出去，剑脱手而飞，屁声音没有，主要就是一个发力的要诀。
<br/>　　当然了，前提是你得有一定的力量为基础。
<br/>　　“我这个丝毫没有学习过任何技击搏斗的人都可以打出拳风了，牛逼啊，这还只是第一重，除此之外，我感觉我的身体各方面素质甚至是我的听力目力还有神秘的第六感都敏锐了许多。甚至是我的小丁丁都似乎变大了一圈，原本只是亚洲人平均尺寸，现在最起码也是黑人级别的了，嘿嘿，苏老师以后可要幸福了。”
<br/>　　林峰看了眼自己裤裆，我了个擦，差点都以为这不是自己的宝贝了，太大了，一只手都握不下，绝对的非洲大黑鸟级别。
<br/>　　“呜呜呜。”
<br/>　　而这时。
<br/>　　那个大块头叫了两声，对方总算是恢复了意识，但是身体上还是酥麻的，意识朦胧。
<br/>　　林峰站起身，这一次空投还算不错，拿着武士刀来到那大块头面前。
<br/>　　“我，我错了，大哥，放，放我一马吧。”
<br/>　　大块头眼巴巴的看着林峰。
<br/>　　林峰目光微微寒冷。
<br/>　　“对不起，斩草不除根春风吹又生。”
<br/>　　林峰屈指一弹，一抹气劲直接犹如刀片一样，刺入了他的咽喉。
<br/>　　咕噜噜。
<br/>　　大块头双眼瞪大如牛，带着怨毒和不甘，抽搐了两下，就彻底气绝而亡了。
<br/>　　林峰摇了摇头，他不是喜欢杀人的变态，但是有时候，局势逼迫他这么做。
<br/>　　就好像末世之前有一句俗话，到了最后，我们都变成了自己所讨厌的人。
<br/>　　离开这个院子之前，林峰搜了一圈，这个院子内有许多食物，还有冰箱，以及一个大功率柴油发电机，除此之外则是些武器。
<br/>　　林峰统统带走尤其是这柴油发电机，非常重要。
<br/>　　林峰能装的都带走，然后离开了这个狼窝的据点。
<br/>　　“还特么吃烤肉？”
<br/>　　林峰把他们没吃完的烤肉全部吃掉填饱肚子后，把这个烤架和煤气罐也都顺便带走，什么都不留下。
<br/>　　离开院子，林峰拔出武士刀杀了几个门口的丧尸，然后一路狂奔。
<br/>　　回到学校。
<br/>　　“峰哥！”
<br/>　　李广等人道。
<br/>　　“狗子他们呢？”
<br/>　　林峰看他们车似乎还没回来。
<br/>　　“之前回来过一次说是出去杀丧尸了。”
<br/>　　李广道。
<br/>　　“杀丧尸？”
<br/>　　林峰知道，估计是苏云和他表哥央求的，毕竟这年头，真正有远见的人，都会想着杀丧尸提升实力。然后等待军警的解救前往幸存者营地。这就是林峰正在做的事情，他一直相信，军警肯定还在。
<br/>　　当然了这种人还是少数。
<br/>　　更多的人，还是在一个地方躲着不敢出来，食物吃完了，然后就吃老鼠药自杀的，这种情况也是很多，末世来了，许多人都会选择自杀，倒不是脆弱，实在是无法面对，与其让丧尸啃的稀烂，倒不如吞药自杀，也算给自己的一种解脱了。
<br/>　　到了下午。
<br/>　　陈勾他们回来了。
<br/>　　有陈勾在，苏云和吴三实力都有所提升。
<br/>　　吴三算是突破到了二阶，而苏云都直接三阶了。
<br/>　　“妈的，这一次老师你可要让我睡大腿了，这一上午都是我当保镖在旁边守着，差点就被丧尸给抓伤了。”
<br/>　　陈勾道。
<br/>　　“我去，表弟，牛逼啊，睡你们老师大腿？我上学那会可从来没有过这么大胆的想法。”吴三伸起大拇指。
<br/>　　“少胡说八道。”苏云瞪了眼陈勾就直接离去了。
<br/>　　“哎，看来，我还是无法走进苏老师的内心啊，这林峰到底给苏老师什么了，居然可以睡大腿。”陈勾叹气。
<br/>　　“表弟，要不今天大表哥的大腿给你睡，也算是感谢你的救命之恩。”
<br/>　　吴三嘿嘿笑道。
<br/>　　“滚吧，我怕把你压死。就你这一米八还不够一百一十斤的，腿比女人都细，我有时候都怀疑是不是姨妈他小时候虐待过你了。”
<br/>　　陈勾翻了个白眼直接离去了。
<br/>　　留下李广等人面面相觑，“现在的男人怎么都这么喜欢搞基。”
<br/>　　回到教室内。
<br/>　　林峰直接拿出了一个大木桶，这个大木桶就是武侠电视之中那些泡澡的大木桶。
<br/>　　林峰也是在那个院子里找到的，直接给他弄回来了。毕竟有物品栏在。
<br/>　　物品栏是个神奇的地方。
<br/>　　放一颗米又或者是放一艘大体积的东西，都算1个物品格子。
<br/>　　“还挺干净。”
<br/>　　林峰拿水稍微冲洗了一下，这若是让其余人看到了估计会说他奢侈，说他是魔鬼，居然用水来洗东西。
<br/>　　林峰随后就拿出了一些晶石，这都是他回来一路上杀的，不多，也就二三十个，林峰把晶石全部捏碎。
<br/>　　里面的液体瞬间就是流入了木桶之中，林峰这一次主要是要试验一下，这个液体是不是只能口服。
<br/>　　林峰三下五除二脱掉衣服就是要做进去。
<br/>　　“林峰，我说……”
<br/>　　也就在这时。
<br/>　　苏云直接推开了教室门进来了，还要说什么顿时看到了林峰，然后就愣住了。
<br/>　　一双好看的美眸之中甚至露出了一抹恐惧。
<br/>　　“这么大？”
<br/>　　“老师，你进来怎么不敲门啊，不是说好了，我们一人一个教室当房间的吗？”
<br/>　　林峰顿时无奈道。
<br/>　　苏云脸色泛红，太大了，现在的女孩子，十个八个看过成人电影，所以她也不例外，毕竟她就算不看，以前读书那会，其余室友也都看，那声音总是可以听到，男人对性好奇，女人也自然，大家都是人。
<br/>　　所以她知道许多东西，可是，她没有见过这么大的，这种尺度的一般都是非洲人了。
<br/>　　“我，我走了。”
<br/>　　苏云一颗小鹿狂跳似的离开了这里，回到自己教室内。
<br/>　　苏云还感觉脸颊发烫，脑海里就是刚才挥之不去的那一幕。黑色草丛之中一条巨龙在怒吼。而且刚才那个巨龙还抖了两下，就好像在怒吼一样。
<br/>　　“怎么回事，他之前的明明没有那么大的啊，怎么现在这么大了，难道他要发育，开始长身体了。”
<br/>　　……
