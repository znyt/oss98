　　连绵的云层之中，两道身影急速飞驰，正是苏莫和秦不死。
<br/>　　两人离开炼器师公会之后，便一路疾驰，向北陵星上一处叫通宇谷的地方飞去。
<br/>　　通宇谷，是一处北陵星通向其他星辰的中转站，其上有星空传送大阵。
<br/>　　通过大阵，便可以前往其他星辰。
<br/>　　这些，柴管事都已经对苏莫交代的一清二楚，他们两人只要按照路线前进便可。
<br/>　　在古灵星河，一般距离较近的星辰，都可以通过星空传送大阵来往。
<br/>　　但有些星辰，间隔太远的话，就算是布置了星空传送大阵，也无法传达到目的地，只能靠飞行。
<br/>　　好在去往古周星，并不需要飞跃星空，只需一直乘用传送阵便可。
<br/>　　但是，即便是乘用传送阵，其中也需要中转几十次，并不能一次性到达。
<br/>　　“苏莫，那巫族的尊者，好像真的没有追来！”秦不死眸光扫视四方，惊讶的说道。
<br/>　　他们已经离开了炼器师公会整整一天了，也没有受到任何袭击，这可能真的甩掉了巫族的尊者。
<br/>　　要知道，他们现在可是没有什么防护，不仅没有用巫族天才做人质，也没有撑起玄力气罩。
<br/>　　若是巫族尊者追来的话，估计早已出手了吧！
<br/>　　“应该没有追来，我们已经安全了！”苏莫点了点头，他还是比较信任柴管事的，想必对方早已拦住了巫族尊者。
<br/>　　“逃亡了这么久，终于自由了！”秦不死满脸豪气，这无穷无尽的星空，将是他大展拳脚的地方。
<br/>　　“对了，苏莫，你去神武学府，那我呢？”秦不死问道，他已经知道了苏莫要去什么地方。
<br/>　　神武学府是武道学院，应该只收年轻的天才，他活了几千年了，肯定不可能加入神武学府。
<br/>　　“我们先到古周星再说吧！到时候我也要安排苍穹神宫的人！”苏莫道。
<br/>　　而后，两人一边闲谈，一边赶路。
<br/>　　北陵星很大，两人足足飞行了五日，才来到了所谓的通宇谷。
<br/>　　这是一座巨大的山谷，占地方圆千里。
<br/>　　山谷中修建了一座白玉平台，白玉平台占地方圆万丈，其上布置着十数个星空传送大阵，不时的闪烁出冲天的光芒。
<br/>　　而在山谷的四周，也有大量的守卫把守，守护着山谷的安全。
<br/>　　谷中武者进进出出，非常的热闹。
<br/>　　唰！唰！
<br/>　　破空声响起，苏莫和秦不死两人落在了谷口。
<br/>　　随后，两人便大步的走进了山谷中。
<br/>　　谷中白玉平台的下方，排放着一张长约八丈的金色案几，十余名强者端坐在案几之后。
<br/>　　所有欲乘用传送阵的人，都需缴纳费用。
<br/>　　苏莫和秦不死，径直便走了过去，来到了案几之前。
<br/>　　“前辈，我们去昌羽星！”苏莫向案几后的一名中年强者道。
<br/>　　昌羽星便是他们的第一站，到了昌羽星之后，还要继续转乘传送阵。
<br/>　　“一人一块下品灵晶！”那名强者淡淡的说道。
<br/>　　“这么贵？”苏莫闻言心中大惊，传送一次居然费用高达一块下品灵晶！
<br/>　　一块下品灵晶啊，这可是相当于十亿极品灵石，这也太贵了吧！
<br/>　　“支付不起就离开吧！”那名强者见苏莫的脸色，立刻淡漠的喝道。
<br/>　　苏莫心中暗叹，他看到其他的人都是支付灵晶，而且有的人还一次支付两块灵晶，也只能支付灵晶了。
<br/>　　就算再贵，他也要离开。
<br/>　　看来星际传送，也不是一般人能享受的待遇。
<br/>　　随即，苏莫取出两块下品灵晶，递给了这名强者。
<br/>　　“右边第三个传送大阵！”
<br/>　　那名强者收取灵晶之后，一指白玉平台的右边区域。
<br/>　　“嗯！”苏莫和秦不死点了点头，随即便登上了白玉平台。
<br/>　　白玉平台上分为十数个区域，刻画着十数个庞大的阵法，每一个阵法都有无数繁杂的纹路构成，复杂无比。
<br/>　　而且每一个阵法中央，都刻有字迹。
<br/>　　而右边第三个大阵，中央刻有‘昌羽’二字，显然这便是通向昌羽星的大阵。
<br/>　　此刻，大阵并未启动，其上已经或伫立、或盘坐着二十余名武者。
<br/>　　苏莫两人走进了大阵之中，静静的等待了起来。
<br/>　　看来这大阵，是要聚集足够多的人，才会启动了。
<br/>　　苏莫没有等待太久，仅仅是小半个时辰的时间，大阵中便聚集了两百余人。
<br/>　　而这个时候，大阵终于启动了。
<br/>　　只见原本毫无动静的大阵，那无数繁杂的纹路之中，突然间有刺目的白光流转。
<br/>　　而后，白光形成了一道直径达百余丈的光柱，冲天而起，洞穿九霄。
<br/>　　苏莫等人均是完全被光柱包裹，他们的身影缓缓的消散了白玉平台上。
<br/>　　苏莫只感觉仿佛身处惊涛骇浪之中，眼前是迷蒙般的光晕，巨大的晕眩感不断的侵蚀着他的大脑，他努力的保持头脑清明。
<br/>　　这个过程无比漫长，一分一秒都仿佛放慢了百倍。
<br/>　　不知道过去了多久，可能是半个时辰，也可能是半日，更可能是半个月。
<br/>　　苏莫眼前迷蒙的光晕消散，他的眼前恢复了清明，转头看向四方，他发现自己身处一座恢弘广阔的巨型大殿之中。
<br/>　　大殿中布满了数百个传送大阵，来来往往的武者众多。
<br/>　　这里居然也是一处星空传送大阵的中转站，这让苏莫微微意外，如此的话他们就能节省很多时间了。
<br/>　　而后，苏莫和秦不死，便在此处再次乘用传送阵，去往下一颗星辰。
<br/>　　时光飞逝，苏莫和秦不死两人，辗转一颗又一颗星辰。
<br/>　　有的时候，他们到达一颗星辰之后，传送到的地方，便是星空传送的中转站。
<br/>　　但有的时候并不是，他们还需自行赶路，前往星空传送大阵的所在。
<br/>　　就这样，苏莫和秦不死，整整耗费了五个多月的时间，辗转了七十多颗星辰，才最终到达了古周星。
<br/>　　七十多颗星辰，耗费了近一百六十块下品灵晶。
<br/>　　本来苏莫身上的下品灵晶并不够，无奈之下，他和秦不死几乎拿出了全部的家当出去售卖，才凑够灵晶。
<br/>　　秦不死已经身无分文，而苏莫将身上所有的战利品，各种兵器、功法等宝物，也几乎售卖一空了。
<br/>　　就连从巫娴手上夺下的那双薄如蝉翼的尊级手套，他都拿去卖了。
<br/>　　一座恢弘的古殿门前，苏莫和秦不死联袂走了出来。
<br/>　　望着天穹之上火红的骄阳，苏莫长长的吁了口气。
<br/>　　“古周星，这是新的起点！”
