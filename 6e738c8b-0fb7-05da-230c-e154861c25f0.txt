　　清长风目光望着远处，喃喃道：“你说的，都是他想让我们看到的，至少还有一些事，是我们看不到的。第一，他怎么知道星君的肉身在我这里？”
<br/>　　白泽停顿了几秒，道：“有内应。”
<br/>　　“不，这件事就没几个知道，都是不可能泄密的。”
<br/>　　“既是内应，自然要百般调查，偶然发现，或也有此可能。”
<br/>　　清长风没纠结这个问题，进而说道：“第二，我跟道风也有仇，但我绝不会跑风之谷去行刺他，哪怕知道他只剩肉身在那，我也不去，否则可能就回不来了，他也不是傻子，为何敢来？”
<br/>　　白泽听他这么一说，也思忖了下，道：“或许他太过自信，以为能全身而退？”
<br/>　　“他还将自己一分为三，只凭元神之体前来刺杀，最要紧的是，他在真相大白之后、元神不走，坚持行刺，哪怕他得手了，又如何能够退走？”
<br/>　　白泽倒吸一口气，沉吟半晌道：“的确，行刺对手，本就是有机便进，无机便退，断没道理把自己赔进去，那么，真人你的意思是？”
<br/>　　“不知道。”清长风摇摇头，“我总感觉，他像是故意寻死似的……”
<br/>　　白泽笑道：“哪里有人会故意寻思，修为越深厚，本就越是怕死。这件事中必有蹊跷，但真人也不必多想，至少道风是真的死了，众目睽睽，这个却是做不了假，纵然精魄飞走，有机缘聚魄而生，一身修为却是没了，倒也无须多虑。”
<br/>　　清长风缓缓点头，也只能这样安慰自己了，端起茶杯，用力吸了一口，冲白泽笑道：“白仙长，咱们是老朋友了，你就直说吧，你来找我，究竟何事。”
<br/>　　白泽笑了笑，“道风一死，鬼域形势必乱，风之谷怕是支撑不住阴司和太阴山的双重压力，早晚解体，那白云城中，却有不少奇珍异宝，真人不动心？”
<br/>　　清长风一听就明白了，道：“凡俗之物，仙长居然也有兴趣？”
<br/>　　“山河社稷图，鱼肠剑，单这两件，便有无上价值，谁人能不心动。”
<br/>　　清长风并没否认，道：“须得我师妹修成神功，从异界返回，届时或可去风之谷搜罗一番，仙长看上什么了？”
<br/>　　白泽微笑。
<br/>　　“神器，我不跟你们争，我只求一人。”
<br/>　　“人？”清长风惊讶。
<br/>　　“龙族公主，椒图。她被道风放出，现今便在风之谷中。”
<br/>　　“龙族公主……”清长风更是不解，“仙长却是要她做什么，难不成要结道侣？”
<br/>　　这本是一句玩笑话，白泽竟然没有否认，徐徐说道：“她与我，却有孽缘，隐秘之事，便不细说，待星君归来，攻打风之谷之日，我当尽一份微薄之力，恳请真人和星君届时帮我擒住南海公主。”
<br/>　　白泽是上古灵兽，在轩辕山地位很高，能这么放低姿态央求自己，清长风不敢怠慢，拱手说道：“白仙长言重了，只要能见到那南海公主，一定助你擒住她。”
<br/>　　心中却是有些不屑，这老家伙修炼了那么多年，居然凡心未泯，什么孽缘，不就是好色么，打人家公主的主意。
<br/>　　阴司，酆都城，冥王大殿。
<br/>　　整个酆都城最为雄伟的建筑，一座宫殿，建在了几十层的白玉石阶上，两扇朱红色的大门，高达数十米，站在大殿门外，所有生灵都会感到自己的渺小。
<br/>　　没有太多金碧辉煌的装饰，青灰色的砖墙，朱红色的屋顶，古朴而又庄重，法度森严。
<br/>　　门外是一个巨大的广场，两边各自长着一棵树，一棵梧桐，一棵榆木，都长得巨大，树干粗得几个人都抱不过来。
<br/>　　两棵树巨大的树冠，将整个门口和附近的院墙都盖住了，更是为这座宫殿增添了神秘的气息。
<br/>　　没有守兵，三届之内，还没有生灵敢来这里找茬，平时住在枉死城里的那些鬼魂，每一个敢来这附近转悠的。
<br/>　　两棵树微微摇晃着树冠，带着某种舒服的节奏。
<br/>　　这时，一个书生打扮的人出现在了空旷的街道上，径直走了过来。
<br/>　　两棵树顿时停止了摇晃，树冠散开，朝着来人的方向倾斜过去，这动作好似人间在弯腰打量。
<br/>　　“原来是府君大人，见过了。”那棵梧桐中间，突然传出来一个女子的声音。
<br/>　　崔府君微微点头致意，又看了看边上那棵榆树，榆树的树冠也晃动了一下，似乎在打招呼，却没有开口。
<br/>　　梧桐树说道：“府君大人不要见怪，他还是老样子。”
<br/>　　崔府君扫了那榆树一眼，在树冠连接树杈的地方，有数道深深的刻痕，形成了四个汉字：
<br/>　　天地不仁。
<br/>　　崔府君当然知道这四个字的来历，那是很久很旧以前，无极鬼王反出阴司的时候，从冥王大殿中离开时，遭到这两棵树的阻拦，无极鬼王制住他们，在榆树上刻下了这四个字，用以嘲讽三界法规。
<br/>　　因为这四个字，是用极阴鬼气刻下去的，是刻在灵魂深处的烙印，要是洗去，树妖也就死了，因此就算是大帝也没办法，这榆树也只能带着这四个字过活。
<br/>　　从那以后，他的情绪就不好了，总默默呆着，不爱跟人说话。
<br/>　　崔府君一路来到两扇朱门前，在朱门上按了两下，大门顿时虚化，形成了一幅扭曲的太极双鱼图，旋转起来，中间是一孔虚空般的所在。
<br/>　　崔府君大步走了进去。
<br/>　　没有经过回廊和前厅等那些冗长的建筑，直接来到正殿门外。
<br/>　　门没关，崔府君直接进去，先到三清神像前面上了一炷香，然后走入偏殿。
<br/>　　房间里已经有好几个人了。
<br/>　　钟馗、招工转轮王，阎罗王，这三位一方大佬，此刻都坐在地上的蒲团上，加上他，三王一判都到齐了。还有一个脸黑得相爱那个到炭、但是身穿一身大红袍的黑须男子，相貌极其威严，跟画了京剧的脸谱似的。
<br/>　　“赵元帅也来了。”崔钰冲他拱了拱手。
<br/>　　“正好回师，师尊召唤，怎能不来。”
<br/>　　这汉子就是大名鼎鼎的赵公明，本职是阴兵元帅，因为总穿一套红衣，给人感觉红红火火的，被人间认为是财神，总当做财神来拜，至于拜他管不管用，只有问他才知道了。
