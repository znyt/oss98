的话，就省下不少的功夫呢。”凌经理道。<br/>　　<br/>　　    “对了，陈老板，不知道你的那个土鸡怎么卖？”<br/>　　<br/>　　    “土鸡现在还没有到出栏的时间，都没有长成，而且数量少，恐怕供应不了你们。”<br/>　　<br/>　　    “啊土鸡不能供应？”凌经理顿时有些失望。<br/>　　<br/>　　    “放心好了，我已经安排工人去采购鸡苗了，等再过一阵子，绝对可以供应给你们。”陈默道。<br/>　　<br/>　　    凌经理点了点头，也只能这样了。他也去鸡舍里看过，也就上千只鸡子。还真不够，所以陈默的也对，等数量上来了再。不过这土鸡真好吃，实在是不能放过啊。<br/>　　<br/>　　    “陈老板，你看这样成不成，鸡子等以后供应也没有问题，但是能不能先每天供应给我们少许的数量呢？”凌经理问道。<br/>　　<br/>　　    陈默有些哭笑不得，这些鸡子也就不到一千只，要怎么供应才能算少量的呢？<br/>　　<br/>　　    “凌经理，一个是鸡子还没有出栏，二个是这些土鸡价格比较贵，三个是少许数量是多少？”<br/>　　<br/>　　    “没有出栏也没有什么，都能下锅！”这话的，让陈默就是有些感觉，这个凌经理似乎有些不靠谱啊<br/>　　<br/>　　    “每天五十只，每斤按照50元？”凌经理接着道，陈默听的满头黑线，这是多欠鸡肉！<br/>　　<br/>　　    “不行，数量太多，价格太少！”陈默道。<br/>　　<br/>　　    “那你想要什么价位？”<br/>　　<br/>　　    “每天只能给你们10只土鸡，价格么每斤按照100元，活鸡！”<br/>　　<br/>　　    凌经理想了想，点头道：“行！就按照陈老板的意思！没有想到陈老板这么会做生意！”就一次几千元的采购，他自己就能做主，尤其是吃过这里的鸡子后，绝对的不能忘怀。凌经理此刻都能想到那帮头头要是吃到如此美味的土鸡后，会是什么表情。<br/>　　<br/>　　    嗯！回去后就让厨师做一下，然后看那些头头们吃鸡肉的表情，最好是拍下来。嘿嘿！<br/>　　<br/>　　    陈默听了凌经理的话，也就笑了笑，道：“都是本生意，靠大家捧场！”<br/>　　<br/>　　    两人有商量了一下供货的时间和打款的时间，然后这合同也就开始执行了。不过首次要货的数量，等凌经理回去后统计好以后，在拉货的时候明。另外就是陈默这边，做好蔬菜成熟表，有新品种蔬菜成熟后，要及时告知。<br/>　　<br/>　　    当然，这些都没有问题，陈默也都是一一应承下来！<br/>　　<br/>　　    如果感觉不错，请推荐和收藏一下，谢谢了！<br/>　　<br/>　　    7***<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
