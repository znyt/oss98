　　苏莫思索了片刻，还是决定走一步看一步吧！
<br/>　　好在神武学府够强大，乃是十三个大势力共同组建，也不惧天冥星巫族。
<br/>　　只是他目前实力太弱了，若是巫族之人暗中杀他，实在是太容易了。
<br/>　　随即，苏莫面部肌肉蠕动，稍稍改变了一番容貌，免得被巫人王认出来。
<br/>　　虽说巫人王没有见过他的真容，但天冥星巫族之人，估计都熟知他的容貌。
<br/>　　“苏兄，你这是？”谢彬见苏莫眨眼间容貌大变，变的面相极为普通，不由得有些疑惑。
<br/>　　“谢兄，碰到个仇人，不想被认出来！”苏莫低声解释道。
<br/>　　谢彬闻言恍然，随即点了点头，他也没有多问。
<br/>　　时间流逝，拍卖会场中的人越聚越多，没过多久便聚集了七八十万人。
<br/>　　苏莫暗暗心惊，这么多人，一人一块下品灵晶的入场费，这拍卖会简直赚飞了。
<br/>　　这个时候，拍卖会场的大门关闭了起来，拍卖会即将开始了。
<br/>　　只见北部方位，那座巨大的宫殿之中，走出了一名麻衣老者，大步的走到了中央拍卖台上。
<br/>　　这名老者鹤发童颜，面色红润，仿佛十五六岁的少年一般。
<br/>　　老者乍一出现，整个拍卖会场，各种嘈杂的声音迅速消失，瞬间安静了下来。
<br/>　　鹤发童颜的老者，伫立在拍卖台之上，见整个拍卖会场已经安静了下来，面上顿时露出了笑意。
<br/>　　“诸位，老夫玄净子，人称玄净尊者，想必在坐的众位，大多都听说过老夫的名号吧？”鹤发童颜老者抱拳环顾四周，朗声说道。
<br/>　　“哈哈！我等久闻玄净尊者的大名！”
<br/>　　“前辈可是古灵星河最有名气的拍卖师之一，早就如雷贯耳了！”
<br/>　　“由玄净尊者前辈主持此次拍卖会，想必非常精彩！”
<br/>　　靠近拍卖台席位的人，不少人纷纷开口，个个面带笑意。
<br/>　　玄净尊者，乃是拍卖会主办方所在大商会的长老，也是最富盛名的拍卖师，在整个古灵星河之内，举办过上千次大型的拍卖会。
<br/>　　所以，古灵星河之内，有见识的武者，基本上都知道玄净尊者。
<br/>　　当然，苏莫不包括在其内，在这个拍卖场内，估计谁都比他有见识。
<br/>　　“呵呵！这一次拍卖会注定不会让众位失望！”
<br/>　　玄净尊者面带和煦笑容，略一沉吟，朗声道：“在拍卖正式开始之前，本次拍卖会要增加一些彩头！”
<br/>　　“哦？什么彩头？”
<br/>　　“对啊！玄净尊者前辈，难道开始拍卖之前，有什么重宝要拿出来观赏？”
<br/>　　“尊者前辈，到底是什么彩头啊？”
<br/>　　众人听闻玄净尊者之言，顿时来了兴趣，既然能被玄净尊者说成是彩头，想必非同小可。
<br/>　　苏莫亦是来了点兴趣，拍卖会能有什么彩头？无非是某种重宝而已。
<br/>　　一时间，所有人都露出了好奇的目光，静等玄净尊者揭晓。
<br/>　　玄净尊者见所有人的兴趣都被调动了起来，面上笑意更甚，他环顾四周，朗声道：“想必众位都知道，真圣界今年就要开启了，这是古灵星河年轻一代的盛会，此次彩头，便是和真圣界有关！”
<br/>　　和真圣界有关？
<br/>　　众人闻言，均是一脸诧异，拍卖会和真圣界有什么关系？
<br/>　　虽然来参加此次拍卖会，有很多人都是想淘一些宝物，为去真圣界做准备，但拍卖会本身和真圣界并无关系吧！
<br/>　　这时，只听玄净尊者继续道：“整个古灵星河之内，无数的年轻天才，都想在真圣碑留名，但是古灵星河何等之大？天才何等之多？而能在真圣碑留名之人，实在是少之又少！”
<br/>　　说到此处，玄净尊者停顿了一瞬，见众人均是在认真的聆听，稍稍提高了音量，道：“为此，这次拍卖会的主办方，几个店铺遍布整个星河的大商会，耗费大量的人力和物力，制作了一份真圣榜！”
<br/>　　“真圣榜？”
<br/>　　所有人闻言均是心中有些疑惑，什么真圣榜？是什么榜单？
<br/>　　“真圣榜上，收录了两百名古灵星河武帝境之内最强大的天才，也是最有希望在真圣碑上留名的妖孽天才，每一位天才的排名，都是根据其天赋、战力、修为等等为参考，而定下最终的名次！”玄净尊者朗声说道。
<br/>　　此话一出，整个拍卖会场短暂的安静了片刻，随即一下子火爆了开来。
<br/>　　“榜单呢？都有哪些天才！”
<br/>　　“我师兄周盛天资妖孽，名震一方，有没有上榜？”
<br/>　　“玄净尊者，到底有哪些天才上榜了？赶快拿出来看看吧！”
<br/>　　“……”
<br/>　　大量的武者开口，均是有些迫不及待了起来，好奇哪些人能上榜。
<br/>　　就连一些真正的绝世天才，都是眸中精光闪耀，对此大感兴趣，暗暗猜测自己有没有上榜。
<br/>　　这可是一个扬名的机会，只要上了真圣榜，那可就是真正的名扬古灵星河了。
<br/>　　苏莫对此倒是没有多大的兴趣，反正他是不可能上榜的，第一他修为较低，第二他也就是前一段时间，才在神武学府扬名而已。
<br/>　　再者，他也不希望上榜，搞得人尽皆知，毕竟他有巫族这个大敌，不宜太过显露声名。
<br/>　　“众位稍安勿躁！请听老夫说完！”
<br/>　　玄净尊者抬起手掌压了压，少倾之后，待众人安静了下来，才继续道：“这份榜单将古灵星河内所有种族的天才全部囊括在内，虽然我们经过了严格的斟选，但可能还是会有纰漏，所以有何不妥之处，还请大家勿怪，全当一个乐子！”
<br/>　　在榜单公布之前，玄净尊者便给了自己一个下去的台阶，也给了背后几个商会势力一个台阶。
<br/>　　毕竟，古灵星河太大了，武帝境的天才太多了。
<br/>　　想要排出武帝境之内，最强的两百名天才，难度非常的大，几乎不可能完全正确，这也是以前没有人做这种榜单的原因。
<br/>　　比如两个上榜的天才，排名肯定有高有低，但排名低之人，肯定不会认为自己不如排名高之人，除非两人实际的交过手。
<br/>　　还有很多没有上榜的妖孽天才，肯定不会认为自己不如上榜的那些天才。
<br/>　　总之，这个真圣榜一旦公布出来，会有太多的不确定性，太多的异议，定然会有很多人不满，甚至嗤之以鼻。
<br/>　　所以，这个榜单的发布，几个参与的大商会，也是要承受不小的压力。
