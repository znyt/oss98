　　“这个家伙太厉害了吧？这么轻松的就击败了阳云！”
<br/>　　“真是厉害啊，看来此人是属于顶尖的妖孽天才！”
<br/>　　“龙腾的两个奴仆，居然天赋都是如此厉害！”
<br/>　　迷宫城之外，很多人看到了苏莫和阳云的战斗，很多龙族之人甚至是奴仆，都是议论纷纷。
<br/>　　阳云来到龙族百余年，本身的修为已达四品，在龙族小有名气。
<br/>　　众人对于阳云的战力，也是极为了解，一般同阶的龙族天才，都不是阳云的对手。
<br/>　　但现在，在同阶之下，阳云简直没有还手之力。
<br/>　　“这么夸张？”龙城的脸色阴沉了下来，因为在他看来，苏莫的实力，比巫铨也要强上一大截，和那个炎空应该相差无几了。
<br/>　　龙洌、龙烨和龙双双三人，均是面色凝重，显然也是惊讶于苏莫实力的强大。
<br/>　　“龙腾，你这个奴仆叫什么？”龙烨突然问道。
<br/>　　“他叫屠霄！”龙腾微笑着说道，这是当初苏莫来到龙族之时，告诉他要用的名字。
<br/>　　毕竟，苏莫在龙族，肯定不能暴露真实姓名。
<br/>　　这个名字，寓意很深，龙腾明白，苏莫将来是要杀紫霄。
<br/>　　“屠霄？”
<br/>　　龙烨闻言，点了点头，继续道：“此奴修炼的武学道统很是奇怪，之前是火属性道统，居然又改变了！”
<br/>　　龙腾闻听此言，眉头微挑，这龙烨居然会注意这些细节。
<br/>　　“呵呵，这应该才是他的真实道统，之前在隐藏实力罢了！”龙腾笑着说道。
<br/>　　同时，他心中暗暗祈祷，希望苏莫不要暴露的太多，尽量隐藏实力。
<br/>　　“阳云身上有两件宝物，这个屠霄总共已经得到了四件宝物，目前的确是最大的赢家了！”龙烨叹声道。
<br/>　　五百多件宝物，在这么大的迷宫城之中，有这么多的人抢夺，可谓是僧多粥少。
<br/>　　目前，迷宫城之中还有数千人，大多数人一件宝物都没有得到。
<br/>　　得到两件宝物的人，都只有寥寥几人。
<br/>　　而这个屠霄，已经得到了四件宝物，可谓是最大的赢家了。
<br/>　　……
<br/>　　迷宫城之中，苏莫击败了阳云之后，继续跟着出口移动。
<br/>　　接下来的一段时间之中，不时的有人来到出口，无一列外，全部被苏莫击败。
<br/>　　小半个时辰的时间，他便又击败了一百多人，无一合之敌。
<br/>　　所有人的兵器，以及身上的储物宝器，全部被他抢夺一空。
<br/>　　他也不再查看，不管有没有宝物，全部一股脑的收了起来，准备回去之后再仔细的查看。
<br/>　　苏莫的实力之强，让得迷宫城之外，无数的人震惊。
<br/>　　而经过这么久的时间，迷宫城之中的争夺，也已经到了尾声。
<br/>　　整个迷宫城之中，只剩下了几百人，其他人全部被淘汰了出去。
<br/>　　剩下的几百人，大多都在争夺之中得到了一件宝物，有少部分人得到了两件，也有少部分人并未得到宝物。
<br/>　　这个时候，所有人都在寻找出口了，因为，随着人数的减少，在浩大的迷宫城之中已经很难找到其他人了。
<br/>　　况且，众人也知道，到了现在还能留在迷宫城的人，大多都是战力超强之人。
<br/>　　所以，除了少部分没有得到宝物之人，其他得到宝物之人，都在寻找出口了。
<br/>　　“居然就得到了两件宝物！”
<br/>　　一处通道之中，司空炎大步而行，面现郁闷之色，忙活了这么久，他只得到了两件宝物。
<br/>　　虽然在二十万人之中，他能抢夺到两件宝物，已经算是非常厉害了，但这还是让他有些失望。
<br/>　　毕竟，他的实力可是碾压所有人，这无数的龙族奴仆，没有人是他的对手。
<br/>　　叹了口气，司空炎开始寻找出口，他知晓出口在最外围，所以只要到达最外围，找到出口便不是难事。
<br/>　　仅剩的一些奴仆，纷纷寻找出口，而苏莫遇到的对手，自然也是越来越强。
<br/>　　他跟着出口不断的移动，所有想从出口出去之人，全部被他击败。
<br/>　　嗖！嗖！
<br/>　　这时，有两道破空声响起，传到了苏莫的耳中，显然有人在飞速而来。
<br/>　　唰！唰！
<br/>　　少倾，两道妙曼的身影，出现在在了苏莫的视线之中，这两人正式龙洌的两位女奴，冰儿和凝雪。

<br/>　　这一刻，迷宫城之外，所有人的目光，都汇聚在了苏莫和冰儿、凝雪的身上。
<br/>　　从之前的表现来看，苏莫的实力非常强大，但是冰儿和凝雪，同样也很强大，而且两人联手之下，实力更强了。

<br/>　　“败了也没有什么，他身后就是出口，随时能逃出去！”
<br/>　　“我倒要看看，他能否战胜龙洌的两位女奴！”
<br/>　　众人议论纷纷，狭路相逢勇者胜，龙洌的两位女奴可不是开玩笑的。
<br/>　　“龙腾，谁能笑到最后，就看这一战了！”龙洌微微一笑，瞥了龙腾一眼。
<br/>　　随着这个屠霄很强大，但是他的两位女奴，同样强大。
<br/>　　只要她们打败屠霄，就能得到屠霄身上的宝物，他两位女奴就是最大的赢家。

<br/>　　因为，龙腾这两个奴仆，都太厉害了，让他的奴仆巫铨，黯然失色。
