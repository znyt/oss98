　　江行泽喜欢看吸血蛇吃人的场景，他觉得很美妙，但他万万没想到，自己有一天也会被吸血蛇吞进肚子里去。
<br/>　　这是黑化版的农夫与蛇，江行泽知道吸血蛇无比凶煞，还惯着它，养着它，帮着它吃人。
<br/>　　当这条蛇不在需要他帮忙的时候，那么他最后也会成为这条蛇的食物。
<br/>　　善有善报恶有恶报，老天何曾放过谁。
<br/>　　咯..．
<br/>　　吸血蛇一口吞，然后满意的打了一个饱嗝，露出了世界都清静了的表情，而涂小安淡漠的看着，无动于衷，也丝毫不意外。
<br/>　　这才是他认识的吸血蛇，无比凶煞，岂能让一个人类在自己面前叽叽歪歪。
<br/>　　对于江行泽被吃，他更是连悲悯都没有，因为原本在涂小安眼中，江行泽早就是个死人了。
<br/>　　只是没想到这个人命那么大，蹦跶了那么久，还将吸血蛇从山野集团给拐了出来。
<br/>　　佘山死了那么多人，陷入吃人怪兽的恐惧中。
<br/>　　这笔账都要算在江行泽的身上，他就这样死了，还算便宜他了。
<br/>　　不过涂小安也是故意让吸血蛇吃了他，因为江行泽助纣为虐，帮着吸血蛇让它吃人，如今他被吸血蛇吃掉，是最好的结果。
<br/>　　跳梁小丑没了，涂小安俨然的看着三十米庞大巍峨的吸血蛇，他甚至觉得吸血蛇比上古蛇精还要强大一丝。
<br/>　　蛇眸对着吸血蛇浑然一眯，：“给我洞察”
<br/>　　“叮...超等变异进化体吸血蛇，觉醒类蛇种，等级二十八级，掌控火焰之力”
<br/>　　“叮...宿主吞噬此蛇，完成系统任务，等级直接提升三级，开启第二阶段性的进化，可掌控对方的火焰之力”
<br/>　　系统的第二道声音随之出现。
<br/>　　闻此声，涂小安蛇眸明亮起来，果然如此，吞噬了吸血蛇，它的火焰之力就是自己的了。
<br/>　　另外光吃了吸血蛇他就可以直接提升三级。
<br/>　　其实这个等级是完成系统任务的奖励，照理说吞噬吸血蛇也应该得到大量的进化能量，但此蛇是作为第二阶段性的进化，所以就不计算在内。
<br/>　　不过吸血蛇整整二十八级，还真的比上古修蛇还要强上一丝。
<br/>　　不得不说吸血蛇的机遇，不比他涂小安逊色。
<br/>　　二十八级呀，比他足足高上八级，两条蛇其实差距是非常的大。
<br/>　　涂小安感受到了压力，但最大的压力还是来自吸血蛇的火焰之力。
<br/>　　这种属性能力，让涂小安又爱又恨，爱是如果自己打败了吸血蛇，它的能力也就成了自己的能力。
<br/>　　恨是打败拥有火焰能力的吸血蛇，可没有那么的简单。
<br/>　　他跟这条蛇就犹如世敌，两者只能活下一条来。
<br/>　　涂小安现在所做的，不单单是为了给自己的报仇，更是为了万千的生物。
<br/>　　此蛇活下来，不管是对人类世界，还是对大自然的生物都是一场巨大的灾难。
<br/>　　此时，两蛇遥遥相望，一个在天上，不过一米，一个在地面，拥有三十米的庞大巍峨蛇躯，但相同的是，两条蛇彼此对峙着，战意越来越浓，都恨不得一口将对方吞进自己的肚子里。
<br/>　　无需废话，今天唯有一战。
<br/>　　谁生谁死，全看实力，涂小安之前失败过一次，这次系统加身，他不会容许自己在失败。
<br/>　　一往无前，战意盎然，洒我热血，务必杀之。
<br/>　　蛇谷中，风吹过，杀意弥漫，两条蛇一天一地，散发出来的恐怖气息愈演愈烈，天地间充满了阴冷绝杀之意。
<br/>　　只见天上的金银蛇浑身闪烁电光，周身有一条一条银蛇在窜动，异象迸发，周围的温度骤然下降，冰天雪地，寒气袭人。
<br/>　　在看吸血蛇，三十米巍峨的蛇躯呈大火焚烧之势，就宛如一头火麒麟在世，炽热的温度将空气都给蒸发了，一丝丝白烟无形中腾腾而升。
<br/>　　“轰...轰...轰...”
<br/>　　一股无比寒冷的气息，跟一股无比炽热的气息在空气中先行碰撞，惊天动地的爆炸声和火浪一道道爆响起噼哩叭啦的震动。
<br/>　　无论那股气息，都能让万物都陷入死亡绝望中，纵然是万物之灵长的人类都不行。
<br/>　　纵然是一般的觉醒者也不行，这种恐怖残暴的气息，你置身其中，能碾压着你直接七孔流血而死。
<br/>　　两条蛇，几乎是站在了二阶的巅峰，三阶生灵不出，它们就是无敌的存在。
<br/>　　“吸血蛇，你可曾记得当初一位少年跟你在佘山密林大战了三天三夜”
<br/>　　激烈的战斗一触即发，可悬浮的金银蛇却发出一道咆哮的声音，宛如地狱归来的嗜血修罗。
<br/>　　吸血蛇微微一愣，气息都一泄，仿佛陷入了一场回忆中。
<br/>　　“你身负重伤逃了，而那位少年身中你的蛇毒，然后死了”
<br/>　　吸血蛇像是想起了，也懂了，暴戾道：“你是帮他来报仇的？”
<br/>　　“不...我是来帮自己报仇，也是帮那些被你残忍吃了的人报仇，你不死，天理不容”
<br/>　　“给我天打雷劈”只见涂小安蛇眸一道异芒闪过，眼中一片寒冷，他含词轻吐，周身电光沸腾起来，如雷神降临，君临天下，天谴于人。
<br/>　　天打雷劈四个字刚落下，吸血蛇的上空平地惊雷一般，炸裂了，一条粗大的银蛇显化，银蛇似乎将天空都给划破了脸，然后雷声隆隆的顺势而下。
<br/>　　劈..．
<br/>　　吸血蛇庞大的蛇躯躲无可躲，但下一秒，地面却出现了一个巨坑，尘土飞扬。
<br/>　　它巍峨如山的蛇躯不见了，一道嘶吼声出现，吸血蛇出现在了涂小安的下方，血盆大口张口，滚滚火焰，上空出现了一片火海，满天横流，疯狂的火浪一个接着一个，张牙舞爪地朝着金银蛇娇小的身躯吞噬而去。
<br/>　　热...太热...比煮沸的开水温度都不知道要高上几百倍。
<br/>　　涂小安放眼望去，四面八方全部被封锁，火焰滔滔，炽热无比，这是想将他直接给熟了。
<br/>　　凝..．
<br/>　　他的蛇眸浑然一眯，无形的意识力形成一道光罩笼罩在自己的身上。
<br/>　　“给我爆”
<br/>　　光罩旋即爆发出一股气浪，这满天横流的火焰，被气浪催枯拉朽的席卷出去，吹的四散开来。
<br/>　　火焰横流一散，涂小安在空中如若幻影，换了一个身位，非常的灵敏。
<br/>　　论体型，吸血蛇三十米，涂小安就算暴涨起来，也不过二十米，不占优势，还不如已小胜大，另外论肉身防御力，跟肉身力量，吸血蛇也好不逊色。
<br/>　　八级的等级差，不是摆设。
