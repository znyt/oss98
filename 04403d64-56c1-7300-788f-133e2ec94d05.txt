　　如今这二人，居然是联合在了一起，抢夺洞府。
<br/>　　不过这洞府，可是足足比起寻常洞府要大上四五倍，灵气也是极为浓郁。
<br/>　　就算是两个人占据，也比起寻常洞府得到的灵气要多得多。
<br/>　　这二人联手，只怕是如今这一届的新人，没有人敢跟他们二人抗衡。
<br/>　　只是，楚风眠却是一个例外。
<br/>　　楚风眠转过头，脸色平静道极点的开口道。
<br/>　　“两位，有何贵干？”
<br/>　　“哼，小子，你也不用装傻，赶紧从这洞府中滚出来，这洞府不是你这种废物可以占据的。”
<br/>　　牛峰看着楚风眠，一脸嘲笑道。
<br/>　　“区区御风境的蝼蚁，也配占据这种洞府？我看山脚下那个洞府，最适合你。”
<br/>　　山脚下的洞府，便是灵气最为稀薄，最为狭小的一个。
<br/>　　“这洞府，便是先到先得，可没有配得上配不上的道理。”
<br/>　　楚风眠淡淡的开口道。
<br/>　　这二人的心思，楚风眠自然是知晓的一清二楚，如今楚风眠在来到北邙学院中，倒是不想要展现出太多实力，懒得对着二人出手。
<br/>　　不然就凭这二人刚刚的话，他们两人，就已经是一个死人了。
<br/>　　不过楚风眠的劝解，这二人却完全没有放在心中，反而是脸上更是露出了几分嘲笑之意。
<br/>　　“先占先得？小子，就你这样一个废物，难道你以为你有资格占有这洞府？”
<br/>　　那牛峰听到楚风眠的话，不由的哈哈大笑起来，仿佛是在嘲笑。
<br/>　　不止是牛峰，周围不少围观的武者，都是纷纷大笑起来。
<br/>　　这洞府，是山脉中最好的洞府，想要窥探的人，可不止牛峰，马允二人，也有不少人，都聚集过来，在一旁观察这。
<br/>　　谁都想要抢夺这最好的一座洞府。
<br/>　　在北邙学院中，一向都是由实力为尊，最大的便是实力，力量便是道理。
<br/>　　什么先来先得的道理，都没有，想要得到，便是可以直接出手抢走。
<br/>　　像是这洞府，虽然按照门规，是不得擅自抢夺的，但是实际上，就算是抢走，也没有人会怪罪。
<br/>　　这牛峰的笑，便是在嘲笑楚风眠，居然是相信门规。
<br/>　　“一个御风境的废物，还是乖乖去最角落的洞府去修炼吧，快滚吧，再不滚，我就将你直接抓出来了，倒时候你可别嫌丢人。”
<br/>　　牛峰大笑这开口道，双眼之中，完全没有将楚风眠放在眼中。
<br/>　　楚风眠这样一尊御风境武者，可以通过考核，自然也是引起了他的注意，不过在牛峰看来，楚风眠最多就是一个运气好的家伙。
<br/>　　坐收渔翁之利才通过考核的，他本身都没有放在眼中。
<br/>　　楚风眠看向牛峰，摇了摇头道。
<br/>　　“实力不强，口气不小。”
<br/>　　楚风眠是不愿意贸然出手，展现实力，但是也不代表楚风眠会隐忍，要是太过隐忍，那楚风眠也就不是楚风眠了。
<br/>　　楚风眠的性格，便是狂傲到极点的人，他已经给了一份情面了，不可能还有第二份。
<br/>　　“是吗？实力不强？我的实力就算是不强，但是对付你，也是绰绰有余了！”
<br/>　　牛峰听到楚风眠的话，脸上动了真怒，一脸通红的道。
<br/>　　“既然你不滚，我就让你滚！给我滚出来！”
<br/>　　“将他的修为废了，直接将他扔出去就行了，这里毕竟是北邙学院，杀人总是不好的。”
<br/>　　旁边的马允也开口道，这言语中的意思，都是在威胁了。
<br/>　　“小子，不想当一个废人，就快点滚出来吧。”
<br/>　　“跟这废物那么多废话干嘛，浪费时间，直接将他抓出来就行了！”
<br/>　　牛峰脸上露出几分不耐烦的神色，突然出手，一身灵力猛然暴起，化为了一道灵力手掌，直接向着洞府中抓了过去。
<br/>　　“出来！”
<br/>　　这牛峰出手，一道灵力大手，瞬间凝聚而出，这一抓，直接向着楚风眠抓了过去。
<br/>　　一抓之间，其中的力量，都在封锁空间，令楚风眠动弹不得，直接要将楚风眠抓走。
<br/>　　“这小子真是找死，居然跟牛峰，马允二人抗衡。”
<br/>　　“哼，这样不知死活的家伙，直接给他个教训好了。”
<br/>　　“牛峰可是现在这一届新人剩下中的佼佼者了，居然敢跟他抗衡。”
<br/>　　“这等吧，那个小子很快就会被扔出来了。”
<br/>　　一些武者，看着楚风眠，也都是脸上露出嘲笑。
<br/>　　在他们看来，一个御风境武者，敢占据这最好的洞府，也是一种不知天高地厚的行为。
<br/>　　给个教训，也是应该的。
<br/>　　牛峰这一抓的力量，虽然不是全力，但是神力境九重武者，只怕都难以抵挡。
<br/>　　用来对付御风境的楚风眠，便是直接相当于必杀之招，毫不留情。
<br/>　　“找死！”
<br/>　　就在这一抓，来到楚风眠面前的一刻，楚风眠突然动了。
<br/>　　楚风眠只是缓缓抬起右手，一股灵力，凝聚在拳锋之上，突然一拳打出。
<br/>　　这一拳，直接砸在了牛峰凝聚出的灵力手臂上。
<br/>　　顿时这灵力手臂，直接出现无数裂痕，然后层层破碎开来，反噬的力量，更是让牛峰都是后退数步，才稳定住了身形。
<br/>　　“你们想要来抢夺我的洞府？敢来打扰我，很好！今日不给你们一个教训看看，还真以为楚某可以任人欺凌不成！”
<br/>　　楚风眠冷哼一声，在逼退了牛峰之后，更是一掌突然打出。
<br/>　　这一掌，无声无息之间突然打出，众人都没有看清，已经化为一道掌印，向着牛峰轰击了过去。
<br/>　　既然这牛峰来了，那这一次，楚风眠就要杀鸡儆猴，也让这些人，再也不敢来打扰楚风眠。
<br/>　　“怎么可能！不过一个御风境的废物，怎么可能打破的了我的灵力！”
<br/>　　牛峰缓缓的站起来，满是震惊的大吼一声。
<br/>　　看到楚风眠一掌袭来，他的身躯，猛然炸裂开来，一身肌肉，高高隆起，浑身的灵力都被激发了出来，爆发出了其中的全力。
<br/>　　“烈火金刚身！”
<br/>　　牛峰的身躯上，突然燃烧起了无数金色火焰。
<br/>　　这火焰，蕴含淡淡佛意，却是真正的佛火。
<br/>　　在灵力加持之下，牛峰便是直接化为了一尊真正的烈火金刚。
